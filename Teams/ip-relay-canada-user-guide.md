# <a name="ip-relay-in-canada---user-guide"></a>加拿大 IP 轉送 - 使用者指南

[!INCLUDE [sfbo-retirement](../Skype/Hub/includes/sfbo-retirement.md)]

## <a name="scenario-a"></a>案例 A
  
如果聽力障礙或語音受損者想要連絡非受損的人員，他們會透過特殊的 Web 用戶端，與稱為「通話接聽者」的專員「聊天」。

通話接聽器會透過聊天會話與聽力障礙或語音受損者通訊，並視需要透過加拿大或美國的任何地方的電話網絡與無障礙人士聯繫。

## <a name="scenario-b"></a>案例 B

如果無障礙人士希望與註冊的聽力障礙或語音受損使用者交談，他們會撥打 Microsoft 提供的免付費電話號碼。

[撥號接聽器] 接著會與無障礙人士通訊，並透過聊天與語音障礙使用者聯繫，並來回溝通。

如果聽力障礙或語音受損者未在聊天程式中連線，如果有可用的電子郵件地址，如果來電者要求，通話者會傳送電子郵件給聽力障礙或語音受損的人員。

## <a name="scenario-c"></a>案例 C

如果聽力障礙或語音受損者需要緊急協助，他或她可能會選取適當的按鈕：911 – 員警 / 911 – 火 / 911 – 醫院。

這會傳送緊急通知給北方 IP 轉送運算子，後者接著會存取為此目的設計的 911 帳戶。 一旦通話接受者確認其位址後，他們便會連絡適當的 PSAP，並視需要隨時保持線上狀態。

## <a name="how-to-place-a-text-chat-to-voice-call"></a>如何將文字聊天設為語音通話

https://aka.ms/IPRelay

若要起始語音通話的簡訊，請移至上述 URL，並使用您的 IP 訊息轉送使用者名稱和密碼登入。

登入後，您會在畫面左側看到顯示的指示。

### <a name="how-to-make-a-text-to-voice-call"></a>如何使用簡訊進行語音通話

1. 在畫面右下角，按一下 [聊天]。
2. 按一下 IPRelay 使用者。
3. 在彈出的新方塊底部，輸入您的訊息。

### <a name="how-to-receive-a-voice-to-text-call"></a>如何接聽語音簡訊通話

- 語音使用者必須知道您的 IP 訊息轉送使用者名稱，才能進行語音簡訊通話。

- 語音使用者可以撥打 (866) 660-8613 與 IP 訊息轉送運算子連線。

- 您必須登入 IP 訊息轉送入口網站，才能接聽來電或聊天。

### <a name="how-to-place-a-911-call"></a>如何撥打 911 電話

如果您遇到緊急狀況，您可以選取位於畫面底部的適當緊急按鈕：911 – 員警 / 911 – 火 / 911 – 緊急服務

IP 訊息轉送運算子會連絡 911、確認您的位址，並與各種緊急中心翻譯，直到他/她不再需要為止，並且已將適當的部門分派到您的位置。

> [!Warning]
> 請勿測試此服務，因為即使您指出這只是一項測試，也會與適當的代理機構連絡，而且可能還會向您收取緊急機構的誤派費用。
