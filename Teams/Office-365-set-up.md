# <a name="enable-teams-in-your-organization"></a>在組織中啟用團隊

> [!IMPORTANT]
> [!INCLUDE [new-teams-sfb-admin-center-notice](includes/new-teams-sfb-admin-center-notice.md)]

根據預設, 小組已開啟所有組織。 如果您在預覽期間使用團隊, 該設定會保持與您在團隊預覽期間所設定的內容相同。 


如果您是組織的系統管理員, 您可以指派使用者授權來控制小組的個別存取, 您可以允許或封鎖在團隊中可以使用的內容來源。 如需詳細資訊, 請參閱[管理貴組織的 Microsoft 團隊設定](enable-features-office-365.md)或[microsoft 團隊中應用程式的系統管理設定](admin-settings.md)。 


 
若要深入瞭解管理個別授權, 請參閱[Microsoft 團隊的 Office 365 授權](Office-365-licensing.md)。
