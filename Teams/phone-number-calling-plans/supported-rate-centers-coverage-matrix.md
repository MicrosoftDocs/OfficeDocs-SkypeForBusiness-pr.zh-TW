# <a name="supported-rate-centers-and-coverage-for-calling-plans"></a>支援的費率中心與通話方案的覆蓋範圍

本文列出支援的費率中心和北美的通話方案覆蓋範圍。 [費率中心] 是一個地理區域的行業期限，傳統上會針對公開交換電話網絡 (PSTN) ，為其定義邊界以進行當地通話、帳單比率及電話號碼指派。 在許多情況下，當行業移至所有包含方案或幾分鐘的組合時，費率中心已變得不太重要，但仍會被其他人使用。

北美洲試算表 [支援的費率中心與覆蓋範圍矩陣](https://www.microsoft.com/download/details.aspx?id=102534) 會列出我們支援的費率中心。 當您收到新的電話號碼，或當您要將電話號碼從現有的提供者轉移至團隊時，請下載試算表，並使用它來查詢 [費率中心]。 如果您不知道您的費率中心，您可以根據您的區功能變數代碼在網際網路上尋找， (NPAs) 及首碼 (NXX) s。
如果您正在取得新號碼，但我們沒有您所要求的號碼，我們會嘗試從相同的費率中心提供您的數位。
