## <a name="microsoft-teams-rooms-feature-introduction-and-issue-resolution"></a>Microsoft 團隊聊天室功能簡介和問題解決
### <a name="401050-07102019"></a>4.0.105.0 (07/10/2019)

此更新中引入的專案:
- Skype 會議室 System store 應用程式 rebranding 到 "Microsoft 團隊聊天室"
- Microsoft 團隊聊天室主控台使用者介面與 Microsoft 團隊的關聯
- [主題更新] 只會將自訂背景圖像保留在會議室前面, 同時顯示 [主控台背景中立色彩], 以確保主控台 UI 控制項符合色彩對比協助工具需求
- 小組通話/會議的會議中通話控制的通用列, 可為 Microsoft 團隊 PC/Web/行動用戶端提供一致的體驗<sup>1</sup>
- 在團隊通話/會議<sup>1</sup>後撥打品質意見反應評分
- 在 Microsoft 團隊聊天室上接收/轉譯 microsoft Whitebord 當從 PC/Web/行動小組共用用戶端<sup>1</sup><sup>2</sup>時, 會顯示在會議室上方  
- 已移除升級至 Windows 10 版本1809的支援, 因為在 Microsoft 小組聊天室用戶端發現相容性問題。 Windows 10 版本19H1 支援將會在未來版本中新增

<sup>1</sup>使用小組鈴響的 Microsoft 團隊服務推出。 此功能可能會在舊版或更新版本的4.0.105.0 用戶端更新中提供

<sup>2</sup>要求 IT 管理員開啟 Microsoft 白板網頁。 此外, 如果您在房間顯示器前有觸控, 您必須使用 Windows 設定來校準多個觸控顯示幕, 才能開始使用 Microsoft 白板 collboration 從會議室顯示共用至團隊會議

### <a name="40850-0482019"></a>4.0.85.0 (04/8/2019)

此更新中引入的專案:

- 修正「提供意見反應」功能的問題 
- 針對即將推出的 MTR 裝置升級到 Windows 10 版本1809的優化準備

### <a name="40780-03142019"></a>4.0.78.0 (03/14/2019)

此更新中引入的專案:

- 修正舊版 Windows 10 RS2 組建中受影響的裝置的「在 app 啟動期間掛起」錯誤。  


### <a name="40760-03042019"></a>4.0.76.0 (03/04/2019)

此更新中引入的專案:

- Microsoft 團隊 P2P 會議和 PSTN 通話的 DTMF 小鍵盤。 若要讓 Microsoft 團隊成為您的預設呼叫用戶端, 系統管理員必須將 IsTeamsDefaultClient 設定為 true
- 將遠端參與者的內送影片釘選到會議室顯示前的全螢幕。 在主機上使用參與者名單中的 [釘選] 命令
- 使用會議室通知的新增功能來改善前廳流覽通知
- 房間系統裝置上未啟用藍牙信標時, 會議室顯示前面會移除 [轉換] 圖示
- 修正團隊會議中的音量控制問題


### <a name="40640-12142018"></a>4.0.64.0 (12/14/2018)

- 聊天室使用者介面的主題和正面改善
- TLS 1.2 用戶端支援。 針對內部部署客戶, 啟用 communciation 超過 TLS 1.2 的 Microsoft 團隊聊天室需要商務用 Skype Server 2015 Cummulative 更新 9 (CU9) 或 Buisness Server 的 Skype 2019 Cummulative 更新 1 (CU1)。

### <a name="release-to-market--1272016"></a>發佈至市場 (12/7/2016)

**功能:**

 **專為商務用 Skype 建立**
  
- Skype 會議的單一觸控式連接
- 針對含螢幕充滿高清影片和 HD wideband 音訊的會議室優化 Skype 會議體驗
- 所有參與者都可以從他們所處的位置使用其選擇的裝置, 連線至 Skype 會議
- 從您的目錄邀請人員, 您可以隨時查看其可用性, 或透過電話撥打電話。
- 支援商務用 Skype PSTN 會議和 PSTN 通話, 以取代您聊天室中的獨立會議電話

  **轉換任何會議室**
  
- 專用 Skype 會議應用程式已針對表格觸控控制器的中央和大房間顯示器進行優化
- 在會議室顯示幕或投影機的前方重複使用現有投資
- 從 huddle 空間到大型會議室, 在所有類型的會議空間中都能運作
- 已認證的商務用 Skype 音訊與視頻裝置可提供各種空間大小
- 內建的有線攝取, 可將專案桌面共用到聊天室和 Skype 會議

  **輕鬆部署、易於管理**
  
- 當裝置在會議室中偵測到其他人時, 可自動喚醒顯示的 [永遠開啟裝置]
- 輕鬆部署並更新 UWP (通用 Windows 平臺) Skype 會議應用程式
- Windows AppLocker 會將裝置鎖定至 Skype 會議應用程式
- 透過 Intune 和 SCCM (MDM) 以 Windows 10 企業版裝置進行監控與管理
- 企業級的可靠性
- 由於您熟悉的 Skype 使用者介面, 因此最終使用者的低訓練努力
- 在 Surface Pro 4 平板電腦上執行
