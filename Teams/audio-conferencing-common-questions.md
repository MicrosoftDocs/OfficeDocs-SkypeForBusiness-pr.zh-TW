# <a name="audio-conferencing-common-questions"></a>音訊會議的常見問題

以下是我們向想要使用音訊會議之客戶提出的一些常見問題。
  
## <a name="what-are-the-benefits-of-audio-conferencing"></a>音訊會議的好處為何？

舉例來說，當人們在路上，而且無法使用膝上型電腦或行動裝置上的商務用 Skype或Microsoft Teams應用程式出席會議時，撥入會議是一項好處。 但在其他情況下，使用電話參加商務用 Skype 或 Microsoft Teams 會議，會比在電腦上使用應用程式的效果更好：
  
- 當網際網路連線能力受限制時。

- 當會議只有音訊時。

- 人員嘗試加入商務用 Skype 會議，但失敗。

- 如果他們撥入電話，通話品質會比較好。

- 與會者可以使用藍牙裝置以「免持」方式加入會議。

- 與會者會發現這對他們而言更為簡單且方便。

## <a name="who-can-attend-an-audio-conferencing-meeting-and-who-can-i-hear"></a>神秘可以參加音訊會議會議嗎？ 我可以聽到誰的聲音？

擁有撥入號碼和會議 ID 的任何人都可以加入商務用 Skype或Microsoft Teams會議，除非會議召集人鎖定會議。
  
無論您是使用電話、商務用 Skype或Microsoft Teams應用程式撥入，您都能聽到通話中的其他人，他們能聽到您的聲音。 如果會議召集人不想聽到會議出席者的聲音，可以將會議出席者設為靜音。
  
## <a name="can-i-add-a-toll-free-number-for-my-audio-conferencing-users"></a>我可以為音訊會議使用者新增免付費電話號碼嗎？

是的，免付費電話號碼 (服務號碼) 只適用于某些國家/地區。 如需可用號碼的清單，請參閱[音訊會議和通話方案的國家/地區和地區可用性](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)。
  
## <a name="how-many-local-dial-in-numbers-are-currently-supported"></a>目前支援多少本機撥入號碼？

當您購買 音訊會議 授權時，會指派本機撥入號碼給您。 撥入號碼會包含在會議邀請中。 這些當地號碼僅適用于您的組織。 指派給貴組織的電話以及該號碼是由該組織內啟用音訊會議的使用者共用。 因此，商務用 Skype或Microsoft Teams由使用者 A 和另一個使用者 B 排程的會議都會有相同的撥入號碼。
  
本機撥入號碼，以及在某些情況下，您組織所在國家/地區的國際撥入號碼，都會包含在會議邀請中。 如果會議出席者使用邀請中包含的不同號碼，則會是共用電話號碼。
  
## <a name="how-many-international-dial-in-numbers-does-audio-conferencing-in-microsoft-365-or-office-365-support"></a>Microsoft 365或Office 365支援音訊會議多少個國際撥入號碼？

如需目前的國家/地區清單，請[參閱Microsoft Teams中的音訊會議電話數位，](phone-numbers-for-audio-conferencing-in-teams.md)或[商務用 Skype Online 中音訊會議的電話數位](/SkypeForBusiness/audio-conferencing-in-office-365/phone-numbers-for-audio-conferencing)。
  
## <a name="can-i-set-up-local-numbers-for-audio-conferencing-from-additional-cities-in-the-country"></a>我可以設定來自國內其他城市之音訊會議的當地號碼嗎？

如果您的區域無法使用音訊會議電話號碼或不符合貴組織的需求，請在[SkypeFeedback 論壇](https://techcommunity.microsoft.com/t5/skype-for-business-blog/welcome-to-the-skype-for-business-community/ba-p/89273)傳送意見反應給我們。
  
## <a name="what-is-the-maximum-length-of-the-audio-conferencing-meetings"></a>音訊會議會議的長度上限為何？

時間長度上限取決於參與會議的人員，以及他們用來加入會議的驗證類型。
  
|**會議出席者**|**會議結束時間**|
|:-----|:-----|
|使用者已使用商務用 Skype或Microsoft Teams應用程式加入，或已撥入會議。  <br/> |如果 24 小時後出席者清單沒有變更，會議便會結束。  <br/> |
|所有使用者都已撥入會議，但有人使用 PIN 進入會議。  <br/> |會議會在 24 小時後結束。  <br/> |
|所有使用者都已撥入會議，但沒有任何人使用 PIN 進入會議。  <br/> | 如果允許匿名使用者開始會議，會議會在四小時後結束。 如果匿名使用者不允許召開會議，而且所有已驗證的參與者都已離開會議，則會議會在最後一個經過驗證的參與者離開 90 分鐘後結束。 <br/> |

## <a name="how-many-total-phone-participants-can-i-have-in-meetings"></a>我可以在會議中總共有多少個電話參與者？

請參閱會議允許出席者總數 [的限制和規格](limits-specifications-teams.md) 。
  
若要瞭解會議限制的相關資訊，請參[閱商務用 Skype線上限制]](/office365/servicedescriptions/skype-for-business-online-service-description/skype-for-business-online-limits#bkmk_Meeting_LyncOnlineLimits)。
  
## <a name="why-did-users-start-receiving-emails-with-their-audio-conferencing-information"></a>為什麼使用者會開始收到含有其音訊會議資訊的電子郵件？

我們新增了一項新功能，可讓系統[管理員](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504)您傳送及更新音訊會議資訊和電子郵件中的 PIN。 若要深入瞭解，包括如何停用，請參閱啟用[或停用在 Microsoft Teams 中音訊會議設定變更時傳送電子郵件，或在 商務用 Skype](enable-or-disable-sending-emails-when-their-settings-change-in-teams.md) [Online 中音訊會議設定變更時啟用或停用傳送電子郵件](/SkypeForBusiness/audio-conferencing-in-office-365/enable-or-disable-sending-emails-when-their-settings-change)。
  
## <a name="can-audio-conferencing-be-used-by-the-users-who-are-part-of-an-on-premises-deployment-of-skype-for-business-server"></a>音訊會議是否可以由內部部署商務用 Skype Server的使用者使用？

使用[[會議第一](meetings-first.md)]，商務用 Skype Server使用者可從Teams會議和Teams 音訊會議獲益。
  
## <a name="can-a-user-get-a-personal-conference-id"></a>使用者可以取得個人會議 ID 嗎？

商務用 Skype和Microsoft Teams使用者會被隨機指派會議識別碼。 使用者無法保留或設定只有他們可以使用的靜態會議 ID。
  
## <a name="can-i-use-audio-conferencing-with-skype-meeting-broadcast"></a>我可以將 音訊會議 與 Skype 會議 廣播搭配使用嗎？

目前不支援使用者音訊會議加入Skype 會議廣播。
  
## <a name="can-a-user-get-operator-assistance-during-a-meeting"></a>使用者可以在會議期間取得電信業者協助嗎？

否，使用者在會議期間按 *0，無法取得任何電信業者協助或支援。 如果音訊會議有問題，組織的系統管理員可以連絡[Microsoft 支援服務以取得Office 365](https://support.office.com/article/Microsoft-support-for-Office-365-32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b)。
  
## <a name="how-does-a-user-access-or-change-their-conference-id"></a>使用者如何存取或變更其會議 ID？

商務用 Skype或Microsoft Teams使用者可以在 Outlook 中排程會議並Outlook 網頁版，找到指派給他們的會議 ID。 此外，使用者可以在設定之後傳送給他們的電子郵件中找到會議 ID。
  
> [!NOTE]
> 使用者將無法重設其會議 ID。 只有組織管理員您才能重設會議 ID。[ ](https://support.office.com/article/admin-eac4d046-1afd-4f1a-85fc-8219c79e1504)
  
我們正在開發一個解決方案，讓使用者無需組織管理員的協助，就能存取及重設會議 ID。
  
## <a name="how-do-users-access-or-change-their-pin"></a>使用者如何存取或變更他們的 PIN？

商務用 Skype或Microsoft Teams使用者可以在設定之後傳送給他們的電子郵件中找到 PIN。
  
> [!NOTE]
> Teams使用者可以變更他們的 PIN 碼。 會議邀請底部有一個連結，可重設其 PIN 碼。
  
我們正在開發解決方案，讓使用者無需組織管理員的協助，就能存取及重設 PIN。

## <a name="can-attendees-dial-out-to-international-phone-numbers-when-they-are-in-a-skype-for-business-or-microsoft-teams-meeting"></a>出席者是否可以在參加商務用 Skype或Microsoft Teams會議時撥出國際電話號碼？

是的，出席者可以撥出國際電話，並邀請其他來電者加入商務用 Skype或Microsoft Teams會議。 請參閱[從Microsoft Teams會議撥出，讓其他人可以加入會議](dialing-out-from-a-teams-meeting-so-other-people-can-join-it.md)或[從商務用 Skype線上會議撥出，讓其他人可以加入。](/SkypeForBusiness/audio-conferencing-in-office-365/dialing-out-from-a-meeting-so-other-people-can-join-it)
  
## <a name="how-does-a-skype-for-business-or-microsoft-teams-user-schedule-a-meeting-with-audio-conferencing-meeting-details"></a>商務用 Skype或Microsoft Teams使用者如何排程包含音訊會議會議詳細資料的會議？

當使用者獲指派 **音訊會議** 授權，且使用者在Outlook或Outlook 網頁版中建立新的商務用 Skype或Microsoft Teams會議時，撥入電話號碼和會議 ID 會自動新增至會議邀請。
  
## <a name="how-does-a-user-schedule-and-start-a-meeting-when-all-attendees-will-be-using-a-phone-to-dial-in"></a>當所有出席者都使用電話撥入時，使用者如何排程並開始會議？

排程所有使用電話撥入的出席者都會加入的會議，與排定一般線上會議並無不同。 不過，有兩種方式可以召開會議，讓所有參與者使用電話撥入：

- **選項 #1**：根據預設，如果會議召集人和所有參與者都是使用電話加入會議，會議召集人必須輸入她音訊會議 PIN 才能開始會議。 當來電者撥打線上會議的電話號碼時，系統會詢問他們是否要驗證為指定會議的召集人。 在召集人開始前透過撥入加入會議的所有參與者都會被置於大廳，並聆聽保留的音樂。 針對商務用 Skype會議，一旦召集人輸入其音訊會議 PIN 啟動會議，大廳內的所有參與者都會自動加入會議。 針對Microsoft Teams會議，參與者會根據召集人會議原則中自動准許人員設定的值加入會議。

- **選項 #2**：預設會停用 [允許未驗證的來電者成為會議中的第一位人員] 設定。 如果已為指定的召集人啟用此設定，則該召集人所排程的所有會議都可以在召集人輸入音訊會議 PIN 之前開始。 啟用此設定時，第一個參與者透過撥入電話號碼加入會議時，會議就會立即開始，且第一個參與者不會被置於大廳。 如需詳細資訊，請參閱[在 Microsoft Teams 中管理使用者的音訊會議設定](manage-the-audio-conferencing-settings-for-a-user-in-teams.md)或[在 商務用 Skype Online 中管理使用者的音訊會議設定](/SkypeForBusiness/audio-conferencing-in-office-365/manage-the-audio-conferencing-settings-for-a-user)。
