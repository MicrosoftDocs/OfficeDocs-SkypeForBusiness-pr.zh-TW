# <a name="availability-in-china"></a>中國的可用性

[尋找其他國家或地區](country-and-region-availability-for-audio-conferencing-and-calling-plans.md)，或進一步瞭解音訊會議與通話方案的國家/地區可用性。

以下是適用于中國的商務用 Skype 和 Microsoft 團隊的 Office 365 服務。

|**功能**|**詳細資料**|
|:-----|:-----|
|音訊會議是否可供購買？  <br/> |否  <br/> |
|是否會自動為音訊會議指派電話號碼？  <br/> | 非地理位置。 電話號碼未連結至特定的城市或位置。 <br/>  如果您需要專用的電話號碼，請參閱[取得商務用 Skype 和 Microsoft 團隊的服務電話號碼](/microsoftteams/getting-service-phone-numbers)。  <br/> |
|使用 [呼叫我] 功能撥出功能  <br/> |離線  <br/> |

|**功能**|**詳細資料**|
