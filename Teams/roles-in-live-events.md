<a name="presenter-producer-and-attendee-capabilities-in-a-teams-live-event"></a>Teams 即時活動中的簡報者和參與者功能和出席者
======================================================

Microsoft Teams 支援可承載大量的線上使用者流量的及時活動。 當參與者角色為被動時，即時活動簡報者和製造者可以存取多個會議功能，包括：  

- 聊天 (包含相片和貼圖)
- 錄製
- 檔案
- 問/答

本文將說明 Teams 即時活動中的可用功能。

## <a name="presenter-and-producer"></a>簡報者和製造者

簡報者和製造者包含下列：

- 來自我的組織的簡報者
- 來自其他組織的使用者。 簡報者由製造者所指定，且需要來自製造者的個人邀請。

簡報者和製造者可存取所有的即時活動功能。

| |  | |||
|---------|----------------|----------------|---------------------|------------|--------------|
|  **功能**       | 活動發生前 | 活動發生中 | 活動發生後 |
| 聊天 | 是 | 是| 是 |
| 錄製 | 不適用 |N/A | 是 |
| 檔案 | 是 |是 | 是 |
| 問/答 (如果已由製造者啟用)| 是 | 是 | 是 |
|||||||

## <a name="live-event-attendee"></a>即時活動出席者

即時活動出席者就是指有即時活動或活動記錄連結的人員。 參加/查看活動不需要特殊許可權。 如果簡報者和/或製造者允許，現實查閱即時活動的出席者可以參加問/答。 

| |  | |||
|---------|----------------|----------------|---------------------|------------|--------------|
|  **功能**       | 活動發生前 | 活動發生中 | 活動發生後 |
| 問/答 (如果已由製造者啟用) | 是 | 是 | 是 |
| 錄製 | 不適用 | N/A| 是 |
| 影片播放控制項 (如果已啟用) | 是 | 是 | 是 |
|||||||
