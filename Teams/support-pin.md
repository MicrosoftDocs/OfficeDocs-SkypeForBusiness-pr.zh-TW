# <a name="how-to-look-up-your-support-pin"></a>如何查詢您的支援 PIN

若要查詢支援 PIN （可透過電話啟用 Microsoft 客戶支援），請遵循下列步驟： 

1. 登入 Office 系統[管理中心](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/homepage)。 

2. 按一下齒輪圖示。

3. 您的支援釘選即會顯示在右下方窗格中。 您也可以視需要選擇重新產生您的 PIN。  

   ![顯示支援 PIN 的圖表](media/support-pin.png)
