# <a name="audio-conferencing-pay-per-minute"></a>音訊會議按分鐘計費

除了標準每個使用者的音訊會議授權之外，大量和授權組織也可以選擇每分鐘付費優惠，為組織啟用音訊會議功能。
  

您可以瞭解音訊會議 [是否支援](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md) 您的國家/地區。

  
> [!NOTE]
> 由於此優惠僅適用于大量和授權客戶，您必須與您的客戶代表合作以取得這些授權。 
  
## <a name="need-some-details"></a>需要一些詳細資料嗎？

雖然音訊會議每個使用者授權方案包含電話撥入使用量和部分撥出使用量，但音訊會議每分鐘付費可讓您以每分鐘支付所有音訊會議使用量，而不需要支付每個使用者的每月授權費用。 這樣一來，所有撥入和撥出電話都會以每分鐘計費，包括：
  
- 撥打貴組織的付費音訊會議號碼的來電。
    
- 撥打貴組織免付費音訊會議號碼的來電。
    
- 從貴組織會議撥打給外部電話號碼的外寄電話。
    
> [!NOTE]
> 您可以在音訊會議中查看通話區段的查看費率，以找出與這類通話相關聯的撥入和撥出[費率](https://products.office.com/microsoft-teams/online-meeting-solutions#Rates)。 
  
貴組織必須設定通訊信用 [額度](what-are-communications-credits.md) ，以每分鐘付費。 這也需要同時指派每分鐘付費授權和通訊信用額度授權給每個使用者，音訊會議才能使用。 如果您需要更多詳細資料，請參閱為[](set-up-communications-credits-for-your-organization.md)您的組織和/或 Microsoft Teams 附加元件授權[設定通訊信用額度](https://docs.microsoft.com/microsoftteams/teams-add-on-licensing/microsoft-teams-add-on-licensing)。
  
若要為貴組織的使用者啟用每分鐘付費的音訊會議，請參閱 [在 Microsoft Office 365 或 Office 365](try-or-purchase-audio-conferencing-in-office-365-for-teams.md)中試用或購買音訊會議。

## <a name="why-is-it-best-for-you"></a>為什麼最適合您？

- 每一位出席者在排定會議期間所撥打的每一個傳入或外寄電話 (費率會因付費電話或免付費電話和目的地) 而異，而不是在行動裝置或 PC 中使用商務用 Skype 或 Teams 應用程式。

- 管理成本的功能，因為系統管理員可以控制限制可于 (會議) 撥出類型。 查看[音訊會議和使用者的 PSTN](/microsoftteams/outbound-calling-restriction-policies)通話輸出通話限制政策

- 如果您擁有音訊會議每分鐘付費授權，則不必針對音訊會議使用量將通訊信用額度授權個別指派給每個使用者 (您可能仍然需要將它們指派給其他服務) 。

- 使用通訊信用額度控制及監控每分鐘的付款費用。

- 針對不需要訂閱所有使用者的客戶，在價格方面更有彈性。 

- 啟用音訊會議每分鐘付費以及 E5 的每月訂閱或獨立音訊會議訂閱，這兩種服務會繼續以相同方式工作。 變更對音訊會議或通訊信用額度的運作沒有任何影響。
