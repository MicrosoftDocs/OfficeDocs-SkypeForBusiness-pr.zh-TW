# <a name="enhance-my-service"></a>加強我的服務

本文提供您組織使用雲端語音服務之需求概觀。 您可以持續監控雲端語音服務功能，確保為貴組織帶來最大的價值。

## <a name="enhance-my-service-overview"></a>增強我的服務概觀
成功將音訊會議或 電話系統 使用通話方案服務部署到您的組織，並且以品質和可靠性執行服務之後，下一個步驟就是定期評估您是否有機會加強服務。 這是持續服務改進規劃的一部分，也是雲端服務管理的現代化標準。 

若要確保您隨時充分利用服務，最好的方法就是監控[Teams 公用](https://products.office.com/business/office-365-roadmap?filters=microsoft%20teams)藍圖，以取得貴組織可能感興趣的新功能增強功能。 如果您有存取權，Microsoft 365 系統管理中心透過我們的訊息中心定期收到我們有關新功能和變更功能的更新。 

此外，由於組織成長與變更，您應定期重新檢查和更新商務使用案例，以確保服務已根據貴組織目前的需求進行配置。  在貴組織中建立意見回饋通道，確保您從使用者和內部支援者取得實際的意見回饋。 這些可以是非正式交談、一般會議，或透過Teams或Yammer社群。 

當您看到提升服務的機會時，請花一些時間正確執行。 重新[探討在 Microsoft Teams](deploy-chat-teams-channels-microsoft-teams-landing-page.md)部署聊天、團隊、頻道和應用程式，並端對端適用相同的指引。 進行小型的漸進式變更時，您不可能需要完整完成所有活動和工作，但請務必一路評估每一個活動和工作，給自己取得成功結果的最佳機會。
