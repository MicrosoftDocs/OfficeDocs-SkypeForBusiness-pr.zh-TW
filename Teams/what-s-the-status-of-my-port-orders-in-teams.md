# <a name="whats-the-status-of-my-port-orders-in-teams"></a>我在團隊中的埠訂單狀態為何？

您可以移至**Microsoft 團隊系統管理中心** > **的舊版入口** > 程式**語音** > **埠訂單**，查看您的埠順序狀態。 每個埠訂單狀態將會列在 [Status] （**狀態**）欄中。 如果您需要協助，請[聯絡商務產品支援-系統管理協助](https://docs.microsoft.com/office365/admin/contact-support-for-business-products?toc=/microsoftteams/toc.json&bc=/microsoftteams/breadcrumb/toc.json)。

下表列出 [埠順序] 狀態，以及視需要您可以採取的動作。

|**狀態值**|**您可以查看訂單嗎？**|**您可以編輯訂單嗎？**|**您可以取消訂單嗎？**|**您可以刪除訂單嗎？**|**說明**|
|:-----|:-----|:-----|:-----|:-----|:-----|
|**後處理** <br/> |是的  <br/> |不  <br/> |是的  <br/> |不  <br/> |系統管理員已建立訂單，且已由 Microsoft 收到。  <br/> |
|**與承運人聯繫** <br/> |是的  <br/> |不  <br/> |是的  <br/> |不  <br/> |訂單已由 Microsoft 接收並獲核准，我們正在與遺失的承運人一起使用，以取得認可。  <br/> |
|**已核准轉接** <br/> |是的  <br/> |確認訂單承諾（FOC）  <br/> |是的  <br/> |不  <br/> |失敗的承運人已接受該訂單，且已設定 FOC 日期。  <br/> |
|**轉接擱置** <br/> |是的  <br/> |不  <br/> |不  <br/> |不  <br/> |傳送時間少於24小時，因此無法再編輯或取消訂單。  <br/> |
|**出錯** <br/> |不  <br/> |是的  <br/> |是的  <br/> |是（此時，如果發生錯誤，您就無法刪除埠順序。 需要重新建立埠順序，或者您需要[與商務產品支援人員-系統管理協助](https://docs.microsoft.com/office365/admin/contact-support-for-business-products?toc=/microsoftteams/toc.json&bc=/microsoftteams/breadcrumb/toc.json)。  <br/> |遺失的承運人已拒絕訂單。  <br/> |
|**完畢** <br/> |是的  <br/> |不  <br/> |不  <br/> |不  <br/> |已成功傳送數位。  <br/> |
|**取消** <br/> |不  <br/> |是的  <br/> |不  <br/> |不  <br/> |系統管理員已取消訂單。  <br/> |
   
 **如需完整的逐步指示，請參閱[將電話號碼轉接至 Office 365](/microsoftteams/transfer-phone-numbers-to-office-365)。**
