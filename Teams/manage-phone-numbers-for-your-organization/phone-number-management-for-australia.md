# <a name="phone-number-management-for-australia"></a>澳洲的電話號碼管理

我們已與 Telstra 合作，以購買及提供澳洲帳單位址的客戶通話方案。 Telstra 將會銷售其通話方案，並提供澳洲使用者號碼。 Telstra 支援自動電話機和通話佇列的使用者與服務號碼電話號碼管理。 不過，音訊會議的服務號碼和授權仍由 Microsoft 365 和 Office 365 銷售和管理。

有關 [Telstra 通話的資訊](https://aka.ms/TelstraVoicePlan)。

使用下表尋找在澳洲取得及管理 Microsoft Teams 電話號碼的資訊。

詳細資訊請參閱管理  [貴組織的電話號碼](manage-phone-numbers-for-your-organization.md)。
  
|**您想要做什麼？**|**執行這些步驟並下載正確的表單**|
|:-----|:-----|
|取得澳洲通話方案的新使用者電話號碼。   <br/> |請 [聯絡 Telstra Calling，](https://aka.ms/TelstraVoicePlan) 為使用者尋找新的電話號碼。 然後他們會將這些數位新增到您的組織，好讓您可以將這些號碼指派給使用者。 <br/>
|如果您利用 Telstra 通話功能，您可能會收到新的服務電話號碼，可在來自 Telstra 的澳洲雲端自動電話機和通話佇列使用。 <br/> |如需 [新電話號碼，](https://aka.ms/TelstraVoicePlan) 請聯絡 Telstra 通話。 接著，他們會將這些數位新增到您的組織，好讓他們指派給自動電話機和通話佇列。 如果您 **不是使用** Telstra 通話，請聯絡我們的 Microsoft Number ManagementService desk，向 Microsoft 取得服務號碼 [。請聯絡我們](mailto:ptnapac@microsoft.com)。 <br/>|
|取得澳洲音訊會議的新服務電話號碼。   <br/> |[請聯絡我們](mailto:ptnapac@microsoft.com)。|
|取得移轉至 Microsoft 365 或 Office 365 的使用者電話號碼、自動電話機和通話佇列。  <br/> | 請 [聯絡 Telstra 通話](https://aka.ms/TelstraVoicePlan) 以轉接電話號碼。 接著，他們會將這些數位加到您的組織。  <br/> |
|將電話號碼移轉到 Microsoft 365 或 Office 365，以用於音訊會議。  |[請與我們聯絡](mailto:ptnapac@microsoft.com) |
|變更指派給有通話方案電話號碼之使用者的緊急位址。 |請 [聯絡 Telstra 通話](https://aka.ms/TelstraVoicePlan) 以變更緊急位址。 他們可以更新這項資訊。|
|將使用者編號變更為服務編號。 |使用 Telstra 通話時支援。|
|將目錄服務更新為通話佇列或自動電話機碼|若要新增或變更通話佇列或自動電話機號碼，請連 [上 PSTN 服務台](contact-pstn-service-desk.md)。 |
