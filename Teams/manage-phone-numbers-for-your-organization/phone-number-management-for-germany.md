# <a name="phone-number-management-for-germany"></a>德國電話號碼管理

使用此表格來尋找在德國針對 Microsoft 團隊取得及管理電話號碼的相關資訊。 
  
如需詳細資訊，請參閱[管理貴組織的電話號碼](manage-phone-numbers-for-your-organization.md)。
  
|**您想要做什麼？**|**下載正確的表單**|
|:-----|:-----|
|取得 Microsoft 團隊系統管理中心未列出的新使用者或服務電話號碼。  <br/> | 在德國，您無法使用 Microsoft 團隊系統管理中心取得貴組織的電話號碼。 您必須提交新的電話號碼要求。 首先，請下載此表單。 <br/> [德國（v. 4.0）的新電話號碼要求（de）](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-germany-(v.4.0)-(de-de).pdf?raw=true) <br/> [德國的新電話號碼要求（v. 4.0）（zh-cn）](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/new-number-request-forms/new-phone-number-request-for-germany-(v.4.0)-(en-us).pdf?raw=true) </br>然後將其填入並[傳送給我們](mailto:ptneu@microsoft.com)。  <br/> |
|取得傳送給我們的電話號碼。  <br/> | 下載其中一個表單。 <br/> 針對使用者編號： <br/> [德國（LOA）的授權信件（地理數位）（v. 1.0）（de）](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-germany-(geographic-numbers)-(v.1.0)-(de-de).pdf?raw=true) <br/><br/> 免付費電話號碼： <br/> [德國的授權信件（LOA）（免付費電話號碼）（v.）（de）](https://github.com/MicrosoftDocs/OfficeDocs-SkypeForBusiness/blob/live/Teams/downloads/LOA-forms/letter-of-authorization-(loa)-for-germany-(toll-free-numbers)-(v.2.0)-(de-de).pdf?raw=true) <br/><br/> 填寫並[傳送給我們](mailto:ptneu@microsoft.com)。  <br/> |
|在國內電話目錄中發佈/取消發佈電話號碼。  <br/> |[傳送](mailto:ptneu@microsoft.com)您想要發佈/取消發佈的電話號碼清單給我們。 <br/> |
|將使用者的號碼變更為服務號碼。  <br/> |[向我們傳送](mailto:ptneu@microsoft.com)您想要轉換成服務號碼的使用者號碼清單。 <br/> |
