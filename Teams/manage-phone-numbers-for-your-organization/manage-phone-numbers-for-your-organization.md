# <a name="manage-phone-numbers-for-your-organization"></a>管理貴組織的電話號碼

您有三種方式可取得貴組織的使用者與服務 (免付費和免付費) 號碼。
- 在 Microsoft [團隊管理中心] 中取得數位。
- 將您現有的號碼轉移或「埠」傳送給我們。
- 針對未列在 Microsoft 團隊系統管理中心的新號碼使用要求表單。

> [!IMPORTANT]
> 在比利時、法國、德國、愛爾蘭、荷蘭和西班牙, 您無法使用 Microsoft 團隊系統管理中心取得手機新電話號碼。 您必須下載並填寫表單。 如需澳大利亞的新電話號碼, 請參閱[Telstra 語音服務](https://aka.ms/TelstraVoicePlan)。

## <a name="heres-a-bit-more-information-about-these-number-request-forms"></a>以下是有關這些數位要求表單的詳細資訊

### <a name="request-forms-for-new-phone-numbers"></a>新電話號碼的要求表單

每個國家或地區都有不同的指示、不同類型的號碼 (地理/非地域及服務 (免付費/免付費), 以及取得電話號碼的規則/規章, 以便在小組中使用。

有時候 (視您的國家/地區而定) 您將無法使用 Microsoft 團隊系統管理中心取得新的使用者 (訂閱者) 或服務 (免付費或免付費) 號碼, 或者您可能需要特定的電話號碼或區功能變數代碼。

如果是這樣, 您將需要下載正確的 PDF 格式 (每個國家/地區和編號類型), 並將其傳送給我們。 我們的服務台會從這裡取得, 並讓您瞭解進度或您的訂單。

### <a name="letters-of-authorization-loas-for-transferring-numbers"></a>轉接號碼的授權 (LOAs)

如果您需要傳送電話號碼, 您將會下載一**份授權 (LOA)**, 讓我們以您代表的許可權, 在其他服務提供者處傳送現有號碼 (如果有爭議的話)。 如果您有下列情況, 請使用 LOA:
- 您已從其他運營商所擁有的使用者號碼。
- [音訊會議橋接]、[自動語音應答] 或 [通話佇列] 的服務 (付費) 號碼。
- 服務 (免費電話) 電話號碼。
- 您必須取得超過999的使用者電話號碼, 但無法使用 [使用本機號碼移植] 嚮導在 Microsoft [團隊管理中心] 中取得。

## <a name="if-you-still-need-help-and-arent-sure-what-to-do"></a>如果您仍需要協助, 而且不確定該怎麼做 ... .。。

如果您仍需要協助, 請告訴我們我們可以為您提供哪些協助。 在您的說明要求中, 包含詳細資料, 例如您的組織識別碼、網域、您需要協助的號碼類型、您所需的數位數量、授權人員與您的問題相關的其他重要詳細資料。
- 如需美國境內電話號碼的說明, 請[傳送您的要求](mailto:ptn@microsoft.com)。
- 如需歐洲電話號碼的說明, 請[傳送您的要求](mailto:ptneu@microsoft.com)。
- 如需亞洲電話號碼的說明, 請[傳送您的要求](mailto:ptnapac@microsoft.com)。
