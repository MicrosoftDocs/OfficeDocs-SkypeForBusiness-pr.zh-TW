# <a name="calling-plans-in-microsoft-365---code-of-practice-for-the-united-kingdom-uk"></a>Microsoft 365中的通話方案 - 英國 (英國) 的實作法

本代碼適用于英國 ( (英國) Office 365通話方案服務 (「服務」) ，且英國境內員工 (Microsoft 365少於 10 名或) 的小型企業客戶。 其他商務用 Skype服務受您與 Microsoft 的服務合約以及您使用特定服務所適用的任何其他條款所規範。

## <a name="1-about-skype-for-business"></a>1. 關於商務用 Skype

商務用 Skype是由 Microsoft Ireland Operations Limited 在英國提供，其註冊辦公室位於愛爾蘭都柏林 2 的 John Defenderon's Quay 70 號。

## <a name="2-customer-service"></a>2. 客戶服務

如果您有任何銷售、帳單或技術支援問題，您可以在一般上班時間或線上透過電話與我們連絡：

Toll-Free 電話號碼：0800 032 6417

線上：登入Microsoft 365 系統管理中心，然後在 [**支援**] 底下選取 **[新增服務要求**] 以提交服務要求。

一般上班時間為星期一至星期五，從上午 9：00 到下午 5：00。

您或許可以流覽[商務用 Skype網站](https://go.microsoft.com/fwlink/?LinkID=820709&amp;clcid=0x809)來回答部分問題。

## <a name="3-complaints-and-dispute-resolution"></a>3. 申訴與爭議解決

我們的首要優先順序是客戶滿意度和優質服務。 不過，萬一您抱怨，我們會迅速協助您解決問題。 我們在[Microsoft 365 或 Office 365 中的通話方案 - 英國 (英國) 頁面的申訴處理常式](complaint-handling-code-for-the-united-kingdom-u-k.md)詳細瞭解小型企業向我們提出申訴的程式，以及我們的獨立爭議解決方案。
  
## <a name="4-number-porting"></a>4. 號碼移轉

商務用 Skype在某些情況下可能無法移轉客戶的電話號碼以搭配服務使用。 如果數位可攜性可用，商務用 Skype會採取所有合理的步驟，以確保在與客戶目前的服務提供者達成移轉號碼的合約後，立即完成移轉號碼和後續啟用。

## <a name="5-service-reliability"></a>5. 服務可靠性

Microsoft 365或Office 365通話方案取決於您的寬頻連線。 如果您的寬頻連線發生斷電或故障，您將無法撥打電話。 造成這些失敗的原因可能超出我們的控制範圍。

## <a name="6-emergency-calls"></a>6. 緊急通話

商務用 Skype提供 999/112 公用緊急電話語音的存取權給英格蘭、蘇格蘭、蘇格蘭和愛爾蘭內的客戶。 不過，撥打 999 或 112 的緊急電話與使用Microsoft 365或Office 365通話方案服務的方式，與傳統電話語音不同。 客戶必須通知每位使用者Microsoft 365或Office 365通話方案服務的差異。

差異包括下列專案： (i) 商務用 Skype 可能不知道來電者的實際位置，這可能會導致緊急服務被移轉到錯誤的位置; (ii) 當商務用 Skype使用者撥打緊急電話時，電信業者可能會要求使用者提供其目前的位置，以協助正確路由緊急電話並轉接緊急服務; (iii) 如果使用者的裝置沒有電源、發生電源中斷，或因任何原因無法存取網際網路，使用者就無法透過Microsoft 365或Office 365通話方案服務撥打緊急電話; (iv) 但Microsoft 365和Office 365 通話方案服務可以用於全世界任何有網際網路連線的地方，使用者不應該撥打英格蘭、蘇格蘭、蘇格蘭或北愛爾蘭以外的 999 或 112，因為通話可能不會路由至適當的緊急服務。

請務必讓每位客戶註冊您使用者的位置，並持續更新這項資訊，以便將資訊提供給緊急服務。 您可以透過商務用 Skype系統管理員入口網站線上變更位置資訊。

## <a name="12-premium-rate-services"></a>12. 進階版費率服務

### <a name="what-are-premium-rate-services"></a>什麼是進階費率服務？

進階版 Rate Services 是轉譯資訊和娛樂的電話號碼，會向您的電話帳單收費。 進階版 Rate Services 包含但不限於慈善捐贈、投票行 (，也就是英國 Got Talent) 、目錄轉場、慈善線和成人服務行， (，也就是基本) 。 撥打至 進階版 Rate Services 的電話比撥打其他有線電話號碼的成本更高，開頭是 0871、0872、0873、118 和 09。

進階版 Rate Services 是由服務提供者 (「服務提供者」) 提供的。 服務提供者 () 供應商品或服務，或 (b) 代表一些這類提供者做為轉銷商或匯總者。

### <a name="how-do-premium-rate-services-work"></a>進階費率服務如何運作？

商務用 Skype將您的 進階版 Rate Services 電話傳送給另一個電話公司， (代表服務提供者接聽來電的「終止通訊提供者」) 。 雖然 進階版 Rate Services 營收除于服務提供者、商務用 Skype和終止通訊提供者之間，但大部分的營收是由服務提供者接收。

### <a name="charges"></a>收費

進階版 Rate Services 的所有使用費用都可以在我們的[網站上](https://go.microsoft.com/fwlink/?LinkID=820709&amp;clcid=0x809)找到。 通話進階費率號碼的通話持續時間應以一分鐘為單位。 分鐘數的分數會無條件進位到下一分鐘。 適用時，會在通話開始時收取連線費用。

### <a name="telephone-preference-service-and-fax-preference-services"></a>電話喜好設定服務和傳真喜好設定服務

電話喜好設定服務 (「TPS」) 是由 Direct Marketing Association 代表 Ofcom 所營運，是一個登錄，可讓客戶退出宣告接收來路不明的銷售電話。 如果您想要向 TPS 註冊，可以 [https://www.tpsservices.co.uk/tps-services-for-consumers/register-on-the-tps.aspx](https://www.tpsservices.co.uk/tps-services-for-consumers/register-on-the-tps.aspx) 撥打 0843 005 9576 來註冊。

與 TPS 一樣，傳真喜好設定服務 (「FPS」) 是註冊公司和個人可退出宣告接收不明銷售和市場傳真的登錄。 您可以撥打 0843 005 9576 或在線上 [https://secure.dma.org.uk/cgi-bin/session.pl?reg_option=fps](https://secure.dma.org.uk/cgi-bin/session.pl?reg_option=fps) 向 FPS 註冊。

### <a name="internet-dialler-protection"></a>網際網路撥號程式保護

網際網路撥號程式類似于電腦病毒，是可下載到電腦上的軟體，可以在您不知情或未經同意的情況下，將電腦連線至 進階版 Rate Services。 為了保護您的電腦，我們建議您在按一下連結或快顯，並下載不明來源的資料時，請小心謹慎。 為了進一步保護，基於電話或電子郵件的要求，我們可能會基於內容和成本考慮而無法存取所有或特定進階版 Rate Services。 此外，建議您下載所有必要的安全性軟體來保護您的電腦，並經常安裝可用軟體的更新。 如果您認為您已因網際網路撥號程式而受害，請透過電話或電子郵件與我們連絡。

所有服務提供者都必須向 PhonepayPlus 註冊，PhonepayPlus是由 Ofcom 核准的獨立組織。 PhonepayPlus 會規範進階版費率服務，並調查消費者提出的抱怨。 所有服務提供者都必須遵守 PhonepayPlus 練習規範，這可在 [練習代碼](https://psauthority.org.uk/for-business/code-15)中找到。 如果服務提供者違反管理辦法，PhonepayPlus 可能會：

- 要求當事人補救違規

- 問題？

- 服務的橫條存取權

### <a name="service-provider-contact-information"></a>服務提供者連絡資訊

若要尋找服務提供者的連絡人資訊，您可以在 [提交線上查詢](https://psauthority.org.uk/For-Consumers/Contact-Us)中使用 PhonepayPlus 的#NumberChecker工具。 如果您認為您因進階版 Rate Service 被不當收費，您應該直接連絡服務提供者。

### <a name="complaints"></a>投訴

如果您想要對使用 PhonepayPlus 的服務提供者提出不滿意見，您可以免費在線上或透過電話提交查詢：

- 電話：0300 30 300 20 (上午 9：30 - 星期一下午 5 點到星期五) 

- 線上查詢： [提交線上查詢](https://psauthority.org.uk/For-Consumers/Contact-Us)

### <a name="dispute-resolution"></a>爭議解決

如果您認為帳單上的 進階版 Rate Service 收費是商務用 Skype一部分的錯誤，而且您想要退款，您可以根據商務用 Skype抱怨處理碼提出申訴。 如果您向商務用 Skype提出申訴，並 () 在提交 8 (8) 周之後，或 (b) 您收到我們的信件表示我們已做出最終決定，則您可能會將抱怨提交給其他服務，這是由 Ofcom 核准的協力廠商處理通訊服務的申訴。

## <a name="13-national-translation-services-personal-numbers-and-0870-numbers"></a>13. 國家/地區翻譯服務、個人號碼和 0870 號碼

National Translation Services 是以 08 或 0500 (Freephone 號碼開頭的電話號碼，) ，而且被國家電話號碼計畫分類為「特殊服務號碼」。 個人號碼的首碼是數位 070。

國家翻譯服務、個人號碼和 0870 號碼的費率均會在我們的 [網站上](https://go.microsoft.com/fwlink/?LinkID=820709&amp;clcid=0x809) 提供，並且可能會根據您撥打電話的一天和一周中的時間而有所變化。 撥打 0800 號碼是免費的。
