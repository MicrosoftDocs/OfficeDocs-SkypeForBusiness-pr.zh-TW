# <a name="how-many-telephone-numbers-can-you-get"></a>您可以取得多少電話號碼？

當您尋找及取得貴組織 Microsoft 提供的電話號碼時，您可以取得比已指派授權更多的電話號碼。 但這取決於您購買和指派的電話號碼類型及授權類型。 

您取得和管理電話號碼方式會根據您的 PSTN 連接選項而不同：Microsoft 通話方案、連線或直接路由。 詳細資訊，請參閱 [管理貴組織的電話號碼](manage-phone-numbers-landing-page.md)。

下表適用于 Microsoft 通話方案、 [音訊](deploy-audio-conferencing-teams-landing-page.md)會議及語音應用程式，例如通話 [佇列和自動語音留言](plan-auto-attendant-call-queue.md)。 如需運算子和連線路由的資訊，請與您的服務提供者聯繫。

您可以在系統管理中心查看在搜尋和查詢程式期間從 Microsoft 取得的電話號碼數量Microsoft Teams取得 (電話號碼) 。
