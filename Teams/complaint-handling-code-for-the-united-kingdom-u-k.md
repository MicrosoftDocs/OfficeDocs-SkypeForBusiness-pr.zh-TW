本代碼適用于英國 (英國)  (擁有 10 名員工或少於) 英國之組織的小型企業客戶，) Microsoft 365根據英國通訊法規 (「服務」 (「服務」) 。 其他商務用 Skype服務受您與 Microsoft 的服務合約以及您使用特定服務所適用的任何其他條款所規範。

## <a name="how-to-make-a-complaint"></a>如何提出抱怨

客戶滿意度是商務用 Skype優先順序。 不過，如果您因任何原因不滿意服務，您可以透過電話或線上提出申訴：

- Toll-Free 電話號碼：0800 032 6417

- 線上：登入Microsoft 365 系統管理中心，然後在 [**支援**] 底下選取 **[新增服務要求**] 以提交服務要求。

## <a name="timelines"></a>時程表

如果您在一般上班時間內透過電話提出申訴，您將會與現場商務用 Skype代表通話，後者會在通話期間嘗試解決問題。

如果您在線上提出申訴，商務用 Skype代表將會在收到抱怨的兩個 (2) 個工作天內回應您的抱怨，並嘗試立即解決問題。

如果商務用 Skype代表無法立即解決問題，他/她將會清楚說明需要採取的進一步步驟來調查問題。 代表會提供您其連絡資訊，以便您在調查過程中，連絡對方並提出任何問題。

商務用 Skype的目的是在收到回條的 7 個 (7 個) 工作天內解決與服務相關的所有抱怨。

如果您對於管理服務的抱怨方式不滿意，您可以要求商務用 Skype代表透過電話或電子郵件，依您的服務合約向上呈報抱怨。 Microsoft 會在收到的七 (7) 天內評估抱怨。

## <a name="alternative-dispute-resolution"></a>替代爭議解決方式

如果您在提交 8 (8) 周之後，若您對服務的抱怨未獲得解決，或者您收到我們發出的信件表示我們已做出最終決定，您可以將抱怨提交給「服務」，這是由 Ofcom 核准的協力廠商，以處理通訊服務的抱怨。 我們也會以書面形式通知您這類替代爭議解決服務的可用性。

方塊服務與 Microsoft 無關，因此會針對抱怨的優點做出偏向的決策。 將抱怨提交給「無線服務」無需支付任何費用。

所有不滿意見都必須在向商務用 Skype提出抱怨的 12 (12) 個月內傳送至文檔服務。
