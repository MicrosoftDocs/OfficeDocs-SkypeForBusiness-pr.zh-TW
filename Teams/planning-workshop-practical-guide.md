# <a name="tools-and-workshops-plan-deliver-and-adopt-microsoft-teams"></a>工具和研討會：規劃、傳遞及採用Microsoft Teams

下載這些工具，以在組織中完成規劃、交付Teams採用。


- [Microsoft Teams客戶成功套件](https://aka.ms/TeamsCustomerSuccess)

    此套件包含各種簡報、電子郵件範本、秘訣&訣竅，以及海報，可用於在組織中推出和推動Teams採用。 它包含兩個適用于系統管理員和小組領導者的快速入門手冊，以及每個部門的使用案例範例。 使用這些資源來充分利用貴Teams的各種功能，包括行銷、銷售、財務、人力資源、IT、工程Project管理。

- [如何推出 Teams](./deploy-overview.md)

    請遵循我們的建議路徑來推出Teams。


- [取得Teams技術訓練](itadmin-readiness.md)

    使用這些技術訓練資源來教育自己、教導其他人或舉辦研討會。

- [Office 365 政府版客戶](plan-for-government-gcc.md)

   您前往Teams，就像我們的企業客戶體驗一樣。 由於Teams一般可供租使用者使用，我們建議您從我們的 Microsoft 365 方案 - GCC[部署](plan-for-government-gcc.md)文章和系統管理快速[入門指南](https://download.microsoft.com/download/F/3/9/F39B4F10-5720-4516-87E1-91E5A5678EFB/MicrosoftTeams-AdminQuickStart-EnableTeams.docx)開始。 我們的[安全性和合規性](security-compliance-overview.md)概觀可協助您Teams環境中的安全性與合規性。

- [Teams 教育版 - 系統管理員的快速入門](./teams-quick-start-edu.yml)

    使用此教學課程來指導您完成Teams教育組織的部署。
