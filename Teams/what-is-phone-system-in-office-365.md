# <a name="what-is-phone-system"></a>什麼是電話系統

電話系統是 Microsoft 在 Microsoft 365 雲端使用 Microsoft Teams 啟用通話控制和專用 Exchange (PBX) 功能的技術。

電話系統可與 Teams 用戶端和認證裝置搭配使用。 電話系統可讓您以直接從 Microsoft 365 提供的一組功能取代現有的 PBX 系統。

貴組織中使用者之間的通話會在電話系統內部處理，永遠不會移至公用交換電話網 (PSTN) ，因而移除內部通話的長途電話成本。 

若要進行外部通話，[電話系統] 提供連線到 PSTN 的附加元件選項。 如需語音解決方案和 PSTN 連線選項的詳細資訊，請參閱 [規劃您的 Teams 語音解決方案](cloud-voice-landing-page.md) 和 [連線到 PSTN](#connect-to-the-public-switched-telephone-network-pstn)。

除了澳門特別行政區以外，手機系統全球皆可使用。 

## <a name="phone-system-features"></a>電話系統功能

透過電話系統，使用者可以使用 Teams 來撥打和接聽電話、轉接電話，以及將通話設為靜音或取消靜音。 電話系統使用者可以按一下通訊錄中的名稱，然後撥打 Teams 電話給那個人。 若要撥打和接聽電話，電話系統使用者可以使用行動裝置、膝上型電腦或電腦耳機，或是可搭配 Teams 使用的眾多 IP 手機之一。 

電話系統管理員可以從用於傳訊、共同作業等的同一部主機上管理通話選項和設定。

如需電話系統功能的詳細資訊，請參閱以下說明 [您可透過電話系統取得的功能](here-s-what-you-get-with-phone-system.md)。
  

## <a name="connect-to-the-public-switched-telephone-network-pstn"></a>連線到公用交換電話網路 (PSTN) 
  
對於外部通話，電話系統可以透過下列其中一種方式連線到 PSTN：
  
-  (國內或國際) 購買 Microsoft 通話方案。 Microsoft 通話方案是全雲解決方案，Microsoft 是您的 PSTN 電信業者。 如需詳細資訊，請參閱 [電話系統和通話方案](calling-plan-landing-page.md)。

- 使用您現有的電話語音基礎結構進行內部部署 PSTN 連線。

  您可以使用運算子連線通話或直接路由，將您的內部部署電話語音基礎結構連線到電話系統。 

如需所有 PSTN 連線選項的詳細資訊，請參閱 [PSTN 連線選項](pstn-connectivity.md)。


## <a name="phone-system-with-services"></a>電話系統與服務

電話系統可用於服務和語音信箱，例如：

- **自動語音應答** ： 自動語音應答可用來為貴組織建立功能表系統，讓外部和內部來電者在系統中移動，以尋找並撥打或轉接電話給公司使用者或組織中的部門。 請參閱 [什麼是雲端自動語音應答？](what-are-phone-system-auto-attendants.md)。

- **通話佇列** - 當某人撥入貴組織的電話號碼時，可以使用通話佇列問候語。 這些問候語包括自動保留通話，以及搜尋下一個可用的來電代理程式來處理通話的功能。 保留的人員也可以在保留時聆聽音樂。 您可以為組織建立單一或多個通話佇列。 請參閱 [建立雲端通話佇列](create-a-phone-system-call-queue.md)。

- **語音信箱**- 系統會自動為所有 Teams 使用者設定和布建雲端語音信箱。 請參閱[設定雲端語音信箱](set-up-phone-system-voicemail.md)。

如需功能的詳細資訊，請參閱 [電話系統](here-s-what-you-get-with-phone-system.md)提供以下功能。 如果您已準備好開始使用，請參閱 [在貴組織中設定電話系統](setting-up-your-phone-system.md)。
