# <a name="what-is-phone-system"></a>什麼是電話系統？

PBX 專用 (PBX) 是公司中的電話系統。 電話系統提供 PBX 功能，但不需要複雜且昂貴的設備。 

電話系統可讓您以 Microsoft 365 或 Office 365 提供的一組功能取代現有的內部部署 PBX 系統，這些功能已緊密整合至您的雲端體驗。 如要取得功能詳細資訊，請參閱使用電話系統 [取得的功能](here-s-what-you-get-with-phone-system.md)。

## <a name="phone-system-technology"></a>電話系統技術

電話系統可在雲端中使用 Microsoft Teams 和商務用 Skype Online 進行通話控制及 PBX 功能。 
  
使用電話系統，使用者可以使用 Teams 或商務用 Skype Online 來撥打和接聽來電、轉接通話，以及將通話設為靜音或取消靜音。 電話系統使用者可以在通訊錄中按一下名稱，並撥打 Teams 或商務用 Skype Online 電話給該人員。 若要撥打和接聽電話，電話系統使用者可以使用他們的行動裝置、膝上型電腦或 PC 的耳機，或是使用 Teams 和商務用 Skype Online 的許多 IP 電話之一。 電話系統管理員可以從用於傳訊、共同合作等的同一個主機管理通話選項和設定。
  
貴組織使用者之間的通話在電話系統內部處理，而且永遠不會在 PSTN (電話) 。 這適用于貴組織中位於不同地理區域的使用者之間的通話，可移除這些內部通話的長途費用。

針對組織外部的通話，Microsoft 提供多種選項，將電話系統連接到公用交換電話網絡 (PSTN) 。

## <a name="connect-to-the-public-switched-telephone-network-pstn"></a>在 PSTN 中 (公用交換電話) 
  
電話系統可以有兩種方式之一連接到 PSTN：
  
- 購買 Microsoft 通話方案 (國內或國內及國際電話) 。 Microsoft 通話方案是一種以 Microsoft 做為 PSTN 電信公司的全雲端解決方案。 詳細資訊，請參閱 [電話系統與通話方案](calling-plan-landing-page.md)。

- 使用現有的電話基礎結構進行內部部署 PSTN 連接。

  對於 Teams 使用者，您可以使用直接路由將您的內部部署電話基礎結構連接到電話系統。 詳細資訊，請參閱 [電話系統直接路由](direct-routing-landing-page.md)。

  有關所有 Microsoft 電話解決方案的更多資訊，請參閱 [Microsoft 電話解決方案](/SkypeForBusiness/hybrid/msft-telephony-solutions)。


## <a name="phone-system-with-services"></a>電話系統與服務

 電話系統可用於服務和語音信箱，例如：

- **自動語音** 機 - 自動語音機可用來為貴組織建立功能表系統，讓外部和內部來電者在系統之間移動，以尋找及撥打或轉接電話給貴組織的公司使用者或部門。 請參閱 [什麼是雲端自動總機？](what-are-phone-system-auto-attendants.md)。

- **通話佇列** - 當有人來電到貴組織的電話號碼時，可以使用通話佇列問候語。 這些問候語包括自動保留通話的能力，以及搜尋下一個可用的通話代理程式來處理通話，而通話者正在聆聽保留的音樂。 您可以為貴組織建立單一或多個通話佇列。 請參閱 [建立雲端通話佇列](create-a-phone-system-call-queue.md)。

- **語音** 信箱 - 當您取得使用者的電話系統授權時，該使用者可以取得來電者留下的語音信箱。 在您指派電話系統授權和電話號碼給使用者之後，系統會自動為使用者設定和設定雲端語音信箱。 請參閱 [設定雲端語音信箱](set-up-phone-system-voicemail.md)。

如要取得功能詳細資訊，請參閱使用電話系統 [取得的功能](here-s-what-you-get-with-phone-system.md)。 如果您已準備好開始使用，請參閱在貴組織中設定 [電話系統](setting-up-your-phone-system.md)。
