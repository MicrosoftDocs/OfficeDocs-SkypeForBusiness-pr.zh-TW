# <a name="raise-your-hand-option-in-teams-meetings"></a>在團隊會議中提出您的手選項

在 Microsoft 團隊會議期間，參與者可以使用 [**提出您的手**] 選項來取得演講者和版主的注意。

此功能非常重要，原因如下：

- 參與者可以在不中斷會議的情況下，發出想要交談的通知。

- [版主] 或 [演講者] 可以查看會議中是否有任何人想要交談。  

- 如果鼓勵參與者參與會議，**您可以提高您的手**讓會議更具包容性。

- [**舉手**] 提供一種簡單快速的方式來投票會議參與者。

團隊管理員無需建立原則，就能在會議中顯示該功能。

## <a name="raise-your-hand-display"></a>升高您的手顯示器

在您加入的任何會議中，或在會議室裝置以外的所有用戶端（此時）都支援**您的手舉手**。

- 所有會議參與者都能看到凸起的手，並以手圖示表示。

- 產生的手會顯示在會議階段、名單（依工期排序），以及共用內容時。

- 凸起的手會顯示為參與者名稱旁邊的黃色。

參與者一經確認之後，就可以使用您的 [**放下**] 選項。 會議簡報者也可以降低參與者的手。
