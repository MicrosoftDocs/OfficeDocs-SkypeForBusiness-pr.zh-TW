<a name="edit-guest-user-information"></a>編輯來賓使用者資訊
===========================

若要編輯來賓帳戶 (例如 [顯示名稱] 或 [個人資料相片]), 請移至您的 Azure Active Directory 入口網站。 如需詳細資訊, 請參閱[瞭解 Office 365 身分識別與 Azure Active Directory](https://support.office.com/article/Understanding-Office-365-Identity-and-Azure-Active-Directory-06a189e7-5ec6-4af2-94bf-a22ea225a7a9)。

您目前無法從 Microsoft 團隊系統管理中心、Microsoft 365 管理中心或 Exchange 系統管理中心編輯來賓資訊。
