# <a name="microsoft-teams-powershell-overview"></a>Microsoft Teams PowerShell 概觀

Microsoft Teams PowerShell 模組是一組 Cmdlet，可直接從 PowerShell 命令列管理Teams。 PowerShell 提供強大的自動化功能，可用於管理您的Teams工作負載。  

您可以使用下列其中一種方法來使用Microsoft Teams PowerShell 模組： 

- 透過PowerShell 資源庫 (建議選項) 安裝[Microsoft Teams](https://www.powershellgallery.com/packages/MicrosoftTeams) PowerShell 模組。 
- 透過[手動下載](https://www.powershellgallery.com/packages/MicrosoftTeams)安裝 Microsoft Teams PowerShell 模組。 

> [!NOTE]
> Microsoft Teams 4.x.x 之前的 PowerShell 模組版本即將淘汰。 請更新至最新版本。 [PowerShell 模組Teams更多詳細資料 - 支援的版本](teams-powershell-supported-versions.md)。

## <a name="features"></a>功能 

Microsoft Teams PowerShell 模組包含下列功能： 

- 使用單一模組來管理Teams管理的所有層面，包括使用者、團隊、原則和設定。  
- 支援驗證機制，例如存取權杖和認證。 

##  <a name="data-collection"></a>資料收集 

Microsoft Teams PowerShell 模組預設設定會收集遙測資料。 Microsoft 會匯總收集的資料，以找出使用模式和常見的問題，例如成功率低的 Cmdlet，以協助我們安排工作優先順序，以改善Teams PowerShell 體驗。 Microsoft Teams PowerShell 模組不會收集任何私人或個人資料。 如需詳細資料，請參閱 [Microsoft 隱私權聲明](https://privacy.microsoft.com/privacystatement)。
