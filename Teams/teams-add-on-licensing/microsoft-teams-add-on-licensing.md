# <a name="microsoft-teams-add-on-licenses"></a>Microsoft Teams附加元件授權

## <a name="which-capabilities-can-i-get-with-add-on-licenses"></a>我可以使用附加元件授權取得哪些功能？

附加元件授權是特定功能Microsoft Teams授權。 它們提供您僅為貴組織中需要這些功能的使用者新增功能彈性。 若要新增功能，請為每位使用此功能的使用者購買一個附加元件授權。

可用的附加元件授權選項取決於您Microsoft 365方案。 To see which voice calling and audio conferencing options are available to you, see the [Which Microsoft capabilities are available with my plan?](#which-microsoft-capabilities-are-available-with-my-plan) section of this article.

## <a name="which-microsoft-capabilities-are-available-with-my-plan"></a>我的方案提供哪些 Microsoft 功能？

所有Microsoft 365方案Teams使用者使用 VoIP 技術在貴組織中進行對等通話。 不過，您可能想要新增不同的授權，以取得功能，例如撥打 PSTN 號碼或設定 PBX。

### <a name="audio-conferencing"></a>音訊會議

您可以免費 **新增音訊會議** 授權給所有Microsoft 365訂閱。

Microsoft 365 E5/A5/G5 方案包含音訊會議授權，因此您不需要使用這些方案購買額外的音訊會議授權。

### <a name="voice-and-pstn-connectivity"></a>語音和 PSTN 連接

無論您是小型企業、企業組織、教育機構、政府機構或非營利組織，您都有三種方式Teams 電話系統 PSTN 連接：

- 購買商務Microsoft 365 E1/E3、A1/A3 或 G1/G3 方案，然後Teams 電話通話方案套件授權 **來新增通話** 方案。
  - 此模型可為您提供Teams 電話系統 PBX 功能和 Microsoft 國內通話方案。
  - 如果您希望 Microsoft 處理所有電話號碼管理和帳單，請選擇此設定。
  - 檢查您的地區Teams 電話通話[方案套件授權。](../country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)

- 購買 E1/E3、A1/A3 或 G1/G3 方案、新增 **Teams 電話 標準** 授權，然後選擇 Microsoft 獨立通話方案或協力廠商 PSTN 電信公司。
  - 此模型可為您提供Teams 電話系統，但您仍然需要連接到 PSTN。
  - 若要查看所有 PSTN 連接選項，請參閱 [PSTN 連接選項](../pstn-connectivity.md)。
  - 若要瞭解所有語音解決方案，請參閱[規劃您的Teams解決方案](../cloud-voice-landing-page.md)。

- 購買 E5/A5/G5 方案並新增 **Microsoft 通話方案** 或連接至協力廠商 PSTN 連接提供者。
  - [Microsoft 365 E5、A5 和 G5 方案Teams 電話系統，但組織仍然需要選擇[PSTN 連接選項](../pstn-connectivity.md)。
