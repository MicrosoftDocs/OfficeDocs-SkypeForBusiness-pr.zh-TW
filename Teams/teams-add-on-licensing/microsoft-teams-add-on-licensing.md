# <a name="microsoft-teams-add-on-licenses"></a>Microsoft Teams附加元件授權

## <a name="what-features-can-i-get-with-add-on-licenses"></a>附加元件授權可以取得哪些功能？

附加元件授權是特定 Microsoft Teams 功能的授權。 它們提供您僅為貴組織中需要這些功能的使用者新增功能彈性。 為了新增一個功能，需為每個將使用此功能的使用者購買一個附加元件授權。

可用的附加元件授權選項取決於您Microsoft 365或Office 365方案。 若要查看哪些語音選項可供您使用，請參閱本文的我的方案提供哪些語音功能 [？ 一](#what-voice-features-are-available-with-my-plan) 節。

以下是您可以新Teams附加元件授權及其功能的清單，Teams訂閱。

|許可證|描述|
|-------|-----------|
|**Microsoft 365 Business Voice** <br/> | Business Voice 是 Microsoft 的中小型企業語音功能套件，但將于 2022 年 3 月 1 日被棄用。 <br/> <br/> 如果您是中小型企業，建議您使用通話方案Teams 電話購買方案 ，如果地區提供。[ ](../business-voice/whats-business-voice.md) |
|**Teams 電話通話方案套件** <br/> |Teams 電話方案是一種電話解決方案，電話系統 PBX 功能和國內 PSTN 連接方案。 <br/> |

## <a name="what-voice-features-are-available-with-my-plan"></a>我的方案提供哪些語音功能？

查看您需要哪些附加元件授權，才能在 Teams中取得語音功能，視您擁有的計畫而不同。

根據您想要的語音功能，建議您比較切換到包含這些功能的計畫是否更具成本效益。 例如，對大多數的大型組織而言，在計畫內購買一Microsoft 365 企業版套件會降低整體成本。 一般來說，當您個別購買多個附加元件授權，而不是作為計畫的一部分時，最終可能會獲得較高的合併成本。
