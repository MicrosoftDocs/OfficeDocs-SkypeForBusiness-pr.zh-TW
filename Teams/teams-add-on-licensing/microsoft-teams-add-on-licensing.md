# <a name="microsoft-teams-add-on-licenses"></a>Microsoft Teams 附加元件授權

## <a name="what-are-add-on-licenses"></a>什麼是附加元件授權？

附加元件授權是特定 Microsoft Teams 功能的授權。 它們提供您僅為貴組織中需要這些功能的使用者新增功能彈性。 為了新增一個功能，需為每個將使用此功能的使用者購買一個附加元件授權。

## <a name="what-features-can-i-get-with-add-on-licenses"></a>附加元件授權可以取得哪些功能？

以下是您可以使用 Teams 附加元件授權新增的功能清單。 可用的附加元件授權選項取決於您的 Microsoft 365 或 Office 365 方案。

## <a name="what-voice-features-are-available-with-my-plan"></a>我的方案提供哪些語音功能？

根據您擁有的計畫，查看您需要哪些附加元件授權才能在 Teams 中取得語音功能。

根據您想要的語音功能，我們建議您比較切換到包含這些功能的計畫是否更具成本效益。 例如，對大多數的大型組織而言，在 Microsoft 365 企業版計畫中購買一組軟體會降低整體成本。 一般來說，當您個別購買多個附加元件授權，而非計畫的一部分時，最終可能會獲得較高的合併成本。

#### <a name="small-business"></a>[**小型企業**](#tab/small-business/)

如果您是小型企業或中型企業， (使用者少於 300) ，您擁有四種取得語音功能的方法：

- 購買 Microsoft 365 商務版方案與 Microsoft 365 商務語音。
- 購買企業版 E1 或 E3 方案及 Microsoft 365 商務語音。
- 購買企業版 E1 或 E3 方案，並個別新增語音功能。
- 購買企業版 E5 方案，其中包含語音功能。

針對 Microsoft 365 商務語音，您可以選擇兩個選項：

- **具有通話方案之 Microsoft 365 商務** 語音：如果您希望所有電話號碼管理和帳單都由 Microsoft 處理，請取得這項功能。
- 沒有通話方案 (直接路由) 的 **Microsoft 365** 商務語音：如果您希望協力廠商電話提供者處理所有電話號碼管理和帳單，請取得這項功能。 您必須設定並部署直接路由，以將您的[](../direct-routing-landing-page.md)內部部署會話邊界控制器 (SBCs) Microsoft 365 或 Office 365。

若要深入瞭解，請參閱使用 [Microsoft 365](../business-voice/what-to-buy.md)商務語音需要購買哪些專案？ 。
