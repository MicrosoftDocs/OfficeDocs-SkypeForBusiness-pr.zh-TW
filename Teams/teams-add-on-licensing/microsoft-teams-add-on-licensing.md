# <a name="microsoft-teams-add-on-licenses"></a>Microsoft 團隊附加元件授權

## <a name="what-are-add-on-licenses"></a>什麼是附加元件授權？

附加元件授權是特定 Microsoft Teams 功能的授權。 它們可讓您靈活地為組織中需要的使用者新增功能。 為了新增一個功能，需為每個將使用此功能的使用者購買一個附加元件授權。

以下是可使用 [團隊附加元件授權] 新增的功能清單。 您可使用的附加元件授權選項，取決於您擁有的 Microsoft 365 或 Office 365 方案。

## <a name="what-voice-features-are-available-with-my-plan"></a>我的方案提供哪些語音功能？

視您所擁有的方案而定，查看在團隊中取得語音功能所需的附加元件授權。

根據您想要的語音功能而定，建議您比較是否能更經濟高效地切換到包含這些功能的方案。 例如，對於大多數大型組織而言，在 Microsoft 365 企業版方案中購買軟體套件，可能會降低總成本。 通常，當您個別購買多個附加元件授權，而不是作為方案的一部分時，可能會有較高的組合成本。

#### <a name="small-business"></a>[**小型企業**](#tab/small-business/)

如果您是小型或中型企業（小於300的使用者），您可以透過四種方式取得語音功能：

- 購買 Microsoft 365 商務版方案和 Microsoft 365 商務版語音。
- 購買企業版 E1 或 E3 方案，以及 Microsoft 365 商務版語音。
- 購買企業版 E1 或 E3 方案，並個別新增語音功能。
- 購買企業版 E5 方案，包括語音功能。

針對 Microsoft 365 商務語音，您有兩個選項可供選擇：

- **使用通話方案的 Microsoft 365 商務語音**：如果您想要由 microsoft 處理所有電話號碼管理和帳單，請取得此情況。
- **沒有通話方案的 Microsoft 365 商務版語音（直接路由）**：如果您希望由協力廠商電話供應商處理所有電話號碼管理和帳單，請取得此情況。 您必須設定並部署[直接路由](../direct-routing-landing-page.md)，將您的內部部署會話邊界控制器（SBCs）連線至 Microsoft 365 或 Office 365。

若要深入瞭解，請參閱[我需要購買什麼才能使用 Microsoft 365 商務版語音？](../business-voice/what-to-buy.md)。

以下是根據授權方案的摘要。

  ![小型企業方案選項](../media/licensing-smallbusiness-options.png)

如果您的國家或地區不支援音訊會議，您可以選擇取得沒有音訊會議方案的 Microsoft 365 企業版 E5，或是沒有音訊會議方案的 Office 365 Enterprise E5。

如果您已經有 Microsoft 365 企業版或 Office 365 企業版方案，您仍然可以為個別的語音功能購買附加元件授權。 不過，您可以選擇切換至[microsoft 365 商務](https://www.microsoft.com/microsoft-365/compare-all-microsoft-365-products?&activetab=tab:primaryr2)方案和[Microsoft 365 商務版語音](https://www.microsoft.com/microsoft-365/business/business-voice)。 在某些情況下，視您想要的功能而定，使用 Microsoft 365 商務語音來取得 Microsoft 365 商務方案可能會有更經濟的效益。

> [!NOTE]
> 如果您需要使用協力廠商電話語音，請在您的組織中部署[直接路由](../direct-routing-landing-page.md)，或者，如果您的國家或地區未提供通話方案，您必須使用直接路由進行語音功能。

#### <a name="trial-offers"></a>[**試用版優惠**](#tab/trial/)

### <a name="trial-offers-for-licensing-plans"></a>授權方案的試用版優惠

為了回應其他人需要遠端作業（或在家工作）的需求，Microsoft 針對政府、教育機構和非盈利性組織提供了幾個免費的團隊試用版。 以下方案提供試用版：

- Office 365 企業[版 E1 試用版](../e1-trial-license.md)、E3 試用版和 E5 試用版
- Office 365 教育[版 A1 試用版](../teams-edu-licensing.md)
- Office 365 政府[版 G1 試用版](../g1-trial-license.md)

若要註冊試用優惠，請參閱[註冊試用版](https://transform.microsoft.com/customer-trials?tab=requestnewtrial)。

### <a name="trial-offers-for-voice-features"></a>語音功能的試用優惠

以下是語音功能試用優惠的摘要。

  ![以試驗為基礎的選項](../media/licensing-trial-offers.png)

 您的基礎方案不一定必須是試用版方案，才能使用這些語音功能的任何實驗。 您可以使用 Microsoft 365 商務方案或企業、政府、教育或非贏利方案，並隨附任何適用于語音功能的試用優惠。 若要查看可用的試用版優惠與附加元件，請在 Microsoft 365 系統管理中心移至 [**帳單** > **購買服務**]，選取**可能感興趣的其他類別**，然後選取 [**附加**元件]。

  針對每個語音與會議選項，如果您有免付費電話號碼，或者您需要額外加入從會議撥出或供使用者撥打國內和國際 PSTN 通話的時間，必須有[通訊點數](../set-up-communications-credits-for-your-organization.md)。
