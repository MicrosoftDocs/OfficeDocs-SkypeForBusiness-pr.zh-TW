# <a name="advanced-communications-add-on-for-microsoft-teams"></a>Microsoft Teams 的 Advanced Communications 附加元件

如果貴組織想要進一步增強您的通訊體驗，請取得進位通訊以自訂進一步的會議，以及管理貴組織的通訊政策的工具。

## <a name="what-is-advanced-communications"></a>什麼是進位通訊？

進級通訊包含進位會議自訂、管理貴組織通訊政策的工具，以及自訂策略套件的進一版可管理性。 其他功能包括自訂品牌會議大廳以量身打造會議，以及監控、追蹤及分析使用者和裝置資料的工具。

- **量身訂做及自訂會議**：使用自訂品牌會議大廳等功能，為您的內部和自訂案例推動跨會議標準化。 在貴組織的部門中彈性地執行。
- **管理貴組織的通訊**：監控、追蹤及分析使用者和裝置上的資料，以確保使用順暢的體驗。
- **自訂策略套件的量身** 打造體驗：您可以為組織中具有類似角色的使用者組合您自己的一群組原則。 新增您想要使用 Teams 系統管理中心或 PowerShell 組合使用之策略類型與政策，以建立您自己的策略套件。 每個使用者都需要授權才能接收自訂策略套件指派。 

所有 Microsoft 365 市場都提供進級通訊，作為任何付費 Microsoft 365 或 Office 365 訂閱的附加元件。 進位通訊中包含的功能不會新增到其他 Microsoft 365 或 Office 365 方案，且無法另行購買。 若要取得功能，您必須購買附加元件。

> [!NOTE]
> 功能將隨著可用而發佈。 請流覽 [Microsoft 365 公用藍圖](https://www.microsoft.com/microsoft-365/roadmap?filters=Microsoft%20Teams) ，瞭解最新的功能時間。

> [!NOTE]
> Microsoft 365 GCC High 或美國商務部在 DoD 環境 (進) 通訊。
