# <a name="messaging-policies-licensing"></a>訊息原則授權

我們會聽到世界各地的客戶聽取 COVID-19 （Coronavirus） pandemic 在工作中的新挑戰，以處理日常作業中的重大變動。 當您在危機回應期間，能夠直接且快速地與小組成員連線，就是在危機回應期間維持業務連續性所必需的，也就是許多人都回到工作場所。

最初是在2019年6月發佈的，Microsoft 團隊的[優先順序通知](https://support.microsoft.com/en-us/office/mark-a-message-as-important-or-urgent-in-teams-ea99d5b6-1317-4550-8d75-86ff14cd4462)可讓使用者持續傳送每兩分鐘重複一次的通知，在所有標示為緊急的郵件中，最多可重複執行20分鐘。 已將優先順序通知提供給所有團隊客戶的促銷活動<sup>1</sup> ，直到2020的第二半部。

我們會立即生效，我們已結束本次促銷，並將優先順序通知提供給所有團隊客戶<sup>1</sup>做為基本功能。 使用者現在可以使用優先順序通知來傳送及接收不受限制的緊急訊息。 [深入瞭解管理優先順序通知](../messaging-policies-in-teams.md)。

<sup>1</sup>對於 GCC、GCC 或 DoD 客戶，目前不提供優先順序通知。
