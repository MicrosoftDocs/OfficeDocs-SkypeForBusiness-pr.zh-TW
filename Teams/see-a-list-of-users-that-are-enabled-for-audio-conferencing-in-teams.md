# <a name="see-a-list-of-users-that-are-enabled-for-audio-conferencing-in-microsoft-teams"></a>查看在 Microsoft Teams 中啟用音訊會議的使用者Microsoft Teams

啟用貴Microsoft Teams音訊會議的使用者之後，您可以查看已啟用的使用者清單。 

> [!NOTE]
> [!INCLUDE [updating-admin-interfaces](includes/updating-admin-interfaces.md)]
  
## <a name="viewing-a-list-of-users"></a>檢視使用者清單

- 在 Microsoft Teams系統管理中心，按一下左側流覽中的 [**使用者。**

## <a name="what-else-should-i-know"></a>我還需要知道什麼？

- 當您查看已啟用的使用者清單時，可以從清單中選取使用者，以編輯該使用者的音訊會議設定。
    
- 當您選取單一使用者時，您可以查看指派給使用者的預設電話號碼，以及任何其他相關的音訊會議設定。    
   
- 您可以使用搜尋按鈕搜尋清單中的個別使用者。
    
## <a name="run-a-self-diagnostics-tool"></a>執行自我診斷工具

Microsoft 365系統管理員使用者可以存取可在租使用者內執行診斷，以確認使用者能夠建立 PSTN 電話會議。 

> [!NOTE]
>這項功能不適用於德國Microsoft 365、由世紀互聯提供的 Microsoft 365或Microsoft 365使用。

選取執行測試，如下所示。 這會在中心填入Microsoft 365 系統管理診斷。
>> [!div class="nextstepaction"]
>> [執行測試：Teams會議](https://aka.ms/TeasConfDiag)

診斷會執行大量的驗證。
