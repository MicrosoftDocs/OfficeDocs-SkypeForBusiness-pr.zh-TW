<a name="manage-the-whiteboard-in-microsoft-teams"></a>管理白板Microsoft Teams
==========================================

Microsoft Whiteboard是一種自由形式的數位畫布，其中人員、內容和想法彙集在一起。 會議中的白板Microsoft Teams由 Whiteboard Web App 提供，可讓Teams參與者在共用的數位畫布上一起繪圖、繪圖和書寫。

使用者可以共用白板，讓會議的所有參與者Teams白板。 同一個白板可同時在 Windows 10、iOS 和 Web App 上的所有 Whiteboard 應用程式中使用。

若要開啟或關閉 Whiteboard 應用程式，請參閱[為Microsoft Whiteboard啟用應用程式。](https://support.office.com/article/1caaa2e2-5c18-4bdf-b878-2d98f1da4b24) 請記住，這項設定會為整個組織啟用或停用 Whiteboard，而不只是Teams。

## <a name="whiteboard-on-surface-hub"></a>白板Surface Hub

在嘗試使用應用程式Microsoft Whiteboard，請確定白板應用程式已安裝于您的Surface Hub裝置上。 如果未安裝 Whiteboard，請Surface Hub裝置上，前往 Microsoft Store 應用程式，[然後取得](https://www.microsoft.com/p/microsoft-whiteboard/9mspc6mp8fm4?activetab=pivot:overviewtab)Microsoft Whiteboard。 若要詳細資訊，請參閱在 Microsoft Whiteboard[上啟用Surface Hub。](https://support.office.com/article/enable-microsoft-whiteboard-on-surface-hub-b5df4539-f735-42ff-b22a-0f5e21be7627)
