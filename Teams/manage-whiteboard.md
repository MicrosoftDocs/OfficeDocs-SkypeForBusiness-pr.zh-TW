# <a name="manage-the-whiteboard-in-microsoft-teams"></a>在 Microsoft Teams 中管理白板

Microsoft Whiteboard 是一種自由形式的數位畫布，其中人員、內容和想法彙集在一起。 Microsoft Teams 會議中的白板整合是由 Whiteboard Web App 提供，可讓 Teams 會議參與者在共用的數位畫布上一起繪圖、繪圖和書寫。

使用者可以共用白板，讓 Teams 會議的所有參與者都可以使用白板。 Windows 10、iOS 和 Web App 上的所有 Whiteboard 應用程式都同時提供相同的白板。

若要開啟或關閉 Whiteboard 應用程式，請參閱為貴組織啟用 [Microsoft Whiteboard](https://support.office.com/article/1caaa2e2-5c18-4bdf-b878-2d98f1da4b24)。 請記住，這項設定將針對整個組織啟用或停用 Whiteboard，而不只是針對 Teams。

## <a name="whiteboard-on-surface-hub"></a>Surface Hub 上的白板

在嘗試使用 Microsoft Whiteboard 之前，請確定您的 Surface Hub 裝置上已安裝 Whiteboard 應用程式。 如果未安裝 Whiteboard，請前往 Surface Hub 裝置上的 Microsoft Store 應用程式，然後取得 [Microsoft Whiteboard](https://www.microsoft.com/p/microsoft-whiteboard/9mspc6mp8fm4?activetab=pivot:overviewtab)。 詳細資訊，請參閱啟用 [Surface Hub 上的 Microsoft Whiteboard](https://support.office.com/article/enable-microsoft-whiteboard-on-surface-hub-b5df4539-f735-42ff-b22a-0f5e21be7627)。
