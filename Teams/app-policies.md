# <a name="overview-of-app-policies-used-to-manage-apps-in-microsoft-teams"></a>管理 Microsoft Teams 中應用程式的應用程式原則概觀

應用程式原則是用來控制Microsoft Teams中的應用程式。

## <a name="app-permission-policies"></a>應用程式權限原則

**應用程式許可權原則** 可用來控制哪些應用程式可供您的使用者使用。 身為系統管理員，您可以允許或封鎖使用者從Teams App Store 安裝的特定應用程式。 [應用程式許可權原則](teams-app-permission-policies.md)是透過Microsoft Teams系統管理中心來管理。

![應用程式許可權原則的螢幕擷取畫面。](media/app-permission-policy.png)

## <a name="app-setup-policies"></a>應用程式設定原則

**應用程式設定原則** 可讓您代表您的使用者釘選和安裝應用程式，並為組織控制這些設定。 [系統會](teams-app-setup-policies.md)在Teams系統管理中心管理應用程式設定原則。

![Teams系統管理中心的應用程式設定原則螢幕擷取畫面。](media/app-setup-policy.png)

## <a name="custom-app-policies"></a>自訂應用程式原則

您可以使用 **自訂應用程式原則** 控制誰可以上傳自訂應用程式Teams。 [自訂應用程式設定和原則](teams-custom-app-policies-and-settings.md)是透過Microsoft Teams系統管理中心來管理。

![自訂應用程式原則的螢幕擷取畫面。](media/custom-app-policy.png)
