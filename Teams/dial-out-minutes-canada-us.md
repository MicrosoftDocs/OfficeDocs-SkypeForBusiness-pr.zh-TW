# <a name="office-365-audio-conferencing-extended-dial-out-minutes-to-us-and-canada"></a>Office 365 音訊會議延伸到美國和加拿大的撥出分鐘數

針對採用我們音訊會議服務的客戶，且需要向美國和加拿大撥打電話時，Microsoft 現在提供的附加元件優惠可讓任何音訊會議使用者使用的是以實際限制的方式使用美國和加拿大撥出時間。 使用此附加元件 SKU 的會議召集人，不需將租使用者的區域加上國家撥出使用方式池或通訊點數，就能讓美國和加拿大不受限制地使用原則撥出通話。 <sup>1</sup>

|提供名稱 | 無限撥出時間給美國加拿大 |
|-----|------|
| 優惠中包含哪些無限制撥出的國家/地區？| 美國和加拿大。|
| 誰需要指派授權？ | 會議召集人需要指派授權。 初始化通話的使用者不需要擁有授權。 |
| 是否有任何先決條件？ | 需要啟用音訊會議。 它可以是訂閱或每分鐘服務。|
| 誰可以購買此優惠？ | 可購買語音會議的任何國家或地區。 [閱讀其他資訊](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)。|
| 是彙集數分鐘嗎？  |不。 在每個月，會議召集人的一般使用原則撥出分鐘數幾乎都不受限制。 |
| 這個新的附加元件 SKU 會取代音訊會議所含的60分鐘嗎？ | 不。 每個使用者每個月的現有音訊會議60分鐘數（在租使用者層級共用）將會繼續套用至所有區域撥出會議國家/地區。|
| 頻道的可用性為何？  | 它可在 EA/EAS、EES、CSP 和 Web Direct 中取得。  |
| 區段的可用性為何？ | 它適用于商業 (，包括 WW 商業公用磁區) 、教育、非贏利及美國 GCC |
| 如果有延長撥出分鐘數的使用者位於美國，並在音訊會議銷售至市場以外的地方傳送到其他國家/地區，該怎麼辦？ 他們的撥出服務是否能正常運作？ | 是的。 如果您是以美國 (為基礎的使用者 (代表其 Office 365 授權是在) 美國旅遊的公司中指派，而不是目前不在音訊會議) 的銷售目標位置中，那麼無論撥打電話發生的位置為何，美國或加拿大的撥出時間都將持續套用至美國/她的延長撥出時間。 |
|||

<sup>1</sup> Microsoft 保留限制或終止使用此功能的權利，以讓您在使用中或因應欺騙性的情況下使用。
