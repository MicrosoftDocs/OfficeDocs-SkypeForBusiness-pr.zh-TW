# <a name="office-365-audio-conferencing-extended-dial-out-minutes-to-us-and-canada"></a>Office 365音訊會議延長美國與加拿大的撥出分鐘數

針對採用我們的音訊會議服務，且需要撥打到美國和加拿大的商務需求較高的客戶，Microsoft 現在提供附加元件優惠，讓任何音訊會議使用者幾乎無限制地享有合理使用原則的美國和加拿大撥出分鐘數。 具有此附加元件 SKU 的會議召集人幾乎可以無限制地撥打合理使用原則撥入美國和加拿大的電話，而不扣除租使用者的區域 A 國家/地區撥出使用量集區或通訊信用額度。 <sup>1</sup>

|優惠名稱 | 無限撥出分鐘到美國和加拿大 |
|-----|------|
| 優惠包含的無限撥出國家/地區是什麼？| 美國和加拿大。|
| 神秘授權嗎？ | 會議召集人必須指派授權。 初始化通話的使用者不需要擁有授權。 |
| 有任何先決條件嗎？ | 音訊會議必須啟用。 它可以是訂閱或每分鐘服務。|
| 神秘購買此優惠嗎？ | 任何可以購買音訊會議的國家/地區。 [閱讀更多](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)。|
| 是否將分鐘數集中？  |不。 會議召集人將擁有幾乎無限制的使用原則撥出分鐘到美國/加拿大每個月。 |
| 這個新附加元件 SKU 會取代音訊會議中包含的 60 分鐘嗎？ | 不。 現有的音訊會議每月每個使用者 60 分鐘 ，以租使用者層級集中，會繼續適用于所有區域 A 撥出式會議國家/地區。|
| 頻道可用性是什麼？  | 它可在 EA/EAS、EES、CSP 和 Web Direct 上使用。  |
| 什麼是區段可用性？ | 它適用于商業 (包括 WW 商業) 、教育、非營利組織及美國GCC |
| 如果具有延長撥出分鐘數到美國/CAN 的使用者位於美國，並前往另一個國家/地區以外的銷售Audio-Conferencing呢？ 他們的撥出服務可以嗎？ | 是的。 如果位於美國 (的使用者表示其 Office 365 授權是在美國) 指派給美國 (目前不在音訊會議) 銷售位置的市場，則所有撥入美國或加拿大的撥出分鐘數都會繼續適用于其美國/CAN 附加元件延伸撥出分鐘數，無論通話地點在哪裡。 |
|||

<sup>1</sup> Microsoft 保留限制或終止功能在異常使用或懷疑用於詐騙用途時之使用的權利。
