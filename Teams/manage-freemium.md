# <a name="manage-the-free-version-of-microsoft-teams"></a>管理免費版 Microsoft Teams

在免費版 Microsoft Teams中，第一個在貴組織中註冊的人具有有限的系統管理員角色。 具有此有限角色的人可以新增和移除小組成員，並指定任何人都可以邀請其他成員，但此使用者沒有進一步管理功能，且無法存取Microsoft Teams中心。 若要深入瞭解，請參閱邀請[人員Teams (免費) 。](https://support.office.com/article/invite-people-to-teams-free-53a9b20c-2ad7-442e-967c-2e9305e96463)

若要取得完整的 Teams 功能 ，包括增強型系統管理，您必須為使用者購買適當的 Microsoft 365 或 Office 365 訂閱者案，以升級至完整版本的[Teams。](upgrade-freemium.md) 

若要進一Teams版本及其功能，請參閱[比較Teams方案](https://products.office.com/microsoft-teams/free)。



## <a name="more-information"></a>詳細資訊

- 如想快速摘要瞭解免費Teams與免費Teams之間的差異，請參閱免費Microsoft Teams Microsoft Teams[的差異](https://support.office.com/article/0b69cf39-eb52-49af-b255-60d46fdf8a9c)。 
- 有關升級至完整版訂閱Teams，請參閱從 Teams 免費升級至[](https://support.office.com/article/29475bbd-a34f-4175-9b33-d44430f8ad39)Teams，Microsoft Teams免費升級至[訂閱版本](upgrade-freemium.md)。
- 若要開始使用與升級使用者相關的其他系統管理員工作，包括新增使用者授權、變更使用者名稱，以及指派臨時密碼，請參閱適用于系統管理員從[Teams](https://support.office.com/article/75a95e7f-001e-42d0-a787-ae8b992d5a52)免費升級到付費訂閱。
