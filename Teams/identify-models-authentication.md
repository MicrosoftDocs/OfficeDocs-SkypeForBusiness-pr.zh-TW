<span data-ttu-id="b9675-105">Microsoft 團隊支援 Office 365 提供的所有身分識別模型。</span><span class="sxs-lookup"><span data-stu-id="b9675-105">Microsoft Teams support all the identity models that are available with Office 365.</span></span> <span data-ttu-id="b9675-106">支援的身分識別模型包括：</span><span class="sxs-lookup"><span data-stu-id="b9675-106">Supported identity models include:</span></span>

-   <span data-ttu-id="b9675-107">**雲端身分識別**：在這個模型中，使用者是在 Office 365 中建立和管理，並儲存在 Azure active directory 中，且密碼是由 Azure active directory 驗證。</span><span class="sxs-lookup"><span data-stu-id="b9675-107">**Cloud Identity**: In this model, a user is created and managed in Office 365 and stored in Azure Active Directory, and the password is verified by Azure Active Directory.</span></span>
