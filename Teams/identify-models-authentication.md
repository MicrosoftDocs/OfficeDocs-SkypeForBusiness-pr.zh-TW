<span data-ttu-id="28175-105">Microsoft 團隊支援 Microsoft 365 和 Office 365 提供的所有身分識別模型。</span><span class="sxs-lookup"><span data-stu-id="28175-105">Microsoft Teams support all the identity models that are available with Microsoft 365 and Office 365.</span></span> <span data-ttu-id="28175-106">支援的身分識別模型包括：</span><span class="sxs-lookup"><span data-stu-id="28175-106">Supported identity models include:</span></span>

-   <span data-ttu-id="28175-107">**雲端身分識別**：在這個模型中，使用者是在 Microsoft 365 或 Office 365 中建立和管理，並儲存在 Azure active directory 中，且密碼是由 Azure active directory 驗證。</span><span class="sxs-lookup"><span data-stu-id="28175-107">**Cloud Identity**: In this model, a user is created and managed in Microsoft 365 or Office 365 and stored in Azure Active Directory, and the password is verified by Azure Active Directory.</span></span>
