# <a name="step-3-assign-business-voice-licenses"></a>步驟 3：指派商務語音授權

每一個需要使用Microsoft 365 商務語音使用者都需要獲得商務語音授權。

1. 在 Microsoft 365[系統管理中心](https://admin.microsoft.com)，前往帳單  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=842264" target="_blank">授權</a>頁面。
2. 選取 **Microsoft 365 商務語音**。
3. 在產品詳細資料頁面上，選取指派 **授權**。
4. 在 " **指派授權給使用者"** 窗格中，開始輸入名稱，然後從結果中選擇名稱，以將其新增到清單中。 一次最多可以新增 20 個使用者。
5. 完成時，請選取 **[指派]**，然後選取 **[關閉]**。
