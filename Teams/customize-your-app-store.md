# <a name="customize-your-organizations-app-store-in-microsoft-teams"></a>在 Microsoft Teams 中自訂貴組織的 App Store

現在您可以新增標誌、自訂背景和自訂文字色彩，讓貴組織在Teams中儲存的應用程式與貴公司的商標一起自訂，讓它更吸引使用者。

> [!NOTE]
> 商標變更最多可能需要 24 小時才會生效。

您可以選取 [自訂市集] **Teams應用程式**  >  ，以存取系統管理中心的自訂 **市集**。

  ![管理員主機自訂醒目提示的市集功能。](media/customize-app-store.png)

本文將說明自訂應用程式的儲存功能，以及如何使用這些功能。

## <a name="customize-your-organization-logo"></a>自訂您的組織標誌

<!-- Bookmark used by Context Sensitive Help (CSH). Do not delete. -->
<a name="orglogo"> </a>
<!-- Do not remove the bookmark link above. -->

您可以在這裡上傳貴組織的標誌。 選取的標誌會顯示在 **[為您的租使用者建置** 的 **應用程式**  >  ] 頁面的Teams用戶端中。

當您選取 [ **選擇圖片]** 選項時，可以上傳您選擇的檔案。 標誌應為 240x60，否則會調整為該大小。 它應該不會大於 5 MB。 支援的格式為 .svg、.png 和 .jpg。

標誌會顯示在Teams市集中租使用者應用程式目錄的右上角。

## <a name="customize-your-small-logo"></a>自訂您的小標誌

<!-- Bookmark used by Context Sensitive Help (CSH). Do not delete. -->
<a name="orglogomark"> </a>
<!-- Do not remove the bookmark link above. -->

貴組織的標誌或小型標誌是代表貴組織品牌的影像或符號，不包含您的公司名稱。 愛心標記可協助小組成員識別 App 市集。 選取的標誌會顯示在 [應用程式] 頁面Teams用戶端中。

選取 **[選擇圖片]** 選項，然後上傳您選擇的檔案。 將標誌標記解析度保持在 32x32 圖元，或將其向下插入到此解析度。 保持檔案大小小於 5 MB。 支援的格式為 .svg、.png 和 .jpg。

Teams Microsoft Store會在 **[為您的租使用者打造**] 區段旁顯示標誌。

## <a name="customize-the-background-color"></a>自訂背景色彩

<!-- Bookmark used by Context Sensitive Help (CSH). Do not delete. -->
<a name="custombackground"> </a>
<!-- Do not remove the bookmark link above. -->

請確定背景與文字有足夠的對比，讓使用者可以清楚閱讀文字。 背景會顯示在 **[專為您的租使用者建立** 的 **應用程式**  >  ] 標題的Teams用戶端中。

您的選擇包括︰

* 使用Teams用戶端預設背景主題
* 選擇圖片。 當您選取背景圖片時，可以上傳您選擇的檔案。 背景應為 1212x100 圖元，否則會調整為該大小。 它應該不會大於 5 MB。 支援的格式為 .svg、.png 和 .jpg。
* 選擇自訂背景。 當您選取 **[選擇自訂色彩**] 時，色彩選擇器會隨即開啟。 調整色彩選擇器的縮放比例，選取您要的色彩。

背景影像或色彩會做為Teams租使用者應用程式目錄中頂端橫幅的背景。

## <a name="customize-the-text-color-of-your-name"></a>自訂名稱的文字色彩

<!-- Bookmark used by Context Sensitive Help (CSH). Do not delete. -->
<a name="textcolor"> </a>
<!-- Do not remove the bookmark link above. -->

提供與背景相較之文字色彩的足夠對比，讓使用者可以清楚讀取貴組織的名稱。 文字會顯示在 **專為您的租使用者標題建置****應用程式的Teams** 用戶端中  >  。

當您選取 **[選擇自訂色彩**] 時，色彩選擇器會隨即開啟。 調整色彩選擇器的縮放比例，選取您要的色彩。

 ![色彩選擇器。](media/choose-a-custom-color.png)

## <a name="complete-the-customization-of-your-team-apps-store"></a>完成團隊應用程式市集的自訂作業

當您完成自訂 App 市集時，請務 **必選取** [儲存] 以儲存變更。
若要預覽您的變更，請選取 **[預覽應用程式市集** ]，以查看您自訂應用程式市集的呈現方式。

![自訂應用程式市集的預覽。](media/PowerAppsInStore650w.png)

> [!NOTE]
> 由於某些預設色彩是根據使用者的Teams版用戶端，因此表示方式可能與您自訂應用程式市集的最終版本有所不同。
