# <a name="types-of-phone-numbers"></a>電話號碼類型

Microsoft Teams根據您要使用電話號碼的用途，使用不同的電話號碼類型。 

Teams **會使用** 使用者號碼，可指派給使用者貴組織的使用者，以及服務號碼，這些號碼會指派給音訊會議、自動語音服務或通話佇列等服務。 服務電話號碼的並行 *通話容量高於*  使用者號碼。 服務電話號碼可供使用，但會依國家/地區及號碼類型 (無論是付費或免付費號碼) 。 如果您需要其他號碼或其他號碼類型，Teams系統管理中心看到的數位類型，您可以提交電話號碼要求給[TNS 服務台](manage-phone-numbers-for-your-organization/contact-tns-service-desk.md)。

> [!NOTE]
> 請注意，非地理號碼的可用性有地區及國家/地區限制。
> 如果您有其他問題或需要取得更多電話號碼，請 [聯絡 TNS 服務台](manage-phone-numbers-for-your-organization/contact-tns-service-desk.md)。
  
### <a name="user-numbers"></a>使用者編號
    
使用者編號會指派給使用者，而且有兩種：
    
- **地理號碼：** 地理數位與地理區域有關系，且最常見。 例如，在大多數情況下，地理位置電話號碼只能在國家/地區的特定位址、城市、省/市或地區內使用。
    
- **非地理號碼：** 非地理數位是國家/地區內與地理區域沒有關系的國家號碼。 例如，從國家/地區內任何位置撥打號碼時，非地理號碼的成本通常相同。 此外，部分國家 /地區 ，例如丹麥，僅提供非地理號碼。
    
### <a name="service-numbers"></a>服務編號 

服務號碼提供數種不同的號碼類型，且可用性會因國家/地區而異。
    
#### <a name="toll-service-numbers"></a>付費服務號碼
    
付費服務號碼可能會產生來電者的付費費用，而且有兩種：
    
- **地理號碼：** 地理數位與地理區域有關系。 例如，在大多數情況下，地理位置電話號碼只能在國家/地區的特定位址、城市、省/市或地區內使用。
        
- **非地理號碼：** 非地理數位是國家/地區內與地理區域沒有關系的國家號碼。 例如，從國家/地區內任何位置撥打號碼時，非地理號碼的成本通常相同。
   
#### <a name="toll-free-service-numbers"></a>免付費服務號碼 

這些服務號碼通常不會向來電者收取付費費用。 Teams提供超過 60 個國家/地區的免付費電話號碼。
    
> [!CAUTION]
> 部分國家/地區及原始號碼類型 ，例如來自行動電話的通話，在某些情況下可能會向來電者收取付費費用。
