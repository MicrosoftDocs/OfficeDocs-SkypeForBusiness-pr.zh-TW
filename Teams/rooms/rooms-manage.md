# <a name="manage-microsoft-teams-rooms"></a>管理 Microsoft Teams 會議室

如果您的組織中有 Microsoft 小組會議室認證的裝置，您可以使用 Microsoft 團隊系統管理中心，從中央位置管理它們。 您可以：

- 執行裝置管理（例如重新開機或封鎖裝置），以及下載裝置記錄
- 套用團隊專用的設定
- 檢查 Microsoft 團隊聊天室裝置及其週邊設備的健康情況狀態，包括相機、顯示器、麥克風等等
- 查看目前與過去的會議活動（例如通話品質、網路健康與連線性，以及參與者人數）
- 查看連線至 Microsoft 球隊房間裝置的週邊設備（例如相機和投影機）

若要管理團隊室裝置，請開啟[Microsoft 團隊系統管理中心](https://admin.teams.microsoft.com)，然後移至 [**裝置**  >  **團隊] 會議室**。

:::image type="content" source="../media/teams-rooms-summary.png" alt-text="小組系統管理中心摘要頁面":::

> [!IMPORTANT]
> 若要使用 [團隊管理中心] 管理裝置，您必須獲指派團隊服務系統管理員或全域系統管理員角色。

## <a name="make-changes-to-teams-rooms-devices"></a>變更團隊聊天室裝置

如果您有一個以上的團隊會議室裝置，您可以在多個裝置上同時執行大部分動作。 例如，您可以在所有裝置上同時設定團隊 app 設定。

### <a name="device-settings"></a>裝置設定

您可以在組織中的一或多台裝置上變更設定。 若要變更設定，請選取您要管理的裝置或裝置，然後選取 [**編輯設定**]。 新窗格隨即開啟，您可以在裝置上變更所有設定。 下表列出您可以使用 [團隊系統管理中心] 變更的設定。 某些設定只有在您選取單一裝置時才可使用。

如果您選取一個以上的裝置，則支援大量編輯的設定會顯示下列兩個選項。

- **保留現有值**如果您選擇這個選項，就不會對您所選的裝置上的設定進行任何變更。
- 將**現有值取代為**如果您選擇此選項，您可以使用您所提供的值來更新您所選取之裝置上的設定。
    > [!CAUTION]
    > 您選擇要更新之設定的現有值將會以您所提供的值取代。 如果您想要新增至現有值的清單，您必須將現有的值包含在您想要新增的值中。 例如，如果某個設定有現有的網域清單 `contoso.com, fabrikam.com` ，而您想要新增 `northwindtraders.com` ，您需要提供的值就是 `contoso.com, fabrikam.com, northwindtraders.com` 。

### <a name="device-restart-options"></a>裝置重新開機選項

裝置設定的變更只會在裝置重新開機後生效。 當您進行需要重新開機的變更時，您可以選擇是否要立即重新開機裝置，或排程重新開機。 以下是可用的重新開機選項：

- **立即重新開機**如果您選擇此選項，當您選取此選項時，所有您要變更的裝置都會立即重新開機。
- **排程重新開機**如果您選擇此選項，您可以重新開機您要變更的裝置，以減少對組織造成中斷的影響。
  - **選取日期和時間**-選擇要重新開機裝置的特定日期和時間。 您選擇的日期和時間是裝置被重新開機的地方。 
  - 在**夜間重新開機時保留更新**裝置會在夜間重新開機以執行維護。 您對裝置所做的變更將會在此重新開機期間生效。

> [!CAUTION]
> 重新開機時，在使用中的裝置將無法在重新開機處理常式期間使用。 它們會與進行中的會議中斷連線，而且無法加入新的會議。

### <a name="remove-or-block-a-device"></a>移除或封鎖裝置

當您**移除**裝置時，裝置會從您的組織中移除，且不會再出現在團隊系統管理中心的 [小組室] 裝置清單中。 

當您**封鎖**裝置時，小組就不會再與裝置進行通訊。 封鎖的裝置不會傳送命令，即使它們包含在正在大容量編輯的裝置群組中也一樣。 它仍列在您的小組室裝置清單中，且其狀態為 [已**封鎖**]。

不論是否已封鎖或移除裝置，如果它仍是使用有效的使用者名稱和密碼進行設定，則會自動將它重新新增至您的小組室裝置清單（如果它連接至 Microsoft 365）。

若要移除一或多個裝置，請執行下列動作：

1. 移至 [**裝置**  >  **團隊聊天室**]，然後選取您要移除的裝置。
1. 選取 [**移除**]。

若要封鎖裝置，請執行下列動作：

1. 移至 [**裝置**  >  **團隊聊天室**]，然後選取您要封鎖的裝置名稱。
1. 在 [裝置詳細資料] 頁面上，選取頁面右上角的 [**動作**]。
1. 選取 [**封鎖**]。

若要解除封鎖裝置，請執行下列動作：

1. 移至 [**裝置**  >  **團隊聊天室**]，然後選取您要封鎖的裝置名稱。
1. 在 [裝置詳細資料] 頁面上，選取頁面右上角的 [**動作**]。
1. 選取 [**解除封鎖**]。

## <a name="download-device-logs"></a>下載裝置記錄

如果要求 Microsoft 支援，您可以下載裝置的診斷記錄檔複本。 將記錄檔壓縮成 zip 檔案，該檔案可從 [小組系統管理中心] 下載。

若要從團隊機房裝置將記錄下載到您的電腦，請執行下列動作：

1. 移至 [**裝置**  >  **團隊聊天室**]，然後選取您要從中下載記錄檔的裝置名稱。
1. 選取 [**下載裝置記錄**]。 可能需要幾分鐘的時間，裝置記錄才能變得可用。
1. 選取 [歷程**記錄**] 索引標籤，然後選取 [**診斷**檔案] 底下的 [記錄檔案]。 包含裝置診斷記錄檔的 zip 檔案將會下載到您瀏覽器的預設 [下載] 資料夾。

## <a name="view-device-information"></a>查看裝置資訊

從 [團隊系統管理中心]，您可以在組織中查看所有裝置的整體狀態，並分別查看每個裝置的詳細資料。

### <a name="teams-rooms-system-dashboard"></a>團隊聊天室系統儀表板

[團隊聊天室] 系統儀表板會快速顯示所有裝置的狀態和健康情況。

### <a name="device-details-view"></a>裝置詳細資料檢視

若要查看裝置的詳細資訊，請從裝置清單中選取它的名稱。 在 [詳細資料] 視圖中，您可以看到裝置的下列相關資訊：

- **健康狀態**顯示團隊室裝置的整體健康情況。 健康情況可能是**正常**或不**正常**。
- **離線自**顯示 Microsoft 365 最後一次可以與裝置通訊的時間。
- **裝置狀態**顯示裝置目前的狀態： [**空閒**]、[**團隊會議**]、[ **Skype 會議**] 或 [**攝取**]。
- **週邊設備**顯示已連接至團隊房間裝置的外設，以及其健康狀態。 健康狀態可以是 [**已連接**] 或 [已**中斷**連線]。
- **健康情況**顯示已連接至團隊聊天室裝置、網路連線、登入至所需服務的相關外設以及軟體版本資訊的詳細資訊。
- **詳細資料**顯示製造商資訊、網路 IP 位址，以及團隊房間裝置串列/MAC 位址。
- **活動**顯示過去的會議詳細資料，包括會議的日期和時間、參與者人數、持續時間及音訊品質。 如需會議詳細資料的詳細資訊，請參閱本文稍後的[會議活動詳細資料](#meeting-activity-details)一節。
- 歷程**記錄**顯示團隊聊天室裝置上的管理活動歷程記錄，包括設定更新、裝置重新開機及裝置記錄下載連結。

#### <a name="meeting-activity-details"></a>會議活動詳細資料

[團隊聊天室] 中的 [**活動**] 索引標籤詳細資料會顯示裝置在一段時間內參與的所有會議的高層次及詳細資訊。 在 [**活動**] 索引標籤中，您可以查看會議的召開時間、參與會議的參與者人數，以及在會議期間音訊品質。

:::image type="content" source="../media/teams-rooms-meeting-activity-summary.png" alt-text="團隊聊天室裝置活動摘要清單":::

若要查看特定會議的詳細資訊，請選取會議的日期和時間，以取得詳細資訊。 如果會議只有兩個參與者，您會看到 [參與者詳細資料] 頁面，否則您會看到 [參與者摘要] 頁面。

##### <a name="participant-summary"></a>參與者摘要

[參與者摘要] 頁面會顯示參加會議的所有參與者。 您可以查看每個參與者加入會議的時間、其名稱、音訊品質，以及在會話期間所使用的功能。 若要查看參與者會話的詳細資料，請選取該參與者的會話開始時間。

:::image type="content" source="../media/teams-rooms-meeting-activity-participant-summary.png" alt-text="團隊會議室裝置會議詳細資料":::

##### <a name="participant-details"></a>參與者詳細資料

[參與者詳細資料] 頁面會顯示該參與者會話的端對端診斷資訊。 如下圖所示，提供給參與者和團隊聊天室裝置的**裝置**、**系統**和**連線資訊。** 此外，也提供參與者與團隊聊天室裝置之間的**網路**診斷資訊。 選取您要取得詳細資訊之內容的圖示。 如需其他診斷資訊，請選取 [**高級**] 索引標籤。

:::image type="content" source="../media/teams-rooms-meeting-activity-participant-details.png" alt-text="團隊會議室裝置通話詳細資料":::
