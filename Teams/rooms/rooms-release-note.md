- 切換至影片庫版面配置（即使內容存在）
- 虛擬為簡報者提供出席者與控制項的實習支援

> 版本4.5 是最新版本，可支援 Windows 10 版本 1803;在 Windows 10 版本1803上，系統將不會提供未來版本。 如需有關 Windows 版本支援的詳細資訊，請參閱 [windows 10 版本支援](https://docs.microsoft.com/microsoftteams/rooms/rooms-lifecycle-support#windows-10-release-support)。

### <a name="401050-07102019"></a>4.0.105.0 (07/10/2019) 

此更新中引入的專案：

- Skype 會議室 System store 應用程式 rebrand 到 "Microsoft 團隊聊天室"

- 在團隊通話/會議<sup>1</sup>後撥打品質意見反應評分
- 在 Microsoft 團隊聊天室上接收/轉譯 Microsoft 球隊會議室在會議室上方從 PC/Web/行動小組共用用戶端<sup>1</sup> <sup>2</sup>
- 已移除 Windows 10 版本1809的支援，因為 Microsoft 團隊聊天室用戶端的相容性問題。 Windows 10 版本19H1 支援將會在未來版本中新增

<sup>1</sup> 使用小組鈴響的 Microsoft 團隊服務推出。 此功能可能會在舊版或更新版本的4.0.105.0 用戶端更新中提供

<sup>2</sup> 要求 IT 管理員開啟 Microsoft 白板。 此外，如果您使用的是觸控式啟用的會議室顯示，您必須在 [裝置管理員登入] 中使用 Windows 設定來校準多個觸控顯示器，才能開始使用 Microsoft 白板在小組會議中共用的聊天室顯示器進行共同作業
