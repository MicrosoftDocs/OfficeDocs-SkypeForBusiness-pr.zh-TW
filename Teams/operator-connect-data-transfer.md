# <a name="data-transfers-between-carriers-and-microsoft-for-operator-connect"></a>電信公司與 Microsoft for Operator 連線

## <a name="provisioned-and-assigned-numbers"></a>已置備和已指派的數位

為了確保接線生 連線 計畫正常運作，Microsoft 會透過 API 通知每個參與的電信企業，而電信公司的電話號碼已在 Teams 系統管理中心內指派，以及該電話號碼的 SIP URI (會話初始通訊協定統一資源識別項) 。

## <a name="call-detail-records-and-quality-data"></a>通話詳細資料記錄和品質資料

Microsoft 會透過 API 提供通話詳細記錄和品質資料給每個參與的電信公司。 這是 Microsoft 和電信服務提供者執行疑難排解，並確保服務正常運作的必要步驟。

## <a name="call-duration-data"></a>通話持續時間資料

每個電信公司都會透過 API 向 Microsoft 提供客戶使用 MICROSOFT TEAMS。 Microsoft 會使用這些記錄為客戶提供使用方式資訊，並確保我們的記錄與電信公司的記錄一致。 Microsoft 會根據相關法律和法規，以及 Microsoft 隱私權聲明來儲存及處理這些記錄。
