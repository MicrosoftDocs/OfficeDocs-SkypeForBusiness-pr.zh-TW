# <a name="data-transfers-between-carriers-and-microsoft-for-operator-connect"></a>電信公司與 Microsoft for Operator 連線

Microsoft 是其處理與 Operator 連線 有關的個人資料的連線。 客戶必須通知其使用者以下資訊：

- Microsoft 的運算子連線功能Microsoft Teams可讓Teams使用者透過通話網路Microsoft Teams。
- 當貴組織啟用此功能時，Microsoft 會與貴組織的通訊服務電信業者 (電信業者) 共用資訊，以確保服務正常運作及疑難排解。

當您使用運算子時，Microsoft 與運算子共用的資訊連線包括：

- 您的電話號碼，以及指派給您電話號碼的 Microsoft 指派的 SIP URI (會話初始通訊協定統一資源識別項) 指派給電話號碼;
- 通話詳細資料和通話品質資料，例如來電和來電收件者電話號碼;通話邀請和結束時間;特定本機號碼識別碼;通話持續時間;服務連接資訊 (例如來電者與來電者之 IP 位址及作業系統) ;和
- 緊急通話位置資訊。

運算子將能夠匯出並下載上述資訊。

此資訊將由您的接線員依照您的接線員隱私權作法處理。 若要詳細資訊，請參閱您的運算子的隱私權政策及使用條款。 如需 Microsoft 隱私權作法的資訊，請參閱此連結中適用于Teams [Microsoft 隱私權聲明](https://go.microsoft.com/fwlink/?LinkId=521839)。
