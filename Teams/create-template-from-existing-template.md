# <a name="create-a-new-template-from-an-existing-team-template-in-microsoft-teams"></a>從 Microsoft 團隊中的現有小組範本建立新範本

[!INCLUDE [preview-feature](includes/preview-feature.md)]

**EDU 客戶尚不支援自訂範本。**

Microsoft 團隊提供預先建立的範本，可供您儲存及修改以符合您特定的組織需求。

1. 登入團隊系統管理中心。

2. 在左側導覽中，展開 [**團隊**  >  **小組範本**]。

3. 在 [ **小組範本** ] 區段中，選取您要複製之範本旁邊的，以醒目提示該範本。

4. 選取 [ **複製**]。

![[團隊範本] 對話方塊的影像，其中醒目提示 [新增]。](media/template-duplicate.png)

 (或者，您可以選取 [**新增**  >  **現有範本的開始**] 來開啟 [**選取要做為起點的範本**]。 ) 

![小組範本開始點畫面的影像，其中醒目提示了現有的範本。](media/template-start-existing-template.png)

5. 在 [ **選取要用來做為起點的範本** ] 畫面中，選取 **[下一步]**。

    重複的範本隨即開啟，並將 [ **複本** ] 附加到名稱上。

6. 完成下欄欄位，然後選取 **[下一步]**：
    - 範本名稱
    - 範本簡要與詳細描述
    - 地區可視性  

7. 在 [ **頻道]、[索引標籤] 和 [應用程式]** 區段中，修改您團隊所需的任何現有通道和應用程式。

    1. 選取要編輯的頻道，然後選取 [ **編輯**]。
    2. **在完成時套用您的** 變更。

8. 新增您團隊所需的任何頻道或應用程式。

    1. 在 [ **頻道** ] 區段中，選取 [ **新增**]。
    2. 在 [ **新增** ] 對話方塊中，命名頻道。
    3. 新增描述。
    4. 決定是否要依預設顯示頻道。
    5. 搜尋您要新增至頻道的 app 名稱。
    6. 選取 **[** 完成後套用]。

7. 完成新增頻道和應用程式後，請選取 [ **提交** ]。

    新範本會儲存在範本庫中。

> [!Note]
> 團隊使用者可能需要長達24小時的時間，才能看到圖庫中的自訂範本。
