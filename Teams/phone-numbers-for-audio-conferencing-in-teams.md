# <a name="phone-numbers-for-audio-conferencing-in-microsoft-teams"></a>Microsoft Teams 音訊會議的電話號碼

當您為Microsoft Teams設定 **音訊會議** 時，撥入電話號碼會自動指派給您的組織。 若要查看指派給音訊會議橋接器的電話號碼，請在Microsoft Teams系統管理中心，移至 **[會議**  >  **會議] 橋接器**。 如需詳細資訊，請移至 [查看音訊會議號碼清單](see-a-list-of-audio-conferencing-numbers-in-teams.md)。

## <a name="audio-conferencing-coverage-and-pricing"></a>音訊會議涵蓋範圍與定價

如需所有可使用音訊會議的國家/地區和城市的完整清單，請參閱 [音訊會議和通話方案的國家/地區和地區可用性](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)。 如需定價資訊，請參閱 [音訊會議定價](https://go.microsoft.com/fwlink/?linkid=799762)。

## <a name="dial-in-phone-numbers-in-a-meeting-invite"></a>會議邀請中的撥入電話號碼

若要管理包含的電話號碼：當Microsoft Teams使用者在 Outlook 或Outlook Web App中排程會議時，請參閱[在 Microsoft Teams 中設定邀請中包含的電話號碼](set-the-phone-numbers-included-on-invites-in-teams.md)。

## <a name="dial-in-phone-numbers-set-on-an-audio-conferencing-bridge"></a>音訊會議橋接器上設定的撥入電話號碼

可指派給您的會議橋接器的音訊會議電話號碼有兩個類型：**共用** 和 **專用**。 任何來電者都可以使用這兩種類型的號碼來加入您組織中舉行的音訊會議。

- **專用電話號碼** 是只有在組織內部使用者才可以使用的電話號碼。 您可以變更某人撥入其中一個號碼時所使用的語言。

- **共用電話號碼** 是可與其他Microsoft 365或Office 365共用的電話號碼。 您無法變更某人撥入其中一個號碼時所使用的語言。

雖然指派給召集人的預設音訊會議號碼只會包含在會議邀請中，呼叫者可以使用指派給您的會議橋接器的任何電話號碼加入會議。 您可以使用每個會議邀請中包含的 [尋找當地電話號碼] 連結，來取得可用來加入會議的電話號碼清單。

## <a name="automatically-assigned-audio-conferencing-phone-numbers"></a>自動指派的音訊會議電話號碼

組織若已啟用音訊會議功能，系統便會自動將共用的音訊會議電話號碼指派給組織。 指派電話號碼時，系統會將電話號碼指派為會議橋接器的預設電話號碼。 指派為橋接器預設號碼的電話號碼，將會是來自組織的國家/地區的電話號碼。

若要查看已自動指派電話號碼給組織之國家/地區的清單，請參閱[音訊會議與通話方案的適用國家/地區](country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans.md)。

> [!NOTE]
> 登入 **Microsoft 365 系統管理中心**，並查看 [組織設定檔] 下，即可找出組織的國家或地區位置。

> [!CAUTION]
> 由於委內瑞拉、印尼和阿拉伯聯合大公國 (UAE) 的付費電話號碼有限，來自這些國家/地區中的組織，將不會自動獲指派音訊會議付費電話號碼。 來自這些位置的免付費電話號碼是否可供使用，視可用庫存而定。

## <a name="what-else-should-you-know"></a>您還應知道什麼？

- 若要查看音訊會議支援的語言清單，請參閱 [音訊會議支援的語言](audio-conferencing-supported-languages.md)。

- 您可以為每個音訊會議電話號碼設定最多四種語言 -一個主要和三個次要語言。 您也可以在專用音訊會議電話號碼上設定語言。

- 若要設定使用者的撥入電話號碼，請參閱 [設定邀請中包含的電話號碼](set-the-phone-numbers-included-on-invites-in-teams.md)。
