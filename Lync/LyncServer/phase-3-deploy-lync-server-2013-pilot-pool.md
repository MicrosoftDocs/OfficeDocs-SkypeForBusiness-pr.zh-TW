</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-19_

本節涵蓋部署 Lync Server 2013 試驗池所需的步驟。 部署 Lync Server 2013 需要使用拓撲建立器來定義您的拓撲及您想要部署的元件，為您的環境準備部署 Lync Server 2013 元件，在第一個前端發佈拓撲設計伺服器，然後針對您部署的元件安裝和設定 Lync Server 2013 軟體。 完成後，您的 Lync Server 2013 試驗池部署會與現有的 Lync Server 2010 池共存。
