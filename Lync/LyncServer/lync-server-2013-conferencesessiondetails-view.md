# <a name="conferencesessiondetails-view-in-lync-server-2013"></a>Lync Server 2013 中的 ConferenceSessionDetails 檢視

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-11-12_

ConferenceSessionDetails 檢視會儲存多方成員工作階段的相關資訊。 每筆記錄都代表一個會議工作階段，它可以是有「焦點」的工作階段，或是特定會議伺服器的工作階段。 Microsoft Lync Server 2013 中已採用此檢視。
