# <a name="backing-up-file-stores-in-lync-server-2013"></a>在 Lync Server 2013 中備份檔案存放區

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-17_

備份 Lync Server 檔案存放區，包括 Lync Server 元件所使用的所有檔案和資料夾。

<div>

## <a name="to-back-up-file-stores"></a>備份檔案存放區

1.  若要尋找 Lync Server 檔案存放區的特定位置，請開啟 [拓撲建立器]，然後在 [檔案**儲存區**] 節點中查看。

2.  使用 Robocopy 或其他檔案系統管理工具，將每個檔案存放區\\複製到 $Backup 的檔儲存。
