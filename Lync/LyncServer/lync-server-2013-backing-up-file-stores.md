# <a name="backing-up-file-stores-in-lync-server-2013"></a>在 Lync Server 2013 中備份檔案存放區

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-17_

備份 Lync Server 檔存放區時，會包含 Lync Server 元件所使用的所有檔案和資料夾。

<div>

## <a name="to-back-up-file-stores"></a>若要備份檔案存放區

1.  若要尋找 Lync Server 檔案存放區的特定位置，請開啟拓撲產生器，然後在 [檔案存放 **區** ] 節點中查看。

2.  使用 Robocopy 或其他檔案系統管理工具，將每個檔案存放區複製到 $Backup 儲存區 \\ 。
