# <a name="backing-up-file-stores-in-lync-server-2013"></a>備份 Lync Server 2013 中的檔案存放區

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改主題：** 2013年-02-17_

備份 Lync Server 檔案存放區包含的所有檔案和 Lync Server 元件所使用的資料夾。

<div>

## <a name="to-back-up-file-stores"></a>若要備份檔案存放區

1.  若要找出您 Lync 伺服器檔案存放區的特定位置，請開啟拓撲產生器，並查看**檔案會儲存**] 節點。

2.  使用 Robocopy 或其他檔案系統管理工具，將每個檔案存放區複製到 $Backup\\檔案存放區。
