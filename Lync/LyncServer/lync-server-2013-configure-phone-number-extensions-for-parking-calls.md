# <a name="configure-phone-number-extensions-for-parking-calls-in-lync-server-2013"></a>設定電話分機號碼的 Lync Server 2013 中駐留通話

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-09-10_

通話駐留應用程式來駐留通話通話駐留軌道表中使用分機號碼。 您需要設定通話駐留軌道表與您的組織會保留駐留通話的分機號碼的範圍。 這些副檔名必須是虛擬分機擴充 （也就是沒有任何使用者或指派給他們的電話的功能）。 通話駐留應用程式已在其中部署及設定每個 Lync 伺服器集區可以有一或多個軌道範圍。 在 Lync Server 部署，軌道範圍必須是全域唯一的。

<div>


> [!IMPORTANT]  
> 您必須語音原則中選取 [<STRONG>啟用通話駐留</STRONG>] 核取方塊，之後才能使用通話駐留。 根據預設，不會選取此選項。
