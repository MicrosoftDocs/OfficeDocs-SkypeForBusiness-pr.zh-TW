# <a name="configure-phone-number-extensions-for-parking-calls-in-lync-server-2013"></a>在 Lync Server 2013 中設定停用通話的電話號碼延伸

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-10_

通話公園應用程式使用 [通話駐留軌道] 表格中的分機號碼來寄存來電。 您需要將 [通話公園軌道] 表格設定為您的組織為寄存通話保留的分機號碼範圍。 這些分機必須是虛擬分機 (也就是，沒有為分機指派任何使用者或電話)。 已部署並設定通話駐留應用程式的每個 Lync 伺服器池都可以有一個或多個軌道範圍。 在 Lync Server 部署中，軌道範圍必須全域唯一。

<div>


> [!IMPORTANT]  
> 您必須先選取語音原則中的 [<STRONG>啟用電話寄存</STRONG>] 核取方塊，然後才能使用通話寄存。 根據預設，不會選取此選項。
