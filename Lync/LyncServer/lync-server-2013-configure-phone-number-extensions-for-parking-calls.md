# <a name="configure-phone-number-extensions-for-parking-calls-in-lync-server-2013"></a>在 Lync Server 2013 中設定休止通話的電話號碼分機號碼

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-10_

通話駐留應用程式會在通話駐留軌道表格中使用分機號碼，以寄存通話。 您必須使用組織為寄存通話所保留的分機號碼範圍，設定通話駐留軌道表格。 這些分機號碼必須是虛擬分機 (亦即，未指派使用者或電話的分機) 。 在其中部署和設定通話駐留應用程式的每個 Lync 伺服器集區，都可以有一或多個軌道範圍。 軌道範圍在 Lync Server 部署中必須是全域唯一的。

<div>


> [!IMPORTANT]  
> 您必須先選取語音原則中的 [ <STRONG>啟用通話駐留</STRONG> ] 核取方塊，才能使用通話駐留。 根據預設，不會選取此選項。



</div>
