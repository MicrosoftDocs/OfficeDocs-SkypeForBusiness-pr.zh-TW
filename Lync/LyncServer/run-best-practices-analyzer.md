# <a name="run-best-practices-analyzer"></a>執行最佳做法分析程式

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-03-21_

Lync Server 2010 最佳做法分析程式工具會從 Lync Server 2010 部署收集配置資訊，並判斷是否已根據 Microsoft 最佳做法設定配置。 您可以在執行 Microsoft .NET Framework 3.5 Service Pack 1 的用戶端電腦上，或直接在執行 Lync Server 2010 的伺服器上安裝該工具。 我們建議您在用戶端電腦上安裝並執行此工具。 您也應該在用戶端電腦上的本機安裝 Lync Server 2010 系統管理工具，讓最佳做法分析程式可以收集完整的資料集。

您可以從 Microsoft 下載中心下載 Lync Server 2010 最佳做法分析程式<http://go.microsoft.com/fwlink/p/?linkid=246173> 。
