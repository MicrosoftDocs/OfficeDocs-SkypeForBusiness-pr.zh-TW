# <a name="run-best-practices-analyzer"></a>執行 Best Practices Analyzer

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-03-21_

Lync Server 2010 最佳做法分析工具會從 Lync Server 2010 部署收集設定資訊，並判斷是否設定是根據 Microsoft 最佳作法來設定。 您可以在執行 Microsoft .NET Framework 3.5 Service Pack 1 的用戶端電腦上安裝工具，或是直接在執行 Lync Server 2010 的伺服器上安裝工具。 您可以在執行 Microsoft .NET Framework 3.5 Service Pack 1 的用戶端電腦或直接在執行 nm-ocs-14-2nd 的伺服器上安裝工具。 Lync Server 2010 系統管理工具也應該安裝在用戶端電腦的本機上，以便最佳作法 Analyzer 可以收集完整的資料集。

您可以從 Microsoft 下載中心下載 Lync Server 2010 最佳作法分析程式 <https://go.microsoft.com/fwlink/p/?linkid=246173> 。
