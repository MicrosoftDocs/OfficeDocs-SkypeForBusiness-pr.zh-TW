# <a name="create-response-group-queues-in-lync-server-2013"></a>在 Lync Server 2013 中建立回應群組佇列

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-01-18_

[佇列] 會按住呼叫者，直到工程師接聽通話。 當回應群組應用程式搜尋可用的代理時，會按照您列出的順序來搜尋代理群組。 您可以選取指派給佇列的代理群組，並指定佇列行為，例如限制該列隊可以保留的呼叫次數，以及呼叫在工程師接聽通話之前等待的時間。
