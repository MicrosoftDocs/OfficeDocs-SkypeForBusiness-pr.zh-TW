# <a name="manage-xmpp-federated-partners-in-lync-server-2013"></a>管理 Lync Server 2013 中的 XMPP 同盟協力廠商

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-10-19_

這是初步文件，後續有可能變更。 空白主題會以預留位置的形式。

若要管理 XMPP 同盟網域使用者的支援，您需要執行下列動作：

  - 設定一個或多個外部存取原則，以支援 XMPP 同盟網域的使用者。

  - 設定「Access Edge 設定」原則以支援同盟。

  - 建立 XMPP 同盟協力廠商定義。

  - 了解 XMPP 同盟可使用的交涉設定。

若要執行這些工作，請使用本節中的程序。
