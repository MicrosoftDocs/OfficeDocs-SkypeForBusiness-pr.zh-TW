# <a name="manage-xmpp-federated-partners-in-lync-server-2013"></a>在 Lync Server 2013 中管理 XMPP 同盟夥伴

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-19_

這是初步檔，可能會變更。 空白主題是以預留位置形式提供。

若要管理 XMPP 聯盟網域使用者的支援，您必須執行下列動作：

  - 設定一或多個外部存取原則，以支援 XMPP 聯盟網域的使用者。

  - 設定存取邊緣配置原則以支援同盟。

  - 建立 XMPP 聯盟夥伴定義。

  - 瞭解可供 XMPP 同盟使用的協商設定。

若要執行這些工作，請使用本節中的程式。
