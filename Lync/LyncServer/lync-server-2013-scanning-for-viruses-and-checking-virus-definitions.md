# <a name="scanning-for-viruses-and-checking-virus-definitions-in-lync-server-2013"></a>在 Lync Server 2013 中掃描病毒及檢查病毒定義

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2014-05-01_

強烈建議安裝 IM 層級的防病毒產品。 IM 是眾所周知的來源，可快速散佈整個組織中的病毒和惡意軟體。 Microsoft Forefront®的 Lync Server 安全性，可提供多引擎掃描，包含病毒、惡意軟體、檔案及關鍵字篩選保護，以及與 Office 通訊伺服器的無縫整合。

除了 Forefront Security for Lync Server 之外，我們也強烈建議安裝檔級的防病毒解決方案，以保護伺服器的檔案系統。

保持掃描程式引擎和病毒定義的更新十分重要。 設定和監控更新的健康情況，可確保使用最新的掃描資訊來保護 Office 通訊伺服器及檔案系統。

<div>


> [!IMPORTANT]  
> 在執行 lync server 2013 及 Forefront Security for Lync Server 的伺服器上使用協力廠商的檔案層級防毒軟體時，請確定未掃描 Lync Server 和 Lync Server 的 Forefront Security 所安裝的資料夾，以避免其損毀。 如需排除的完整清單，請參閱 <A class=uri href="https://support.microsoft.com/kb/943620">https://support.microsoft.com/kb/943620</A> 。
