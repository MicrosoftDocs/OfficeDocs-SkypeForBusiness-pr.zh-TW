# <a name="managing-response-groups-in-lync-server-2013"></a>管理 Lync Server 2013 中的回應群組

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2018年-02-01_

回應群組是一種通話管理功能，可讓您保留對特定區域所撥出的通話 (例如客服中心)，然後將這些通話路由傳送至指定的人員群組 (稱為 *「代理人」*)。

若要管理回應群組，請設定代理人群組、佇列及工作流程，藉以定義從通話保留至代理人應答這段期間對通話的處理方式。

<div>


> [!NOTE]  
> 如果您有超過 300 個工作流程的單一的集區中的回應群組部署中，最好是使用 Lync Server 管理命令介面指令程式來建立工作流程。 如果您使用「回應群組設定工具」，為內含 300 多個工作流程的集區建立更多工作流程，該網頁需要一段時間載入。 代理程式相關聯間接透過佇列的工作流程數目也有調和間距影響載入頁面。



</div>

本節中的主題提供逐步程序的工作，您可以執行來自訂和維護您的部署中的回應群組應用程式
