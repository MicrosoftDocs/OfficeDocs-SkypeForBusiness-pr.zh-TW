# <a name="new-unified-contact-store-feature-in-lync-server-2013"></a>Lync Server 2013 中新增整合連絡人存放區功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-13_

Lync Server 2013 引入了整合連絡人存放區。 「整合連絡人存放區」可讓使用者在 Microsoft Exchange Server 2013 中保留所有連絡人資訊。 將使用者的 Lync 連絡人遷移至 Exchange 2013 之後，使用者就可以從 Lync 2013、Outlook 或 Outlook Web App 存取及管理其連絡人，而且他們的最愛仍保持同步處理。 例如，如果使用者在 Outlook 中新增連絡人至 [我的最愛]，該連絡人會出現在 Lync 2013 的 [我的最愛] 群組中。 使用者不需要登入 Lync 即可從 Outlook 或 Outlook Web App 管理其連絡人。

「整合連絡人存放區」預設為啟用。 您可以在全球、由網站、由租使用者，或由個人或個人群組來啟用或停用整合連絡人存放區的使用者。
