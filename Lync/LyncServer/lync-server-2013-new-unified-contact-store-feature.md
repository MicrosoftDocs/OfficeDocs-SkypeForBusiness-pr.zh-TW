# <a name="new-unified-contact-store-feature-in-lync-server-2013"></a>Lync Server 2013 的新增整合連絡人存放區功能

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-13_

Lync Server 2013 引進整合連絡人存放區。 整合連絡人存放區可讓使用者在 Microsoft Exchange Server 2013 中保留所有的連絡人資訊。 將使用者的 Lync 連絡人遷移至 Exchange 2013 之後，使用者就可以從 Lync 2013、Outlook 或 Outlook Web App 存取及管理其連絡人，其最愛保持同步。 例如，如果使用者將連絡人新增至 Outlook 中的 [我的最愛]，該連絡人會出現在 Lync 2013 的 [我的最愛] 群組中。 使用者不需要登入 Lync 以從 Outlook 或 Outlook Web App 管理其連絡人。

預設會啟用整合連絡人存放區。針對整合連絡人存放區啟用或停用使用者的方式可以為全域、依網站、依租用戶、依個人或依個人群組。
