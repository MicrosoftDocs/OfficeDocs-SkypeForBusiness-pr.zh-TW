# <a name="new-unified-contact-store-feature-in-lync-server-2013"></a>Lync Server 2013 中的新整合連絡人存放區功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-09-13_

Lync Server 2013 引進了整合連絡人存放區。 整合連絡人存放區可讓使用者可在 Microsoft Exchange Server 2013 中保留所有其連絡人資訊。 使用者的 Lync 連絡人移轉至 Exchange 2013 之後，使用者可以存取並管理其連絡人從 Lync 2013、 Outlook 或 Outlook Web App，以及其 [我的最愛] 保持同步。 例如，如果使用者將連絡人新增至 [我的最愛] 在 Outlook 中，連絡人會出現在 Lync 2013 中的 [我的最愛] 群組中。 使用者不需要管理其連絡人從 Outlook 或 Outlook Web App 的 Lync 登入。

預設會啟用整合連絡人存放區。針對整合連絡人存放區啟用或停用使用者的方式可以為全域、依網站、依租用戶、依個人或依個人群組。
