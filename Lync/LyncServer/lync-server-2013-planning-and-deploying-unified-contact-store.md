# <a name="planning-and-deploying-unified-contact-store-in-lync-server-2013"></a>規劃及部署整合的連絡人儲存在 Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-06-14_

整合連絡人存放區是 Microsoft Office 產品提供的一致聯絡體驗的功能。 這項功能可讓使用者儲存所有聯絡人資訊在 Exchange 2013，讓資訊皆可全域 Lync、 Exchange、 Outlook 和 Outlook Web Access。
