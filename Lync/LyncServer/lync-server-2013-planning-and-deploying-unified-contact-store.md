_**主題上次修改日期：** 2012-06-14_

整合連絡人存放區是一項功能，可提供跨 Microsoft Office 產品的一致連絡人體驗。 這項功能可讓使用者將所有連絡人資訊儲存在 Exchange 2013 中，使其可在 Lync、Exchange、Outlook 和 Outlook Web Access 之間擁有全域資訊。
