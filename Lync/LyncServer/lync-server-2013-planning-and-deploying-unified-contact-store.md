# <a name="planning-and-deploying-unified-contact-store-in-lync-server-2013"></a>在 Lync Server 2013 中規劃與部署整合連絡人存放區

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-06-14_

[整合連絡人存放區] 功能可在 Microsoft Office 產品中提供一致的連絡人體驗。 此功能可讓使用者將 Exchange 2013 中的所有連絡人資訊儲存在一起，以便在 Lync、Exchange、Outlook 和 Outlook Web Access 中全域提供資訊。
