# <a name="unassigned-number-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的未分派數位 Cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-03-21_

未指定的號碼是屬於組織的電話號碼，但尚未指派給使用者或電話。 在撥打電話給那些號碼時，可以設定未指派的號碼播放各種類型的宣告。

<div>
