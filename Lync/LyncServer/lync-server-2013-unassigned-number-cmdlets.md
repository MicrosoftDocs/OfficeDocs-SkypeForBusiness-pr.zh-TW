# <a name="unassigned-number-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的未指派號碼 cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-03-21_

未指派號碼是屬於組織但尚未指派給使用者或電話的電話號碼。未指派號碼可設定為當撥打這些號碼時播放各種宣告。
