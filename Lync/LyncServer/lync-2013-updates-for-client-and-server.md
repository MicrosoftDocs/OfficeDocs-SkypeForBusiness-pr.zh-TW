## <a name="determining-your-lync-desktop-client-version"></a>決定您的 Lync 桌面用戶端版本

若要找出您要使用哪個版本的 Lync 桌面用戶端，請開啟**有關 Lync** ] 對話方塊依序按一下 [選項] 圖示。 在 [**選項**] 功能表中，選取 [**說明**]，然後選取**有關 Microsoft Lync**。 用戶端版本號碼通常會顯示在 [**關於 Lync** ] 對話方塊上的第一行。

</div>

<div>

## <a name="lync-client-updates"></a>Lync 用戶端更新

最常見其中一項檢查，Lync 用戶端問題的疑難排解，會決定是否正在執行最新版的 Lync 用戶端時，並安裝所有的修補程式。 要取得報告的 Lync 用戶端問題多數就是升級至最新版本或更新的產品來解決。 因此，我們強烈建議您安裝最新版的 Lync 用戶端，以及最新的更新，再連絡 Microsoft 產品支援服務。 本[文章](https://docs.microsoft.com/SkypeForBusiness/sfb-client-updates#lync-2013-client-updates)將協助您識別正確版本的版本號碼。

Lync 用戶端每次更新有它自己的版本號碼。 您可以決定可以查看相關 KB 文章與更新相關聯的版本號碼連結到下表中。

</div>
