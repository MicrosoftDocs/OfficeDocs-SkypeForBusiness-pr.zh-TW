# <a name="updates-for-lync-2013-client-and-lync-server-2013"></a>Lync 2013 用戶端和 Lync Server 2013 的更新

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2014-11-12_

若要確保您擁有最新版本的 Lync 用戶端，請參閱此經常更新的 Microsoft Lync 2013 所有用戶端發行版本本清單。

<div>

## <a name="determining-your-lync-desktop-client-version"></a>決定您的 Lync 桌面用戶端版本

若要找出您使用的 Lync 桌面用戶端版本，請按一下 [選項] 圖示以開啟 [**關於 lync** ] 對話方塊。 在 [**選項**] 功能表上 **，選取 [說明]，然後**選取 [**關於 Microsoft Lync**]。 用戶端版本號碼通常會顯示在 [**關於 Lync** ] 對話方塊的第一行。

</div>

<div>

## <a name="lync-client-updates"></a>Lync 用戶端更新

疑難排解 Lync 用戶端問題時，最常見的一項檢查就是判斷您是否正在執行最新版本的 Lync 用戶端，以及是否已安裝所有的修補程式。 透過升級到產品的最新版本或更新，就能解決大部分已報告的 Lync 用戶端問題。 因此，我們強烈建議您在聯繫 Microsoft 產品支援人員之前，先安裝最新版本的 Lync 用戶端及最新的更新。 [本文](https://docs.microsoft.com/SkypeForBusiness/sfb-client-updates#lync-2013-client-updates)將協助您識別正確的發行版本本號碼。

Lync 用戶端的每個更新都有自己的版本號碼。 您可以查看相關的知識庫文章（連結至下表），以判斷與更新相關聯的版本號碼。

</div>

<div>

## <a name="lync-server-updates"></a>Lync Server 更新

如需 Lync Server 2013 更新的相關資訊，請參閱 Microsoft 支援網站上的[Lync Server 2013 更新](https://support.microsoft.com/help/2809243/updates-for-lync-server-2013)。
