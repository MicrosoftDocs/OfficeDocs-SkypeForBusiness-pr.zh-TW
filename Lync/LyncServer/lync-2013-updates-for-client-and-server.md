<div>

## <a name="determining-your-lync-desktop-client-version"></a>決定 Lync 桌面用戶端版本

若要找出您使用的 Lync desktop 用戶端版本，請按一下 [選項] 圖示，開啟 [ **關於 lync** ] 對話方塊。 在 [ **選項** ] 功能表上 **，選取 [說明]，然後** 選取 [ **關於 Microsoft Lync**]。 用戶端版本號碼通常會顯示在 [ **關於 Lync** ] 對話方塊的第一行。

</div>

<div>

## <a name="lync-client-updates"></a>Lync 用戶端更新

疑難排解 Lync 用戶端問題時，最常見的檢查之一，是要判斷您是否正在執行最新版本的 Lync 用戶端，以及是否已安裝所有的修補程式。 透過升級至產品的最新版本或更新，即可解決大部分取得報告的 Lync 用戶端問題。 因此，強烈建議您先安裝最新版本的 Lync 用戶端，以及最新的更新，再聯繫 Microsoft 產品支援人員。 [本文](https://docs.microsoft.com/SkypeForBusiness/sfb-client-updates#lync-2013-client-updates)將協助您識別正確的發行版本本號碼。

Lync 用戶端的每個更新都有自己的版本號碼。 您可以查看相關的知識文庫文章，以判斷與更新相關聯的版本號碼，如下表所示。

</div>
