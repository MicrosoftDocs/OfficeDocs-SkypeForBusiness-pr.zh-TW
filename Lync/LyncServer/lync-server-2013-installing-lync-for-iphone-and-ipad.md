## <a name="installing-lync-for-iphone-and-lync-for-ipad"></a>安裝 Lync for iPhone 和 Lync for iPad

您可以指示使用者安裝 iPhone 版 Lync 2013 和 iPad 版 Lync 2013，方法是從裝置將其導向至 App Store。 您也可以在線上取得每個裝置的 App Store。

  - Lync for iPhone 可在 [ \< <span> </span>Ttp://www.apple.com/iphone/from-the-app-store/] 的 App Store 中取得 >

  - Lync for iPad 可在\< ht<span></span>tp://www.apple.com/ipad/from-the-app-store/的 App Store 中使用 >

<div>


> [!IMPORTANT]  
> 尚未安裝 Lync 2013 應用程式的 iPhone 使用者，以及嘗試從會議邀請加入 Lync 會議的使用者，都會被重新導向到 [加入啟動器] 頁面。 此頁面包含安裝 Lync 2013 應用程式的連結。 不過，這個連結會開啟一個空白的 Safari 瀏覽器頁面，而不是將使用者導向 App Store。 使用者可以執行下列其中一項動作來解決此問題： 
> <UL>
> <LI>
> <P>使用 [<STRONG>主</STRONG>畫面] 按鈕，將 safari 頁面傳送到背景，然後重新開啟 safari。 出現「在 App Store 中開啟此頁面」通知，請在 App Store 中，按一下 [<STRONG>開啟</STRONG>] 以導向 Lync 2013 下載。</P>

## <a name="verifying-mobile-client-installation"></a>驗證行動用戶端安裝

在您設定用戶端並成功登入之後，請使用下列測試確認您的 Lync 安裝在您的行動裝置上正常運作。

**在公司目錄中搜尋連絡人**

1.  在連絡人清單中，在頂端的搜尋列內點一下，然後開始輸入僅存在於全域通訊清單（GAL）中之連絡人的名稱。

2.  確認連絡人名稱出現在搜尋結果中。

**測試立即訊息和目前狀態**

1.  在連絡人清單中，輕觸連絡人。

2.  在連絡人卡片中，按一下 [ **IM** ] 圖示。

3.  確認立即訊息（IM）視窗出現，而且您可以輸入並傳送即時消息。

**測試撥出式會議**

1.  在 Outlook 中，排程 Lync 會議。

2.  在行動裝置上，開啟會議邀請。

3.  按一下會議中要加入的連結。

4.  從會議服務接聽通話，並確認您已連線到會議音訊。

**測試推播通知**

1.  在使用者 A 的行動裝置上，使用使用者 A 的帳戶登入 Lync。

2.  在行動裝置上開啟另一個應用程式。

3.  在其他用戶端上，使用使用者 B 的帳戶登入 Lync。

4.  從使用者 B 傳送 IM 給使用者 A。

5.  確認 IM 通知出現在使用者 A 的行動裝置上。
