# <a name="setting-who-can-post-messages-in-an-auditorium-chat-room-in-lync-server-2013"></a>在 Lync Server 2013 中設定能夠在視聽中心聊天室張貼訊息的人員

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-06_

在 auditorium 聊天室中，只有已獲「簡報者」角色的使用者才能張貼訊息。 所有其他成員只能讀取訊息。 Auditorium 聊天室中的簡報者必須是聊天室的成員。

如需使用 Windows PowerShell 命令列介面來管理 auditorium 聊天室的詳細資料，請參閱在部署檔中[管理會議室](manage-rooms.md)。

雖然 Persistent 聊天室管理員和聊天室管理員可以管理聊天室設定，但他們無法在 auditorium 聊天室中張貼，除非他們是**簡報者**。
