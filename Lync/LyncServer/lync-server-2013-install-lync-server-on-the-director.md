</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改主題：** 2014年-05-05_

若要在 Director 上安裝 Lync Server 2013 元件中使用這些步驟。

<div>

## <a name="to-install-lync-server-components-on-a-director"></a>若要在 Director 上安裝 Lync Server 元件

1.  在 [Lync Server 部署精靈] [Lync Server 2013] 頁面上下, 一步] 為**步驟 2： 安裝或移除 Lync Server 元件**，按一下 [**執行**]。

2.  在 [**安裝 Lync Server 元件**] 頁面上，按 [**下一步**設定元件，如已發行的拓撲中所定義。

3.  Lync Server 元件安裝完成時，按一下 [**完成]**。
