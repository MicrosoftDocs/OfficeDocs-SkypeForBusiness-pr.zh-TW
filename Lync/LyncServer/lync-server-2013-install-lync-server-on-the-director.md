使用下列步驟，在 Director 上安裝 Lync Server 2013 元件。

<div>

## <a name="to-install-lync-server-components-on-a-director"></a>若要在 Director 上安裝 Lync Server 元件

1.  在 [Lync Server 部署嚮導] 的 [Lync Server 2013] 頁面上，按一下 [ **步驟2：安裝或移除 Lync Server 元件**] 旁邊的 [ **執行**]。

2.  在 [ **安裝 Lync Server 元件** ] 頁面上，按 **[下一步]** ，依照您發佈的拓撲中所定義的方式來設定元件。

3.  Lync Server 元件安裝程式完成時，按一下 **[完成]**。
