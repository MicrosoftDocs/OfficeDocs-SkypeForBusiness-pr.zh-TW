# <a name="windows-powershell-cmdlets-for-address-book-services-in-lync-server-2013"></a>Lync Server 2013 中通訊錄服務的 Windows PowerShell cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-11-01_

Lync Server 提供數個 Windows PowerShell 命令列介面 cmdlet 來管理及設定通訊錄服務。 這些 cmdlet 的某些是取代的舊版 Office Communications Server 中使用的 ABServer.exe 命令。 下列主題中的 Cmdlet 是用來設定、建立及擷取通訊錄服務資訊、其設定，以及通訊錄服務在用戶端擷取通訊錄服務檔案及設定時使用的 Web 服務資訊。

所有這些 cmdlet 會發出透過位於系統管理工具中已安裝所在的伺服器或工作站上的 Lync Server 工具在 Lync Server 管理命令介面。
