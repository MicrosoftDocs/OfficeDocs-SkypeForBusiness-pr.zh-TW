# <a name="performing-physical-environmental-checks"></a>執行實體環境檢查

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2014-04-30_

在檢查 Lync Server 2013 部署的效能、可用性及功能前，您應該先檢查實際環境。 例如，伺服器機房溫度可能必須降低，否則可能必須更換網線。 為了獲得最佳結果，請執行下列實際環境檢查：

  - **物理安全性會測量**   物理安全性保護（例如鎖、門和限制存取的聊天室）必須受到保護。 檢查是否有任何未經授權且強行強制的專案和裝置損壞跡象。

  - **溫度與濕度**   高溫、氣流差弱及濕度都可能導致硬體元件過熱。 檢查溫度與濕度，以協助確保環境系統（例如加熱與空調）可以在硬體製造商規格中維持可接受的條件和功能。 最近安裝新的裝置時，也請檢查伺服器的空中流程是否不受歡迎且符合製造商的規格。

  - ****    Lync Server 2013 組織所依賴的裝置和元件是由正常運作的物理網路和相關硬體所組成。 確認路由器、交換器、中樞、物理電纜及連接器都能正常運作。

如何執行這些檢查的詳細資訊，會大大影響您的安裝網站與所選的伺服器硬體。 第一次執行此檢查時，請參閱硬體檔，並記下所需的參數以供日後參考。
