_**主題上次修改日期：** 2012-06-27_

Persistent Chat Cmdlet 可讓您管理及設定 Microsoft Lync Server 2013 Persistent Chat service (以前稱為 Group Chat 服務) 。 常設聊天讓使用者能夠參與線上立即訊息工作階段。 儘管這些工作階段是即時發生的，但每個工作階段的內容都是持續性；這表示任何人隨時都可以繼續進行這些交談。
