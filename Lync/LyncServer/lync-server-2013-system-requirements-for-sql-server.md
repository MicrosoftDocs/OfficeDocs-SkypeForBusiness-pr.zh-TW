# <a name="system-requirements-for-sql-server-in-lync-server-2013"></a>Lync Server 2013 中 SQL Server 的系統需求

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-10-25_

在您部署 Enterprise Edition server 之前，請先在符合硬體需求的專用電腦上安裝 Microsoft SQL Server 2008 R2 或 Microsoft SQL Server 2012。 如需硬體需求的詳細資訊，請參閱支援檔中的[Lync server 2013 伺服器硬體平臺](lync-server-2013-server-hardware-platforms.md)。 如需軟體需求的詳細資訊，請參閱可支援檔中的[Lync Server 2013 資料庫軟體支援](lync-server-2013-database-software-support.md)。 如需部署所需許可權的相關資訊，請參閱[Lync server 2013 中的 SQL Server 部署許可權](lync-server-2013-deployment-permissions-for-sql-server.md)。

在建立 [前端] 池之前，您也必須將 Windows 防火牆設定為使用 SQL Server Configuration Manager，並在 Windows 防火牆中開啟埠，以允許 Lync Server 2013 透過特定埠存取 SQL Server[高級安全性]。
