# <a name="call-detail-recording-cdr-in-lync-server-2013"></a>Lync Server 2013 中的詳細通話記錄 (CDR) 

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-22_

詳細通話記錄 (CDR) 會記錄對等活動的使用和診斷資訊，包括立即訊息、Voice over Internet Protocol (VoIP) 通話、應用程式共用、檔案傳輸和會議。 使用資料可用來計算投資報酬率 (ROI)，而診斷資料可用來疑難排解對等活動和會議。 當您安裝 Lync Server 2013 時，也會為 CDR 安裝一個預先定義的全域設定設定集合。 請使用本節中的主題來設定 CDR。
