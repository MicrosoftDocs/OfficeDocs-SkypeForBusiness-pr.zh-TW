# <a name="new-enterprise-voice-features-in-lync-server-2013"></a>Lync Server 2013 中的新企業語音功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-05-01_

Lync Server 2013 引進了數種新的路由與通話管理功能，可加強企業語音。

Lync Server 2013 支援在中繼伺服器和閘道之間進行多個 trunks。 *主幹*是埠號碼與中繼伺服器與埠號碼和閘道之間的邏輯關聯。 這表示中繼伺服器可以有多個 trunks 至不同的閘道，而閘道可以有多個 trunks 至不同的中繼伺服器。 Intertrunk 路由可讓 Lync Server 2013 將 IP PBX 互連至公用交換式電話網絡（PSTN）閘道，或與多個 IP PBX 系統相互連接。 Lync Server 2013 在不同的電話系統之間充當粘附（也就是互連）。

Microsoft Lync Server 2013 可改善來電轉接、同時撥打、語音信箱處理及本機號碼簡報等方面的改良功能。 這些功能豐富了企業語音通話體驗。

Lync Server 2013 引進了下列新的企業語音增強功能：

  - [Lync Server 2013 中的新通話功能](lync-server-2013-new-call-features.md)

  - [Lync Server 2013 中的新來電顯示功能](lync-server-2013-new-caller-id-feature.md)

  - [Lync Server 2013 中的新語音信箱功能](lync-server-2013-new-voice-mail-feature.md)

  - [Lync Server 2013 中的新主幹功能](lync-server-2013-new-trunk-feature.md)

  - [Lync Server 2013 中的新主幹間功能](lync-server-2013-new-intertrunk-feature.md)

  - [Lync Server 2013 中的新通話管理功能](lync-server-2013-new-call-management-features.md)
