_**主題上次修改日期：** 2012-10-01_

在使用 Office 通訊伺服器 2007 R2 在共存狀態中部署 Lync Server 2013 之前，您必須執行一些額外的 Active Directory 工作，以設定 Lync Server 2013 的架構、樹系和網域。 架構擴充新增 Lync Server 所需的 Active Directory 類別和屬性。 如需其他資訊，請參閱 [準備 Active Directory 網域服務以進行 Lync Server 2013](lync-server-2013-preparing-active-directory-domain-services.md)的主題。

**準備 Lync Server 2013 的 Active Directory**

1.  在 Lync Server 2013 前端伺服器上，執行 Lync Server 2013 設定。

2.  選取 [**準備 Active Directory** ]

3.  完成步驟 1 到 5。
