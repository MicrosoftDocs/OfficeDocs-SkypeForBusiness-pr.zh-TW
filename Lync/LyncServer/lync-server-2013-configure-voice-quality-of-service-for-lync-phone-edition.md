# <a name="configure-voice-quality-of-service-for-lync-phone-edition-in-lync-server-2013"></a>在 Lync Server 2013 中設定 Lync Phone Edition 的語音服務品質

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-29_

您可以設定連線至 Lync Server 2013 之 IP 電話的 QoS 層級，在池中設定 Lync Phone Edition 裝置的語音服務品質（QoS）需求。

<div>

## <a name="to-configure-voice-quality-of-service-for-lync-phone-edition"></a>若要設定 Lync Phone Edition 的語音服務品質

1.  開啟瀏覽器視窗，然後輸入系統管理員 URL，以開啟 Lync Server [控制台]。 如需可用於啟動 Lync Server [控制台] 的不同方法的詳細資訊，請參閱[開啟 Lync server 2013 系統管理工具](lync-server-2013-open-lync-server-administrative-tools.md)。

2.  在左側導覽列中，按一下 [**用戶端**]，然後按一下 [**裝置**設定]。

3.  在 [**裝置**設定] 頁面上，于裝置配置清單中，按兩下您要變更 QoS 設定的配置。

4.  在 [**編輯裝置**設定] 中的 [**語音服務品質**] 底下，指定 QoS 等級。 預設層級為**40**。
