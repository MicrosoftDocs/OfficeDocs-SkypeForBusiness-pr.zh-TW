# <a name="iis-support-in-lync-server-2013"></a>Lync Server 2013 中的 IIS 支援

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-06-29_

幾個 Lync Server 2013 元件需要網際網路資訊服務（IIS）。 支援的 IIS 版本包括下列各項：

  - IIS 8。0

  - IIS 7。5

如需有關 Lync Server 2013 元件的 IIS 需求的詳細資訊，請參閱規劃檔中的[Lync server 2013 中的網際網路資訊服務（IIS）需求](lync-server-2013-internet-information-services-iis-requirements.md)。
