# <a name="enhanced-9-1-1-e9-1-1-and-mediation-server-in-lync-server-2013"></a>Lync Server 2013 中的增強型 9-1-1 (E9-1-1) 和轉送伺服器

_**主題上次修改日期：** 2012-09-29_

轉送伺服器具有擴充的功能，可與增強型 9-1-1 (E9-1-1) 服務提供者正確互動。 轉送伺服器上不需要特殊設定;E9-1-1 互動所需的 SIP 擴充預設會包含在轉送伺服器的 SIP 通訊協定中，以用於與閘道對等 (PSTN 閘道、IP-PBX 或內部網際網路電話語音服務提供者（包括 E9-1-1）服務提供者的 SBC 進行互動) 

E9-1-1 服務提供者的 SIP 主幹是否可在現有的轉送伺服器集區上終止，或是否需要獨立轉送伺服器，取決於 E9-1-1 SBC 是否可以與轉送伺服器集區互動。 如需詳細資訊，請參閱 [M:N 主幹 In Lync Server 2013](lync-server-2013-m-n-trunk.md)。
