# <a name="new-role-based-access-control-features-in-lync-server-2013"></a>Lync Server 2013 中的新角色存取控制功能

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-20_

Lync Server 2013 以兩個主要的方式，增強以角色為基礎的存取控制 (RBAC) 功能。 您現在可以建立自訂角色，每個角色都只具有您指定之一組 Cmdlet 的許可權。 您也可以提供這些自訂角色，以執行 Cmdlet 的腳本。

此外，Lync Server 2013 包含兩個新的預先定義角色。

  - 具備「回應群組管理員」** 角色的使用者可以管理組織中特定的回應群組佇列，但他們卻不一定要擁有其他佇列或完整回應群組應用程式的管理權限。

  - 擁有 *Persistent Chat Manager* 角色的使用者可以管理組織中特定的持續聊天室，但不一定要有其他聊天室或整體聊天功能的管理許可權。
