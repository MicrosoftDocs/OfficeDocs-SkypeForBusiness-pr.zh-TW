# <a name="backing-up-lync-server-2013"></a>備份 Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-17_

本節中的程式說明如何備份 Lync Server，讓您可以在發生停機或失敗時復原服務。

您應該針對您的組織開發備份與復原戰略及規劃，如[開發適用于 Lync Server 2013 的備份與還原策略及規劃](lync-server-2013-developing-a-backup-and-restoration-strategy-and-plan.md)中所述。 此策略和規劃應包含您打算使用的特定程式。 您可以使用本節主題中所包含的程式，以及[Lync Server 2013 備份和還原工作](lync-server-2013-backup-and-restoration-worksheets.md)表中的工作表，來記錄如何規劃備份特定的 Lync server 部署。
