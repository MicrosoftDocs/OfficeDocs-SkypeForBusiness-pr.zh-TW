# <a name="system-usage-reports-in-lync-server-2013"></a>Lync Server 2013 中的系統使用量報告

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-21_

系統使用量報告可根據 Lync Server 收集的呼叫詳細資料錄製（CDR）資料，提供系統使用量資訊。

<div>

## <a name="in-this-section"></a>本節內容

  - [Lync Server 2013 中的使用者註冊報告](lync-server-2013-user-registration-report.md)
    
    根據註冊事件（例如使用者登錄），提供 Lync Server 2013 部署的使用者連線摘要。 報告提供一種方式來查看內部和外部登入，並將登入 Lync Server 2013 的使用者數與實際在該服務登入時所用的使用者數目進行比較。

  - [Lync Server 2013 中的對等活動摘要報告](lync-server-2013-peer-to-peer-activity-summary-report.md)
    
    提供對等立即訊息（IM）、音訊、影片、檔案傳輸及應用程式共用會話摘要。 點對點工作階段是只涉及兩個使用者的會話。

  - [Lync Server 2013 中的 [會議摘要] 報告](lync-server-2013-conference-summary-report.md)
    
    提供所有會議活動的摘要。 會議是與三個或更多人進行的會議。

  - [Lync Server 2013 中的 PSTN 會議摘要報告](lync-server-2013-pstn-conference-summary-report.md)
    
    提供所有 PSTN 會議的摘要。 在這些會議中，至少有一個使用者使用公用的交換電話網絡（PSTN）撥入，也稱為*電話撥入式會議*。

  - [Lync Server 2013 中的 [回應群組使用方式] 報告](lync-server-2013-response-group-usage-report.md)
    
    提供回應群組用法摘要。 回應群組應用程式提供一種方式，讓您自動將電話來電路由至實體，例如服務台或客戶支援行。

  - [Lync Server 2013 中的 IP 電話清點報告](lync-server-2013-ip-phone-inventory-report.md)
    
    提供目前在組織中使用之 IP 電話的相關資訊。 報告是以電話登記和登錄為基礎。 不應將它視為完整的庫存。 例如，您可能已移除仍列在報表中的手機，因為他們至少已登入一次。 同樣地，您也可能會有不會顯示在報表中的新手機，只是因為使用者尚未登入 Lync Server 並使用新的手機。

  - [Lync Server 2013 中的呼叫許可控制報告](lync-server-2013-call-admission-control-report.md)
    
    提供對等的清單，以及使用呼叫許可控制的會議活動。 [通話許可控制] （CAC）是根據頻寬限制，決定是否要允許即時通訊會話，例如語音或視頻通話。
