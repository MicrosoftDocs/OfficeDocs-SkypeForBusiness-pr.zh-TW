# <a name="back-up-systems-and-data"></a>備份系統與資料

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

</div>

_**主題上次修改日期：** 2012-09-18_

在您開始遷移至 Lync Server 2013 之前，強烈建議您執行完整系統備份並記錄現有系統，包括位於每個集區的使用者帳戶清單，這樣您就可以在必要時復原至 Lync Server 2010。 您可使用多種工具和程式來備份和還原資料、設定和系統。

如需詳細資訊與程式，請參閱 [備份及還原 Lync Server 2010](https://go.microsoft.com/fwlink/p/?linkid=265417) 。
