# <a name="back-up-systems-and-data"></a>備份系統與資料

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-09-18_

開始移轉至 Lync Server 2013 之前，我們強烈建議您執行完整系統備份並記錄現有系統，以便您可以將回復到 Lync Server 2010 如果它包括位於每個集區的使用者帳戶的清查會成為必要項目。 您可使用多種工具和程式來備份和還原資料、設定和系統。

如需詳細資訊和程序，請參閱[備份和還原 Lync Server 2010](https://go.microsoft.com/fwlink/p/?linkid=265417) 。
