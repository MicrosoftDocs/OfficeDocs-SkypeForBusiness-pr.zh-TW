# <a name="customizing-the-online-meeting-add-in-in-lync-server-2013"></a>在 Lync Server 2013 中自訂線上會議增益集

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-06-28_

Lync 2013 的線上會議增益集支援 Outlook 訊息與共同作業用戶端內的會議管理。 您可以透過幾種方式自訂增益集的行為。 在這些選項中，您可以新增標誌和文字至線上會議邀請的正文。
