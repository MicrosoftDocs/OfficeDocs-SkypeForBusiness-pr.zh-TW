# <a name="infrastructure-requirements-for-call-admission-control-in-lync-server-2013"></a>Lync Server 2013 中的通話許可控制的基礎結構需求

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-08-21_

不需要額外的基礎結構需求（例如特殊網路路由器）。 部署 Lync Server 2013 會自動安裝用於強制執行呼叫許可控制（CAC）原則的頻寬原則服務。 [通話許可控制] 只適用于執行 Lync 的用戶端。
