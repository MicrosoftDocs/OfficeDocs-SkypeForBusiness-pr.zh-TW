# <a name="configuring-group-call-pickup-in-lync-server-2013"></a>在 Lync Server 2013 中設定群組呼叫收取

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-01_

Lync Server 2013 的累計更新：2月2013將群組呼叫收取引進為新的 Enterprise Voice 功能。 群組呼叫收取可讓使用者透過撥打呼叫收取群組號碼來挑選撥打給另一位使用者的通話。

當您部署企業語音時，會自動在前端伺服器或 Standard Edition server 上安裝並啟用群組呼叫收取所使用的元件。 不過，您必須先設定群組呼叫收取，使用者才能使用它。

本節會引導您完成群組呼叫收取設定。
