# <a name="requirements-for-unified-contact-store-in-lync-server-2013"></a>Lync Server 2013 中統一連絡人存放區的需求

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-01_

下列清單說明在 Lync Server 2013 中實施整合連絡人存放區的需求：

  - 您必須執行 Microsoft Lync Server 2013 和 Exchange 2013。

  - 使用者必須使用 Lync 2013，以從 Lync Server 2013 開始將連絡人遷移至 Exchange 2013。

  - 使用者信箱必須遷移至 Exchange 2013。

  - 您必須在 Lync Server 2013 和 Exchange 2013 之間設定伺服器對伺服器驗證。
    
    <div>
    

    > [!NOTE]  
    > 如需在 Lync Server 2013 和 Exchange 2013 之間設定驗證的詳細要求，請參閱 Operations 檔 <A href="lync-server-2013-managing-server-to-server-authentication-oauth-and-partner-applications.md">中的管理伺服器對伺服器驗證 (OAuth) 和協力廠商的 Lync Server 2013 中的合作夥伴應用程式</A> 。
