# <a name="requirements-for-unified-contact-store-in-lync-server-2013"></a>Lync Server 2013 中的整合連絡人存放區的需求

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-01_

下列清單說明在 Lync Server 2013 中實現整合連絡人存放區的需求：

  - 您必須執行 Microsoft Lync Server 2013 和 Exchange 2013。

  - 使用者必須使用 Lync 2013 來啟動從 Lync Server 2013 到 Exchange 2013 的連絡人的遷移。

  - 使用者信箱必須遷移至 Exchange 2013。

  - 您必須在 Lync Server 2013 與 Exchange 2013 之間設定伺服器對伺服器驗證。
    
    <div>
    

    > [!NOTE]  
    > 如需在 Lync Server 2013 與 Exchange 2013 之間設定驗證的詳細需求，請參閱在 Operations 檔中<A href="lync-server-2013-managing-server-to-server-authentication-oauth-and-partner-applications.md">管理 Lync server 2013 中的伺服器到伺服器驗證（OAuth）與合作夥伴應用程式</A>。
