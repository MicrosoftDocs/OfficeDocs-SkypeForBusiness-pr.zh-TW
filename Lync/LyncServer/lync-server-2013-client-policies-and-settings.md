# <a name="client-policies-and-settings-in-lync-server-2013"></a>Lync Server 2013 中的用戶端原則和設定

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-06-18_

本主題概要說明您可以在 Lync Server 2013 中設定的用戶端相關設定和原則。 Lync Server 2013 包含下列管理及設定用戶端的工具：

  - **Lync Server 2013 控制台**    以 web 為基礎的圖形使用者介面，用來管理及設定伺服器、使用者、用戶端和裝置。

  - **Lync Server 管理命令**     介面包含一組豐富 Windows PowerShell 命令列介面 Cmdlet 和大量預先定義腳本的管理介面。

  - **Lync 2013 群組原則**    您可以使用 Office 群組原則系統管理範本為用戶端設定的一組原則。 在您部署 Lync 2013 用戶端之前，必須先設定某些用戶端引導原則。 Lync 2010 的其他選擇性設定仍可在 Lync 2013 中進行處理。

本節說明 Lync Server 2013 中用戶端相關設定的變更。
