# <a name="client-policies-and-settings-in-lync-server-2013"></a>用戶端原則和 Lync Server 2013 中設定

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-06-18_

本主題提供用戶端相關的設定和原則，您可以設定 Lync Server 2013 中的概觀。 Lync Server 2013 包含下列工具以管理及設定用戶端：

  - **Lync Server 2013 控制台**   管理及設定伺服器、 使用者、 用戶端和裝置的 web 型圖形使用者介面。

  - **Lync Server 管理命令介面**   管理介面，包含一組豐富的 Windows PowerShell 命令列介面指令程式及許多預先定義的指令碼。

  - **Lync 2013 群組原則**   一組您可以使用 Office 的群組原則系統管理範本設定用戶端的原則。 部署 Lync 2013 用戶端之前，必須設定特定用戶端啟動載入原則。 從 Lync 2010 其他選用設定繼續在 Lync 2013 生效。

本節說明在 Lync Server 2013 中的用戶端相關設定的變更。
