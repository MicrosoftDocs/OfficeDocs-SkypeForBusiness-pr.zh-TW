# <a name="back-up-systems-and-data"></a>備份系統與資料

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-19_

開始遷移至 Lync Server 2013 之前，我們強烈建議您執行完整的系統備份並記錄您現有的系統，包括駐留在每個池中的使用者帳戶清單，以便回滾到 Office 通訊伺服器 2007 R2 （如果需要的話）。 有多個工具和程式可用於備份及還原資料、設定和系統。 如需詳細資訊與程式，請參閱「Office 通訊伺服器 2007 R2 備份與還原<http://go.microsoft.com/fwlink/p/?linkid=168162>指南」。
