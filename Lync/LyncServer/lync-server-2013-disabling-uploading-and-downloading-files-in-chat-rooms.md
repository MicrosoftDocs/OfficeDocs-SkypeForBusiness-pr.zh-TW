# <a name="disabling-uploading-and-downloading-files-in-chat-rooms-in-lync-server-2013"></a>停用 Lync Server 2013 的聊天室中的上傳和下載檔案

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-12_

根據預設，使用者可以上傳和下載檔案投遞的郵件。 您可以停用此功能，並防止使用者以下列兩種方式上傳和下載檔案：

  - 防止所有使用者在特定的持續聊天室或聊天室類別中上傳和下載檔案。

  - 防止某些使用者在任何聊天室中上傳和下載檔案。 使用者無法上傳和下載任何聊天室中的檔案，包括允許檔案上傳和下載的聊天室。

如需啟用和停用檔案上傳與下載的詳細資訊，請參閱部署檔中的 [Configure The Lync Server 2013](lync-server-2013-configure-categories.md) 和 [Manage 類別](manage-categories.md) 。
