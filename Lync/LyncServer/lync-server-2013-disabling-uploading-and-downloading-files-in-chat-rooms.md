# <a name="disabling-uploading-and-downloading-files-in-chat-rooms-in-lync-server-2013"></a>在 Lync Server 2013 中停用聊天室的上傳和下載檔案功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-12_

根據預設，使用者可以在張貼的訊息中上傳和下載檔案。 您可以停用這項功能，並防止使用者上傳和下載檔案的方式有兩種：

  - 防止所有使用者在特定的持續聊天室或聊天室類別中上傳和下載檔案。

  - 防止特定使用者上傳或下載任何聊天室中的檔案。 使用者無法在任何聊天室中上傳和下載檔案，包括允許檔案上傳和下載的聊天室。

如需啟用和停用檔案上傳及下載的詳細資料，請參閱在[Lync Server 2013 中設定類別](lync-server-2013-configure-categories.md)，以及在部署檔中[管理類別](manage-categories.md)。
