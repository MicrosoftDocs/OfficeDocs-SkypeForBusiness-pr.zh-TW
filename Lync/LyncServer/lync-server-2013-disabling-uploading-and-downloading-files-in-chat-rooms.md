# <a name="disabling-uploading-and-downloading-files-in-chat-rooms-in-lync-server-2013"></a>停用 [上傳及下載檔案在 Lync Server 2013 中的聊天室數

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-09-12_

根據預設，使用者可以上傳和下載它們張貼在郵件中的檔案。 您可以停用此功能，並防止使用者上傳及下載檔案以兩種方式：

  - 避免讓所有使用者上傳和下載特定常設聊天室或聊天室類別中的檔案。

  - 避免讓特定使用者上傳及下載檔案在任何聊天室中。 使用者無法上傳與下載任何聊天室，包括允許檔案上傳和下載的聊天室中的檔案。

如需關於啟用和停用檔案上傳和下載的詳細資訊，請參閱部署文件中的[Lync Server 2013 中的 Configure categories](lync-server-2013-configure-categories.md)和[Manage categories](manage-categories.md) 。
