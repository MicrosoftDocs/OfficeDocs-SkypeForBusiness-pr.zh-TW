# <a name="configure-sql-server-in-lync-server-2013"></a>在 Lync Server 2013 中設定 SQL Server

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2015-01-22_

針對您部署的每個資料庫，您可以針對您的 Lync Server 2013 部署（可在資料庫伺服器上 collocated 的所有資料庫）使用單一 SQL Server 實例。 如需資料庫 collocation 的詳細資訊，請參閱可支援性檔中的[Lync server 2013 支援的伺服器 collocation](lync-server-2013-supported-server-collocation.md) 。

此外，您必須先安裝並提供每個 SQL Server 實例，才能完成拓撲建立器中設定資料庫的步驟，或使用 Windows PowerShell Cmdlet 手動建立資料庫。 如需 SQL Server 支援的詳細資訊，請參閱[Lync Server 2013 的硬體設定](lync-server-2013-hardware-setup.md)。
