# <a name="new-video-features-in-lync-server-2013"></a>Lync Server 2013 中的新影片功能

<div data-xmlns="http://www.w3.org/1999/xhtml">

_**主題上次修改日期：** 2012-08-16_

Lync Server 2013 引進下列新的影片功能：

  - **HD 影片**    使用者可在雙方通話和多方會議中體驗高達 HD 1080P 的解析度。

  - **圖庫視圖**    在包含超過兩個人的影片會議中，使用者可以查看會議中的參與者影片。 如果會議的參與者超過五位，最活躍的參與者只會出現在頂端列的影片，而且會顯示其他參與者的相片。

  - 上**的264影片**    現在，在 Lync 2013 用戶端上的編碼影片的預設值為 H-p 影片。 H.264 視訊支援更大範圍的解析度和畫面播放速率，並提升了視訊延展性。
