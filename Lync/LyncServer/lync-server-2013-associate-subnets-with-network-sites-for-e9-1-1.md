# <a name="associate-subnets-with-network-sites-for-e9-1-1-in-lync-server-2013"></a>將子網與 E9-1-1 在 Lync Server 2013 中的網站產生關聯

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-02_

您的網路中要為 E9-1-1 啟用的每個子網都必須與特定網路網站產生關聯。 這是因為我們需要子網路資訊來判斷端點所在的網路網站。

E9-1-1 沒有將子網與網站產生關聯的特殊需求。 若要在拓撲中建立子網與網站之間的關聯性，請遵循在 [Lync Server 2013 中將子網與網路網站關聯](lync-server-2013-associate-a-subnet-with-a-network-site.md)的程式。
