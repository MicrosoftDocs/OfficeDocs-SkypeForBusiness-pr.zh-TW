# <a name="services-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的服務 Cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-06-20_

隨附于 Microsoft Lync Server 2013 的伺服器角色和服務 Cmdlet 可讓您傳回已安裝在 Lync Server 2013 部署中之各個元件的相關資訊。 此外，這些 Cmdlet 可讓您輕鬆地停止並啟動 Lync Server 服務。

<div>

以下是直接與管理服務相關的 Cmdlet 清單：

**服務**

    [Get-CsService](https://technet.microsoft.com/en-us/library/Gg413038(v=OCS.15))

<!-- end list -->

  - <span></span>  
    [CsWindowsService](https://technet.microsoft.com/en-us/library/Gg398803(v=OCS.15))

  - <span></span>  
    [開始-CsWindowsService](https://technet.microsoft.com/en-us/library/Gg398561(v=OCS.15))

  - <span></span>  
    [停止 CsWindowsService](https://technet.microsoft.com/en-us/library/Gg398426(v=OCS.15))
