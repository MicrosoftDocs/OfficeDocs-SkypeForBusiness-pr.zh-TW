# <a name="in-lync-server-2013-add-kerberos-authentication-to-other-sites"></a>在 Lync Server 2013 中新增 Kerberos 驗證至其他站台

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2010-11-03_

如果您需要將網站新增至已為 Kerberos 驗證設定的現有部署，您可以針對新網站使用現有的 Kerberos 驗證帳戶，或建立新帳戶。 如需建立網站新帳戶的詳細資料，請參閱[在 Lync Server 2013 中建立 Kerberos 驗證帳戶](lync-server-2013-create-a-kerberos-authentication-account.md)。 如需將現有帳戶指派給網站，以取得新網站使用相同帳戶的詳細資料，請參閱[將 Kerberos 驗證帳戶指派給 Lync Server 2013 中的網站](lync-server-2013-assign-a-kerberos-authentication-account-to-a-site.md)。
