# <a name="planning-for-call-park-in-lync-server-2013"></a>在 Lync Server 2013 中規劃通話駐留

<div data-xmlns="http://www.w3.org/1999/xhtml">

</div>

_**主題上次修改日期：** 2012-09-07_

Lync Server 通話駐留應用程式可讓企業語音使用者呼叫暫止，然後再從任何電話取回。 停用通話的使用者可以撥打通話保留所提供的軌道編號，以取得寄存通話或使用外部機制（如立即訊息或分頁系統）來要求其他人取得通話。 本節包含通話駐留特有的規劃資訊。
