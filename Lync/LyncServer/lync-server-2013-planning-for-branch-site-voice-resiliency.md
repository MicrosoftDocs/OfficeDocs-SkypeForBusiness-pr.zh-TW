# <a name="planning-for-branch-site-voice-resiliency-in-lync-server-2013"></a>規劃 Lync Server 2013 中的分支網站語音彈性

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-09-21_

如果您想要提供分支網站恢復能力，也就是高可用性企業語音服務，您有三個選項這麼做：

  - Survivable Branch Appliance

  - Survivable Branch Server

  - 完整的 Lync Server 部署在分支網站

此指南會協助您評估最適合您組織的恢復能力解決方案，以及根據恢復能力解決方案，評估要使用的 PSTN 連線解決方案。也會協助您說明先決條件及其他規劃考量，以準備部署選擇的解決方案。
