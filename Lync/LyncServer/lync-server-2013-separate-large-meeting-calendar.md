# <a name="separate-large-meeting-calendar-in-lync-server-2013"></a>在 Lync Server 2013 中分隔大型會議行事曆

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-01_

您應該針對每個大型會議集區，另外維護一份以該集區進行排程的大型會議行事曆。 例如，您可以在大型會議集區上家用單一使用者帳戶，並且使用 Outlook 搭配 Lync 2013 的 Exchange 和線上會議增益集來維護個別的行事曆。 如果您為了讓支援人員建立大型會議而使用多個使用者帳戶，您可以設定個別行事曆，彙整所有由支援人員所建立的大型會議。

維護個別的大型會議行事曆有助於防止衝突，並確保任何時候只有一場大型會議正在舉辦中。
