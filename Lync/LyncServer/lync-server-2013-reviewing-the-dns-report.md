# <a name="reviewing-the-dns-report-in-lync-server-2013"></a>檢閱 DNS 報告在 Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改主題：** 2013年-02-21_

[DNS] 報告，也就是系統管理員報告的一部分，詳細資料的所有建議和已知的項目網域名稱系統 (DNS) 中之內部、 周邊，及外部網路。 如果設計工具已完成編輯網狀圖與所有 IP 位址及完整的網域名稱 (Fqdn) 定義為其實際執行值，DNS 報告提供的極佳設定資源。 這份報告也可以做為作業的疑難排解文件。

![DNS 系統管理報告](images/Gg558641.9dd1e810-ddc7-4816-a806-4239baf9ec51(OCS.15).jpg "DNS 系統管理報告")

您應該檢閱 DNS 報告徹底確保沒有任何錯誤，可能會造成部署時發生困難，或，可能會使疑難排解工作階段的變得複雜您 DNS 管理團隊。
