# <a name="test-voice-routing-in-lync-server-2013"></a>在 Lync Server 2013 中測試語音路由

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-24_

您可以使用 Lync Server 控制台的 [ **測試語音路由** ] 索引標籤來設定測試案例案例。 若要定義測試案例，您可以指定要測試指定電話號碼的撥號對應表、語音原則、PSTN 使用方式及語音路由。

在您實際部署語音路由設定之前，我們建議您在不同的電話號碼上進行測試，以確定結果是您預期的結果。

<div>


> [!TIP]  
> 您可以使用 [ <STRONG>匯出測試案例</STRONG> ] 和 [匯 <STRONG>入測試案例</STRONG> ] 命令儲存語音路由測試案例，並將其匯入供其他電腦使用。



</div>

<div>


> [!WARNING]  
> 如果您刪除任何部分的語音路由設定，例如撥號對應表、語音原則、語音路由或電話使用狀況，您應該檢查並更新您的語音路由測試案例。 Lync Server 控制台不會提醒您由於設定變更而不再有效的測試案例。



</div>
