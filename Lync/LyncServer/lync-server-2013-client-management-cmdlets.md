# <a name="client-management-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的用戶端管理 Cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-27_

用戶端管理主要是決定允許哪些用戶端應用程式（例如 Microsoft Lync 2013）登入 Lync Server 2013，並決定這些用戶端應用程式在登入後可使用的功能。

> [!NOTE]
> 如需有關 Cmdlet 的其他資訊，請參閱&nbsp;Lync Server Windows PowerShell <A href="http://go.microsoft.com/fwlink/p/?linkid=263432">http://go.microsoft.com/fwlink/p/?linkId=263432</A>博客（位於）。 每個博客及其 URL 的內容可能會變更，恕不另行通知。

## <a name="client-management-cmdlets"></a>用戶端管理 Cmdlet
