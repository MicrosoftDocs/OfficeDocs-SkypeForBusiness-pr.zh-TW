# <a name="verify-federation-and-remote-access-for-external-users"></a>確認外部使用者的同盟及遠端存取

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-18_

將同盟路由轉換為 Lync Server 2013 Edge 伺服器之後，您應該執行一些功能測試，以驗證同盟是否如預期方式執行。 外部使用者存取的測試應該包括貴組織支援的每一種外部使用者類型，包括下列任一或所有類型的使用者。

<div>

## <a name="test-connectivity-of-external-users-and-external-access"></a>測試外部使用者與外部存取的連通性

  - 至少有一個聯盟網域的使用者，也就是 Lync Server 2013 上的內部使用者，以及 Lync Server 2010 上的使用者。 測試立即訊息（IM）、目前狀態、音訊/視頻（A/V）與桌面共用。

  - 貴組織支援的每個公用 IM 服務提供者的使用者（以及已完成哪些預配）與 Lync Server 2013 上的使用者以及 Lync Server 2010 上的使用者進行通訊。

  - 確認匿名使用者可以加入會議。

  - 在 lync Server 2010 上使用遠端使用者存取（從內部網路外部2010（不含 VPN），使用 Lync server 2013 上的使用者以及 Lync Server 2010 上的使用者來託管的使用者。 測試 IM、目前狀態、A/V 與桌面共用。

  - 在 lync Server 2013 上使用遠端使用者存取（從內部網路外部2013（不含 VPN），使用 Lync server 2013 上的使用者以及 Lync Server 2010 上的使用者來託管的使用者。 測試 IM、目前狀態、A/V 與桌面共用。
