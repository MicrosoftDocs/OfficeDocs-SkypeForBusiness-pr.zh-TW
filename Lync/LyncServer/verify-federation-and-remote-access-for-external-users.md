# <a name="verify-federation-and-remote-access-for-external-users"></a>驗證同盟及外部使用者的遠端存取

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-09-18_

轉換至 Lync Server 2013 Edge Server 同盟路由之後，您應該執行一些功能測試以確認同盟如預期般。 外部使用者存取的測試應該包含組織支援的每種外部使用者類型，包括下列任一項或所有項目。

<div>

## <a name="test-connectivity-of-external-users-and-external-access"></a>測試外部使用者與外部存取連線能力

  - 來自至少一個同盟的網域、 內部使用者在 Lync Server 2013 和 Lync Server 2010 上的使用者的使用者。 測試立即訊息 (IM)、目前狀態、音訊/視訊 (A/V) 及桌面共用。

  - 每個公用 IM 服務提供者的組織支援 （且其佈建完成） 的使用者與使用者在 Lync Server 2013 及 Lync Server 2010 上的使用者進行通訊。

  - 確認匿名使用者可以加入會議。

  - Lync Server 2010 上的使用者及使用者的主控 Lync Server 2010 上使用 Lync Server 2013 上的使用者的遠端使用者存取 （登入 Lync Server 2010 從內部網路外，但沒有 VPN）。 測試 IM、目前狀態、A/V 及桌面共用。

  - Lync Server 2010 上的使用者及使用者的主控在 Lync Server 2013 與 Lync Server 2013 上的使用者使用遠端使用者存取 （登入 Lync Server 2013，從內部網路外，但沒有 VPN）。 測試 IM、目前狀態、A/V 及桌面共用。
