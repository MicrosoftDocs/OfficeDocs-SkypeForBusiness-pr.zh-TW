# <a name="planning-for-monitoring-in-lync-server-2013"></a>在 Lync Server 2013 中規劃監控

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-05_

Microsoft Lync Server 2013 中的監控服務提供一種方法，讓系統管理員可以收集組織中所發生之通訊會話的使用狀況、趨勢及服務品質資料。 Lync Server 2013 中的監控不再需要個別的伺服器角色;相反地，監控服務是內置於每一部前端伺服器。 不過，在 Lync Server 2013 中未啟用預設監控。 本文件將協助您決定是否要在組織中啟用監視。
