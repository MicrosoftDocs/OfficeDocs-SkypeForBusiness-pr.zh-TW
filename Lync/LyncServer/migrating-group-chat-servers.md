# <a name="migrating-group-chat-servers"></a>遷移群組聊天伺服器

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-29_

如果您在舊版 Lync Server 2010 環境中部署了 [群組聊天伺服器]，您必須部署 Lync Server 2013 持久聊天伺服器。 群組聊天伺服器與持久聊天伺服器可以共存，但內容（例如，聊天室等）不會在這些伺服器之間共用。 若要從持久性聊天伺服器存取舊版群組聊天伺服器內容，您必須將群組聊天伺服器遷移至永久性聊天伺服器。 如需遷移至永久性聊天伺服器的詳細資料，請參閱[從 Lync server 2010 遷移、群組聊天或 Office 通訊伺服器 2007 R2 群組聊天至 Lync server 2013、持續聊天伺服器（](migration-from-lync-server-2010-group-chat-or-office-communications-server-2007-r2-group-chat-to-lync-server-2013-persistent-chat-server.md)在遷移檔中）。
