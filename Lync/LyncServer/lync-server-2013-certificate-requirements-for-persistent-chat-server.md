# <a name="certificate-requirements-for-persistent-chat-server-in-lync-server-2013"></a>Lync Server 2013 中 Persistent Chat server 的憑證需求

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-03_

若要安裝 Persistent Chat Server，您必須要有與 Lync Server 2013 內部伺服器搭配使用的憑證，供每一部執行 Persistent Chat Web 服務以進行檔案上傳/下載之伺服器使用。 在您開始 Persistent Chat 安裝之前，請確定您已具備必要的憑證 (s) ，尤其是當您使用外部 CA 時。
