# <a name="certificate-requirements-for-persistent-chat-server-in-lync-server-2013"></a>Lync Server 2013 中的常設聊天室伺服器的憑證需求

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-03_

若要安裝永久性聊天伺服器，您必須擁有與 Lync Server 2013 內部伺服器所使用的相同 CA 所頒發的憑證，以進行檔案上傳/下載。 在開始持續進行聊天安裝之前，請務必先擁有所需的憑證，尤其是在您使用外部 CA 時。
