# <a name="installing-the-operation-manager-agent-files-on-a-watcher-node-in-lync-server-2013"></a>在 Lync Server 2013 的 [觀察程式] 節點上安裝 Operation Manager 代理檔案

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-20_

與設定 Lync 伺服器以報告元件警示類似，Lync Server 2013 觀察程式節點需要安裝 System Center Operations Manager 代理檔案。 這可讓綜合交易執行，並將警報報告給 System Center Operations Manager 根管理伺服器。

若要安裝代理程式檔案，請遵循下列各節所列的程式：

1.  [在 Lync Server 2013 周邊網路以外的監視者節點上安裝憑證](lync-server-2013-installing-a-certificate-on-a-watcher-node-located-outside-the-perimeter-network.md)

2.  [在 Lync Server 2013 中安裝 Operation Manager 代理程式檔](lync-server-2013-installing-the-operation-manager-agent-files.md)

3.  [將 Lync Server 2013 電腦設定為參與 System Center 探索](lync-server-2013-configuring-the-lync-server-computer-to-participate-in-system-center-discovery.md)
