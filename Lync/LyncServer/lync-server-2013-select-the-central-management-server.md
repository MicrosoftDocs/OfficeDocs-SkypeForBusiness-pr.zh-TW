# <a name="select-the-central-management-server-in-lync-server-2013"></a>Lync Server 2013 中選取中央管理伺服器

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-01-02_

您可以定義和設定拓撲之前，您必須先定義要安裝中央管理伺服器的位置。

<div>


> [!NOTE]  
> 這將不會生效，直到您已在拓撲產生器發佈拓撲。 若要建立並發行拓撲之前，請設定中央管理伺服器，請執行<STRONG>Set-csconfigurationstorelocation</STRONG>。



</div>

<div>

## <a name="to-select-the-central-management-server"></a>若要選取中央管理伺服器

1.  啟動拓撲產生器： 按一下 [**開始]**、 [**所有程式]**、 [ **Microsoft Lync Server 2013**]，然後按一下**Lync Server 拓撲產生器]**。

2.  以滑鼠右鍵按一下 [Lync Server 2013] 節點，然後按一下 [**編輯內容]**。

3.  在中央管理伺服器] 窗格中，選取前端伺服器上安裝中央管理伺服器，然後按一下 **[確定]**。

</div>
