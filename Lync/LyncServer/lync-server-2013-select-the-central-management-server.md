# <a name="select-the-central-management-server-in-lync-server-2013"></a>在 Lync Server 2013 中選取中央管理伺服器

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-01-02_

您必須先定義安裝中央管理伺服器的位置，才能定義並設定您的拓撲。

<div>


> [!NOTE]  
> 除非您已在拓撲建立器中發佈拓撲，否則這將不會生效。 若要在建立和發佈拓撲之前先設定中央管理伺服器，請執行<STRONG>CSConfigurationStoreLocation</STRONG>。



</div>

<div>

## <a name="to-select-the-central-management-server"></a>選取中央管理伺服器

1.  啟動拓撲產生器：按一下 [**開始**]，按一下 [**所有程式**]，按一下 [ **Microsoft Lync server 2013**]，然後按一下 [ **Lync server 拓撲**建立器]。

2.  以滑鼠右鍵按一下 Lync Server 2013 節點，然後按一下 [**編輯屬性**]。

3.  在 [中央管理伺服器] 窗格中，選取前端伺服器以安裝中央管理伺服器，然後按一下 **[確定]**。

</div>
