# <a name="select-the-central-management-server-in-lync-server-2013"></a>在 Lync Server 2013 中選取中央管理伺服器

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-01-02_

在定義及設定拓撲之前，您必須先定義安裝中央管理伺服器的位置。

<div>


> [!NOTE]  
> 除非您已在拓撲產生器中發佈拓撲，否則此項不會生效。 若要在建立及發行拓撲之前設定中央管理伺服器，請執行 <STRONG>remove-csconfigurationstorelocation</STRONG>。



</div>

<div>

## <a name="to-select-the-central-management-server"></a>若要選取中央管理伺服器

1.  啟動拓撲產生器：依序按一下 [ **開始**]、[ **所有程式**]、[ **Microsoft Lync server 2013**]，然後按一下 [ **Lync server 拓撲**產生器]。

2.  在 [Lync Server 2013] 節點上按一下滑鼠右鍵，然後按一下 [ **編輯屬性**]。

3.  在 [中央管理伺服器] 窗格中，選取要在其上安裝中央管理伺服器的前端伺服器，然後按一下 **[確定]**。
