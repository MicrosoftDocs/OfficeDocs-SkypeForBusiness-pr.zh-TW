# <a name="configuring-support-for-archiving-in-lync-server-2013"></a>在 Lync Server 2013 中設定存檔支援

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-01_

在您將封存新增到拓撲併發布新的拓撲之後，您必須針對在您的部署中最初實現封存的方式設定選項，然後設定一或多個封存原則，為您的部署啟用歸檔，以及您也可以選擇針對特定的網站和使用者。 您可以使用 Lync Server 2013 [控制台] 來執行這項作業。

<div>


> [!NOTE]  
> 部署之後，您可以變更 [封存] 設定以停用或啟用封存。 如需有關如何針對日常管理實施封存支援或在部署之後符合貴組織中新需求的詳細資料，請參閱在作業檔中<A href="lync-server-2013-managing-archiving.md">管理 Lync Server 2013 歸檔</A>。



</div>
