# <a name="configuring-support-for-archiving-in-lync-server-2013"></a>在 Lync Server 2013 中設定封存支援

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-01_

將封存新增至您的拓撲併發布新的拓撲之後，您必須設定選項，以瞭解如何在部署中開始實施封存，然後設定一或多個封存原則，以啟用部署的封存，也就是針對特定網站和使用者（選用）。 您可以使用 Lync Server 2013 控制台來執行這項作業。

<div>


> [!NOTE]  
> 部署之後，您可以變更封存設定，以停用或啟用封存。 如需如何針對日常管理執行封存支援，或在部署後符合組織中新需求的詳細資訊，請參閱 Operations 檔中的 <A href="lync-server-2013-managing-archiving.md">管理 Lync Server 2013</A> 封存。



</div>
