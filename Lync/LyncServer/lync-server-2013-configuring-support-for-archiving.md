# <a name="configuring-support-for-archiving-in-lync-server-2013"></a>在 Lync Server 2013 中設定封存的支援

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-10-01_

將封存新增至您的拓撲及之後發佈新拓撲，您需要為封存最初如何實作在您部署中，設定選項，然後設定一或多個封存原則，以啟用封存的部署，（選用） 之特定網站及使用者。 若要這麼做，您可以使用 Lync Server 2013 控制台。

<div>


> [!NOTE]  
> 部署之後，您可以變更停用或啟用封存的封存設定值。 如需如何實作封存支援日常管理工作，或符合部署之後，您組織中的新需求的詳細資訊，請參閱作業文件中的<A href="lync-server-2013-managing-archiving.md">管理 Lync Server 2013 進行封存</A>。
