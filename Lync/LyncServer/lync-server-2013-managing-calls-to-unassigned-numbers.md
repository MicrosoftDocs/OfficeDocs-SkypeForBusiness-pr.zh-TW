# <a name="managing-calls-to-unassigned-numbers-in-lync-server-2013"></a>管理 Lync Server 2013 中未指派號碼的呼叫

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-11-01_

Lync Server 可讓您設定撥入電話號碼對您的組織有效，但不會指派給使用者或電話。 您可以使用宣告應用程式，將這些來電轉接至預先確定的目的地（電話號碼、SIP URI 或語音信箱），或播放音訊公告或兩者。 您也可以將來電轉接到 Exchange UM 自動語音應答電話號碼。 使用下列其中一種方式處理未指定編號的呼叫可協助您避免來電者 misdials 或聽到占線的情況，或 SIP 用戶端收到錯誤訊息。

本節說明如何管理未指定的數位範圍，以處理未指派電話號碼的呼叫。 本節也說明如何在災害復原期間管理宣告（如果您在中斷期間想要此功能）。

<div>


> [!NOTE]  
> 在中斷期間使用未指定的數位處理是選用的。
