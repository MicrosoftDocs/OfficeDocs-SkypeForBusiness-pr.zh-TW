# <a name="managing-lync-server-2013-archiving"></a>管理 Lync Server 2013 封存

_**主題上次修改日期：** 2012-10-10_

當您為組織部署封存時，會在部署期間指定初始設定。 然而，有時您可能會想要改變您對例行管理實作封存支援的方式，或符合組織中的新需求。 例如，您可能會需要對特定網站、集區或組織內的使用者以不同的方式設定封存支援。 在 Lync Server 2013 上的使用者，您可以建立及自訂封存原則和設定。 如果您使用 Microsoft Exchange 整合，您也必須設定 Exchange 2013 設定。 本節所提供的資訊和程序可讓您變更您的封存部署。
