# <a name="developing-a-backup-and-restoration-strategy-and-plan-for-lync-server-2013"></a>開發 Lync Server 2013 的備份與還原策略及規劃

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-17_

Lync Server 備份和還原作業的效能取決於您的備份與還原策略及規劃。 您應該建立一個策略來備份及還原與貴組織的整體策略相符的 Lync Server，以及備份資料和設定的完整簡明方案，以及在發生中斷時，還原服務的方案。

如需最強健的前端池災害復原，請使用 Lync Server 2013 中所述的成對式池災害復原拓撲。 如需詳細資訊，請參閱[在 Lync Server 2013 中規劃高可用性和災害復原](lync-server-2013-planning-for-high-availability-and-disaster-recovery.md)。
