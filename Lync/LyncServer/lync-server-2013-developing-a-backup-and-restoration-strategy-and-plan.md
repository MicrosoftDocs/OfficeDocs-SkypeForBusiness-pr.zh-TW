# <a name="developing-a-backup-and-restoration-strategy-and-plan-for-lync-server-2013"></a>開發 Lync Server 2013 的備份與還原策略及規劃

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-17_

您的 Lync Server 備份和還原作業的效能取決於您的備份與還原策略及規劃。 您應該建立一個策略，以備份及還原與組織的整體策略相符的 Lync Server，以及備份資料和設定的完整、簡明計畫，而且在中斷時，是要還原服務的計畫。

如需最強健的前端集區災難復原，請使用 Lync Server 2013 中所引進的配對集區嚴重損壞修復拓撲。 如需詳細資訊，請參閱 [在 Lync Server 2013 中規劃高可用性和嚴重損壞修復](lync-server-2013-planning-for-high-availability-and-disaster-recovery.md)。
