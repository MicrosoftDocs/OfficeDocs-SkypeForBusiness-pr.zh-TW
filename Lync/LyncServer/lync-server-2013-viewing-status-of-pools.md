# <a name="viewing-status-of-pools-in-lync-server-2013"></a>在 Lync Server 2013 中查看池狀態

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2014-05-01_

針對每個企業版池及每個標準版 Server 伺服器的池子，您可以透過 Lync Server 2013 控制台，查看有關池狀態的資訊，以及該池所使用之其他元件的狀態。 我們建議您使用 Lync Server [控制台] 來實現 System Center Operations Manager 2012，以協助管理員在其環境中查看池的健康情況。

<div>

## <a name="view-pools"></a>[查看] 池

1.  使用 Lync Server [控制台] 開啟 [Lync Server 控制台]。

2.  按一下 [拓撲]，然後檢查拓撲中所有伺服器的狀態。
