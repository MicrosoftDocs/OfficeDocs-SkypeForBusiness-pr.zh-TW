# <a name="reviewing-the-summary-report-in-lync-server-2013"></a>在 Lync Server 2013 中查看摘要報告

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-21_

Lync Server 管理員報告是四個重要報表中的第一筆，以詳細說明您的設計。 此報告中的資訊以及其他三個相關聯的報表，都是您的資訊技術小組的絕佳檔：

![一般摘要管理員報告](images/Gg558626.9c529ef7-cb1b-4ce1-a8bc-3ec79aba2377(OCS.15).jpg "一般摘要管理員報告")

  - 憑證報告

  - 防火牆報告

  - DNS 報告

[摘要報告] 會列出與 Edge 網路相關的一般配置資訊。 [位置]、[完全合格的功能變數名稱（FQDN）] 和 [IP 位址]、[網路類型]，以及特定于指定角色的批註，都有記錄。

設計人員以及要部署、管理及維護基礎結構的每個小組，都應該檢查摘要報告的準確性，並確認錯誤最低。
