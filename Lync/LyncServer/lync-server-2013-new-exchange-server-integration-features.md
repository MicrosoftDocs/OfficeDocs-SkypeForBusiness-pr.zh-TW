# <a name="new-exchange-server-integration-features-in-lync-server-2013"></a>Lync Server 2013 中的新 Exchange Server 整合功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-24_

Lync Server 2013 在與 Microsoft Exchange Server 2013 一起部署時支援新功能，如下列清單所述。 若要使用這些功能，必須部署 Lync Server 2013 和 Exchange 2013。

  - 您可以使用「**整合連絡人存放區**」，其中 lync 連絡人清單儲存在 Exchange 2013 中，您可以在 lync 2013、Outlook 2013 和 Outlook Web App 中管理連絡人存放區。

  - 您可以將**高解析度相片**用於連絡人。 含最高648x648 圖元的相片會儲存在 Exchange 2013 中，並提供給用戶端，包括 Lync 2013、Outlook 2013、Microsoft Lync Web App 及 Outlook Web App。

  - 您可以啟用**lync 封存整合**，將 lync Server 2013 存檔整合至 Exchange 2013 就地保留功能（適用于 exchange 2013 的使用者），這可讓系統管理員熟悉合規性與 eDiscovery。
