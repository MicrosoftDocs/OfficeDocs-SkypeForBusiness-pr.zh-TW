# <a name="new-exchange-server-integration-features-in-lync-server-2013"></a>Lync Server 2013 中的新 Exchange Server 整合功能

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-24_

Lync Server 2013 支援新功能（與 Microsoft Exchange Server 2013 一起部署），如下列清單所述。 為使上述各項功能運作，必須部署 Lync Server 2013 和 Exchange 2013。

  - 您可以使用 **整合連絡人存放區**，其中的 lync 連絡人清單儲存在 Exchange 2013 中，您可以在 Lync 2013、outlook 2013 和 Outlook Web App 中管理連絡人存放區。

  - 您可以使用 **高解析度的相片** 處理連絡人。 具有最多648x648 圖元的相片會儲存在 Exchange 2013 中，並可供用戶端（包括 Lync 2013、Outlook 2013、Microsoft Lync Web App 和 Outlook Web App）使用。

  - 您可以為位於 Exchange 2013 的使用者啟用 **lync 封存整合**，以整合 lync Server 2013 封存至 exchange 2013 In-Place 保留功能，讓系統管理員針對法規遵從性和 eDiscovery 提供一種通用的體驗。
