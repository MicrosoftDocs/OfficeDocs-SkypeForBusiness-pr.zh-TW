# <a name="planning-for-response-groups-in-lync-server-2013"></a>在 Lync Server 2013 中規劃回應群組

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-06-19_

如果您的組織有一組人接聽並管理特定類型的通話（例如客戶服務、內部技術支援人員或部門的一般電話支援），您可以部署 Lync Server 回應群組應用程式來管理這些類型通話。 回應群組應用程式會將來電路由並列隊給指定的人員（稱為代理者）。 您可以增加電話支援服務的使用，並使用回應群組來減少執行這些服務的額外負荷。 本節說明回應群組的規劃考慮。
