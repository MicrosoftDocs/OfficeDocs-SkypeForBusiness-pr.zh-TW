# <a name="apply-office-communications-server-2007-r2-updates"></a>套用 Office 通訊伺服器 2007 R2 更新

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-19_

在您遷移至 Lync Server 2013 之前，必須先將更新套用到您的 Office 通訊伺服器 2007 R2 環境。 如需 Office 通訊伺服器 2007 R2 的最新資訊，請參閱**通訊伺服器 2007 R2** [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802)**的更新**。

若要安裝 Office 通訊伺服器 2007 R2 的更新，我們建議您遵循 Microsoft 知識庫文章968802中所述的**方法1累計伺服器更新程式**，即「通訊伺服器 2007 R2 的更新」 [http://go.microsoft.com/fwlink/p/?linkid=3052\&kbid=968802](http://go.microsoft.com/fwlink/p/?linkid=3052%26kbid=968802)。
