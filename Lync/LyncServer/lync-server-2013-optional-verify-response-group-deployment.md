# <a name="optional-verify-response-group-deployment-in-lync-server-2013"></a>在 Lync Server 2013 中 (選用) 驗證回應群組部署

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-11_

在您設定回應群組之後，您必須驗證設定，以確保回應群組如預期般運作。 至少要使用下列類型的使用者驗證下列案例：

**Users**

  - 位於 Lync Server 2013 的使用者

  - 使用公用交換電話網路的外部使用者 (PSTN) 

  - 位於 Lync Server 2013 的代理人

**Scenarios**

  - Lync Server 2013 使用者呼叫回應群組。

  - 外部使用者呼叫回應群組。

  - 當專員在接聽另一通電話而排入佇列時，使用者呼叫回應群組。
