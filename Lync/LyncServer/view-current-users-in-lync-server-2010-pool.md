_**主題上次修改日期：** 2012-09-26_

在深入瞭解您可以在集區之間移動使用者的各種方式之前，必須先判斷舊版 Lync Server 2010 集區中存在的使用者。 在下圖中，[註冊集區集區] 欄可識別為舊版 Lync Server 2010 集區設定的六位使用者。 這些是我們將要移至 Lync Server 2013 集區的測試使用者。

**若要查看 Lync Server 2010 集區中的使用者清單**

1.  使用 RTCUniversalServerAdmins 群組成員的帳戶或 CsAdministrator 或 CsUserAdministrator 系統管理角色的成員帳戶，登入 Lync Server 2010 前端伺服器。

2.  開啟 [ **Lync Server 控制台**]。

3.  按一下 [ **使用者**]，按一下 [搜尋]，然後按一下 [ **尋找**]。
