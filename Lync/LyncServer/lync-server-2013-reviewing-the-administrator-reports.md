# <a name="reviewing-the-administrator-reports-in-lync-server-2013"></a>在 Lync Server 2013 中檢閱管理員報告

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-21_

系統管理員報告是部署與作業的詳細資訊。 報告是根據**設計網站**中標示的選取產生的。 設計工具可以編輯網狀圖並定義伺服器、池及負載平衡器的完整 IP 位址及完整功能變數名稱（Fqdn），進一步為系統管理員報告增添價值。
