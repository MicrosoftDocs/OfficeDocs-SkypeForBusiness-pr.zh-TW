# <a name="e9-1-1-support-in-lync-server-2013"></a>Lync Server 2013 中的 E9-1-1 支援

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-09-21_

Lync Server 2013 支援增強型 9-1-1 (E9-1-1) 為企業部署的一部分。 E9-1-1 是緊急通知功能，會讓來電者的電話號碼與市鎮 (即街道) 地址產生關聯。 E9-1-1 支援僅適用於美國。

若要支援 E9-1-1 為 Lync Server 2013 部署的一部分，您必須從認證的緊急服務提供者取得 E9-1-1 路由服務，或使用緊急位置識別號碼 (ELIN) 閘道。 緊急服務提供者或電訊廠商路由緊急通話，來自 Lync Server 2013 到正確公用安全回應點 (PSAP)，依據通話中所含的位置資訊。 如需詳細資訊 E9-1-1 支援，請參閱規劃文件中的[Planning for Lync Server 2013 中的緊急服務 (E9-1-1)](lync-server-2013-planning-for-emergency-services-e9-1-1.md) 。
