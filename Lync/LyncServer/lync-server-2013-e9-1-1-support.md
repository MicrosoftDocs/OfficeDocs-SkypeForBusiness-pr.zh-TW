# <a name="e9-1-1-support-in-lync-server-2013"></a>Lync Server 2013 中的 E9-1-1 支援

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-21_

Lync Server 2013 支援增強的9-1-1 （E9-1-1）作為企業部署的一部分。 E9-1-1 是緊急通知功能，可將通話方的電話號碼與市政（即街道）位址進行關聯。 E9-1-1-1 支援僅適用于美國。

若要支援 E9-1 （作為 Lync Server 2013 部署的一部分），您必須從認證緊急服務提供者取得 E9-1-1 路由服務，或使用緊急位置識別號碼（ELIN）閘道。 緊急服務提供者或電信公司會根據呼叫中所包含的位置資訊，將源自 Lync Server 2013 的緊急呼叫傳送給正確的公用安全應答點（PSAP）。 如需 E9-1-1 支援的詳細資料，請參閱規劃檔中的[Lync Server 2013 中的緊急服務規劃（E9-1）](lync-server-2013-planning-for-emergency-services-e9-1-1.md) 。
