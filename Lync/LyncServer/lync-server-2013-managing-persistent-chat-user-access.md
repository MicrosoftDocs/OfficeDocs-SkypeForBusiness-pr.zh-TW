# <a name="managing-persistent-chat-user-access-in-lync-server-2013"></a>在 Lync Server 2013 中管理常設聊天室使用者存取

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-06_

下列主題說明如何使用 Lync Server 2013 [控制台] 和 [**永久聊天**] 頁面來管理持續聊天中的使用者存取。 若要管理持久聊天伺服器，您必須在系統中擁有持久的聊天管理員許可權和許可權。 當您安裝持續聊天伺服器時，您可以指定一個或多個具有必要持續聊天管理員權力和許可權的使用者。 若要將系統管理員權利和許可權授與其他使用者帳戶，請使用 [**管理使用者和使用者群組**]。 如需詳細資訊，請參閱[讓使用者在 Lync Server 2013 中管理類別、聊天室，以及使用者權利和許可權](lync-server-2013-enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions.md)。
