# <a name="managing-persistent-chat-user-access-in-lync-server-2013"></a>管理 Lync Server 2013 中的 [常設聊天室使用者存取

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-10-06_

下列主題說明如何使用 Lync Server 2013 控制台和**常設聊天室**] 頁面來管理常設聊天室中的使用者存取。 若要管理 Persistent Chat Server，您必須常設聊天室管理員權利和權限系統中。 當您安裝 Persistent Chat Server 時，您會指定一或多個使用者具備必要的常設聊天室系統管理員權限和權限。 若要將系統管理員權利和權限授與其他使用者帳戶，請使用**管理使用者和使用者群組**。 如需詳細資訊，請參閱[讓使用者管理類別、 聊天室和使用者權限和 Lync Server 2013 中的權限](lync-server-2013-enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions.md)。
