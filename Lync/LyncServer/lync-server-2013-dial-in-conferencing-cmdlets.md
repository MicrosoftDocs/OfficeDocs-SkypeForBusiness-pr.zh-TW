## <a name="dial-in-conferencing-cmdlets"></a>Dial-In Conferencing Cmdlets

如果您不想要允許電話撥入式會議，您可以使用 Set-CsConferencingPolicy Cmdlet 來停用此功能，並將 EnableDialInConferencing 屬性設定為 False。 根據預設，允許所有使用者主控包含電話撥入式會議的會議。
