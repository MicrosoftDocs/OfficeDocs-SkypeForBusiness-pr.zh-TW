# <a name="troubleshooting-server-roles-and-services-cmdlets-in-lync-server-2013"></a>在 Lync Server 2013 中疑難排解伺服器角色和服務 Cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-08-27_

疑難排解 Cmdlet 提供不同的方式來驗證 Microsoft Lync Server 2013 是否如預期運作。 例如，您可以使用 CsHealthMonitoringConfiguration Cmdlet 來設定註冊機構和主管池的測試帳戶。 接著，您可以使用這些測試帳戶來驗證使用者是否能夠成功完成一般工作，例如登入系統、交換立即訊息，或撥打電話至位於公用交換電話網絡（PSTN）的電話。

<div>


> [!NOTE]
> 如需有關 Cmdlet 的其他資訊，請參閱&nbsp;Lync Server Windows PowerShell <A href="http://go.microsoft.com/fwlink/p/?linkid=263432">http://go.microsoft.com/fwlink/p/?linkId=263432</A>博客（位於）。 每個博客及其 URL 的內容可能會變更，恕不另行通知。
