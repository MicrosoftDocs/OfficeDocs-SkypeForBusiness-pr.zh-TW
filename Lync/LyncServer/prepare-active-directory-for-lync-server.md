# <a name="prepare-active-directory-for-lync-server"></a>準備 Lync Server 的 Active Directory

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-17_

在使用 Lync Server 2010 的共存狀態中部署 Lync Server 2013 之前，您必須執行一些其他的 Active Directory 工作，以設定 Lync Server 2013 的架構、林及網域。 架構延伸會新增 Lync Server 2013 所需的 Active Directory 類別和屬性。 如需其他資訊，請參閱[準備 Lync Server 2013 的 Active Directory 網域服務](lync-server-2013-preparing-active-directory-domain-services.md)主題。

**準備 Lync Server 2013 的 Active Directory**

1.  在 Lync Server 2013 前端伺服器上，執行 Lync Server 2013 設定。

2.  選取 [**準備 Active Directory**]。
    
    ![Lync Server 2013 部署精靈，歡迎頁面](images/JJ205265.5f88ae18-9c3c-42ea-a91a-836ecf5d515f(OCS.15).jpg "Lync Server 2013 部署精靈，歡迎頁面")

3.  完成步驟1到5。
    
    ![部署精靈、Active Directory 準備](images/JJ205265.eddd9e94-fa70-453f-8810-b99a2bf0844a(OCS.15).jpg "部署精靈、Active Directory 準備")
