# <a name="quality-of-experience-qoe-in-lync-server-2013"></a>Lync Server 2013 的經驗品質 (QoE)

_**主題上次修改日期：** 2012-11-01_

經驗品質 (QoE) 會記錄數字資料，指出有關通話與工作階段中所包含參與者、裝置名稱、驅動程式、IP 位址和端點類型的媒體品質和資訊。 當您安裝 Lync Server 2013 時，也會為 QoE 安裝通用設定的預先定義集合。 請使用本節中的主題來設定 QoE 設定。
