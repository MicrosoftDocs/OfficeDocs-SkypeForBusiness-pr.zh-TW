# <a name="managing-call-park-in-lync-server-2013"></a>在 Lync Server 2013 中管理通話駐留

<div data-xmlns="http://www.w3.org/1999/xhtml">

_**主題上次修改日期：** 2012-09-10_

通話駐留應用程式可讓企業語音使用者從一部電話接聽來電，然後在稍後從任何電話取回通話。 當使用者公園通話時，Lync Server 會將來電轉接至暫時號碼（稱為 *軌道*），在此呼叫中，來電會保留，直到有人將通話或超時。

本節中的主題會針對您可以在部署中執行來自訂和維護通話駐留應用程式的工作，提供逐步程式。
