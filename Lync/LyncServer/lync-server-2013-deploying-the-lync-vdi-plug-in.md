# <a name="deploying-the-lync-vdi-plug-in-in-lync-server-2013"></a>在 Lync Server 2013 中部署 Lync VDI 外掛程式

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-03_

Lync 2013 用戶端支援虛擬桌面基礎結構中的音訊和影片 (VDI) 環境。 使用者可以將音訊和視訊裝置 (例如，耳機或相機) 連線至本機電腦 (例如，精簡型用戶端或重新設定功能的電腦)。 使用者可以連線至虛擬機器，登入虛擬機器上執行的 Lync 2013 用戶端，並參與即時音訊和視頻通訊，如同用戶端在本機執行。

Lync VDI 外掛程式是安裝在本機電腦上的獨立應用程式，可讓本機音訊和影片裝置使用於虛擬機器上執行的 Lync 2013 用戶端。 外掛程式不需要在本機電腦上安裝 Lync。 在使用者登入虛擬機器上執行的 Lync 2013 用戶端後，Lync 會提示使用者重新輸入其認證，以與本機電腦上執行的 Lync VDI 外掛程式建立連線。 在建立此連線之後，使用者就可以開始和接收音訊和影片通話。
