# <a name="deploying-the-lync-vdi-plug-in-in-lync-server-2013"></a>部署 Lync VDI 外掛程式 Lync Server 2013 中

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-10-03_

Lync 2013 用戶端支援音訊和視訊虛擬桌面基礎結構 (VDI) 環境中。 使用者可以將音訊和視訊裝置 (例如，耳機或相機) 連線至本機電腦 (例如，精簡型用戶端或重新設定功能的電腦)。 使用者可以連線至虛擬機器、 登入是虛擬機器執行，Lync 2013 用戶端和一樣用戶端在本機上執行，參與即時音訊及視訊通訊。

Lync VDI Plug-in 是獨立的應用程式，會在本機電腦上安裝，並使用本機音訊和視訊裝置允許與 Lync 2013 用戶端在虛擬機器上執行。 外掛程式不需要在本機電腦上安裝 Lync。 使用者登入 Lync 2013 用戶端在虛擬機器上執行後，Lync 會提示使用者重新輸入他/她的認證，以建立與本機電腦執行 Lync VDI 外掛程式的連線。 建立此連線之後，使用者已準備好撥打及接聽音訊和視訊通話。
