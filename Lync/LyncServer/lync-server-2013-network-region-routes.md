# <a name="network-region-routes-in-lync-server-2013"></a>Lync Server 2013 中的網路區域路由

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-21_

*網路區域路由*定義一對網路區域之間的路由。 通話許可控制部署中的每一組網路區域都需要網路區域路由。 這可讓部署中的每個網路區域存取每一個其他區域。 使用本節中的程式來查看、建立、修改或刪除網路區域路由。
