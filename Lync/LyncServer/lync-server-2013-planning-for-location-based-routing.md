# <a name="planning-for-location-based-routing-in-lync-server-2013"></a>在 Lync Server 2013 中規劃 Location-Based 路由

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

_**主題上次修改日期：** 2013-07-31_

本主題中的資訊適用于 Lync Server 2013 的累計更新：二月份2013。

Location-Based 路由可讓您根據通話中各方的位置，限制 VoIP 端點和 PSTN 端點之間通話的路由。 Location-Based 路由是 Lync Server 2013 Enterprise Voice 基礎結構的一部分。 Location-Based 路由是通話管理功能，可控制 Lync Server 2013 CU1 路由傳送通話的方式。 它會根據 Lync 來電者的地理位置，強制執行呼叫授權規則，以判斷通話是否可以路由傳送至 PBX 或 PSTN 端點。
