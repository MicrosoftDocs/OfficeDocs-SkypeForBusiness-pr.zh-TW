# <a name="planning-for-location-based-routing-in-lync-server-2013"></a>在 Lync Server 2013 中規劃位置基礎路由

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-07-31_

本主題中的資訊適用于 Lync Server 2013 的累積更新：2月2013。

以位置為基礎的路由可以根據通話中各方的位置，限制 VoIP 端點與 PSTN 端點之間的通話路由。 以位置為基礎的路由是 Lync Server 2013 企業語音結構的一部分。 位置式路由是呼叫管理的功能，可控制 Lync Server 2013 CU1 路由呼叫的方式。 它會強制進行呼叫授權規則，讓您可以根據 Lync 來電者的地理位置，將呼叫路由到 PBX 或 PSTN 端點。
