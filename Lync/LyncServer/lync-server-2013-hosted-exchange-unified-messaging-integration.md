# <a name="hosted-exchange-unified-messaging-integration-in-lync-server-2013"></a>Lync Server 2013 中的主控的 Exchange Unified Messaging 整合

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-09-20 個_

除了先前的 Lync Server 2013 版本的支援已提供*內部*部署的 Exchange 整合通訊 (UM)，與整合 Lync Server 2013 引進與*裝載*Exchange UM 整合的支援。 主控 Exchange UM 可讓 Lync Server 2013 提供語音訊息給使用者，如果您將部分或全部轉移到裝載的 Exchange 服務提供者，例如 Microsoft Exchange Online。

Lync Server 2013 Enterprise Voice 會使用 Exchange UM 基礎結構來提供來電接聽、 來電通知、 語音存取 （包括語音信箱） 和自動語音應答服務。 如需詳細資訊，請參閱[整合的整合通訊與 Lync Server 2013 的功能](lync-server-2013-features-of-integrated-unified-messaging.md)。
