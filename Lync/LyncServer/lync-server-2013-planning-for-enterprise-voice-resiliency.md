# <a name="planning-for-enterprise-voice-resiliency-in-lync-server-2013"></a>規劃 Lync Server 2013 中的企業語音恢復能力

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改主題：** 2014年-10-17_

語音恢復能力指的是使用者能夠繼續了與接收的通話，如果中央網站的主應用程式 Lync Server 2013 變成無法使用，不論是透過廣域網路 (WAN) 失敗或其他原因的網路。 如果中央站台失敗，企業語音服務必須繼續不中斷地透過無縫的容錯移轉至備份網站。 WAN 故障時分支站台通話必須重新導向到本機的 PSTN 閘道。 本章節將討論規劃中央網站或 WAN 故障時的語音恢復能力。
