# <a name="optional-verify-announcement-deployment-in-lync-server-2013"></a>可選驗證 Lync Server 2013 中的宣告部署

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-25_

安裝並設定宣告之後，您必須驗證設定，以確保呼叫未指派的號碼會如預期的那樣運作。 至少請確認下列事項：

  - 撥打對您組織有效但不是未指派號碼的號碼。

  - 撥打未指定的電話號碼，並確認正確的宣告是否可播放。
