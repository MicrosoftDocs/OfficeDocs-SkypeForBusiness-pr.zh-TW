# <a name="lync-for-windows-phone-requirements-in-lync-server-2013"></a>Lync for Lync Server 2013 中的 Windows Phone 需求

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-20_

Microsoft Lync 2013 for Windows Phone 提供立即訊息（IM）、增強的目前狀態，以及由您組織中的使用者從智慧型手機或 Windows 專業行動裝置連線的電話。 行動裝置可讓使用者延伸 Lync 2013 的範圍。 本主題說明 Windows Phone 版 Lync 2013 的規劃考慮，包括識別先決條件與技術需求、必要元件，以及部署指導方針。

<div>

## <a name="lync-for-windows-phone-prerequisites"></a>Windows Phone 版 Lync 先決條件

以下是適用于 Windows Phone 系統必備的 Lync 2013。

  - Windows Phone 8、codenamed "Apollo" 或最新版本。

  - Windows Phone 裝置必須具備 Microsoft 提供的最新更新。 如需詳細資訊，請參閱 Windows Phone 8 [http://go.microsoft.com/fwlink/p/?LinkID=281961](http://go.microsoft.com/fwlink/p/?linkid=281961)更新歷程記錄。

  - 裝置必須有 12 MB 的可用磁碟空間。

  - 使用者必須具備來自承運人的語音和通話方案。
