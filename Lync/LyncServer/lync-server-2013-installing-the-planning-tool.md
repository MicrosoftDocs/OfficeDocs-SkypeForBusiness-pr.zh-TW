# <a name="installing-the-planning-tool-in-lync-server-2013"></a>Lync Server 2013 中安裝規劃工具

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改主題：** 2013年-11-07_

之前開始設計和規劃 Lync Server 2013 基礎結構使用 Microsoft Lync Server 2013 規劃工具，您必須先安裝規劃工具。 規劃工具不需要部署至工作站或伺服器的網域或基礎結構一部分您預計要安裝 Lync Server 2013。 規劃工具時所附帶的讀我檔案詳細說明安裝及使用此工具的重要資訊。 系統會以方便識別以下重複部分讀我檔案中的資訊。

<div>


> [!IMPORTANT]  
> 規劃工具需要由系統管理員權限和安裝此工具的電腦上的權限的使用者安裝。



</div>

支援的作業系統安裝和規劃工具的作業包括：

  - Windows 8

  - Windows 8.1

  - Windows Server 2012

  - Windows Server 2012 R2

  - Windows 7，32 位元版本

  - Windows 7 64 位元版本，使用 Windows on Win32 (WOW)

  - Windows Server 2008 R2，使用 WOW

此外，規劃工具需要 Microsoft.NET Framework 4.5。

符合安裝前需求之後，您可以再安裝規劃工具。

<div>

## <a name="to-install-the-planning-tool"></a>若要安裝規劃工具

1.  系統管理員群組的成員身分登入本機電腦。

2.  使用 Windows 檔案總管] 或 [命令視窗，找出您下載的規劃工具安裝檔案所在的目錄。

3.  找出 LyncPlanningTool.msi。 在 [Windows 檔案總管]，連按兩下檔案。 在命令視窗中，輸入檔案名稱，然後按**Enter**來執行檔案。

4.  在 [ **Microsoft Lync Server 2013、 規劃工具安裝嚮導**的 [歡迎] 頁面中，按一下 [**下一步**]。

5.  檢閱**授權合約**，然後選取 [**我接受授權合約中的條款]** 如果您選擇以接受授權合約]，使用中的條款，然後按一下 [**下一步]**。

6.  選擇要安裝規劃工具檔案的位置。 預設位置為 c:\\Program Files (x86)\\Microsoft Lync Server 2013\\規劃工具。 如果您想要變更的安裝位置，按一下 [**變更**]。 在**變更目的地資料夾**中，瀏覽或輸入要安裝的檔案，按一下 [**確定**]，然後按一下 [**下一步**的位置。

7.  安裝程式已準備好安裝規劃工具。 按一下 [**安裝**] 以開始安裝程序。

8.  開始安裝，並會顯示進度。 成功完成安裝之後，按一下 [**完成**]。

9.  規劃工具可供使用。
