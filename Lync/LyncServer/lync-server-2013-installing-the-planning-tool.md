# <a name="installing-the-planning-tool-in-lync-server-2013"></a>在 Lync Server 2013 中安裝規劃工具

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-11-07_

在您開始設計及規劃 Lync Server 2013 基礎結構之前，使用 Microsoft Lync Server 2013 （規劃工具）之前，您必須先安裝規劃工具。 規劃工具不需要部署到屬於您計畫安裝 Lync Server 2013 之網域或基礎結構一部分的工作站或伺服器上。 規劃工具附帶的讀我檔案詳細說明安裝及使用此工具的重要資訊。 為了清楚起見，自述檔中的部分資訊是重複的。

<div>


> [!IMPORTANT]  
> 規劃工具需要安裝要安裝該工具之電腦的系統管理員權利和許可權的使用者。



</div>

規劃工具的安裝及作業支援的作業系統包括：

  - Windows 8

  - Windows 8.1

  - Windows Server 2012

  - Windows Server 2012 R2

  - Windows 7、32位版本

  - 使用 Windows on Win32 的 windows 7、64位版本 (WOW) 

  - Windows Server 2008 R2，使用 WOW

此外，規劃工具需要 Microsoft .NET Framework 4.5。

符合預先安裝需求之後，即可安裝規劃工具。

<div>

## <a name="to-install-the-planning-tool"></a>安裝規劃工具

1.  以 Administrators 群組成員的身分登入本機電腦。

2.  使用 Windows Explorer 或命令視窗，找到您下載規劃工具安裝檔的目錄。

3.  找出 LyncPlanningTool.msi。 在 [Windows Explorer] 中，按兩下檔案。 在命令視窗中，輸入檔案名，然後按 **enter** 執行該檔案。

4.  在 [ **Microsoft Lync Server 2013，規劃工具安裝精靈]** 的 [歡迎] 頁面上，按 **[下一步]**。

5.  檢查 **End-User 授權合約**，如果您選擇接受授權合約中的使用條款，請選取 [ **我接受授權合約中的條款** ]，然後按 **[下一步]**。

6.  選擇安裝規劃工具檔案的位置。 預設位置是 C： \\ Program Files (x86) \\ Microsoft Lync Server 2013 \\ 規劃工具。 若要變更安裝位置，請按一下 [ **變更**]。 在 [ **變更目的地資料夾**] 中，流覽或輸入要安裝盤案的位置，按一下 **[確定]**，然後按 **[下一步**]。

7.  安裝程式現在已準備好安裝規劃工具。 按一下 [ **安裝** ] 以開始安裝程式。

8.  安裝會開始，並會顯示進度。 安裝順利完成後，按一下 **[完成]**。

9.  規劃工具可供使用。

</div>
