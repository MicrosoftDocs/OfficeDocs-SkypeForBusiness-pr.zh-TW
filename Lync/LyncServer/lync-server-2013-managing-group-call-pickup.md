# <a name="managing-group-call-pickup-in-lync-server-2013"></a>在 Lync Server 2013 中管理群組呼叫收取

_**主題上次修改日期：** 2013-02-22_

Lync Server 2013 的累計更新：2月2013將群組呼叫收取引進為新的 Enterprise Voice 功能。 群組叫用權可讓企業語音使用者透過撥打呼叫收取群組號碼來挑選撥打給另一位使用者的通話。

本節中的主題會針對您執行的工作設定逐步程式，以在部署中設定群組呼叫收取。
