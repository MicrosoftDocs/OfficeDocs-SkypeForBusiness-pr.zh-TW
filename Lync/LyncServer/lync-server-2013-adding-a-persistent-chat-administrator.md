# <a name="adding-a-persistent-chat-administrator-in-lync-server-2013"></a>在 Lync Server 2013 中新增 Persistent Chat 系統管理員

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-06_

在 Lync Server 2013 中，執行特定工作的使用者必須指派為一或多個特定群組的成員。 以角色為基礎的存取控制 (RBAC) 也可以用來授與許可權，方法是將使用者指派至預先定義的 Lync Server 2013 系統管理角色。

在設定及管理 Persistent Chat Server 之前，請確定適當的使用者權限及許可權已存在，而且任何歸類為 Persistent Chat 系統管理員的使用者都會新增至 CsPersistentChatAdministrator 安全性群組。
