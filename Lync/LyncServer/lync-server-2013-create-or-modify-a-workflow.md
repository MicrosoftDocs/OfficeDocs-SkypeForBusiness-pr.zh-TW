# <a name="create-or-modify-a-workflow-in-lync-server-2013"></a>在 Lync Server 2013 中建立或修改工作流程

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-02_

Lync Server 2013 支援兩種類型的工作流程：查尋群組和互動式語音回應（IVR）。 當您建立工作流程時，您會使用 [回應群組設定] 工具來指定要使用的佇列及其他設定，例如 [歡迎訊息]、[保留中的音樂]、[上班時間]，以及回應群組應用程式要求呼叫者的問題。

<div>


> [!NOTE]  
> 您必須先建立代理群組和佇列，才能建立使用它們的工作流程。 如果您想要建立可用於多個工作流程的預先定義的商務時數和假日，您也必須先定義這些小時與假日，然後才能建立使用它們的工作流程。
