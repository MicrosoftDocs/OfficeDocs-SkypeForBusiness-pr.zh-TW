# <a name="planning-for-mobile-clients-in-lync-server-2013"></a>規劃 Lync Server 2013 中的行動用戶端

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-04-02_

Lync 2013 行動用戶端可在網際網路或行動裝置連線中提供增強的目前狀態、IM、Lync 會議以及語音及視頻通話功能。 如需列出行動用戶端功能與功能的矩陣，並將它們與桌面用戶端進行比較，請參閱[Lync Server 2013 的行動用戶端比較表](lync-server-2013-mobile-client-comparison-tables.md)。

<div>


> [!NOTE]  
> Lync Server 2013 也支援 Lync 2010 行動用戶端。 如需詳細資訊，請參閱 Lync Server 2010 TechNet 文件庫中的「規劃行動客戶<A href="http://go.microsoft.com/fwlink/p/?linkid=235955">http://go.microsoft.com/fwlink/p/?LinkID=235955</A>端」。
