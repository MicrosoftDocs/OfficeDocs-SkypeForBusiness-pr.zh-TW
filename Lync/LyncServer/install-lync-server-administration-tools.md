# <a name="install-lync-server-administration-tools"></a>安裝 Lync Server 管理工具

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-02_

本主題說明如何安裝您需要用來部署和管理 Lync Server 2013 的管理工具。 您也可以在其他電腦上安裝系統管理工具，例如專用的系統管理主控台。

<div>

## <a name="to-install-the-lync-server-2013-administrative-tools"></a>若要安裝 Lync Server 2013 系統管理工具

1.  在 Lync Server 2013 前端伺服器上，執行 Lync Server 2013 設定。

2.  從 [Lync Server 2013 部署嚮導] 頁面中，選取 [**安裝管理工具**]
    
    ![Lync server 2013 部署嚮導，歡迎頁面](images/JJ205265.5f88ae18-9c3c-42ea-a91a-836ecf5d515f(OCS.15).jpg "Lync Server 2013 部署嚮導，歡迎頁面")
