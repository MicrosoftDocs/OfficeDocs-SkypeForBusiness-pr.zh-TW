# <a name="active-directory-schema-extensions-classes-and-attributes-used-by-lync-server-2013"></a>Lync Server 2013 使用的 Active Directory 結構描述擴充功能、類別及屬性

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-06-19_

本參考章節包含下列資訊：

  - Lync Server 2013 的新功能或已變更的 Active Directory 架構擴充功能
    
    Active Directory 架構包含可在 Active Directory 林中建立的每個物件類別的正式定義。 架構也包含可在 Active Directory 物件上存在的每個屬性的正式定義。 Active Directory 通用類別目錄包含樹林所有物件的複本，以及每個物件的屬性子集。 本節將說明 Lync Server 2013 中新增或變更的類別和屬性。

  - Lync Server 所使用的所有類別，以及每個類別的描述

  - Lync Server 所使用的所有屬性，以及每個屬性的描述

  - Lync Server 所使用之類別的清單，其中每個類別都可能包含

  - 全域設定和物件，以及在林準備期間建立的泛型服務和管理群組

  - 在網域準備期間，在網域根目錄和內建容器中建立的存取控制專案（Ace）

  - 由 Grant\_CsSetupPermission Cmdlet 在 Active Directory 組織單位（OU）上所做的變更。

  - 由 Grant\_CsOUPermission Cmdlet 在 ACTIVE Directory OU 上所做的變更。
