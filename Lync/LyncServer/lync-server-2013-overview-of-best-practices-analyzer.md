# <a name="overview-of-best-practices-analyzer-in-lync-server-2013"></a>Lync Server 2013 中的最佳做法分析程式概述

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-19_

您可以使用 Lync Server 2013、最佳做法分析程式來找出並解決 Lync Server 2013 部署的問題。 Lync Server 2013，最佳做法分析程式會收集 Lync Server 2013 元件的設定資訊。

有了適當的網路存取，最佳做法分析工具就能檢查執行 Active Directory 網域服務、Exchange Server 整合通訊（UM）和 Lync Server 2013 的伺服器。 您可以使用最佳做法分析程式來執行下列動作：

  - 主動執行檢查，以確認設定已根據建議的最佳做法進行設定。

  - 自動偵測 Lync Server 2013 所需的更新。

  - 產生問題的清單，例如不理想的設定、不支援的選項、缺少的更新，或我們建議的做法。

  - 協助您疑難排解及修正特定問題。

最佳做法分析程式提供下列功能：

  - 最低安裝必備元件。

  - 已報告問題的線上檔，包括疑難排解秘訣。

  - 您可以儲存以供日後查看的配置資訊。

  - 一流的系統分析。

最佳做法分析工具會使用一組 XML 設定檔來判斷要從 Lync Server 2013 環境收集的資訊。 除了檢查 Active Directory 網域服務之外，它還會檢查資源，例如 Windows Server 作業系統註冊表和 Windows 管理規範（WMI）中的設定。

最佳做法分析程式會將所收集的資料與 Lync Server 2013 部署設定和設定的一組預定義規則進行比較。

將收集的資料與預先定義的規則進行比較之後，該工具會報告問題。 針對報告的每個問題，最佳做法分析程式會提供有關在已掃描的 Lync Server 2013 環境中找到的專案，以及建議的設定的相關資訊。 最佳做法分析程式也提供特定問題的詳細資訊連結。

<div>


> [!NOTE]  
> Lync Server 2013、最佳做法分析程式只會收集 Lync Server 2013 元件的配置資訊。 您可以使用舊版的工具來掃描先前的環境。 如需詳細資訊，請參閱<A href="lync-server-2013-requirements-for-running-best-practices-analyzer.md">在 Lync Server 2013 中執行最佳做法分析程式的需求</A>。
