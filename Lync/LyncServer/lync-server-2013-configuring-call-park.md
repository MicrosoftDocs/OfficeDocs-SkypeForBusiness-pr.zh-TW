# <a name="configuring-call-park-in-lync-server-2013"></a>在 Lync Server 2013 中設定通話駐留

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

</div>

<div id="mainSection">

<span> </span>

_**主題上次修改日期：** 2012-10-30_

通話駐留可讓企業語音使用者從一部電話接聽來電，然後再撥打此通話的內部號碼 (稱為通話駐留 *軌道*) 從任何電話）。

當您部署企業語音時，會自動在前端伺服器或 Standard Edition server 上安裝及啟用呼叫駐留所使用的元件。 不過，您必須先設定通話駐留，使用者才能使用它。

本節會引導您完成通話駐留的設定。
