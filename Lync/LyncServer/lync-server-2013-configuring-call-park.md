# <a name="configuring-call-park-in-lync-server-2013"></a>在 Lync Server 2013 中設定通話駐留

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-10-30_

通話駐留可讓 Enterprise Voice 使用者從一個電話將呼叫放入保留及更新版本來撥打內部號碼 （亦即通話駐留*軌道*） 透過任何電話擷取通話。

通話駐留使用的元件會自動安裝並啟用在前端伺服器或 Standard Edition 伺服器，當您部署企業語音。 不過，您必須先設定通話駐留，再可供使用者。

本節會引導您完成的通話駐留組態。
