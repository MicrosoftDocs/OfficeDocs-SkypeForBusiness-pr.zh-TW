# <a name="configuring-an-on-premises-deployment-for-hybrid-with-lync-online"></a>設定內部部署以混合使用 Lync Online

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2014-05-28_
