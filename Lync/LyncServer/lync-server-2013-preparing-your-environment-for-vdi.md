# <a name="preparing-your-lync-server-2013-environment-for-vdi"></a>針對 VDI 準備 Lync Server 2013 環境

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-22_

若要為 Lync VDI 外掛程式準備環境，系統管理員必須執行下列步驟。

1.  在 Lync Server 2013 中，確保所有 VDI 使用者的 EnableMediaRedirection 都設定為 TRUE。 如需詳細資訊，請參閱[新版 CsClientPolicy](https://docs.microsoft.com/powershell/module/skype/New-CsClientPolicy) Cmdlet 和[CsClientPolicy](https://docs.microsoft.com/powershell/module/skype/Set-CsClientPolicy) Cmdlet 的說明主題。

2.  在資料中心電腦上，在所有虛擬機器上安裝 Lync 2013 用戶端。

3.  在本機電腦上，安裝 Lync VDI 外掛程式。
