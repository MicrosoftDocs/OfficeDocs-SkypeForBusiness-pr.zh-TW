# <a name="setting-up-the-infrastructure-for-archiving-in-lync-server-2013"></a>在 Lync Server 2013 中設定封存基礎結構

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-01_

封存的基礎結構需求與 Lync Server 部署的基礎結構需求相同（儲存區除外）。 不需要其他基礎結構設定，除非使用 Exchange 2013 儲存區、封存資料庫或兩者來設定儲存區。 如需 Lync Server 2013 基礎結構需求的詳細資訊，請參閱規劃檔中的「 [判斷您的 Lync server 2013 基礎結構需求](lync-server-2013-determining-your-infrastructure-requirements.md) 」和「在部署檔中 [準備 lync server 2013 的基礎結構和系統](lync-server-2013-preparing-the-infrastructure-and-systems.md) 」。 如需有關封存儲存需求的詳細資訊，請參閱規劃檔中的「封存 [技術2013需求](lync-server-2013-technical-requirements-for-archiving.md) 」，在部署檔中 [設定用於封存的系統平臺](lync-server-2013-setting-up-system-platforms-for-archiving.md) ，在 [lync Server 2013] 中設定儲存，以在部署檔中 [設定 lync server 2013](lync-server-2013-setting-up-storage-for-archiving.md) 中的封存。
