# <a name="setting-up-the-infrastructure-for-archiving-in-lync-server-2013"></a>針對 Lync Server 2013 中的封存設定基礎結構

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-10-01_

封存的基礎結構需求是與您的 Lync Server 部署，但不包括儲存相同。 安裝任何額外的基礎結構不是必要的但不包括設定使用 Exchange 2013 儲存體、 封存資料庫，或兩者的存放區。 如需 Lync Server 2013 的基礎結構需求的詳細資訊，請參閱[判斷您的基礎結構需求，針對 Lync Server 2013](lync-server-2013-determining-your-infrastructure-requirements.md)中規劃文件並[準備基礎結構和 Lync Server 2013 的系統](lync-server-2013-preparing-the-infrastructure-and-systems.md)中部署文件。 如需封存的儲存需求的詳細資訊，請參閱規劃文件、 部署文件中的[設定系統平台的 Lync Server 2013 中的封存](lync-server-2013-setting-up-system-platforms-for-archiving.md)和[Lync Server 2013 中的封存儲存區設定](lync-server-2013-setting-up-storage-for-archiving.md)部署 > 文件中的[Lync Server 2013 中的封存的技術需求](lync-server-2013-technical-requirements-for-archiving.md)。
