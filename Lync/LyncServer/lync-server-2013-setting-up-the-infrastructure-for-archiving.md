# <a name="setting-up-the-infrastructure-for-archiving-in-lync-server-2013"></a>在 Lync Server 2013 中設定存檔的基礎結構

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-01_

存檔的基礎結構需求與您的 Lync Server 部署（除了儲存空間之外）是一樣的。 除了使用 Exchange 2013 儲存空間、封存資料庫或兩者設定儲存空間之外，不需要額外的基礎結構設定。 如需有關 Lync Server 2013 基礎結構需求的詳細資訊，請參閱在規劃檔中[決定您的 Lync server 2013 基礎結構需求](lync-server-2013-determining-your-infrastructure-requirements.md)，以及在部署檔中[準備 lync server 2013 的基礎結構和系統](lync-server-2013-preparing-the-infrastructure-and-systems.md)。 如需有關存檔的儲存需求的詳細資料，請參閱在部署檔中的 lync server 2013 中針對存檔[2013](lync-server-2013-setting-up-system-platforms-for-archiving.md)所需的[技術需求](lync-server-2013-technical-requirements-for-archiving.md)，以及在部署檔中設定[儲存空間以進行 lync server 2013](lync-server-2013-setting-up-storage-for-archiving.md)中的歸檔。
