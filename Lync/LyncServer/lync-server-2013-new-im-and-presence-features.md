# <a name="new-im-and-presence-features-in-lync-server-2013"></a>Lync Server 2013 中的新 IM 和顯示狀態功能

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-19_

Microsoft Lync Server 2013 新增下列新的立即訊息 (IM) 和顯示狀態功能，以豐富使用者的 Lync 體驗。

  - 如果您的組織也會執行 Exchange 2013，使用者可以利用整合的連絡人存放區。 使用者可以在 Outlook 2013、Outlook Web App 和 Lync 2013 中管理其連絡人。

  - 您的使用者可以透過使用可延伸訊息和顯示狀態通訊協定（如 Google 談話）的公用 IM 提供者的使用者來交換立即訊息和目前狀態資訊，因為 Lync Server 2013 的 **XMPP 整合** 功能。 XMPP 整合內置於前端伺服器和 Edge Server 中，您可以啟用它，並將其設定為允許這項功能。
