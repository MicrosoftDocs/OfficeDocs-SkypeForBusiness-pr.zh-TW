# <a name="lync-for-iphone-and-ipad-requirements-in-lync-server-2013"></a>Lync for iPhone 和 iPad 在 Lync Server 2013 中的需求

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2015-06-01_

若要支援 iPhone 版 Microsoft Lync 2013 或 iPad 版 Microsoft Lync 2013，裝置必須符合下列需求：

  - iPhone 6 mobile 數位裝置，含 Apple iOS 8 或最新版本。

  - iPhone 5 mobile 數位裝置，含 Apple iOS 7 或最新版本。

  - iPhone 4 手機數位裝置（含 Apple iOS 7 或最新版本）。

  - iPad 2 或更新版本的行動數位裝置，含 Apple iOS 7 或最新版本。

  - iPad 迷你 mobile 數位裝置，含 Apple iOS 7 或最新版本。
