# <a name="enable-or-disable-group-call-pickup-for-users-in-lync-server-2013"></a>在 Lync Server 2013 中啟用或停用使用者的群組呼叫裝貨

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-22_

當您將群組呼叫挑選範圍新增到 [通話公園軌道] 表格中時，您可以視需要啟用和停用使用者的群組通話分揀。 本節說明如何使用 SEFAUtil 資源套件工具來啟用和停用群組通話分揀給使用者。

<div>


> [!NOTE]  
> SEFAUtil 參數（/enablegrouppickup）會啟用群組呼叫拾取並指派群組號碼。
