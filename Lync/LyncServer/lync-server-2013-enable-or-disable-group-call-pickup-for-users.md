# <a name="enable-or-disable-group-call-pickup-for-users-in-lync-server-2013"></a>啟用或停用 Lync Server 2013 中的使用者的群組來電接聽

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改主題：** 2013年-02-22_

當群組來電接聽範圍新增至通話駐留軌道表後時，您可以啟用及停用視使用者的群組來電接聽。 本節說明如何使用 SEFAUtil resource kit 工具來啟用及停用使用者的群組來電接聽。

<div>


> [!NOTE]  
> SEFAUtil 參數，/enablegrouppickup，可讓群組來電接聽，並指派群組編號。
