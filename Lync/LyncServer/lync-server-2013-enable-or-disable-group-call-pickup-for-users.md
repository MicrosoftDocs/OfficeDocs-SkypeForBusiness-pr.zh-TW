# <a name="enable-or-disable-group-call-pickup-for-users-in-lync-server-2013"></a>在 Lync Server 2013 中啟用或停用使用者的群組呼叫收取功能

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-22_

當群組呼叫收取範圍新增至通話駐留軌道表時，您可以視需要啟用或停用使用者的群組呼叫收取。 本節說明如何使用 SEFAUtil resource 工具組工具，啟用及停用使用者的群組呼叫收取。

<div>


> [!NOTE]  
> SEFAUtil 參數/enablegrouppickup 可啟用群組呼叫收取並指派群組號碼。
