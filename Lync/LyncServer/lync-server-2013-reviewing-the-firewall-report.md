# <a name="reviewing-the-firewall-report-in-lync-server-2013"></a>在 Lync Server 2013 中查看防火牆報告

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-21_

Lync Server 2013 有一組可能複雜的防火牆規則。 規劃工具會根據設計人員的輸入準則，透過產生一個定義來詳細說明所有防火牆需求的報告，來減少這項複雜性。 IT 防火牆管理員將能使用此報告來設定及定義必要的規則。

從防火牆管理的角度來看，應仔細檢查報告，以確保現有的防火牆規則沒有任何衝突，且沒有任何原則或程式可能違反。
