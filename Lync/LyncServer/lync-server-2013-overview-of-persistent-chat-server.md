# <a name="overview-of-persistent-chat-server-in-lync-server-2013"></a>Lync Server 2013 中的常設聊天室伺服器概觀

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-29_

Lync Server 2013，Persistent 聊天伺服器可讓使用者參與多方、依時間保留的、以主題為基礎的交談。 持續聊天伺服器可協助您的組織執行下列動作：

  - 改善地理位置分散且跨職能團隊之間的溝通。 透過使用持續聊天，小組可以高效地與其他人共用資訊、想法和決策。 張貼到聊天室（討論論壇）的郵件可以持續（也就是可在一段時間內提供），讓來自不同位置和部門的人員都可以參與，即使他們不在線上也一樣。 當使用者連線到聊天室時，系統會自動在聊天室中載入 backchat （可設定的聊天記錄訊息數），為使用者提供交談的內容。

  - 改善資訊認識。 使用者可以使用用戶端篩選器，在郵件內容中定義關鍵字，或在郵件中的「寄件者」欄位值中定義條件，以便在永久聊天立即訊息或聊天室訊息中滿足這些條件時接收通知。 如此一來，使用者就能隨時掌握最新感興趣的內容。

  - 改善與其延伸組織的溝通。 您可以輕鬆地與組織中的其他人員共同作業主題，以及提供共用資訊的持久位置，持續聊天協助改善溝通。

  - 減少資訊超載。 使用者可以使用用戶端篩選來追蹤聊天室和最感興趣的訊息，並可在其連絡人清單中新增他們想要追蹤的聊天室。

  - 增加重要知識與資訊的散佈。 您可以在交談中加入檔和連結，以供所有團隊存取。 透過將問題張貼到更廣泛的小組，使用者可以透過主題專家的回應獲益。 與其他資訊系統整合能讓重要的組織資料輕鬆地傳達給大型群組。

若要在 Lync Server 2013 中啟用聊天室，請部署 Lync Server 2013 持續聊天。 如需啟用聊天室的相關資訊，請參閱持續聊天說明<http://go.microsoft.com/fwlink/p/?linkid=270945>。 如果使用者已啟用 Lync Server，且已部署 Lync Server 支援，則使用者可以安裝並使用 Lync 2013 持續聊天來提供聊天室支援。

如果您的組織必須遵循合規性規範，您可以選擇部署持久的聊天合規性服務。
