# <a name="windows-update-for-lync-server-2013"></a>Lync Server 2013 的 Windows 更新

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-12-05_

經常使用 Windows Update Services 檢查並套用更新和安全性修補程式。 這樣做可協助避免其他系統元件中的漏洞，這可能會導致攻擊者能夠透過系統管理員許可權取得執行 Microsoft Lync Server 2013 之伺服器的存取權，進而危及 Lync Server 2013。

在每個 Lync Server 2013 Standard Edition server (上的後端資料庫) 和所有其他 Lync Server 2013 server (role 上，如果您已將這些資料庫升級至 SQL Server 2008 R2 Express，則 Microsoft SQL Server 2008 Express (64-bit edition) 會在每個 Lync Server Standard Edition server 上執行的更新。 您應將這些資料庫，包括前端集區之後端資料庫上的 SQL Server、監控資料庫以及封存資料庫，都視為例行安全性更新維護的一部分。

<div>

## <a name="best-practice"></a>最佳作法

  - 使用 Windows Update 維持最新狀態。
