# <a name="determining-your-system-requirements-for-lync-server-2013"></a>決定 Lync Server 2013 的系統需求

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2014-01-02_

所有執行 Lync Server 的伺服器必須符合某些最低系統需求。 Lync Server 的系統需求包括伺服器硬體、要安裝在每個伺服器上的作業系統，以及相關的軟體需求，例如 Windows 更新和必須安裝在伺服器上的其他軟體。

<div>


> [!IMPORTANT]  
> Lync Server 只適用于64位的版本，需要有64位硬體和一個64位版本的 Windows Server。 例外狀況是 Microsoft Lync Server 2013，規劃工具，可在32位版中使用。



</div>

<div>


> [!NOTE]  
> 如需 Active Directory 支援、支援的拓撲、伺服器組合及其他支援問題的詳細資訊，請參閱 <A href="lync-server-2013-supportability.md">Lync server 2013 的</A>支援。



</div>
