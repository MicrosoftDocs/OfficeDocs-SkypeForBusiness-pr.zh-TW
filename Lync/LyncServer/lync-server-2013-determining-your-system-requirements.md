# <a name="determining-your-system-requirements-for-lync-server-2013"></a>判定 Lync Server 2013 的系統需求

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2014-01-02_

所有執行 Lync Server 的伺服器都必須符合特定的最低系統需求。 Lync 伺服器的系統需求包括伺服器硬體、要在每個伺服器上安裝的作業系統，以及相關的軟體需求，例如 Windows 更新及必須安裝在伺服器上的其他軟體。

<div>


> [!IMPORTANT]  
> Lync Server 只能在64位版本中使用，這需要64位硬體和64位版本的 Windows Server。 例外情況是 Microsoft Lync Server 2013、規劃工具（在32位版本中提供）。



</div>

<div>


> [!NOTE]  
> 如需有關 Active Directory 支援、支援的拓撲、伺服器 collocation 及其他支援問題的詳細資訊，請參閱<A href="lync-server-2013-supportability.md">Lync server 2013 的支援服務</A>。
