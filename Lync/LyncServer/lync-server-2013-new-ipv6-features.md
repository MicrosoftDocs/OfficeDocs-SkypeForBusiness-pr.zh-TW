# <a name="new-ipv6-features-in-lync-server-2013"></a>Lync Server 2013 中的新 IPv6 功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-08-16_

Lync Server 2013 包括對 IPv6 位址的支援。 由於需要 IP 位址的裝置數量不斷增加，因此可用的 IPv4 位址數量（即32位位址）就會用盡。IPv6 提供大量的可用位址，因為它會使用128位位址。 在您的環境中使用 IPv6 位址的方式，取決於您所使用的裝置中的 IPv6 支援。

因為許多現有的裝置還不支援 IPv6，所以從 IPv4 到 IPv6 的完整轉換可能需要幾年的時間。 因此，Lync Server 2013 支援僅包含 IPv4 位址、僅限 IPv6 位址和雙堆疊（IPv4 與 IPv6）位址的網路環境支援。
