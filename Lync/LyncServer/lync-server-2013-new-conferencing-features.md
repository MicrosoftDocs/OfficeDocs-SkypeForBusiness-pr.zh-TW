# <a name="new-conferencing-features-in-lync-server-2013"></a>Lync Server 2013 中的新會議功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-11-08_

Lync Server 2013 引進了數種新功能來加強會議，如下列清單所述。

  - **加入啟動器**
    
    Lync Server 2013 更新聯接啟動器，以在啟動用戶端前驗證每個會議，並提供在下列用戶端中開啟會議的支援：
    
      - Windows Phone 7
    
      - Android 裝置
    
      - Apple iOS 裝置
    
      - Windows 8
    
      - Internet Explorer 10

  - **已更新的 PowerPoint 共用**
    
    Lync Server 2013 現在使用 Office Web Apps 和 Office Web Apps 伺服器（以前稱為 WAC 伺服器）來處理 PowerPoint 簡報。 使用 Office Web Apps Server 可提供更高解析度的顯示效果，以及更好的 PowerPoint 功能支援（Lync Server 2013 使用標準 DHTML 和 JavaScript 來廣播 PowerPoint 簡報），以及具備適當許可權的使用者能夠獨立于簡報本身來流覽 PowerPoint 簡報。

  - **畫廊視圖和 HD 視訊會議**
    
    在 [視訊會議] 中，使用者可以同時查看最多五個會議參與者的影片。
    
    <div>
    

    > [!NOTE]  
    > 最多75個參與者的會議中出現圖庫視圖。 當會議的參與者超過75時，體驗會還原為單一視圖。

    
    </div>

  - **HD 影片**
    
    使用者可以在雙方通話和多方會議中體驗最多 HD 1080P 的解析度。

  - **僅限簡報者影片模式**
    
    簡報者可以設定會議，只顯示簡報者的影片。 如果有多個視頻資料流程可供使用，且鎖定至不同的來源，此模式可避免大型會議中的干擾。 此模式也適用于透過會議裝置捕獲並提供的影片。

  - **影片焦點**
    
    簡報者可以設定會議，讓會議中的每個人都看得到來自所選參與者的影片。 此模式也適用于透過全景影片的會議裝置捕獲並提供的影片。

  - **非企業語音使用者的撥出式會議**
    
    Lync Server 2013 現在允許未啟用企業語音的參與者從會議會議啟動撥出通話。 此功能可由系統管理員進行設定。

  - **封存**
    
    如果 Exchange Server 整合已啟用封存功能，則在會議期間共用的任何檔都會存檔到 Exchange 2013 資料儲存區中。 這包括 PowerPoint 簡報、附件、白板和投票。

  - **自訂會議邀請**
    
    系統管理員可以使用 Lync Server [控制台] 或 Lync Server 管理命令介面來自訂線上會議的電子郵件邀請。 自訂專案可以包含標誌的 Url、解說文字、法律文字及頁尾文字。 所有後續邀請將包含自訂專案。
