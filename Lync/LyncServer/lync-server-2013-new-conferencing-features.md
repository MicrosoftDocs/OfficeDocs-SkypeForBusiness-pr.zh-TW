    <span data-ttu-id="0a144-107">Lync Server 2013 更新聯接啟動器，以在啟動用戶端之前驗證每個會議，並提供支援，以在下列用戶端中開啟會議：</span><span class="sxs-lookup"><span data-stu-id="0a144-107">Lync Server 2013 updates the Join launcher to validate each meeting before launching a client, and to provide support for opening a meeting in the following clients:</span></span>

  - <span data-ttu-id="0a144-113">**更新 PowerPoint 共用**</span><span class="sxs-lookup"><span data-stu-id="0a144-113">**Updated PowerPoint Sharing**</span></span>

  - <span data-ttu-id="0a144-116">**圖庫 View and HD Video 會議**</span><span class="sxs-lookup"><span data-stu-id="0a144-116">**Gallery View and HD Video Conferencing**</span></span>

  - <span data-ttu-id="0a144-126">**影片聚光燈**</span><span class="sxs-lookup"><span data-stu-id="0a144-126">**Video Spotlight**</span></span>
