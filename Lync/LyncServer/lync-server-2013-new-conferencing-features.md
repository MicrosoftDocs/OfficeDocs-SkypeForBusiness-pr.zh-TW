# <a name="new-conferencing-features-in-lync-server-2013"></a>Lync Server 2013 中的新會議功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-11-08_

Lync Server 2013 引進數項新功能可增強會議，如下列清單所述。

  - **Join Launcher**
    
    Lync Server 2013 更新 Join launcher，以驗證之前啟動用戶端，每個會議，並提供支援下列用戶端中開啟會議：
    
      - Windows Phone 7
    
      - Android 裝置
    
      - Apple iOS 裝置
    
      - Windows 8
    
      - Internet Explorer 10

  - **更新的 PowerPoint 共用**
    
    Lync Server 2013 現在會使用 Office Web Apps 與 Office Web Apps Server （以前稱為 WAC 伺服器） 來處理 PowerPoint 簡報。 使用 Office Web Apps Server 允許更高解析度會顯示與更佳的 PowerPoint 功能支援存取至多個類型的行動裝置 (Lync Server 2013 使用標準 DHTML 和 JavaScript 廣播 PowerPoint 簡報），以及能夠以適當的權限的使用者捲動獨立簡報本身的 PowerPoint 簡報。

  - **圖庫檢視與 HD 視訊會議**
    
    在視訊會議中，使用者可以同時看到最多五位會議參與者的視訊。
    
    <div>
    

    > [!NOTE]  
    > 圖庫檢視被發生在具有多達 75 位參與者的會議。 當會議取得超過 75 位參與者時，經驗會還原成單一檢視。

    
    </div>

  - **HD 視訊**
    
    使用者可以在雙方通話和多方會議中體驗高達 HD 1080P 的解析度。

  - **簡報者只視訊模式**
    
    簡報者可以設定會議，以便只有簡報者從影片會顯示。 當多個視訊資料流，而可鎖定不同的來源，此模式可防止在大型會議期間將干擾。 這種模式也適用於擷取和會議裝置所提供的視訊。

  - **視訊焦點**
    
    簡報者可以設定會議，以便從視訊來源所選參與者的視訊會出現在會議中的每個使用者皆。 這種模式也適用於擷取和提供者的全景視訊會議裝置的影片。

  - **非 Enterprise Voice 使用者的撥出式會議**
    
    Lync Server 2013 現在可讓未啟用，以啟動從會議撥出電話的 Enterprise Voice 的參與者。 這項功能是由系統管理員可設定。

  - **封存**
    
    如果 Exchange Server 整合啟用封存與任何會議期間共用的文件封存到 Exchange 2013 資料儲存區。 這包括 PowerPoint 簡報、 附件、 白板與投票。

  - **自訂會議邀請**
    
    系統管理員可以自訂使用 Lync Server Control Panel] 或 [Lync Server 管理命令介面的線上會議的電子郵件邀請。 自訂可以包括標誌的 Url，協助文字、 法律、 頁尾文字。 所有後續邀請將包含自訂項目。
