# <a name="new-conferencing-features-in-lync-server-2013"></a>Lync Server 2013 中的新會議功能

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-11-08_

Lync Server 2013 引進數項新功能，可強化會議，如下列清單所述。

  - **加入啟動器**
    
    Lync Server 2013 更新聯接啟動器，以在啟動用戶端之前驗證每個會議，並提供支援，以在下列用戶端中開啟會議：
    
      - Windows Phone 7
    
      - Android 裝置
    
      - Apple iOS 裝置
    
      - Windows 8
    
      - Internet Explorer 10

  - **更新 PowerPoint 共用**
    
    Lync Server 2013 現在使用 Office Web Apps 與 Office Web Apps Server (以前稱為 WAC Server) 來處理 PowerPoint 簡報。 Office Web Apps Server 的使用可讓高解析度顯示及更好的 PowerPoint 功能的支援、存取更多類型的行動裝置 (Lync Server 2013 使用 standard DHTML 和 JavaScript 來廣播 PowerPoint 簡報) ，以及具有適當許可權的使用者能夠在獨立于簡報的情況下流覽 PowerPoint 簡報。

  - **圖庫 View and HD Video 會議**
    
    在 [影片會議] 中，使用者可以同時查看五個會議參與者的影片。
    
    <div>
    

    > [!NOTE]  
    > 在最多有75位參與者的會議中，圖庫視圖已有經驗。 當會議超出75參與者時，體驗會回復為單一模式。

    
    </div>

  - **HD 影片**
    
    使用者可以在雙方通話和多方會議中體驗高達 HD 1080P 的解析度。

  - **僅限簡報者影片模式**
    
    簡報者可以設定會議，只顯示簡報者的影片。 當多個影片可供使用，並且鎖定于不同來源時，此模式可防止大型會議中的干擾。 這種模式也適用于會議裝置所捕獲並提供的影片。

  - **影片聚光燈**
    
    簡報者可以設定會議，使會議中的每一位使用者都看得到來自選取之參加影片來源之參與者的影片。 這種模式也適用于透過全景影片的會議裝置所捕獲並提供的影片。

  - **非企業語音使用者的撥出式會議**
    
    Lync Server 2013 現在允許未啟用 Enterprise Voice 的參與者從會議會議中啟動撥出通話。 這項功能可由系統管理員設定。

  - **封存**
    
    如果 Exchange Server 整合已啟用封存，則在會議期間共用的任何檔都會封存成 Exchange 2013 資料儲存區。 這包括 PowerPoint 簡報、附件、白板及投票。

  - **自訂會議邀請**
    
    管理員可以使用 Lync Server 控制台或 Lync Server 管理命令介面，自訂線上會議的電子郵件邀請。 自訂專案可以包含徽標、協助文字、法律文字和頁尾文字的 URLs。 所有後續邀請會包含自訂專案。
