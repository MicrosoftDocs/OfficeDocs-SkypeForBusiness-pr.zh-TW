  - <span data-ttu-id="a1d8a-112">**已更新的 PowerPoint 共用**</span><span class="sxs-lookup"><span data-stu-id="a1d8a-112">**Updated PowerPoint Sharing**</span></span>
    
    <span data-ttu-id="a1d8a-113">Lync Server 2013 現在使用 Office Web Apps 和 Office Web Apps 伺服器（以前稱為 WAC 伺服器）來處理 PowerPoint 簡報。</span><span class="sxs-lookup"><span data-stu-id="a1d8a-113">Lync Server 2013 now uses Office Web Apps and the Office Web Apps Server (formerly known as WAC Server) to handle PowerPoint presentations.</span></span> <span data-ttu-id="a1d8a-114">使用 Office Web Apps Server 可提供更高解析度的顯示效果，以及更好的 PowerPoint 功能支援（Lync Server 2013 使用標準 DHTML 和 JavaScript 來廣播 PowerPoint 簡報），以及具備適當許可權的使用者能夠獨立于簡報本身來流覽 PowerPoint 簡報。</span><span class="sxs-lookup"><span data-stu-id="a1d8a-114">The use of Office Web Apps Server allows for higher-resolution displays and better support for PowerPoint capabilities, access to more types of mobile devices (Lync Server 2013 uses standard DHTML and JavaScript to broadcast PowerPoint presentations), and the ability for users with the appropriate privileges to scroll through a PowerPoint presentation independent of the presentation itself.</span></span>

  - <span data-ttu-id="a1d8a-115">**畫廊視圖和 HD 視訊會議**</span><span class="sxs-lookup"><span data-stu-id="a1d8a-115">**Gallery View and HD Video Conferencing**</span></span>

  - <span data-ttu-id="a1d8a-125">**影片焦點**</span><span class="sxs-lookup"><span data-stu-id="a1d8a-125">**Video Spotlight**</span></span>
    
    <span data-ttu-id="a1d8a-126">簡報者可以設定會議，讓會議中的每個人都看得到來自所選參與者的影片。</span><span class="sxs-lookup"><span data-stu-id="a1d8a-126">Presenters can configure the conference so that only the video from a selected participant who is a video source is seen by everyone in the conference.</span></span> <span data-ttu-id="a1d8a-127">此模式也適用于透過全景影片的會議裝置捕獲並提供的影片。</span><span class="sxs-lookup"><span data-stu-id="a1d8a-127">This mode also applies to video captured and provided by conferencing devices for panoramic video.</span></span>

  - <span data-ttu-id="a1d8a-128">**非企業語音使用者的撥出式會議**</span><span class="sxs-lookup"><span data-stu-id="a1d8a-128">**Dial-out Conferencing for non-Enterprise Voice users**</span></span>
    
    <span data-ttu-id="a1d8a-129">Lync Server 2013 現在允許未啟用企業語音的參與者從會議會議啟動撥出通話。</span><span class="sxs-lookup"><span data-stu-id="a1d8a-129">Lync Server 2013 now allows participants that are not Enterprise Voice enabled to initiate dial-out calls from a meeting conference.</span></span> <span data-ttu-id="a1d8a-130">此功能可由系統管理員進行設定。</span><span class="sxs-lookup"><span data-stu-id="a1d8a-130">This feature is configurable by the administrator.</span></span>
