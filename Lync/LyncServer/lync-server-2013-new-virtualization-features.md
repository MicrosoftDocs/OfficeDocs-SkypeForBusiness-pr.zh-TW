# <a name="new-virtualization-features-in-lync-server-2013"></a>Lync Server 2013 中的新虛擬化功能

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-11-07_

Lync Server 2013 支援在 Windows Server 2012、Windows Server 2012 R2 和 Windows Server 2008 R2 上進行虛擬化。 在 Windows Server 2012 和 Windows Server 2012 R2 上的支援包括對單一根 I/O 虛擬化 (SR-IOV) 功能的支援。 使用 SR-IOV 時，實體網路介面卡的虛擬功能會直接指派給虛擬機器。 這會增加網路輸送量並減少網路延遲，同時減少處理網路流量所需的主機 CPU 負荷。 若要利用 SR-IOV，您必須使用具有支援 SR-IOV 之 BIOS 的主機伺服器，以及使用支援 SR-IOV 的網路介面卡。
