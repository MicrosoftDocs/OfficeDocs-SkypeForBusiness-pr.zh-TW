# <a name="enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions-in-lync-server-2013"></a>讓使用者管理類別、 聊天室和使用者權限和 Lync Server 2013 中的權限

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-11-01_

**CsPersistentChatAdministrator**角色 （常設聊天室系統管理員） 的成員可以授與聊天室管理員權利和權限給其他使用者及其本身。

常設聊天室系統管理員可以執行下列動作：

  - 建立聊天室類別和聊天室。

  - 設定所有類別和聊天室的成員資格。

  - 管理所有類別和聊天室的所有設定。

  - 啟用或停用原則 for Persistent Chat Server。

  - 設定及管理 Persistent Chat Server 集區上的組態設定。

如需詳細資訊，請參閱部署文件中[新增 Lync Server 2013 常設聊天室管理員](lync-server-2013-adding-a-persistent-chat-administrator.md)。
