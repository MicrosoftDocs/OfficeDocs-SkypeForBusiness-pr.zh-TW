# <a name="enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions-in-lync-server-2013"></a>在 Lync Server 2013 中讓使用者管理類別、聊天室及使用者權限

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-11-01_

**CsPersistentChatAdministrator**角色的成員（持續聊天系統管理員）可將聊天室管理員權利和許可權授與其他使用者及他們自己。

持續聊天管理員可以執行下列動作：

  - 建立會議室類別和聊天室。

  - 設定所有類別和聊天室的成員資格。

  - 管理所有類別和聊天室的所有設定。

  - 啟用或停用持久聊天伺服器的原則。

  - 在持久性聊天伺服器池中設定及管理設定。

如需詳細資訊，請參閱在部署檔中[新增 Lync Server 2013 中的持續聊天系統管理員](lync-server-2013-adding-a-persistent-chat-administrator.md)。
