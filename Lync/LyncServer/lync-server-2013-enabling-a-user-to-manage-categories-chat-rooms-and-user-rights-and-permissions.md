# <a name="enabling-a-user-to-manage-categories-chat-rooms-and-user-rights-and-permissions-in-lync-server-2013"></a>讓使用者在 Lync Server 2013 中管理類別、聊天室及使用者權限

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-11-01_

**CsPersistentChatAdministrator**角色的成員 (Persistent Chat administrators) 可將聊天室管理員權力和許可權授與其他使用者及自身。

Persistent Chat 系統管理員可以執行下列作業：

  - 建立聊天室類別和聊天室。

  - 設定所有類別和聊天室的成員資格。

  - 管理所有類別和聊天室的所有設定。

  - 啟用或停用持久聊天伺服器的原則。

  - 設定和管理 Persistent Chat Server 集區上的設定。

如需詳細資訊，請參閱部署檔中的在 [Lync Server 2013 中新增 Persistent Chat administrator](lync-server-2013-adding-a-persistent-chat-administrator.md) 。
