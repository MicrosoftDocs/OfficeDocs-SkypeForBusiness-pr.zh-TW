# <a name="web-server-and-services-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的 Web 服務器和服務 Cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-18_

許多 Microsoft Lync Server 2013 元件都是以 web 為基礎：這些元件可以使用 Web 服務或網頁來執行他們的工作。 [Web 服務器] 和 [Web 服務] Cmdlet 可讓您執行設定 Web 服務器設定和管理簡單 Url 等操作。 簡單的 Url 可讓使用者更容易加入會議與會議，並讓系統管理員能更輕鬆地登入 Lync Server 2013 [控制台]。
