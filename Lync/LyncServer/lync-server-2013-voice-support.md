# <a name="voice-support-in-lync-server-2013"></a>Lync Server 2013 中的語音支援

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-06-29_

如果您的部署包含前端集區，您可以部署 Enterprise Voice、 Voice over IP (VoIP) 解決方案 Microsoft 所提供的支援。 Voice over IP (VoIP) 是傳統的 PBX 電話語音的軟體型替代方式。 VoIP 通話體驗類似的傳統電話語音經驗，但 Enterprise Voice 會包含可讓更豐富的通訊和共同作業功能。 例如，Enterprise Voice 部署可以設定為讓 Lync 2013 和 Lync Phone Edition 的使用者，以檢視貴組織的通訊錄中的增強顯示狀態資訊或連絡人的位置資訊。 透過整合與其他 Lync Server 2013 工作負載和與 Exchange 整合通訊 (UM) 啟用某些 Lync Server 2013 功能。 如需功能隨附於 Enterprise Voice，以及如何規劃部署的詳細資訊，請參閱規劃文件中的[Planning for Lync Server 2013 中的企業語音](lync-server-2013-planning-for-enterprise-voice.md)。
