# <a name="voice-support-in-lync-server-2013"></a>Lync Server 2013 中的語音支援

_**主題上次修改日期：** 2012-06-29_

如果您的部署包含前端集區，您可以在 Microsoft 所提供的) 方案中，部署 Enterprise Voice、Voice over IP (VoIP 方案的支援。 Voice over IP (VoIP) 是傳統 PBX 型電話軟體的軟體替代方案。 雖然 VoIP 的呼叫經驗類似傳統的電話語音體驗，但 Enterprise Voice 包含的功能可讓您進行更豐富的通訊和協同作業。 例如，您可以設定您的企業語音部署，讓 Lync 2013 和 Lync Phone Edition 使用者可以查看組織通訊錄中連絡人的增強顯示狀態資訊或位置資訊。 某些 Lync Server 2013 功能可透過與其他 Lync Server 2013 工作負載搭配使用，以及使用 Exchange 整合通訊 (UM) 來啟用。 如需有關 Enterprise Voice 可用之功能及功能的詳細資訊，以及如何規劃部署，請參閱規劃檔中的在 [Lync Server 2013 中規劃 Enterprise Voice](lync-server-2013-planning-for-enterprise-voice.md) 。
