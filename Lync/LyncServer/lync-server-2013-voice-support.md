# <a name="voice-support-in-lync-server-2013"></a>Lync Server 2013 中的語音支援

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-06-29_

如果您的部署包含 [前端] 池，您可以部署 Microsoft 所提供的企業語音支援（VoIP）方案。 [語音 over IP （VoIP）] 是傳統 PBX 電話的軟體替代方案。 雖然 VoIP 通話體驗與傳統的電話經驗類似，但企業語音也包含可讓您更豐富的通訊與共同作業的功能。 例如，您可以將企業語音部署設定為允許 Lync 2013 和 Lync Phone Edition 使用者查看貴組織通訊錄中的連絡人的目前狀態資訊或位置資訊。 有些 Lync Server 2013 功能是透過與其他 Lync Server 2013 工作負載以及 Exchange 整合通訊（UM）來啟用。 如需企業語音及如何規劃部署的功能和功能的詳細資訊，請參閱規劃檔中的[Lync Server 2013 規劃企業語音](lync-server-2013-planning-for-enterprise-voice.md)。
