# <a name="supported-server-collocation-for-edge-components-in-lync-server-2013"></a>適用於 Lync Server 2013 中的 edge 元件支援的伺服器共同配置

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-09-08_

Access Edge 服務、 Web Conferencing Edge service 為 A / V Edge service 和 XMPP Proxy 服務組合在 Edge Server 上。 下列伺服器提供外部使用者存取所需的各項功能，而且必須部署在專用伺服器：

  - Edge Server

  - Director (選擇性)

  - 反向 Proxy

<div>


> [!IMPORTANT]  
> 反向 proxy 不需要是專門提供 Lync Server 2013。 例如，您可以提供服務給發佈 Lync 伺服器 Web 服務，並同時提供另一個網站，具有不具有 Lync 伺服器上所有已發佈的網站。 如果您已在周邊網路，以支援其他服務中的反向 proxy 伺服器，您可以將它用於 Lync Server 2013。
