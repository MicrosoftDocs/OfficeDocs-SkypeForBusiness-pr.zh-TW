# <a name="customizing-client-installation-in-lync-server-2013"></a>在 Lync Server 2013 中自訂用戶端安裝

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-03_

企業系統管理員可以使用本節中討論的方法來自訂 Office 2013 Windows 安裝程式（.msi）安裝。 因為沒有任何單一工具提供所有的自訂選項，所以您可以在 Lync 2013 部署中結合使用這些方法。 您最少有可能使用下列各節所述的工具：

  - [使用 Lync Server 2013 中的 Office 自訂工具（OCT）](lync-server-2013-using-the-office-customization-tool-oct.md) ，來自訂 Lync 及其他 Office 程式的安裝選項和功能。

  - [使用 config.xml 在 Lync Server 2013 中執行安裝](lync-server-2013-using-config-xml-to-perform-installation-tasks.md)工作，以指定網路安裝點的路徑，並執行緘默安裝。

  - [使用 Lync Server 2013 中的 Setup 命令列選項](lync-server-2013-using-setup-command-line-options.md)，指定要在安裝期間使用的 config.xml 檔案。

  - 使用 [群組原則物件編輯器] MMC 管理單元，[在 Lync Server 2013 中設定用戶端引導原則](lync-server-2013-configuring-client-bootstrapping-policies.md)。

在您部署 Office 產品套件時，您可能會想要設定其他的選項。 本節中的主題提供這些自訂工具的概覽，並討論 Lync 的特定考慮。 [隨附] 是每個工具的詳細 Office 說明的連結。
