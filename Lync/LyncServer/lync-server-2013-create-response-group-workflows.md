# <a name="create-response-group-workflows-in-lync-server-2013"></a>在 Lync Server 2013 中建立回應群組工作流程

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-12_

工作流程會定義從電話鈴聲開始響起到有人接聽該通電話這段時間的呼叫行為。工作流程指定了用於保留通話的佇列，並指定用於群組搜尋的路由方法，或讓互動回應群組使用的問題和回答。工作流程也定義歡迎訊息、等候音樂、上班時間和假日之類的設定值。

您可以使用回應群組設定工具來建立工作流程。 您可以從 Lync Server 控制台的 [回應群組] 頁面，存取回應群組設定工具。

<div>


> [!NOTE]  
> 您必須先建立代理群組和佇列，再建立使用這些項目的工作流程。
