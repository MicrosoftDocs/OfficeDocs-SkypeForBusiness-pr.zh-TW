# <a name="provisioning-guide-for-lync-skype-connectivity-in-lync-server-2013"></a>適用於 Lync Server 2013 中的 Lync Skype 連線佈建指南

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改主題：** 2014年-11-26_

Lync Server 2013 支援與 Skype 進行連線。 此連線可以讓您使用 Skype 使用者的 Microsoft 帳戶 (MSA) 新增 Skype 連絡人的 Lync 2013 使用者。 Skype 用戶端也可以將 Lync 使用者新增至其連絡人清單。 根據行政規定必須在 Lync Server 中設定的原則、 Lync 與 Skype 使用者能夠使用立即訊息通訊，請參閱彼此的目前狀態及啟動音訊和視訊通話。 Lync Skype 連線也是 Lync Online 的功能，並可啟用從 Office 365 入口網站內 Lync 系統管理中心的 Lync Online 客戶。

<div>

> [!IMPORTANT]  
> Lync Server 已設為使用公用立即訊息連線 (PIC) 連線與 Windows Messenger，如果您的部署已設定 Lync Skype 連線。 您可能要考慮的唯一變更是重新命名為 Skype 您現有 Messenger PIC 項目。 如需詳細資訊，請參閱本指南稍後設定 lync Skype PIC 提供者設定。
