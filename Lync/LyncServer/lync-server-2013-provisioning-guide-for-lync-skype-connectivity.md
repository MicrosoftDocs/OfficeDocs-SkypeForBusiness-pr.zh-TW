# <a name="provisioning-guide-for-lync-skype-connectivity-in-lync-server-2013"></a>Lync Server 2013 中的 Lync-Skype 連線布配指南

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2014-11-26_

Lync Server 2013 支援與 Skype 的連線能力。 這種連線功能可讓您的 Lync 2013 使用者使用 Skype 使用者的 Microsoft 帳戶 (MSA) 新增 Skype 連絡人。 Skype 用戶端也可以將 Lync 使用者新增至其連絡人清單。 根據在 Lync Server 中以管理方式設定的原則，Lync 和 Skype 使用者將能夠使用立即訊息進行通訊、查看彼此的狀態，以及啟動音訊和影片通話。 Lync-Skype 連線也是 Lync Online 的一項功能，可在 Microsoft 365 系統管理中心內的 Lync 系統管理中心啟用 Lync Online 客戶。

<div>

> [!IMPORTANT]  
> 如果 Lync Server 已設定為使用公用立即訊息連線功能與 Windows Messenger 進行連線， (PIC) ，您的部署已設定為 Lync-Skype 連線性。 您可能想要考慮的唯一變更是將現有的 Messenger PIC 專案重新命名為 Skype。 如需詳細資訊，請參閱本指南稍後的設定 Lync 的 Skype PIC 提供者設定。

</div>
