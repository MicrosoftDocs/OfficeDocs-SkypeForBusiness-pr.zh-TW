# <a name="user-migration"></a>使用者移轉

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-19_

通常已接受的遷移最佳做法是建立數個測試使用者，並使用它們來執行系統測試。 當您成功地移動並測試這些帳戶之後，您應該找出一組試驗式生產使用者，並在其上移動帳戶並進行驗證測試。 當您取得滿意的結果時，您可以將其餘的使用者移至新的部署。

如需有關啟用 Lync Server 2013 使用者的其他資訊，請參閱在部署 Lync Server 2013 檔中[停用或重新啟用 Lync server 2013 的使用者帳戶](lync-server-2013-disable-or-re-enable-user-account-for-lync-server.md)一主題。
