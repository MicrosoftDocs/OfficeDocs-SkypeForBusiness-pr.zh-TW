# <a name="remove-a-legacy-authorized-host-in-lync-server-2013-optional"></a>在 [Lync Server 2013 （選用） 移除舊版授權的主機

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-09-05_

當您移轉至 Lync Server 部署的遠端呼叫控制時，您必須移除舊版授權的主機項目 （又稱為*受信任的應用程式項目*在 Lync Server 中），任何 SIP/CSTA 閘道舊版部署中。 您必須使用系統管理工具，從這些部署中分別移除授權主機項目。
