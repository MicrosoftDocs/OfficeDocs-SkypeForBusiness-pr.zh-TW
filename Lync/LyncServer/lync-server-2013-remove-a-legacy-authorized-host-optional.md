# <a name="remove-a-legacy-authorized-host-in-lync-server-2013-optional"></a>在 Lync Server 2013 中移除舊版授權主機 (選用) 

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-05_

當您將遠端呼叫控制遷移至 Lync Server 部署時，您必須在舊版部署中移除舊版授權主項目目 (稱為「Lync Server) 中的 *信任應用程式專案* 」，以取得任何 SIP/CSTA 的閘道。 您必須使用系統管理工具，從這些部署中分別移除授權主機項目。
