# <a name="remove-a-legacy-authorized-host-in-lync-server-2013-optional"></a>在 Lync Server 2013 中移除舊版授權主機 (選用)

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-05_

當您將遠端呼叫控制轉移到 Lync Server 部署時，您必須在舊版部署中，針對任何 SIP/CSTA 閘道，移除舊版授權主項目目（在 Lync Server 中稱為*受信任的應用程式專案*）。 您必須使用 [管理工具]，分別從這些部署中移除已授權的主項目目。
