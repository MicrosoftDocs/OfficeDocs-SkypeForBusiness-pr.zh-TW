# <a name="managing-response-group-workflows-in-lync-server-2013"></a>在 Lync Server 2013 中管理回應群組工作流程

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-01_

回應群組工作流程會定義電話在撥打電話到工程師接聽通話時間的時間的行為。 工作流程包含佇列和路由資訊，並包含 [查尋] 群組或 [互動式語音回應（IVR）] 的資訊。

本節中的主題可識別設計 IVR 工作流程的最佳做法，並說明如何建立或修改工作流程，以及如何刪除工作組。
