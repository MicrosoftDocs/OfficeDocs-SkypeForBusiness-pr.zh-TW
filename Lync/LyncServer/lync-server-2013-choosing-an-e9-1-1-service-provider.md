# <a name="choosing-an-e9-1-1-service-provider-for-lync-server-2013"></a>選擇 Lync Server 2013 的 E9-1 服務提供者

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-06-06_

E9-1-1 服務提供者會根據呼叫中所包含的位置資訊，將源自 Lync Server 的緊急呼叫路由到正確的公用安全應答點（PSAP）。

若要支援 E9-1 作為 Lync Server 部署的一部分，您必須從 Lync 開啟互通性計畫合格 E9-1-1 服務提供者取得 E9-1-1 路由服務。 選擇最符合貴組織需求的提供者。
