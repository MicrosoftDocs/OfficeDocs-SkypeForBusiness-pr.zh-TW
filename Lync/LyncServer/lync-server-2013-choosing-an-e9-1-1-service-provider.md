</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-06-06_

E9-1-1 服務提供者的路由緊急通話從 Lync Server 以正確公用安全回應點 (PSAP) 依據通話中所含的位置資訊。
