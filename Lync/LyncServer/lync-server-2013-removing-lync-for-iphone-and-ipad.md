# <a name="removing-lync-for-iphone-and-ipad-in-lync-server-2013"></a>在 Lync Server 2013 中移除 iPhone 和 iPad 版 Lync

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-19_

若要從裝置移除 iPhone 版 Lync 2013 或 iPad 版 Microsoft Lync 2010，請執行下列步驟：

1.  在行動裝置的主畫面上，輕觸並按住 Lync 圖示。

2.  當磚開始搖動並出現**x**時，請輕觸**x**以刪除應用程式。
