# <a name="setting-up-the-director-in-lync-server-2013"></a>在 Lync Server 2013 中設定 Director

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2014-05-05_

如果您要透過部署邊緣伺服器來啟用外部使用者存取權，其中一個選項就是部署控制器。 控制器是執行 Microsoft Lync Server 2013 的伺服器，可驗證使用者要求，但不會家用任何使用者帳戶。 現在這不是必要的做法，但如果您擔心效能並想要協助簡化驗證要求，這會很有説明。 如果您認為這對於您的組織而言是一個不錯的想法，設定主管或主管池的步驟與設定企業版前端池或標準版伺服器類似。 在拓撲建立器中定義主管之後，您必須執行本節中的步驟。
