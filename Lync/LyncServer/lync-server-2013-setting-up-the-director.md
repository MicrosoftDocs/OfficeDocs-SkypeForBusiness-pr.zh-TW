# <a name="setting-up-the-director-in-lync-server-2013"></a>在 Lync Server 2013 中設定 Director

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

_**主題上次修改日期：** 2014-05-05_

如果您是透過部署 Edge Server 來啟用外部使用者的存取，則有一個選項是部署 Director。 Director 是執行 Microsoft Lync Server 2013 的伺服器，用來驗證使用者要求，但不會家用任何使用者帳戶。 現在這不是必要條件，但如果您擔心效能並想要協助簡化驗證要求，它會非常有用。 如果您認為這對您的組織來說很不錯，設定 Director 或 Director 集區的步驟，與設定 Enterprise Edition 前端集區或 Standard Edition server 類似。 在拓撲產生器中定義 Director (s) 之後，您必須執行本節中的步驟。
