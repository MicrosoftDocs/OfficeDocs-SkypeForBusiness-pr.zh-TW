# <a name="setting-up-the-director-in-lync-server-2013"></a>設定 Lync Server 2013 中 Director

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改主題：** 2014年-05-05_

如果您正在部署 Edge Server 啟用外部使用者的存取，其中一個選項是部署 Director。 Director 是執行 Microsoft Lync Server 2013 的驗證使用者要求，但不會首頁的任何使用者帳戶的伺服器。 現在，這不是必要項，但是如果您擔心效能，並且想要協助簡化驗證要求，它是很有幫助。 如果您決定這是您的組織很好的作法，Director 或 Director 集區設定的步驟會類似於設定 Enterprise Edition 前端集區或 Standard Edition server。 您已定義您的 director 拓撲產生器中之後，您需要執行本節中的步驟。
