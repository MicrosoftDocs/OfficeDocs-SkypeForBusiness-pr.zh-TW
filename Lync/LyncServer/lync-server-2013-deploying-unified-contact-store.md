# <a name="deploying-unified-contact-store-in-lync-server-2013"></a>在 Lync Server 2013 中部署整合連絡人存放區

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2016-06-06_

在 Lync Server 2013 中啟用整合連絡人存放區時，不需要任何拓撲設定。 啟用使用者的整合連絡人存放區時，需要執行下列項目：

  - 啟用整合連絡人存放區原則 (預設為啟用)。

  - 使用者使用 Lync 2013 至少登入一次。

在使用者的連絡人已遷移之後，當使用者登入 Lync 2013 時，使用者就可以從 Lync 2013、Outlook 2013 或 Outlook Web Access 存取及管理 Lync 連絡人。 使用者不必登入 Lync 以從 Outlook 或 Outlook Web Access 管理其連絡人。

<div>


> [!IMPORTANT]  
> 如果使用者在遷移之後從 Lync 2010 登入，則連絡人和群組可供使用且是最新的，但是使用者無法管理 (，例如新增、刪除、移動、標記、untag 或修改) 那些連絡人。
