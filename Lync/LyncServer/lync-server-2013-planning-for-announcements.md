# <a name="planning-for-announcements-in-lync-server-2013"></a>規劃 Lync Server 2013 中的宣告

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-09-13_

Lync Server 宣告應用程式可讓您設定的電話上的來電處理時撥打的號碼適用於您的組織，但不指派給使用者或電話。 您可以這些將來電轉接給預定的目的地 （電話號碼、 SIP URI 或語音信箱），或播放音訊宣告，或兩者。 宣告應用程式可協助您避免情況下，來電者時，並聽到忙線中音，或 SIP 用戶端會收到錯誤訊息。 此區段包含規劃是專屬於宣告應用程式的資訊。
