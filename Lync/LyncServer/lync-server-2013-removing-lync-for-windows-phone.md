# <a name="removing-lync-for-windows-phone-in-lync-server-2013"></a>在 Lync Server 2013 中移除 Lync for Windows Phone

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-19_

若要從行動裝置移除 Windows Phone 版 Lync 2013 應用程式，請執行下列步驟：

1.  在行動裝置上，從 [開始] 畫面滑動以查看應用程式清單。

2.  敲擊並按住 Lync 應用程式，然後選取 [**卸載**]。
