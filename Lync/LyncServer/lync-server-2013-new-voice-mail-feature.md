# <a name="new-voice-mail-feature-in-lync-server-2013"></a>Lync Server 2013 中的新語音信箱功能

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-05_

Lync Server 2013 引進語音信箱轉義程式，這是一種管理語音信箱的增強功能。 這項新功能可以偵測來電何時路由傳送至語音信箱，並防止來電立即路由傳送到使用者的行動電話語音信箱，而不會讓使用者接聽通話的機會。 當使用者同時向其行動電話進行同時震鈴，且其行動電話關閉、電池計量不足或超出範圍時，就會發生此案例。 語音信箱 Escape 會偵測到呼叫已由使用者的行動電話語音信箱立即接聽，並將通話中斷與行動電話語音信箱的通話。 通話繼續撥打使用者的其他端點，讓使用者有機會接聽通話。 若使用者沒有接聽來電，來電會路由傳送到公司語音信箱。
