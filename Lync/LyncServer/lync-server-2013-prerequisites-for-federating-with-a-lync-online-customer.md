# <a name="prerequisites-for-federating-with-a-lync-online-customer-in-lync-server-2013"></a>在 Lync Server 2013 中與 Lync Online 客戶進行同盟的必要條件

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-19_

若要與 Lync Online 2010 客戶同盟，您應該已在組織中完成 Lync Server 2013 的初始部署和設定。 其中包括下列項目：

  - 在您的組織中部署至少一部 Standard Edition server 或單一 Enterprise Edition 前端集區。 如需部署內部伺服器的詳細資訊，請參閱部署檔中的 [部署 Lync Server 2013](lync-server-2013-deploying-lync-server.md) 。

  - 啟用 Lync Server 2013 的內部使用者帳戶。 如需詳細資訊，請參閱部署檔或作業檔中的 [停用或重新啟用 Lync Server 2013 的使用者帳戶](lync-server-2013-disable-or-re-enable-user-account-for-lync-server.md) 。

  - 部署至少一部 Edge Server，以及支援外部使用者存取所需的其他元件。 如需詳細資訊，請參閱部署檔中的 [管理 Lync Server 2013 的同盟與外部存取](lync-server-2013-managing-federation-and-external-access-to-lync-server-2013.md) 。

  - 在組織內啟用同盟支援，並設定適當的方法來控制同盟網域的存取。 如需詳細資訊，請參閱 Operations 檔中的在 lync server [2013 中啟用或停用遠端使用者存取](lync-server-2013-enable-or-disable-remote-user-access.md) 和 [管理您組織2013的 SIP 同盟提供者](lync-server-2013-manage-sip-federated-providers-for-your-organization.md) 。

  - 為組織中的使用者啟用外部使用者存取。 如需詳細資訊，請參閱在 Lync Server 2013 和部署檔或作業檔 [中，將外部使用者存取原則指派給啟用 Lync 功能的使用者](lync-server-2013-assign-an-external-user-access-policy-to-a-lync-enabled-user.md) 。
