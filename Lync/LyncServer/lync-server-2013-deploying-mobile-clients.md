# <a name="deploying-mobile-clients-in-lync-server-2013"></a>部署 Lync Server 2013 中的行動用戶端

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改主題：** 2013年-02-19_
