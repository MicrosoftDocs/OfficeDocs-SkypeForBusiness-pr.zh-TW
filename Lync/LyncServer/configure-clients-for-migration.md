# <a name="configure-clients-for-migration"></a>設定用戶端以進行移轉

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-24_

本主題包含在您遷移至 Lync Server 2013 之前，您應該採取的建議用戶端部署步驟。 這些設定變更應該在 Lync Server 2010 上進行。

<div>

## <a name="to-configure-clients-before-migration"></a>在遷移前先設定用戶端

1.  部署 Lync Server 2010 的最新伺服器、用戶端和裝置更新（修補程式）。 如需有關 Lync Server 2010 的最新資訊，請參閱更新 Lync 版的資源中心<http://go.microsoft.com/fwlink/p/?linkid=232630>。

2.  在 Lync Server 2010 上，使用用戶端版本篩選，只允許已安裝最新更新的用戶端。
