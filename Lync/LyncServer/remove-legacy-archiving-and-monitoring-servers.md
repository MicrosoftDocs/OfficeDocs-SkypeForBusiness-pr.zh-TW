# <a name="remove-legacy-archiving-and-monitoring-servers"></a>移除舊版封存和監控伺服器

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-10-19_

如果舊版部署包含封存或監控伺服器，移轉至 Lync Server 2013 之後，這些伺服器可以移除舊版環境中，前提是已從任何剩餘的舊版集區中移除所有使用者。 您可以在任何序列中移除封存伺服器或監控伺服器。 主要需求是已從任何剩餘的舊版集區移除所有使用者。

您可以移動使用者從 Lync Server 2010 to Lync Server 2013 中所述的程序[階段 4： 將測試使用者移至試驗集區](phase-4-move-test-users-to-the-pilot-pool.md)。

您已確認的所有使用者都已經都移除剩餘集區之後，請遵循 「 解除安裝 Microsoft Lync Server 2010 和移除伺服器角色，「 可在下載中的程序[https://go.microsoft.com/fwlink/p/?linkId=246227](https://go.microsoft.com/fwlink/p/?linkid=246227)。
