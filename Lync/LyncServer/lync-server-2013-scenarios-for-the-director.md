Director 是執行 Microsoft Lync Server 2013 通訊軟體的伺服器，可驗證使用者要求，但不會家用任何使用者帳戶。 Director 也會主控類似前端伺服器的 web 服務，並會驗證 web 票證要求並提供其他服務。
