# <a name="setting-up-kerberos-authentication-account-passwords-in-lync-server-2013"></a>設定 Lync Server 2013 中的 Kerberos 驗證帳戶密碼

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2010年-11-03_

建立 Kerberos 驗證帳戶的電腦物件之後，您可以設定帳戶的密碼。 您在執行 Windows PowerShell cmdlet，在一部伺服器上設定 Kerberos 帳戶密碼。 您可以在您建立的 Kerberos 驗證的物件上設定密碼。 密碼可以設為已知的值，但預設會選取隨機密碼。 Password 是可使用之帳戶的所有 Kerberos 驗證來源。 您可以使用 Windows PowerShell cmdlet 設定及管理 Kerberos 帳戶密碼。

<div>


> [!NOTE]  
> Kerberos 帳戶物件是電腦物件，但使用 UserAccount 參數所參考的 Windows PowerShell cmdlet 中的作業。 請注意這不是發生錯誤，但指令程式的 Kerberos 帳戶建立和維護搭配使用時的預期的行為。
