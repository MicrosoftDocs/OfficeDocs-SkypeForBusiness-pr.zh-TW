# <a name="location-based-routing-for-conferencing-in-lync-server-2013"></a>在 Lync Server 2013 中為會議 Location-Based 路由

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-07-31_

Location-Based 路由可讓您根據通話中各方的位置，限制 VoIP 端點和 PSTN 端點之間通話的路由。 使用 Lync Server 2013 的累計更新2，可在 Lync 會議上強制執行 Location-Based 的路由規則 (也就是說，可) 防止 PSTN 免付費旁路的會議。 應用程式會監視作用中的會議，並根據參與的使用者位置，強制執行 Location-Based 路由限制。 Location-Based 路由會議應用程式，此外也可強制執行 Location-Based 路由限制，以進行涉及 PSTN 端點的諮詢式轉接。
