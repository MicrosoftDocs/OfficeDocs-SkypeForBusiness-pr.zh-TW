# <a name="phase-4-merge-topologies"></a>階段4：合併拓撲

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-03-29_

下列主題概述將 Microsoft Office 通訊伺服器 2007 R2 池合併至 Microsoft Lync Server 2013 池所需的步驟。 首先，您可以使用 [拓撲建立器合併] 嚮導來合併拓撲資訊。 這個工具會收集 Office 通訊伺服器 2007 R2 環境的相關資訊（包括邊緣伺服器資訊），並將該資訊發佈至使用 Lync Server 2013 共用的資料庫。 發佈合併後的拓撲之後，拓撲建立器會用來查看 Office 通訊伺服器 2007 R2 拓撲資訊，以及新部署的 Lync Server 2013 拓撲結構的相關資訊。 最後，您使用 Lync Server 管理命令介面 Cmdlet 匯入原則和設定設定。
