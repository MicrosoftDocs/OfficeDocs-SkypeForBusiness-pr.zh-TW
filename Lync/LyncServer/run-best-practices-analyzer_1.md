# <a name="run-best-practices-analyzer"></a>執行 Best Practices Analyzer

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-11_

Office 通訊伺服器 2007 R2 最佳作法分析工具會從 Office 通訊伺服器 2007 R2 部署收集設定資訊，並決定是否要根據 Microsoft 最佳作法設定設定。 您可以在執行 Microsoft .NET Framework 2.0 的用戶端電腦上安裝工具，或是直接在執行 Office 通訊伺服器 2007 R2 的伺服器上安裝工具。 您可以在執行 Microsoft .NET Framework 3.5 Service Pack 1 的用戶端電腦或直接在執行 nm-ocs-14-2nd 的伺服器上安裝工具。 您也可以在用戶端電腦上將 Office 通訊伺服器 2007 R2 系統管理工具安裝在本機上，使最佳作法 Analyzer 可以收集完整的資料集。

您可以從 Microsoft 下載中心下載 Office 通訊伺服器 2007 R2 最佳作法分析程式 [https://go.microsoft.com/fwlink/p/?LinkId=268702](https://go.microsoft.com/fwlink/p/?linkid=268702) 。
