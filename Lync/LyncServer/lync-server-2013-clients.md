# <a name="clients-for-lync-server-2013"></a>Lync Server 2013 的用戶端

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2014-01-23_

Lync Server 2013 支援多種類型的用戶端軟體，您可以將它部署到貴組織的使用者，包括電腦安裝的用戶端軟體、web 型用戶端及行動裝置。 本主題概述您可以使用的用戶端。 如需 Lync Server 2013 用戶端提供之功能的詳細比較，請參閱[Lync server 2013 的用戶端比較表](lync-server-2013-desktop-client-comparison-tables.md)。

<div>

## <a name="lync-2013"></a>Lync 2013

Lync 2013 是 Lync Server 的完整功能用戶端。 Lync 2013 使用者介面已完全重新設計，包括新的整合功能2010（例如，持續聊天）、分頁式交談、視頻預覽，以及多方影片。 如需變更摘要，請參閱[Lync Server 2013 中的客戶新功能](lync-server-2013-what-s-new-for-clients.md)。

Lync 2013 用戶端設定是安裝媒體上的 Office 安裝程式的一部分。

</div>

<div>

## <a name="online-meeting-add-in-for-lync-2013"></a>Lync 2013 的線上會議增益集

Lync 2013 的線上會議增益集支援在 Microsoft Outlook 訊息與共同作業用戶端中進行會議管理。 Lync 2013 軟體的線上會議增益集會自動安裝 Lync 2013。

</div>

<div>

## <a name="lync-web-scheduler"></a>Lync Web 排程器

Lync Web 排程器是一種針對沒有 Microsoft Outlook 存取權的使用者，或在不是以 Windows 為基礎的作業系統上使用的網路會議排程與管理工具。 使用者可以使用 Lync Web 排程器建立新的會議、修改現有的會議，以及使用其慣用的電子郵件程式傳送邀請。

</div>

<div>

## <a name="lync-web-app"></a>Lync Web App

Lync Web App 是 Lync Server 2013 會議的網路式會議用戶端。 在這個版本中，將電腦音訊和影片新增至 Lync Web App，可為未在本機安裝 Lync 用戶端的任何人提供完整的會議體驗。 會議參與者可存取所有共同作業及共用功能及簡報者會議控制項。

如果未在使用者的電腦上安裝 Lync 2013，且使用者按一下會議邀請中的會議連結，Lync Web App 就會開啟。 您也可以設定會議加入頁面，以允許使用者使用舊版用戶端加入會議;請參閱在部署檔中的[Lync Server 2013 中的 [設定會議加入] 頁面](lync-server-2013-configuring-the-meeting-join-page.md)。

由於 Lync Web App 的增強功能，因此無法使用 Lync Server 2013 更新版本的 Microsoft Lync 2010 出席者。 Lync Web App 是您組織外部參與者所選擇的用戶端。 使用 Lync Web App 時，不需要安裝本機用戶端，不過音訊、影片和共用功能需要在第一次使用期間安裝外掛程式。

</div>

<div>

## <a name="lync-2013-basic"></a>Lync 2013 基本版

Lync 2013 Basic 是一種可下載的客戶，可供擁有授權、內部部署 Lync Server 2013 部署的客戶，以及訂閱 Microsoft Office 365 方案（不含完整 Lync 2013 用戶端）的客戶。 Lync Basic 用戶端包括 [增強的目前狀態]、[連絡人]、[立即訊息（IM）]、[Lync 會議] 和 [基本語音功能]。 Lync Basic 中不支援的功能包括多方影片、OneNote 整合、虛擬桌面基礎結構（VDI）支援、技能搜尋、錄製、企業語音功能，以及高級通話處理（例如，來電轉接與團隊通話）。 如需詳細資訊，請參閱[Lync Server 2013 的用戶端比較表](lync-server-2013-desktop-client-comparison-tables.md)。

</div>

<div>

## <a name="lync-windows-store-app"></a>Lync Windows Store 應用程式

Lync Windows Store 應用程式是專為 Windows 8.1、Windows 8 和 Windows RT 設計的觸控優化 Lync app。 使用者可以搜尋「Lync」，透過 Windows 網上商店下載 app。 如需詳細資訊，請參閱[Lync server 2013 的用戶端比較表](lync-server-2013-desktop-client-comparison-tables.md)、lync [server 2013 的 lync windows store 應用程式需求](lync-server-2013-lync-windows-store-app-requirements.md)，以及[在 Lync Server 2013 中部署 lync windows store 應用程式](lync-server-2013-deploying-lync-windows-store-app.md)。

</div>

<div>

## <a name="lync-2013-for-mobile-devices"></a>適用于行動裝置的 Lync 2013

Lync 2013 行動應用程式現在除了連絡人、目前狀態和 IM 功能之外，還包括 [語音傳遞 IP （VoIP）] 和 [經由 IP 的視頻] 功能。 行動使用者可以使用 Wi-fi 或行動資料連線，選擇透過 IM、語音通話或視頻通話與其他人進行通訊。 只要按一下行事曆專案中的會議連結，行動使用者就能加入語音及視訊會議。 如需有關 Lync 2013 行動應用程式的詳細資訊，請參閱[在 Lync Server 2013 中規劃行動用戶端](lync-server-2013-planning-for-mobile-clients.md)。

</div>

<div>

## <a name="supported-clients-from-previous-releases"></a>先前版本支援的用戶端

Lync Server 2013 支援來自舊版伺服器發行的下列用戶端。 您可以讓使用者在加入會議時，將某些先前的用戶端提供給使用者。 如需詳細資訊，請參閱部署檔中的[Lync Server 2013 中的 [設定會議加入] 頁面](lync-server-2013-configuring-the-meeting-join-page.md)。

  - **Lync 2010**   lync 2010 提供完整的桌面體驗，包括 IM、增強的目前狀態、語音、影片、共用及電話。 不過，Lync Server 2013 中推出的新功能，只有在使用者的用戶端升級至 Lync 2013 之後，才可使用。

  - **Lync 2010**   行動裝置版 lync Server 2013 支援所有 Microsoft Lync 2010 行動裝置 app。 Microsoft Lync 2010 行動裝置可為貴組織中的使用者提供 IM、增強的目前狀態，以及使用專業版 Windows Mobile 從手機連線的電話。 您可以透過將 Microsoft Lync 2010 行動裝置導向給其行動電話的 app marketplace，來指示使用者安裝該應用程式。 如需詳細資訊，請參閱 Lync Server 2010 檔中的「規劃行動客戶[http://go.microsoft.com/fwlink/p/?LinkID=235955](http://go.microsoft.com/fwlink/p/?linkid=235955)端」。

  - **Lync phone edition**   lync 手機版軟體（例如 USB 連接的手機）尚未針對 Lync Server 2013 進行更新。 Lync Phone Edition 繼續受到支援，以便在會議中撥打及接聽通話、增強目前狀態及用戶端音訊功能。

  - **Lync 2010**   應答 Microsoft lync 2010 助理整合式通話管理程式可讓接待員透過快速通話處理、IM 和螢幕路由，同時管理多個交談。

</div>
