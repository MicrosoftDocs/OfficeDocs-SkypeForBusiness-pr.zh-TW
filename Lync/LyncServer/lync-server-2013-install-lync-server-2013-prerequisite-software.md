# <a name="install-lync-server-2013-prerequisite-software"></a>安裝 Lync Server 2013 必要軟體

_**主題上次修改日期：** 2013-02-21_

Persistent Chat Server 的必要條件軟體，與 Lync server 2013 前端伺服器及已安裝 Persistent Chat Server 功能的 Lync Server 2013 Standard Edition server 相同。

Persistent Chat Server 檔案存放區所需的必要軟體與 Lync Server 2013 相同。

適用于 Persistent Chat Server content 和合規性的 SQL Server 資料庫必要條件軟體，也與 Lync Server 2013 的必要條件相同。

如需 Lync Server 2013 伺服器的所有軟體需求的詳細資訊，請參閱支援檔中的 [其他伺服器支援和 Lync server 2013](lync-server-2013-additional-server-support-and-requirements.md) 中的需求。
