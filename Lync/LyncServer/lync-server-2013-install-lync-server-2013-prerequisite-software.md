# <a name="install-lync-server-2013-prerequisite-software"></a>安裝 Lync Server 2013 先決條件軟體

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改主題：** 2013年-02-21_

Persistent Chat Server 的必要條件軟體為 Lync Server 2013 Front End Servers and Lync Server 2013 Standard Edition 伺服器安裝 Persistent Chat Server 功能的必要軟體相同。

Persistent Chat Server 檔案存放區所需的必要條件軟體會是相同的 Lync Server 2013。

Persistent Chat Server 內容和符合性的 SQL Server 資料庫的必要條件軟體也是相同的 Lync Server 2013。

如需 Lync Server 2013 伺服器的所有軟體需求的詳細資訊，請參閱支援文件中的[其他伺服器支援和 Lync Server 2013 中的需求](lync-server-2013-additional-server-support-and-requirements.md)。
