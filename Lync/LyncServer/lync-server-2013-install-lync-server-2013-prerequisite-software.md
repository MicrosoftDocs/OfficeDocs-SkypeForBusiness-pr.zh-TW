# <a name="install-lync-server-2013-prerequisite-software"></a>安裝 Lync Server 2013 先決條件軟體

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-21_

持續聊天伺服器的必備軟體與 Lync Server 2013 前端伺服器的必備軟體和安裝持久聊天伺服器功能的 Lync Server 2013 標準版伺服器相同。

永久聊天伺服器檔案存放區所需的必備軟體與 Lync Server 2013 的需求軟體相同。

持久聊天伺服器內容與合規性的 SQL Server 資料庫必備元件，也與 Lync Server 2013 的必備軟體相同。

如需有關 Lync Server 2013 伺服器所有軟體需求的詳細資料，請參閱支援檔中的[Lync server 2013 中的其他伺服器支援與需求](lync-server-2013-additional-server-support-and-requirements.md)。
