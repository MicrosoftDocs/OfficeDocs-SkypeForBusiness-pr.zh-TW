# <a name="managing-call-management-features-in-lync-server-2013"></a>管理 Lync Server 2013 中的通話管理功能

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012年-12-18_

Enterprise Voice 通話管理功能可控制來電的路由傳送與應答方式。 Lync Server 2013 提供下列通話管理功能：

  - **通話駐留：** 可讓語音使用者暫時駐留通話，然後從相同電話或其他電話接聽。

  - **群組收取：** 可讓使用者来揀選響鈴其他使用者撥出通話收取群組號碼的來電。

  - **回應群組：** 來電路由傳送給群組的代理程式使用搜尋群組或互動語音回應 (IVR) 問題與解答。

  - **宣告：** 播放一段訊息對撥打給未指派的號碼，或將通話路由傳送到其他地方，或兩者。

本節說明如何管理 Enterprise Voice 部署中的這些通話管理功能。
