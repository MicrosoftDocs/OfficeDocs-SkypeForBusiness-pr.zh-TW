# <a name="setting-up-a-backup-location-for-lync-server-2013"></a>設定 Lync Server 2013 的備份位置

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-17_

在您第一次備份 Lync Server 之前，請先設定所需的硬體和軟體，以便儲存及維護備份。 您必須視情況取得媒體和內容的存取權，並提供每部要備份伺服器與備份媒體之間的網路連線。 您應在備份和還原策略中定義您使用的媒體和位置。 您用於定期備份的位置可以是本機或遠端的，但必須是安全的，而且必須可存取備份與還原。 我們建議使用遠端位置，以防範主要網站上的災難性事件。

當您設定好並測試過個別元件後，請從每部伺服器驗證備份的可存取性。
