# <a name="checking-for-updates-to-best-practices-analyzer-in-lync-server-2013"></a>在 Lync Server 2013 中檢查最佳做法分析程式的更新

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-02_

當您啟動最佳做法分析程式時，工具會提供一個選項，讓您搜尋工具的最新更新。 如果有可用的更新，您可以下載更新。 如果您選擇不下載更新，或最佳做法分析程式無法存取網際網路，您可以繼續使用電腦上已經存在的版本。

<div>


> [!NOTE]  
> 如果您需要 proxy 驗證才能存取網際網路，最佳做法分析程式無法存取您要下載的新更新。 不過，您可以從 Microsoft 下載中心手動下載最新版本的 RtcBPA <A href="http://go.microsoft.com/fwlink/p/?linkid=266539">http://go.microsoft.com/fwlink/p/?linkId=266539</A>。 下載檔案之後，您可以將檔案複製到您想要更新最佳做法分析程式的電腦上，並使用 .msi 檔案在該電腦上安裝新版工具。



</div>

若要更新最佳做法分析程式規則，您必須在本機電腦上以系統管理員身分執行該工具。 如果您未使用系統管理員群組成員的帳戶登入，且偵測到更新，請關閉最佳做法分析程式，然後使用下列程式來啟動程式。

<div>

## <a name="to-open-best-practices-analyzer-as-administrator-to-check-for-updates"></a>若要以系統管理員身分開啟最佳做法分析程式來檢查更新

1.  在已安裝最佳做法分析程式的電腦上，按一下 [**開始**]，指向 [ **Microsoft Lync Server 2013**]，以滑鼠右鍵按一下 [**最佳做法分析**程式]，然後按一下 [**以系統管理員身分執行**]。

2.  指定屬於 [管理員] 群組成員的帳號憑證。

</div>
