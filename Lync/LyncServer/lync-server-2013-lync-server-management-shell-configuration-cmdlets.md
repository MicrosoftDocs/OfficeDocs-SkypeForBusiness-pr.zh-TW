# <a name="lync-server-management-shell-configuration-cmdlets-in-lync-server-2013"></a>Lync server 管理命令介面配置 Cmdlet （Lync Server 2013）

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-03-21_

Lync Server 管理命令介面是 Windows PowerShell 命令視窗的一個實現，具有專用來管理在環境中預先載入的 Microsoft Lync Server 2010 的 Cmdlet。 這些 Cmdlet 可讓您自訂該環境。

<div>

## <a name="lync-server-management-shell-configuration-cmdlets"></a>Lync Server 管理命令介面配置 Cmdlet

使用這些 Cmdlet 來設定 Lync Server 管理命令介面的特定自訂屬性。

  - <span></span>  
    [CsUICulture](https://technet.microsoft.com/en-us/library/Gg412900(v=OCS.15))

  - <span></span>  
    [Set-CsUICulture](https://technet.microsoft.com/en-us/library/Gg398354(v=OCS.15))

<!-- end list -->

  - <span></span>  
    [CsManagementConnection](https://technet.microsoft.com/en-us/library/Gg412849(v=OCS.15))

  - <span></span>  
    [移除-CsManagementConnection](https://technet.microsoft.com/en-us/library/Gg425803(v=OCS.15))

  - <span></span>  
    [Set-CsManagementConnection](https://technet.microsoft.com/en-us/library/Gg413045(v=OCS.15))

</div>
