# <a name="database-and-management-server-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的資料庫和管理伺服器 Cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-10-09_

資料庫和管理伺服器 Cmdlet 是用來管理您的 Microsoft Lync Server 2013 後端資料庫和前端管理服務。 您也可以使用這些 Cmdlet 來安裝或卸載 Lync Server 2013 所使用的任何資料庫，除了為集中式管理存放區設定 Active Directory 服務控制點以外。
