# <a name="running-large-meetings-in-lync-server-2013"></a>在 Lync Server 2013 中執行大型會議

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2019-01-17_

在會議中有幾百位以上的使用者，是一種很好的做法，那就是讓專屬人員成為大型會議的線上會話是個不錯的做法。 這個專用人員可以是會議召集人的代理人，或是組織的大型會議支援人員的成員。 在排程會議時，將專門的會議版主新增為簡報者，不過在會議進行中時，可以將線上會議出席者宣傳給簡報者角色是很重要的。

會議仲裁者可以使用 Lync Server 2013 用戶端的下列簡報者功能來管理大型會議：

- 變更會議存取類型。

- 在會議期間使用 Lync 拖放功能、電話撥出或電子郵件邀請其他參與者。

- 管理會議內容，包括上傳內容、刪除內容，以及切換活動內容。
