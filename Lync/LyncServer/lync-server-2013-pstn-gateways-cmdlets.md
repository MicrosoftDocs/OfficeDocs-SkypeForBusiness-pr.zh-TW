# <a name="pstn-gateways-cmdlets-in-lync-server-2013"></a>Lync Server 2013 中的 PSTN 閘道 Cmdlet

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-03-21_

PSTN 閘道可讓您的企業語音使用者撥打電話給 PSTN 網路上的人員，並接聽電話撥打電話給其他人（也就是公用交換電話網絡）。 這些閘道充當中繼伺服器與 PSTN 網路之間的橋樑。

<div>

## <a name="pstn-gateways-cmdlets"></a>PSTN 閘道 Cmdlet

[CsPstnOutboundCall](https://technet.microsoft.com/en-us/library/Gg398207(v=OCS.15))和[CsPstnPeerToPeerCall](https://technet.microsoft.com/en-us/library/Gg398662(v=OCS.15)) Cmdlet 可讓您確認使用者可以在 PSTN 網路撥打電話。

**PSTN 閘道**

  - <span></span>  
    [Set-CsPstnGateway](https://technet.microsoft.com/en-us/library/Gg398408(v=OCS.15))
