# <a name="clients-supported-for-call-park-in-lync-server-2013"></a>通話駐留 Lync Server 2013 中支援的用戶端

</div>

<div id="mainSection">

_**主題上次修改日期：** 2012年-09-13_

本節說明可用於駐留通話的用戶端和用戶端可以用來擷取駐留的通話。

<div>

## <a name="clients-supported-for-parking-calls"></a>駐留通話支援的用戶端

呼叫來自任何 IP、 專用交換機 (PBX)、 公用交換電話網路 (PSTN)，或可以駐留行動電話。

<div>


> [!NOTE]  
> 您可以駐留僅音訊通話。 立即訊息和會議，無法駐留通話。



</div>

下列用戶端可用於駐留通話的通話駐留：

  - Lync 2013

  - Lync 2010

  - Lync 2010 Attendant

  - Lync Phone Edition

<div>


> [!NOTE]  
> 行動電話無法使用通話駐留駐留通話。



</div>

</div>

<div>

## <a name="clients-supported-for-retrieving-calls"></a>擷取通話支援的用戶端

軌道範圍設定為虛擬分機 （而不需要指派的使用者或電話分機） 區塊。 當您為虛擬分機設定軌道時，行動電話和 PSTN 電話無法擷取駐留的通話。

同盟的使用者無法擷取駐留的通話。

下列用戶端可以擷取駐留的通話駐留的通話：

  - Lync 2013

  - Lync 2010

  - Lync 2010 Attendant

  - Lync Phone Edition

  - IP 公共區域電話

  - 非 IP 電話連線至 Lync Server 2013 基礎結構，包括公共區域電話和專用交換機 (pbx) 電話
