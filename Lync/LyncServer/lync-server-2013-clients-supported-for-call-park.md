# <a name="clients-supported-for-call-park-in-lync-server-2013"></a>Lync Server 2013 中支援通話駐留的用戶端

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-13_

本節識別可用於寄存通話的用戶端，以及可用於檢索寄存通話的用戶端。

<div>

## <a name="clients-supported-for-parking-calls"></a>支援駐留通話的用戶端

來自任何 IP、專用交換機 (PBX) 、公用交換電話網路 (PSTN) 或行動電話可以寄存的電話。

<div>


> [!NOTE]  
> 只有音訊通話可以寄存。 立即訊息和會議無法停用。



</div>

下列用戶端可以將通話駐留用於駐留通話：

  - Lync 2013

  - Lync 2010

  - Lync 2010 Attendant

  - Lync Phone Edition

<div>


> [!NOTE]  
> 行動電話無法將通話駐留用於寄存通話。



</div>

</div>

<div>

## <a name="clients-supported-for-retrieving-calls"></a>支援檢索呼叫的用戶端

軌道範圍會設定為不需要指派使用者或電話) 的虛擬分機區塊 (擴充。 當您將軌道式設定為虛擬分機時，行動電話和 PSTN 電話便無法檢索寄存的呼叫。

同盟使用者無法檢索寄存的呼叫。

下列用戶端可以檢索寄存在通話駐留上的通話：

  - Lync 2013

  - Lync 2010

  - Lync 2010 Attendant

  - Lync Phone Edition

  - IP 公共區域電話

  - 連線至 Lync Server 2013 基礎結構的非 IP 電話，包括通用區域電話和專用交換機 (PBX) 電話

</div>
