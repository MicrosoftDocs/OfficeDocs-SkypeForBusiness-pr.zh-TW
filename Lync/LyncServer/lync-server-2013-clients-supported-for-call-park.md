# <a name="clients-supported-for-call-park-in-lync-server-2013"></a>Lync Server 2013 中支援通話駐留的用戶端

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-13_

本節會識別可以用來寄存通話的用戶端，以及可用於檢索暫停通話的用戶端。

<div>

## <a name="clients-supported-for-parking-calls"></a>支援停用通話的用戶端

來自任何 IP、私人分支 exchange （PBX）、公用交換電話網絡（PSTN）或行動電話的呼叫都可以停用。

<div>


> [!NOTE]  
> 只有音訊通話可以停用。 無法停用立即訊息和會議。



</div>

下列用戶端可以使用通話駐留撥出電話：

  - Lync 2013

  - Lync 2010

  - Lync 2010 助理

  - Lync Phone Edition

<div>


> [!NOTE]  
> 行動電話無法使用通話駐留撥出電話。



</div>

</div>

<div>

## <a name="clients-supported-for-retrieving-calls"></a>支援的用戶端檢索通話

軌道範圍是設定為虛擬延伸的區塊（不含指派使用者或電話的延伸）。 當您將 [軌道式] 設定為 [虛擬延伸] 時，行動電話和 PSTN 手機無法取得停用的通話。

同盟使用者無法取得停用的通話。

下列用戶端可以檢索寄存在通話寄存上的通話：

  - Lync 2013

  - Lync 2010

  - Lync 2010 助理

  - Lync Phone Edition

  - IP 通用區域電話

  - 連線至 Lync Server 2013 基礎結構的非 IP 電話，包括通用的區域電話和私人分支 exchange （PBX）電話

</div>
