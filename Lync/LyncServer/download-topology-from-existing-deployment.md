# <a name="download-topology-from-existing-deployment"></a>從現有部署下載拓撲

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-29_

建立 Lync Server 2013 池時，您將會使用與 Lync Server 2010 相關聯的中央管理儲存。 當您在第一次使用及後續編輯會話時啟動拓撲產生器時，系統會提示您輸入您想要拓撲建立器載入目前設定檔的位置。 因為您已定義 Lync Server 2010 拓撲，且已建立中央管理儲存區，所以您應該選擇從現有的部署下載拓撲。 拓撲建立器將會讀取資料庫並檢索目前的定義。

**從現有的部署下載拓撲**

1.  開啟 [ **Lync Server 部署] 嚮導**。

2.  從 [ **Lync Server 2013 –部署嚮導]** 頁面，按一下 [**安裝管理工具**]。

3.  啟動拓撲產生器：按一下 [**開始**]，按一下 [**所有程式**]，按一下 [ **Microsoft Lync server 2013** ]，然後按一下 [ **Lync server 拓撲**建立器]。

4.  選取 [**從現有的部署下載拓撲**。
    
    ![部署嚮導拓撲產生器設定](images/JJ721913.d5b39fd9-3c13-422e-a06c-25d2568fe781(OCS.15).jpg "部署嚮導拓撲")建立器設定

5.  選擇檔案名，然後以預設的 tbxml 檔案類型儲存拓撲。

6.  展開 Lync Server 節點（如圖所示），以在部署中顯示各種伺服器角色。
    
    ![拓撲建立器伺服器角色一般屬性](images/JJ721913.af99ead3-676b-47fd-8369-5a5f9717383f(OCS.15).jpg "拓撲建立器伺服器角色一般屬性")
