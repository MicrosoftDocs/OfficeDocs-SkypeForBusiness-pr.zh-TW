# <a name="device-update-logs-and-files-in-lync-server-2013"></a>Lync Server 2013 中的裝置更新記錄和檔案

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-20_

裝置更新記錄檔包含重要資訊，可供您用來管理裝置更新 Web 服務並加以疑難排解。 您可以變更所記錄的內容，並移除不需要或不再需要的裝置記錄檔和更新。 本節說明如何使用 Lync Server 控制台或 Lync Server 管理命令介面來修改記錄設定、清除裝置更新記錄檔，或從伺服器移除記錄檔。

<div>


> [!NOTE]  
> 如需裝置更新記錄檔的詳細資訊，請參閱 Lync Server 2010 TechNet 程式庫中的 <A href="https://technet.microsoft.com/library/gg398250(v=ocs.14).aspx">記錄檔類型和位置</A> 。  (請注意，裝置更新 Web 服務（如所有 Lync Phone Edition 元件）的運作方式與 lync server 2010 2013 相同。 ) 



</div>
