# <a name="device-update-logs-and-files-in-lync-server-2013"></a>Lync Server 2013 中的裝置更新記錄和檔案

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-20_

裝置更新記錄包含重要資訊，您可以用來管理裝置更新 Web 服務並為其進行疑難排解。 您可以變更記錄的內容，並移除您不想要或不再需要的裝置記錄和更新。 本節說明如何使用 Lync Server [控制台] 或 [Lync Server 管理命令介面] 來修改記錄設定、清除裝置更新記錄檔，或從伺服器移除記錄檔。

<div>


> [!NOTE]  
> 如需裝置更新記錄檔的詳細資訊，請參閱 Lync Server 2010 TechNet Library 中的<A href="http://technet.microsoft.com/en-us/library/gg398250(v=ocs.14).aspx">記錄檔案類型和位置</A>。 （請注意，裝置更新 Web 服務（例如所有 Lync 手機版本元件）的運作方式2013與 Lync Server 2010 相同。）



</div>
