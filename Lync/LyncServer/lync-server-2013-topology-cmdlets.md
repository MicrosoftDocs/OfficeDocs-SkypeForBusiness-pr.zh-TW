# <a name="topology-cmdlets-jn-lync-server-2013"></a>拓撲 Cmdlet jn Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-06-20_

Microsoft Lync Server 2013 中包含的許多拓撲 Cmdlet 都是專為搭配安裝與拓撲結構建立器而設計;因此，系統管理員很少會直接呼叫一些拓撲 Cmdlet。 不過，在某些情況下，系統會要求系統管理員使用這些 Cmdlet;例如，在建立新的 Kerberos 帳戶之後，您必須執行[Enable CsTopology](https://technet.microsoft.com/en-us/library/Gg398398(v=OCS.15)) Cmdlet，才能使變更生效。 此外，系統管理員可能會執行 Cmdlet （例如[Test CsTopology](https://technet.microsoft.com/en-us/library/Gg398127(v=OCS.15))和[test CsComputer](https://technet.microsoft.com/en-us/library/Gg398162(v=OCS.15)) ），以協助確保 Lync Server 2013 已正確安裝且正常運作。
