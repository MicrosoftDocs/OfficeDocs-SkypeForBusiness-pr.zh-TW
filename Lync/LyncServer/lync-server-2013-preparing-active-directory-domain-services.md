# <a name="preparing-active-directory-domain-services-for-lync-server-2013"></a>為 Lync Server 2013 準備 Active Directory 網域服務

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-21_

在您部署並操作 Lync Server 2013 之前，您必須先延伸架構，然後建立及設定物件，以準備 Active Directory 網域服務。 架構延伸會新增 Lync Server 所需的 Active Directory 類別和屬性。

本節中的主題說明如何準備用來部署 Lync Server 的 AD DS，以及如何指派設定與組織單位（OU）許可權。 如需 Lync Server 所需架構變更的詳細資料，請參閱[Lync server 2013 使用的 Active Directory 架構擴充功能、類別及屬性](lync-server-2013-active-directory-schema-extensions-classes-and-attributes-used-by-lync-server.md)。
