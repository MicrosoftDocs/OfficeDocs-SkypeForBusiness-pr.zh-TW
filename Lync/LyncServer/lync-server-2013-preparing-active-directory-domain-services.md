# <a name="preparing-active-directory-domain-services-for-lync-server-2013"></a>為 Lync Server 2013 準備 Active Directory 網域服務

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**上次修改主題：** 2013年-02-21_

您部署及操作 Lync Server 2013 之前，您必須準備 Active Directory 網域服務延伸架構和再建立及設定的物件。 架構延伸模組新增的 Active Directory 類別和 Lync Server 所需的屬性。

本節中的主題說明如何先準備 AD DS 部署 Lync Server，以及如何指派設定和組織單位 (OU) 權限。 如需 Lync Server 所需的架構變更的詳細資訊，請參閱[Active Directory 架構延伸模組、 類別及 Lync Server 2013 所使用的屬性](lync-server-2013-active-directory-schema-extensions-classes-and-attributes-used-by-lync-server.md)。
