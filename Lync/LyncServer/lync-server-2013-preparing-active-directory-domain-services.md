# <a name="preparing-active-directory-domain-services-for-lync-server-2013"></a>為 Lync Server 2013 準備 Active Directory 網域服務

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2013-02-21_

在您部署及操作 Lync Server 2013 之前，您必須先擴充架構，然後建立及設定物件，以準備 Active Directory 網域服務。 架構擴充新增 Lync Server 所需的 Active Directory 類別和屬性。

本節中的主題說明如何準備用於部署 Lync Server 的 AD DS，以及如何指派安裝和組織單位 (OU) 許可權。 如需 Lync Server 所需之架構變更的詳細資訊，請參閱 [Lync server 2013 所使用的 Active Directory 架構擴充功能、類別和屬性](lync-server-2013-active-directory-schema-extensions-classes-and-attributes-used-by-lync-server.md)。
