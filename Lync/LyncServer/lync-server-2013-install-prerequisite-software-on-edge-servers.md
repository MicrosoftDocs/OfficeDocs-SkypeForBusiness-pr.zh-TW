# <a name="install-prerequisite-software-on-edge-servers-for-lync-server-2013"></a>在 Lync Server 2013 的 Edge Server 上安裝必要軟體

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-08_

在安裝 Lync Server 2013 之前，您必須在部署的每個 Edge Server 上部署必要軟體。 這包括在符合系統需求的伺服器上安裝作業系統。 如需系統需求的詳細資訊，包括支援的作業系統，請參閱 [Lync Server 2013 之外部使用者存取元件的系統需求](lync-server-2013-system-requirements-for-external-user-access-components.md)。
