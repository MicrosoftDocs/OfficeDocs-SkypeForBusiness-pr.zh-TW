# <a name="install-prerequisite-software-on-edge-servers-for-lync-server-2013"></a>針對 Lync Server 2013 在 Edge Server 上安裝必要軟體

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**主題上次修改日期：** 2012-09-08_

在安裝 Lync Server 2013 之前，您必須先在部署的每個 Edge 伺服器上部署必備軟體。 這包括在符合系統需求的伺服器上安裝作業系統。 如需系統需求的詳細資料（包括支援的作業系統），請參閱[Lync Server 2013 的外部使用者存取元件的系統需求](lync-server-2013-system-requirements-for-external-user-access-components.md)。
