# <a name="start-cclogging"></a>Start-CcLogging
 
Start-CcLogging Cmdlet 會為商務用 Skype Cloud Connector Edition 裝置產生傳入和傳出通話記錄。 
  
```powershell
Start-CcLogging
```

## <a name="parameters"></a>參數

無
  
## <a name="examples"></a>範例
<a name="Examples"> </a>

### <a name="example-1"></a>範例 1

下列範例會產生傳入和撥出的通話記錄：
  
```powershell
Start-CcLogging
```

## <a name="detailed-description"></a>詳細描述
<a name="DetailedDescription"> </a>

Start-CcLogging Cmdlet 提供一種方式，讓系統管理員可以在雲端連接器裝置上開始登入來電和撥出電話。 根據預設，記錄會在四小時後自動停止。
  
## <a name="input-types"></a>輸入類型
<a name="InputTypes"> </a>

無。 Start-CcLogging Cmdlet 不接受管線傳送的輸入。
  
## <a name="return-types"></a>傳回類型
<a name="ReturnTypes"> </a>

無
