# <a name="validate-your-cloud-connector-deployment"></a>驗證您的 Cloud Connector 部署

> [!Important]
> 雲端連接器 Edition 會在2021年7月31日和商務用 Skype 線上時終止。 當您的組織升級至 Teams 後，請瞭解如何使用[直接路由](/MicrosoftTeams/direct-routing-landing-page)將您的內部部署電話網絡連線至 Teams。
 
完成部署及設定商務用 Skype Cloud Connector Edition 的步驟後，請確認您可以撥打和接聽電話。
  
## <a name="validate-the-deployment"></a>驗證部署

1. 使用剛才設定的使用者帳戶，登入您的商務用 Skype 用戶端。
    
2. 從用戶端呼叫 PSTN 號碼，並確認呼叫成功。
    
3. 使用您的電話或其他與 PSTN 號碼相關聯的裝置呼叫商務用 Skype 用戶端，並確認呼叫成功。
    
    > [!NOTE]
    > 您應該確認您可以在新增其他網站或轉換網站後順利進行通話，以啟用 HA 支援。
