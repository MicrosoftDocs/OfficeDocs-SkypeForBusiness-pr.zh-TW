# <a name="validate-your-cloud-connector-deployment"></a>驗證您的 Cloud Connector 部署
 
完成部署及設定商務用 Skype 雲端連接器版本的步驟後，請確認您可以撥打及接聽來電。
  
## <a name="validate-the-deployment"></a>驗證部署

1. 使用您剛剛設定的使用者帳戶登入商務用 Skype 用戶端。
    
2. 從用戶端呼叫 PSTN 號碼，並確認呼叫成功。
    
3. 使用您的手機或與 PSTN 號碼相關的裝置撥打商務用 Skype 用戶端，並確認呼叫成功。
    
    > [!NOTE]
    > 您應該確認在新增其他網站或轉換網站之後，您可以順利進行通話，以啟用 HA 支援。
