# <a name="integrate-skype-for-business-server-with-exchange-server"></a>將商務用 Skype Server 與 Exchange Server 整合

**摘要:** 查看 Exchange Server 2013 或更新版本及商務用 Skype Server 的整合步驟。

Exchange Server 2013 或更新版本與商務用 Skype 伺服器相容, 且能順利整合。 例如, 您可以在 Microsoft Outlook 中報告商務用 Skype 使用者目前狀態資訊。同樣地, 商務用 Skype 可以存取使用者的 Outlook 行事曆, 請注意使用者已排程會議, 並在會議期間顯示使用者目前狀態為忙碌。 雖然您不需要執行 Exchange Server, 就能執行商務用 Skype 伺服器 (或反之), 但這兩個產品一起加強了彼此的使用者體驗。

本檔提供整合商務用 Skype Server 與 Exchange Server 2016 或 Exchange Server 2013 的相關資訊, 但前提是這兩個產品的初始設定和配置已經發生。 如需有關部署商務用 Skype 伺服器的詳細資訊, 請參閱[商務用 Skype Server 技術中心](https://go.microsoft.com/fwlink/p/?LinkId=246127)。 如需有關部署 Exchange Server 的詳細資訊, 請參閱適用于您的 Exchange 版本的部署檔。

如果您要將商務用 Skype Server 的內部部署安裝與 Microsoft Exchange Online 整合, 請參閱[設定內部部署商務用 Skype server 和 Outlook Web App 之間的整合](outlook-web-app.md)。

如果您要將商務用 Skype Online 與 Exchange Server 內部部署整合, 請參閱在[商務用 Skype online 與 exchange 內部部署之間設定 OAuth](oauth-with-online-and-on-premises.md)。
