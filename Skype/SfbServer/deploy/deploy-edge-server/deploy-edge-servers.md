# <a name="deploy-edge-servers-in-skype-for-business-server"></a>在商務用 Skype Server 中部署邊緣伺服器
 
**摘要：** 瞭解如何將 Edge 伺服器部署到商務用 Skype Server 環境。
  
下列各節包含[在商務用 Skype server](../../plan-your-deployment/edge-server-deployments/edge-server-deployments.md)檔的商務用 Skype server 方案已經過審查之後，應遵循的步驟。 部署步驟如下所示：
  
- 網路介面
    
- 安裝
    
- 證書
    
- 啟動邊緣伺服器
    
## <a name="network-interfaces"></a>網路介面

就像在規劃中所述，您可以在託管您邊緣伺服器的周邊網路中，或在周邊網路中不使用 DNS，來設定您的網路介面。
  
### <a name="interface-configuration-with-dns-servers-in-the-perimeter-network"></a>在周邊網路中使用 DNS 伺服器的介面設定

1. 針對每個邊緣伺服器安裝兩個網路介面卡，一個用於內部介面，另一個用於面向外部介面。
    
    > [!NOTE]
    > 內部和外部子網必須彼此無法路由。 
  
2. 您可以在外部介面上設定下列**其中一項**：
    
   是. 外部周邊網路子網上的三個靜態 IP 位址，並將預設閘道指向外部防火牆的內部介面。 將 [配接器 DNS 設定] 設定為指向一對週邊 DNS 伺服器。
    
   乙. 外部周邊網路子網上的一個靜態 IP 位址，並將預設閘道指向外部防火牆的內部介面。 將 [配接器 DNS 設定] 設定為指向一對週邊 DNS 伺服器。 只有在您先前已將您的拓撲設定為在埠指派中有非標準值，這項設定才是可接受的，這會在為[商務用 Skype Server 文章建立您的邊緣拓撲](create-your-edge-topology.md)中所述。
    
3. 在您的內部介面上，在內部周邊網路子網上設定一個靜態 IP，然後不要設定預設閘道。 將 [配接器 DNS 設定] 設定為指向至少一個 DNS 伺服器，但最好是一對週邊 DNS 伺服器。
    
4. 在內部介面上建立永久靜態路由至用戶端、商務用 Skype 伺服器以及 Exchange 整合通訊（UM）伺服器所駐留的所有內部網路。
    
### <a name="interface-configuration-without-dns-servers-in-the-perimeter-network"></a>周邊網路中不含 DNS 伺服器的介面配置

1. 針對每個邊緣伺服器安裝兩個網路介面卡，一個用於內部介面，另一個用於面向外部介面。
    
    > [!NOTE]
    > 內部和外部子網必須彼此無法路由。 
  
2. 您可以在外部介面上設定下列**其中一項**：
    
   是. 外部周邊網路子網上的三個靜態 IP 位址。 您也需要在外部介面上設定預設閘道，例如，將網際網路式路由器或外部防火牆定義為預設閘道。 將 [配接器 DNS 設定] 設定為指向外部 DNS 伺服器，最好是一對外部 DNS 伺服器。
    
   乙. 外部周邊網路子網上的一個靜態 IP 位址。 您也需要在外部介面上設定預設閘道，例如，將網際網路式路由器或外部防火牆定義為預設閘道。 將 [配接器 DNS 設定] 設定為指向外部 DNS 伺服器，或 [理想] 是一對外部 DNS 伺服器。 只有在您先前已將您的拓撲設定為在埠指派中有非標準值，這項設定才是可接受的，這會在為[商務用 Skype Server 文章建立您的邊緣拓撲](create-your-edge-topology.md)中所述。
    
3. 在您的內部介面上，在內部周邊網路子網上設定一個靜態 IP，然後不要設定預設閘道。 同時將配接器 DNS 設定保留為空白。
    
4. 在內部介面上建立永久靜態路由至用戶端、商務用 Skype 伺服器以及 Exchange 整合通訊（UM）伺服器所駐留的所有內部網路。
    
5. 在每個邊緣伺服器上編輯主機檔案，以包含下一個躍點伺服器或虛擬 IP （VIP）的記錄。 此記錄將是您在拓撲建立器中設定為 Edge 伺服器下一個躍點位址的 Director、Standard Edition 伺服器或前端池。 如果您使用的是 DNS 負載平衡，請為下一個躍點池的每個成員包含一條線。
    
## <a name="installation"></a>安裝

若要成功完成這些步驟，您必須遵循在[商務用 Skype Server 文章中建立邊緣拓撲](create-your-edge-topology.md)中的步驟。
  
1. 使用本機管理員群組中的帳戶登入您針對 Edge 伺服器角色配置的伺服器。
    
2. 您需要在此電腦上的 Edge 伺服器拓撲檔結尾複製您複製的拓撲設定檔案。 存取您放入該設定檔案的外部媒體（例如 USB 磁片磁碟機或共用）。
    
3. 啟動 [**部署] 嚮導**。
    
4. 嚮導開啟後，按一下 [**安裝或更新商務用 Skype Server 系統**]。
    
5. 嚮導會執行檢查，以查看是否有任何已安裝的專案。 如第一次執行此嚮導，您會想從**步驟1開始。安裝本機配置存放區。**
    
6. 隨即會出現 [**設定中央管理儲存的本機複本**] 對話方塊。 您需要按一下 **[從檔案匯入] （適用于 Edge 伺服器的建議）**。
    
7. 在這裡，流覽至您先前匯出的拓撲位置，選取該 .zip 檔案，按一下 [**開啟**]，然後按一下 **[下一步]**。
    
8. [部署嚮導] 會讀取設定檔案，並將 XML 設定檔寫入本機電腦。
    
9. 完成**執行命令**程式後，請按一下 **[完成]**。
    
10. 在 [部署嚮導] 中，按一下 [**步驟 2]。設定或移除商務用 Skype Server 元件**。 然後，此嚮導會安裝在儲存在本機電腦上的 XML 設定檔中指定的商務用 Skype Server Edge 元件。
    
11. 安裝完成後，您可以移至下方 [**憑證**] 區段中的步驟。
    
## <a name="certificates"></a>證書

Edge 伺服器的憑證需求可以在邊緣認證規劃檔中找到。 設定憑證的步驟如下。
  
> [!NOTE]
> 在執行 [憑證] 嚮導時，您必須以具有您要使用之憑證範本類型之正確許可權的帳戶登入。 根據預設，商務用 Skype 伺服器憑證要求將會使用 Web 服務器憑證範本。 如果您是使用 RTCUniversalServerAdmins 群組成員的帳戶登入，以透過這個範本申請憑證，請重複檢查以確認群組已獲指派 [註冊] 許可權，才能使用該範本。
