# <a name="deploy-instant-messaging-and-presence-in-skype-for-business-server"></a>在商務用 Skype Server 中部署立即訊息與顯示狀態
 
**摘要：** 深入瞭解在商務用 Skype Server 中進行立即訊息與顯示狀態的部署步驟。
  
## <a name="deployment-steps-for-instant-messaging-and-presence"></a>立即訊息與顯示狀態的部署步驟

立即訊息與顯示狀態的規劃文章，可引導您瞭解規劃部署的詳細資訊。 當您安裝商務用 Skype Server 時，立即訊息和目前狀態會啟用。 若要深入瞭解規劃立即訊息和目前狀態，請參閱 [在商務用 Skype Server 中規劃立即訊息和目前狀態](../../plan-your-deployment/instant-messaging-and-presence.md)。
  
在本節中，您將瞭解您可以自訂的一些部署步驟，以微調立即訊息和目前狀態案例。
