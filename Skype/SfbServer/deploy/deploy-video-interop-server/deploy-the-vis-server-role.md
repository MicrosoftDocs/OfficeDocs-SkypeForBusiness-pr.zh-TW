# <a name="deploy-the-vis-server-role-in-skype-for-business-server"></a>在商務用 Skype Server 中部署 VIS 伺服器角色
 
**摘要：** 在商務用 Skype Server 中部署 (VIS) 角色的視頻 Interop 伺服器。
  
若要在剛剛在拓撲產生器中建立的伺服器上設定 VIS 服務，請啟動商務用 Skype Server 部署嚮導，按下 **安裝或更新商務用 Skype Server 系統** ，然後在嚮導中執行下列步驟：
  
1.  選取 [ **安裝本機設定存放區**]。
    
2. 選取 [ **設定] 或 [移除商務用 Skype Server 元件**]。
    
3. 選取 [ **要求]，安裝或指派憑證**。
    
4. 選取 [ **啟動服務**]。
    
目前已安裝及執行此服務的軟體。 您可以開啟服務 mmc 工具，以查看商務用 **Skype Server Video Interop 伺服器** 服務是否與其他商務用 skype server 服務一起執行。 接下來，您必須設定 VIS 伺服器或集區。
