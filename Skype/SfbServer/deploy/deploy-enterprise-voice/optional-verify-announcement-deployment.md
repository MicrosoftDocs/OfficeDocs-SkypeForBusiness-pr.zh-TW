# <a name="optional-verify-announcement-deployment-in-skype-for-business"></a>可選在商務用 Skype 中驗證宣告部署
 
在商務用 Skype Server Enterprise Voice 中驗證發佈應用程式的部署。 這會影響如何處理未指派數位的呼叫。
  
安裝並設定宣告之後，您必須驗證設定，以確保呼叫未指派的號碼會如預期的那樣運作。 至少請確認下列事項：
  
- 撥打對您組織有效但不是未指派號碼的號碼。
    
- 撥打未指定的電話號碼，並確認正確的宣告是否可播放。
