# <a name="optional-verify-announcement-deployment-in-skype-for-business"></a> (選用) 在商務用 Skype 中驗證宣告部署
 
在商務用 Skype Server Enterprise Voice 中驗證宣告應用程式的部署。 這會影響如何處理未指派號碼的呼叫。
  
一旦您安裝並設定好「宣告」，您需要確認設定以確保可以順利撥打未指派的號碼。請至少確認下列項目：
  
- 撥打對您的組織有效的號碼，但它是未指派的號碼。
    
- 撥打未指派號碼，並確認播放正確的宣告。
