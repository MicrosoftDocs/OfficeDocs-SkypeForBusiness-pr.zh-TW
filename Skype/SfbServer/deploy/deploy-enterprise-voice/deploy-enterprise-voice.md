**摘要：** 瞭解如何在中央網站部署商務用 Skype Server 的企業語音。

使用本主題在中央網站部署企業語音。 若要在分支網站上部署企業語音，請跳過[部署分支網站](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-branch-sites)。
