# <a name="create-or-modify-a-voice-policy-and-configure-pstn-usage-records-in-skype-for-business"></a>在商務用 Skype 中建立或修改語音原則及設定 PSTN 使用記錄

**摘要：** 使用商務用 Skype Server 的 [控制台] 建立或修改語音原則，並設定 PSTN 使用記錄。

> [!NOTE]
> 每個語音原則都必須至少有一個相關聯的公用交換電話網絡（PSTN）使用量記錄。 若要查看企業語音部署中所有可用的 PSTN 使用記錄清單，並查看其屬性，請參閱[在商務用 Skype 中查看 pstn 使用狀況記錄](view-pstn-usage-records.md)。

### <a name="to-create-a-voice-policy"></a>建立語音原則

1. 開啟商務用 Skype Server 的 [控制台]。

2. 在左側導覽列中，按一下 [**語音路由**]，然後按一下 [**語音策略**]。

3. 在 [**語音原則**] 頁面上，按一下 [**新增**]，然後選取新原則的範圍：

   - 除了任何指派給使用者原則的使用者或群組之外，**網站原則**也適用于整個網站。 如果您為原則範圍選取 [網站]，請從 [**選取網站**] 對話方塊中選擇網站。 如果已為網站建立語音原則，該網站不會出現在 [**選取網站**] 對話方塊中。

   - **使用者原則**可以套用至指定的使用者或群組。

4. 如果語音原則範圍是 User，請在 [Name] （**名稱**）欄位中，輸入原則的描述性名稱。

    > [!NOTE]
    > 如果語音原則範圍是 [網站]，**新語音原則**中的 [Name] （**名稱**）欄位會預先填入網站名稱，且無法變更。

5. 可選輸入語音原則的其他描述性資訊。

6. 選取或清除下列核取方塊，以啟用或停用此語音原則的每個**通話功能**：

   - **語音信箱轉義**可防止呼叫在同時進行同時撥打，且手機已關閉、不在電池或超出範圍時，立即路由到使用者的行動電話語音信箱系統。

     > [!NOTE]
     > 此功能僅可透過商務用 Skype Server Management 命令介面進行設定

   - [**來電轉接**] 可讓使用者將來電轉接至其他電話與用戶端裝置。 商務用 Skype 伺服器為來電轉接提供了相當廣泛的配置選項。 例如，如果組織不想允許將來電轉寄至 PSTN，系統管理員可以套用特殊的語音原則來部署此限制。 預設為啟用。

   - **委派**可讓使用者指定其他使用者代表自己傳送和接收來電。 在商務用 Skype Server 中，代理人可以設定同時撥打的電話，讓其主管的來電撥打所有代理人同時撥打的目標。 這可為委派提供更大的彈性，以回應導向給 manager 的呼叫。 預設為啟用。

   - [**來電轉接**] 可讓使用者將來電轉接給其他使用者。 預設為啟用。

   - **通話寄存**可讓使用者保留通話，然後從不同的電話或用戶端挑選通話。 預設為停用。

   - **同時撥打**可讓來電撥打其他手機（例如行動電話）或其他端點裝置。 商務用 Skype 伺服器提供大量的配置選項來同時撥打。 預設為啟用。

   - **團隊通話**可讓已定義的小組中的使用者接聽小組其他成員的來電。 預設為啟用。

   - **PSTN 重新路由**可讓指派此原則的使用者所進行的呼叫，如果 WAN 擁塞或無法使用，則會在 PSTN 上重新路由。 預設為啟用。

   - [**頻寬原則覆蓋**] 可讓系統管理員覆寫特定使用者的呼叫許可控制原則決定。 預設為停用。

     > [!NOTE]
     > 此原則只會覆寫給來電給使用者，而不會覆寫給使用者發出的撥出通話。 會話建立之後，就會精確地記錄頻寬使用量。 此設定應該謹慎使用，且應保留適當的呼叫許可控制決定。

   - **惡意的通話追蹤功能**可讓使用者使用用戶端 UI 來舉報惡意通話（例如威脅），進而在通話詳細資料記錄（CDRs）中標示通話。 預設為停用。

   - [**忙碌] 選項**可讓您啟用或停用指定語音原則的 Busy 選項。 [忙碌] 選項可讓來電傳送至語音信箱，或當通話的目標使用者在電話上時，使用占線信號來拒絕來電。 [忙碌選項] 是2016年7月累計更新引進的新語音原則。 核取此參數可啟用 Busy 選項，取消核取則會停用 [忙碌] 選項。 如需詳細資訊，請參閱[規劃商務用 Skype server 的繁忙選項](../../plan-your-deployment/enterprise-voice-solution/busy-options.md)，以及[安裝及設定商務用 Skype Server 的 busy 選項](install-and-configure-busy-options.md)。

7. 若要關聯及設定此語音原則的 PSTN 使用記錄，請執行下列任何一項操作：

   - 若要從企業語音部署中所有可用的 PSTN 使用記錄清單中選擇一或多筆記錄，請按一下 [**選取**]。 醒目提示您想要與此語音原則相關聯的記錄，然後按一下 **[確定]**。

     是. 按一下 [**新增**]。

     乙. 在 [**名稱**] 欄位中，為記錄輸入唯一的描述性名稱。 例如，您可能會想要為位於雷蒙德的全職員工建立 PSTN 使用記錄 namedRedmond，以及針對暫時員工的另一個 namedRedmondTemps。

     > [!NOTE]
     > PSTN 使用記錄名稱在企業語音部署中必須是唯一的。 在儲存記錄之後，[**名稱**] 欄位就無法進行編輯。

   - 若要從企業語音部署中所有可用路由的清單中選擇一或多個路由，請按一下 [**選取**]，然後醒目提示您要與此 PSTN 使用記錄建立關聯的路由，然後按一下 **[確定]**。

   - 若要從 PSTN 使用量記錄移除路由，請將路由反白顯示，然後按一下 [**移除**]。

   - 若要定義新的路由，並將它與此 PSTN 使用量記錄建立關聯，請按一下 [**新增**]。 如需詳細資訊，請參閱[在商務用 Skype 中建立或修改語音路線](create-or-modify-a-voice-route.md)。

   - 若要從企業語音部署中的所有可用路由清單中選擇一或多個路由，請按一下 [**選取**]，然後醒目提示您要與此 PSTN 使用記錄建立關聯的路由，然後按一下 **[確定]**。

   - 若要從這個 PSTN 使用記錄移除路由，請將路由反白顯示，然後按一下 [**移除**]。

   - 若要定義新的路由，並將它與此 PSTN 使用量記錄建立關聯，請按一下 [**新增**]。 如需詳細資訊，請參閱[在商務用 Skype 中建立或修改語音路線](create-or-modify-a-voice-route.md)。

   - 若要編輯已與此 PSTN 使用記錄相關聯的路線，請醒目提示 [路線]，然後按一下 [**顯示詳細資料**]。

     c-clip. 按一下 [確定]****。

8. 排列 PSTN 使用狀況記錄，以獲得最佳效能。 若要變更記錄在清單中的位置，請醒目提示記錄名稱，然後按一下向上或向下箭號。

    > [!IMPORTANT]
    > PSTN 使用記錄在語音原則中列出的順序非常重要。 商務用 Skype 伺服器會從上到下遍歷清單。 我們建議您依使用頻率來組織清單，例如： RedmondLocal、RedmondLongDist、RedmondInternational、RedmondBackup。

   - 若要允許來電轉接及同時撥打至內部商務用 Skype 使用者，請從下拉式功能表中選取 [**僅限內部商務用 skype 使用者**] 的選項。 通話不會轉寄到外部 PSTN 號碼。

    > [!NOTE]
    > 在建立語音原則時，會設定範圍和名稱。 它們無法變更。

4. 可選在 [**編輯語音原則**] 中，輸入語音原則的其他描述性資訊。
