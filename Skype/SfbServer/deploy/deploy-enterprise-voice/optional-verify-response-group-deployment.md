# <a name="optional-verify-response-group-deployment-in-skype-for-business"></a>可選在商務用 Skype 中驗證回應群組部署
 
在商務用 Skype Server Enterprise Voice 中驗證您的回應群組部署成功與否。
  
在您設定回應群組之後，您必須驗證設定，以確保您的回應群組如期運作。 至少，您可以使用下列類型的使用者來驗證下列案例：
  
 **使用者**
  
- 在商務用 Skype 中託管的使用者
    
- 使用公用交換電話網絡（PSTN）的外部使用者
    
- 在商務用 Skype 中託管的代理人
    
  **案例**
  
- 商務用 Skype 使用者會呼叫回應群組。
    
- 外部使用者呼叫回應群組。
    
- 當代理程式位於另一個通話中並移至佇列時，使用者會呼叫回應群組。

    > [!NOTE]
    > 如果回應群組無法如期運作，請選取 [下一步] 連結：https://support.office.com/en-us/article/troubleshooting-for-response-groups-ca72d8f8-4054-4974-b832-4f173611bd89
