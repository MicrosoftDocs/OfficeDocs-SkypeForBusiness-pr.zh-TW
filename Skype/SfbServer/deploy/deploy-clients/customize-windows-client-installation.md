# <a name="customize-windows-client-installation-in-skype-for-business-server"></a>在商務用 Skype Server 中自訂 Windows 用戶端安裝
 
**摘要：** 商務用 Skype 之安裝方法及工具的概述。
  
> [!NOTE]
> 如需 Microsoft 365 和 Office 365 隨附的商務用 Skype 相關安裝資訊，請參閱[在 microsoft 365 或 office 365 中部署商務用 skype 用戶端](https://support.office.com/article/8c563b81-22c9-4024-9efe-9fe28c7bbc96)。 
  
企業系統管理員可以使用本節所述的方法，來自訂 Windows Installer 型商務用 Skype 版本授權版本的 Windows Installer （.msi）安裝。 由於沒有單一工具提供所有自訂選項，因此您可能會在商務用 Skype 部署中使用這些方法的組合。 您可以使用下列各節所述的工具：
  
- [使用商務用 Skype Server 中的 Office 自訂工具（OCT）](use-the-office-customization-tool-oct.md) ，為商務用 skype 和其他 Office 程式自訂安裝選項及功能。
    
- [使用 config.xml 在商務用 Skype 伺服器中執行安裝](use-config-xml-to-perform-installation-tasks.md)工作，以指定網路安裝點的路徑，並執行無訊息安裝。
    
- [使用商務用 Skype Server 中的安裝程式命令列選項](use-setup-command-line-options.md)，指定要在安裝期間使用的 config.xml 檔案。
    
- 使用群組原則物件編輯器 MMC 嵌入式管理單元，[在商務用 Skype Server 中設定用戶端引導原則](configure-client-bootstrapping-policies.md)。
    
當您部署 Office 產品套件時，可能會有其他的選項可供您設定。 本節中的主題會概括介紹這些自訂工具，並討論商務用 Skype 所特有的考慮。 包含的連結為每個工具的詳細 Office 說明。
