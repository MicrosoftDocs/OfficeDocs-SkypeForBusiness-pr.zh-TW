# <a name="install-sql-server-reporting-services-in-skype-for-business-server"></a>在商務用 Skype Server 中安裝 SQL Server Reporting Services 
 
**摘要：** 瞭解在哪裡可以找到商務用 Skype Server 所使用之 SQL Server Reporting Services 的相關資訊。
  
商務用 Skype 伺服器可以使用 SQL Server Reporting Services （SSRS）來查看和監控報告。 若要使用此功能，您必須安裝並設定 Reporting Services。
  
## <a name="install-sql-server-reporting-services"></a>安裝 SQL Server Reporting Services

如果您想要使用商務用 Skype Server Monitoring 報告（請參閱本檔的下一篇文章以取得詳細資訊），您必須先安裝 SQL Server Reporting Services;您可以在安裝 Microsoft SQL Server 或安裝 SQL Server 之後的任何時間安裝 Reporting Services。 如果您尚未安裝 SQL Server，請依照本檔前面所提供的指示進行。 安裝 SQL Server 時，請確定在 [功能選擇] 頁面上選取 [Reporting Services]。 這會安裝 SQL Server Reporting Services。
  
若要瞭解如何安裝 SQL Server Reporting Services，請參閱[Sql Server Reporting services （SSRS）](https://technet.microsoft.com/en-us/library/ms159106.aspx)。
  
如果您已安裝 SQL Server，但未安裝 SQL Server Reporting Services，您可以遵循適當的 SQL Server Reporting Services 指示集來新增該功能。
