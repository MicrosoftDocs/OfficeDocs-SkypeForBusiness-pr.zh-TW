<span data-ttu-id="76b1b-591">服務品質 (QoS) 是在一些組織中使用的網路技術, 可協助為音訊與視頻通訊提供最佳的使用者體驗。</span><span class="sxs-lookup"><span data-stu-id="76b1b-591">Quality of Service (QoS) is a networking technology that is used in some organizations to help provide an optimal end-user experience for audio and video communications.</span></span> <span data-ttu-id="76b1b-592">QoS 最常用於頻寬有限的網路上: 有大量網路資料包爭用相當少量的可用頻寬, QoS 可讓系統管理員為攜帶音訊或視頻資料的資料包指派較高的優先順序。</span><span class="sxs-lookup"><span data-stu-id="76b1b-592">QoS is most frequently used on networks where bandwidth is limited: with a large number of network packets competing for a fairly small amount of available bandwidth, QoS enables administrators to assign higher priorities to packets carrying audio or video data.</span></span> <span data-ttu-id="76b1b-593">透過將這些資料包設為較高的優先順序, 音訊與視頻通訊可能會更快且中斷較少, 而不是包括檔案傳輸、網頁流覽或資料庫備份等專案的網路會話。</span><span class="sxs-lookup"><span data-stu-id="76b1b-593">By giving these packets a higher priority, audio and video communications are likely to complete faster, and with less interruption, than network sessions involving things such as file transfers, web browsing, or database backups.</span></span> <span data-ttu-id="76b1b-594">這是因為用於檔案傳輸或資料庫備份的網路資料包已獲指派「最大努力」的優先順序。</span><span class="sxs-lookup"><span data-stu-id="76b1b-594">That's because network packets used for file transfers or database backups are assigned a "best effort" priority.</span></span>
