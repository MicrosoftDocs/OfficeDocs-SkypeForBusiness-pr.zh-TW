# <a name="requirements-for-your-skype-for-business-2015-environment"></a>商務用 Skype 2015 環境的需求
 
**摘要：** 針對商務用 Skype Server 2015 的系統需求和網路或基礎結構需求的主題大綱。 如果您需要安裝或設定任何專案作為安裝及部署商務用 Skype Server 2015 的先決條件，請參閱這裡。
  
這是適用于商務用 Skype Server 2015 部署之需求之文章的 [登陸] 頁面。 這包括伺服器需求及準備的一般事項，包括您的網域和網路結構，以及憑證。
  
## <a name="whats-in-this-topic"></a>本主題內容

您的環境需求區段中的兩篇文章概括說明您需要掌握及/或設定的專案，才能在貴組織的伺服器環境中繼續部署或升級至商務用 Skype Server 2015。
  
### <a name="server-requirements"></a>伺服器需求

[商務用 Skype server 2015 的伺服器需求](server-requirements.md)是您準備好要在部署中使用的個別伺服器上所需的專案。
  
### <a name="environmental-requirements"></a>環境需求

[商務用 Skype server 2015 的環境需求](environmental-requirements.md)，是在部署商務用 Skype Server 2015 之前，必須準備好的基礎結構。
