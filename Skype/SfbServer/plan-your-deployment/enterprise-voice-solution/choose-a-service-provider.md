# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>為商務用 Skype Server 選擇 E9-1-1 服務提供者
 
在商務用 Skype Server 企業語音中為 E9-1-1 部署選擇服務提供者。
  
E9-1-1 服務提供者會根據通話中所含的位置資訊，將源自商務用 Skype Server 的緊急通話路由傳送到正確的公用安全回應點 (PSAP) 。 
  
若要支援 E9-1-1 做為商務用 Skype Server 部署的一部分，您必須從 Lync Open 互通性計畫合格的 E9-1-1 服務提供者取得 E9-1-1 的路由服務。 請選擇最符合您組織需求的提供者。

若要查看 Lync 的 E9-1-1 路由服務清單，請參閱 [E-911 service 合格的 lync](../../../SfbPartnerCertification/lync-cert/e-911-service-providers.md)。
