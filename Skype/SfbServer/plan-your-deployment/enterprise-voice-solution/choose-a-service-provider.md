# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>選擇商務 Server skype E9-1-1 服務提供者
 
選擇 E9-1-1 部署的服務供應商 skype 商務 Server Enterprise Voice。
  
E9-1-1 服務提供者的路由緊急通話源自 Skype for Business Server 到正確公用安全回應點 (PSAP) 依據通話中所含的位置資訊。 
  
若要支援 E9-1-1 Skype for Business Server 部署的一部分，您必須取得 E9-1-1 路由服務 Lync Open Interoperability Program 合格 E9-1-1 服務提供者。 請選擇最符合您組織需求的提供者。

若要查看的 Lync 的 E9-1-1 路由服務清單，請參閱[E-911 Services 完整 lync](https://docs.microsoft.com/skypeforbusiness/lync-cert/e-911-service-providers)。
