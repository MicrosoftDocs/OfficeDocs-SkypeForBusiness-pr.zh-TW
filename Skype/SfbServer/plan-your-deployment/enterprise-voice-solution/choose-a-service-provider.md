# <a name="choose-an-e9-1-1-service-provider-for-skype-for-business-server"></a>選擇適用于商務用 Skype Server 的 E9-1 服務提供者
 
在商務用 Skype Server Enterprise Voice 中, 為 E9-1-1 部署選擇您的服務提供者。
  
E9-1-1 服務提供者會根據通話中所包含的位置資訊, 傳送從商務用 Skype 伺服器產生的緊急通話。 
  
若要支援 E9-1 作為商務用 Skype Server 部署的一部分, 您必須從 Lync 開啟互通性計畫合格 E9-1-1 服務提供者處取得 E9-1-1 路由服務。 選擇最符合貴組織需求的提供者。

若要查看 Lync 的 E9 路由服務清單, 請參閱適用[于 lync 的 E-911 服務](https://docs.microsoft.com/en-us/skypeforbusiness/lync-cert/e-911-service-providers)。
