# <a name="mobile-client-feature-comparison-for-skype-for-business"></a>商務用 Skype 的行動用戶端功能比較
 
**摘要:** 在規劃商務用 Skype Server 時, 請複習行動用戶端的功能支援。
  
本文將比較商務用 Skype 行動用戶端與商務用 Skype 桌面用戶端在下列類別中的功能和功能:
  
- [登入]、[推播通知] 和 [一般] 功能
    
- 增強的目前狀態
    
- 連絡人與連絡人群組
    
- 立即訊息 (IM)
    
- 商務用 skype 的商務用 skype 音訊和影片
    
- 會議
    
- $
    
- 外部使用者
    
- 封存與合規性
    
-  新式驗證
    
下表列出商務用 Skype Server 內部部署中的商務用 Skype 使用者所能使用的功能。 商務用 Skype Online 和 Microsoft Office 365 使用者也可以使用相同的功能, 除非表格註腳中另有指示。
  
> [!NOTE]
> 如需適用于使用者的線上說明和資源, 請參閱[探索商務用 Skype](https://go.microsoft.com/fwlink/p/?LinkId=528686)。 
  
> [!NOTE]
> 若要比較其他商務用 Skype 用戶端中提供的功能, 請參閱[商務用 skype 的桌面用戶端功能比較](desktop-feature-comparison.md)。 

> [!NOTE]
> MCX (行動服務) 對舊版行動用戶端的支援已不再提供給商務用 Skype Server 2019。 所有目前的商務用 Skype 行動用戶端都已使用整合通訊 Web API (UCWA) 來支援立即訊息 (IM)、目前狀態和連絡人。 使用 MCX 的舊版用戶端的使用者將需要升級至目前的用戶端。
