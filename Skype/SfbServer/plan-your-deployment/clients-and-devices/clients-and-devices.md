# <a name="plan-for-clients-and-devices"></a>規劃用戶端和裝置

**摘要：** 複查商務用 Skype 支援的用戶端和應用程式。

如今的工作力會持續移動。 員工必須進行通訊及共同作業，不論是從公司辦公室、地區位置、家庭辦公室還是在旅途中運作。 商務用 Skype Server 會透過您可以部署至組織使用者的用戶端介面集合，支援這些需求。 周密的規劃可確保員工能夠取得所需的內容，而且在任何地方都可以使用商務用 Skype。

## <a name="available-clients"></a>可用的用戶端

商務用 Skype Server 支援多種類型的用戶端，包括電腦安裝的用戶端軟體、網頁型用戶端，以及行動裝置的用戶端。 在此區段中會引入主要用戶端，針對所有支援的用戶端，請參閱[商務用 Skype Server 2015 的桌面用戶端功能比較](desktop-feature-comparison.md)或[商務用 Skype Server 2019 的桌面用戶端功能比較](../../../SfBServer2019/plan/feature-comparison.md)。 如果您先前使用的是 Lync 用戶端的組合，請注意，不受支援的[舊版用戶端](clients-and-devices.md#Legacy)與商務用 Skype Server 2019 不相容。 更新定期進行，因此請定期查看此主題以取得最新的用戶端資訊。

### <a name="skype-for-business-2019"></a>商務用 Skype (2019) 

商務用 Skype (2019) 是建議的完整功能用戶端，以供商務用 Skype Server 2015 和2019使用。 如需新功能的描述，請參閱[遵循商務用 Skype 中的最新更新](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781)。 [商務用 Skype 的桌面用戶端功能比較](desktop-feature-comparison.md)中詳細說明了用戶端功能支援，使用者檔則是[商務用 Skype](https://support.office.com/Skype-for-business)的說明。 當使用者安裝 Microsoft 365 或 Office 365 時，會包含此用戶端。

也可以使用可支援較少功能的免費基本用戶端。 這兩個版本都可以在[所有裝置](https://products.office.com/skype-for-business/download-app?tab=tabs-3)上下載商務用 Skype 下載。 「 [基本用戶端限制](desktop-feature-comparison.md#Full-Basic) 」一節將說明完全和基本用戶端之間的差異。

### <a name="skype-for-business-2016"></a>商務用 Skype 2016

商務用 Skype 2016 是商務用 Skype Server 2015 或2019的全功能用戶端。 如需新功能的說明，請參閱[商務用 Skype 2016 的新](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781)功能。 [商務用 Skype 的桌面用戶端功能比較](desktop-feature-comparison.md)中詳細說明了用戶端功能支援，使用者檔則是[商務用 Skype](https://support.office.com/Skype-for-business)的說明。 當使用者安裝 Office 365 時會包含此用戶端。

也可以使用可支援較少功能的免費基本用戶端。 這兩個版本都可以在[所有裝置](https://products.office.com/skype-for-business/download-app?tab=tabs-3)上下載商務用 Skype 下載。 「 [基本用戶端限制](desktop-feature-comparison.md#Full-Basic) 」一節將說明完全和基本用戶端之間的差異。

### <a name="skype-for-business-2015"></a>商務用 Skype 2015

商務用 Skype 2015 是商務用 Skype Server 2015 或2019的全功能用戶端。 商務用 Skype 使用者介面已完全重新設計，包含新的整合功能（例如通話監視器、Skype 目錄整合、圖釋等等）。 如需變更的摘要，請參閱[Lync 現在商務用 Skype —請參閱](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612)最近更新。 [商務用 Skype 的桌面用戶端功能比較](desktop-feature-comparison.md)中詳細說明了用戶端功能支援，使用者檔則是[商務用 Skype](https://support.office.com/Skype-for-business)的說明。 當使用者安裝 Office 365 時會包含此用戶端。

### <a name="skype-for-business-on-mac"></a>Mac 版商務用 Skype

Mac 用戶端[上的商務用 Skype](https://www.microsoft.com/download/details.aspx?id=54108)可供下載。 請參閱[商務用 Skype Mac 用戶端需求](mac-requirements.md)，以複查必要條件。

### <a name="skype-for-business-for-mobile-devices"></a>行動裝置的商務用 Skype

用戶端可用於 Windows Phone、iPhone/iPad 和 Android。 使用者可以在[所有裝置上下載商務用 Skype](https://products.office.com/skype-for-business/download-app?tab=tabs-3)。 在商務用 Skype 的行動[用戶端功能比較](mobile-feature-comparison.md)中詳細說明這些用戶端的功能支援。

> [!NOTE]
> 商務用 Skype Server 2019 不再提供舊版行動用戶端的 MCX (行動服務) 支援。 所有目前商務用 Skype 的行動裝置都已經使用整合通訊 Web API (UCWA) 以支援立即訊息 (IM) 、目前狀態及連絡人。 具有使用 MCX 之舊版用戶端的使用者，必須升級至目前的用戶端。

### <a name="online-meeting-add-in-for-skype-for-business"></a>商務用 Skype 的線上會議增益集

商務用 Skype 的線上會議增益集支援來自 Microsoft Outlook 郵件與共同作業用戶端 Windows 上的會議管理。 商務用 Skype 軟體的線上會議增益集會自動安裝商務用 Skype。

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>商務用 Skype Web 應用程式和 Skype 會議應用程式

如果使用者的電腦上未安裝商務用 Skype，且使用者在 Windows 電腦上按一下會議邀請中的會議連結，則會安裝及開啟 Skype 會議應用程式或商務用 Skype Web 應用程式。  Skype會議應用程式是您組織外部參與者的選擇用戶端。  (請注意，mac 上的商務用 Skype 會安裝及開啟。 ) 請參閱[Plan for meeting 用戶端 (Web 應用程式和會議應用程式) ](meetings-clients.md) ，以瞭解使用這些用戶端的需求。


### <a name="skype-for-business-web-scheduler"></a>商務用 Skype Scheduler

[商務用 Skype Scheduler](https://sched.lync.com)是商務用 Skype 線上使用者的 Web 會議排程和管理工具，其使用者無法存取 Microsoft Outlook，或是不是以 Windows 的作業系統為基礎的使用者。 透過商務用 Skype Scheduler，使用者可以使用其偏好的電子郵件程式，建立新的會議、修改現有的會議及傳送邀請。 商務用 Skype Scheduler[檔](https://support.office.com/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US)提供進一步的詳細資料。

### <a name="vdi-plugins"></a>VDI 外掛程式

在安全性與合規性問題尤其敏感的某些組織中，會使用 (VDI) 環境的虛擬桌面基礎結構。 在連線時使用商務用 Skype 搭配完整的音訊和影片，需要在主機電腦上的用戶端上大量載入音訊和視頻處理。 其他的 VDI 外掛程式軟體可讓您將處理常式降至使用者的本機電腦，並減少虛擬機器的負載。 如需使用這些外掛程式的詳細資訊，請參閱[規劃 VDI 環境中的商務用 Skype](vdi-environments.md) 。

### <a name="microsoft-teams-rooms"></a>Microsoft Teams 會議室

Microsoft Teams 會議室是 Microsoft 最新的會議解決方案，其使用熟悉的介面，而且易於部署和管理，利用現有的設備（如 LCD 面板）進行安裝，以方便安裝。 Microsoft Teams 會議室使用專門建立的 UWP 應用程式，在主控台模式中執行 Surface Pro 4 或 Surface Pro 時 (部署的 uwp 應用程式是在裝置上執行的唯一應用程式) 而且需要在您的實施上使用自己的裝置帳戶。 軟體會透過 Windows 儲存區和 Windows 更新進行更新。 https://aka.ms/MTRDocs如需在您的部署中使用這些會議室主控台的詳細資訊，請參閱。 

### <a name="skype-for-business-on-surface-hub"></a>Surface Hub 上的商務用 Skype

Microsoft Surface Hub 是一種全一體生產力裝置，其適用于集體討論會、共同作業及簡報。 其商務用 Skype 用戶端的小小小小[Microsoft Surface Hub 系統管理員手冊](/surface-hub/)所述。

## <a name="choosing-your-organizations-preferred-client"></a>選擇您組織的慣用用戶端
<a name="BK_client_choose"> </a>

如果您的組織購買了適當的授權，請選擇完整的用戶端，否則請選擇基本用戶端。

您的使用者可以自行安裝用戶端，以在[所有裝置上下載商務用 Skype](https://products.office.com/skype-for-business/download-app?tab=tabs-3)。 當使用者在 Windows 上安裝 Microsoft 365 或 Office 365 時，也會安裝用戶端。 如果您的部分使用者有 Mac，這些使用者會有一組不同的功能，如先前章節所述。

商務用 Skype 線上或商務用 Skype Server 2019 中無法使用商務用 Skype Server 2015 的某些功能，請參閱[線上或混合使用者帳戶的 2015](desktop-feature-comparison.md#Online-Hybrid)或[線上或](desktop-feature-comparison.md#Online-Hybrid)混合使用者帳戶限制，以取得2019的詳細資料。 商務用 Skype線上系統管理員可能會想要參考[商務用 Skype 線上服務說明](/office365/servicedescriptions/skype-for-business-online-service-description/skype-for-business-online-service-description)，以瞭解可用之不同方案的詳細資訊。

 在您部署或升級至商務用 Skype 之前，請檢查您的組織中已使用的用戶端。 [針對商務用 Skype 使用桌面用戶端功能比較](desktop-feature-comparison.md)，以瞭解對這些用戶端的功能支援影響。 這可協助您向使用者傳達變更、節奏向外延展程式，以及充分瞭解升級為最新用戶端的益處。

## <a name="ways-to-deploy-the-client-to-your-users"></a>將用戶端部署至使用者的方式
<a name="BK_User_Deploy"> </a>

用戶端安裝程式適用于 MSI 及隨選即用類型安裝程式。 商務用 Skype 用戶端 sustainment 策略可能會影響您的選擇，所以您應該瞭解下列幾點：

- 一般商務用 Skype 不會將新功能新增至先前發佈的用戶端

- 一般說來商務用 Skype 不會在初始發行後，在商務用 Skype MSI 中計畫新功能。 版本間的 MSI 增強功能主要是高品質/安全性。

- 在商務用 Skype 2019 隨選即用安裝程式中，會找到最新且最佳的商務用 Skype 用戶端體驗。

您可以依照商務用 Skype Server 中的[自訂 Windows 用戶端安裝](../../deploy/deploy-clients/customize-windows-client-installation.md)所述，執行用戶端的自訂部署。 [部署商務用 Skype Server 的用戶端](../../deploy/deploy-clients/deploy-clients.md)時，詳細說明安裝方法

### <a name="click-to-run"></a>隨選即用

隨選即用是 Microsoft 資料流程和虛擬化技術，您可以用來安裝及更新包括商務用 Skype 在內的 Office 產品。 這些流式處理和虛擬化功能是以 Microsoft Application Virtualization (App-V) 的技術為基礎。 隨選即用具有下列優點：

- Office 套件的資料流程安裝產生短安裝時間

- 已集成的更新及修補程式

- 佔用較少的磁碟空間

- 使用者基礎授權：每位使用者5安裝5個

- 可透過 XML 編輯器自訂，以安裝獨立程式

您可能想要使用此安裝程式類型的[Office 部署工具](https://www.microsoft.com/download/details.aspx?id=49117)。

您可以使用隨選即用安裝程式來 (基本和完整用戶端版本) ，也可以透過隨 64 32 選即用安裝程式來下載，您的使用者可以在[所有裝置](https://products.office.com/skype-for-business/download-app?tab=tabs-3)上下載商務用 Skype。

### <a name="msi"></a>MSI

MSI 是一種較為傳統的安裝方法，用於商務用 Skype 2015 和2016用戶端。 它可讓您手動安裝更新及修補程式、使用大量授權和啟用，也可以透過[Office 自訂工具](https://www.microsoft.com/download/details.aspx?id=49030)加以自訂。 您可以運用群組原則，使用 Microsoft Endpoint Configuration Manager，或使用協力廠商工具來散佈用戶端。
