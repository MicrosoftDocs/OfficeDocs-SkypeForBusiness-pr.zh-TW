# <a name="plan-for-clients-and-devices"></a>規劃用戶端和裝置

**摘要：** 檢查支援的用戶端和商務用 Skype 相關應用程式。

如今的工作力會持續移動。 員工必須進行通訊及共同作業，不論是從公司辦公室、地區位置、家庭辦公室還是在旅途中運作。 商務用 Skype 伺服器支援這些需求，透過您可以部署至組織之使用者的用戶端介面集合。 周密的規劃可確保員工能夠取得所需的內容，以及商務用 Skype 在任何地方都可供他們使用。

## <a name="available-clients"></a>可用的用戶端

商務用 Skype 伺服器支援多種類型的用戶端，包括電腦安裝的用戶端軟體、網頁型用戶端，以及行動裝置的用戶端。 在此區段中會引入主要用戶端，針對所有支援的用戶端，請參閱商務用 skype [server 2015 的桌面用戶端功能比較](desktop-feature-comparison.md)或[商務用 skype Server 2019 的桌面用戶端功能比較](../../../SfBServer2019/plan/feature-comparison.md)。 如果您先前使用的是 Lync 用戶端的組合，請注意，不受支援的[舊版用戶端](clients-and-devices.md#Legacy)與商務用 Skype Server 2019 不相容。 更新定期進行，因此請定期查看此主題以取得最新的用戶端資訊。

### <a name="skype-for-business-2019"></a>商務用 Skype （2019）

商務用 skype （2019）是商務用 Skype Server 2015 和2019的完整功能用戶端。 請參閱[依照商務用 Skype 中的最新更新](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781)，以取得新功能的描述。 您可以在商務用 skype 的[桌面用戶端功能比較](desktop-feature-comparison.md)中詳細說明用戶端功能支援，以及使用者檔位於[商務用 skype 協助](https://support.office.com/Skype-for-business)。 當使用者安裝 Microsoft 365 或 Office 365 時，就會包含此用戶端。

也可以使用可支援較少功能的免費基本用戶端。 這兩種版本都可以在[所有裝置上下載商務用 Skype](https://products.office.com/skype-for-business/download-app?tab=tabs-3)。 「[基本用戶端限制](desktop-feature-comparison.md#Full-Basic)」一節將說明完全和基本用戶端之間的差異。

### <a name="skype-for-business-2016"></a>商務用 Skype 2016

商務用 skype 2016 是商務用 Skype Server 2015 或2019的全功能用戶端。 如需新功能的說明，請參閱[商務用 Skype 2016 的新](https://support.office.com/article/What-s-new-in-Skype-for-Business-2016-cece9f93-add1-4d93-9a38-56cc598e5781)功能。 您可以在商務用 skype 的[桌面用戶端功能比較](desktop-feature-comparison.md)中詳細說明用戶端功能支援，以及使用者檔位於[商務用 skype 協助](https://support.office.com/Skype-for-business)。 當使用者安裝 Office 365 時，就會包含此用戶端。

也可以使用可支援較少功能的免費基本用戶端。 這兩種版本都可以在[所有裝置上下載商務用 Skype](https://products.office.com/skype-for-business/download-app?tab=tabs-3)。 「[基本用戶端限制](desktop-feature-comparison.md#Full-Basic)」一節將說明完全和基本用戶端之間的差異。

### <a name="skype-for-business-2015"></a>商務用 Skype 2015

商務用 skype 2015 是商務用 Skype Server 2015 或2019的全功能用戶端。 商務用 Skype 使用者介面已完全重新設計，包含新的整合式功能，例如通話監視器、Skype 目錄整合、圖釋等等。 如需變更的摘要，請參閱[Lync 現在是商務用 Skype-查看](https://support.office.com/en-in/article/aba02d7e-c801-4a82-bccd-e7207240f612)最近更新。 您可以在商務用 skype 的[桌面用戶端功能比較](desktop-feature-comparison.md)中詳細說明用戶端功能支援，以及使用者檔位於[商務用 skype 協助](https://support.office.com/Skype-for-business)。 當使用者安裝 Office 365 時，就會包含此用戶端。

### <a name="skype-for-business-on-mac"></a>Mac 版商務用 Skype

Mac 用戶端的[商務用 Skype](https://www.microsoft.com/download/details.aspx?id=54108)可供下載。 請參閱[Mac 用戶端需求的商務用 Skype](mac-requirements.md)以查看必要條件。

### <a name="skype-for-business-for-mobile-devices"></a>行動裝置的商務用 Skype

用戶端可用於 Windows Phone、iPhone/iPad 和 Android。 使用者可以在[所有裝置上下載商務用 Skype](https://products.office.com/skype-for-business/download-app?tab=tabs-3)。 適用于商務用 Skype 的行動[用戶端功能比較](mobile-feature-comparison.md)詳述這些用戶端的功能支援。

> [!NOTE]
> 商務用 Skype Server 2019 不再提供舊版行動用戶端的 MCX （行動服務）支援。 所有目前的商務用 Skype mobile 用戶端都已經使用整合通訊 Web API （UCWA）來支援立即訊息（IM）、目前狀態及連絡人。 具有使用 MCX 之舊版用戶端的使用者，必須升級至目前的用戶端。

### <a name="online-meeting-add-in-for-skype-for-business"></a>商務用 Skype 的線上會議增益集

適用于商務用 Skype 的線上會議增益集支援來自 Windows 的 Microsoft Outlook 郵件和協同作業用戶端中的會議管理。 商務用 skype 軟體的線上會議增益集軟體會自動隨商務用 Skype 安裝。

### <a name="skype-for-business-web-app-and-skype-meetings-app"></a>商務用 skype Web App 和 Skype 會議應用程式

如果使用者的電腦上未安裝商務用 Skype，且使用者按一下 Windows 電腦上會議邀請中的會議連結，則會安裝及開啟 Skype 會議應用程式或商務用 Skype Web App。  Skype 會議應用程式是您組織外部參與者的選擇用戶端。 （請注意，mac 上的商務用 Skype 會安裝及開啟。）如需使用這些用戶端的需求，請參閱[規劃會議用戶端（Web 應用程式和會議應用程式）](meetings-clients.md) 。


### <a name="skype-for-business-web-scheduler"></a>商務用 Skype Web 排程器

[商務用 Skype web](https://sched.lync.com)排程器是一種 Web 式會議排程和管理工具，可供商務用 skype Online 的使用者使用，但沒有 Microsoft Outlook 的存取權，或是不是以 Windows 為基礎的作業系統。 使用商務用 Skype Web 排程器，使用者可以建立新的會議、修改現有的會議，以及使用其偏好的電子郵件程式傳送邀請。 商務用 Skype Web 排程程式[檔](https://support.office.com/article/Skype-for-Business-Web-Scheduler-3b24a211-6470-4a2d-81b7-22d9399d0fec?ui=en-US&amp;rs=en-US&amp;ad=US)提供進一步的詳細資料。

### <a name="vdi-plugins"></a>VDI 外掛程式

虛擬桌面基礎結構（VDI）環境適用于某些組織，其中安全性和合規性問題尤其敏感。 在連接上使用商務用 Skype （如這類連線）需要大量載入在虛擬機器上之用戶端的音訊和視頻處理。 其他的 VDI 外掛程式軟體可讓您將處理常式降至使用者的本機電腦，並減少虛擬機器的負載。 如需使用這些外掛程式的詳細資訊，請參閱[規劃 VDI 環境中的商務用 Skype For business](vdi-environments.md) 。

### <a name="microsoft-teams-rooms"></a>Microsoft Teams 會議室

Microsoft 團隊會議室是 Microsoft 最新的會議解決方案，其使用熟悉的介面，易於部署和管理，利用現有的設備（如 LCD 面板）進行安裝，以方便安裝。 Microsoft 團隊聊天室使用以主控台模式在 Surface Pro 4 或 Surface Pro 上執行的專門建立的 UWP 應用程式（部署 UWP 應用程式是裝置上只會執行的應用程式），而且需要在您的實施上使用自己的裝置帳戶。 軟體會透過「Windows store」和「Windows 更新」進行更新。 https://aka.ms/MTRDocs如需在您的部署中使用這些會議室主控台的詳細資訊，請參閱。 

### <a name="skype-for-business-on-surface-hub"></a>Surface Hub 上的商務用 Skype

Microsoft Surface Hub 是一種一體式生產力裝置，適用于集體討論會、共同作業及簡報。 它擁有自己的商務用 Skype 用戶端反覆運算，已在[Microsoft Surface Hub 系統管理中心手冊](https://docs.microsoft.com/surface-hub/)中記錄。

## <a name="choosing-your-organizations-preferred-client"></a>選擇您組織的慣用用戶端
<a name="BK_client_choose"> </a>

如果您的組織購買了適當的授權，請選擇完整的用戶端，否則請選擇基本用戶端。

您的使用者可以自行安裝用戶端，以在[所有裝置上下載商務用 Skype](https://products.office.com/skype-for-business/download-app?tab=tabs-3)。 當使用者在 Windows 上安裝 Microsoft 365 或 Office 365 時，也會安裝用戶端。 如果您的部分使用者有 Mac，這些使用者會有一組不同的功能，如先前章節所述。

在商務用 skype Online 或商務用 Skype Server 2019 中無法使用商務用 Skype Server 2015 的部分功能。如需詳細資訊，請參閱[線上或混合的使用者帳戶限制](desktop-feature-comparison.md#Online-Hybrid)（適用于2019的2015或[線上或混合使用者帳戶限制](desktop-feature-comparison.md#Online-Hybrid)）。 商務用 skype Online 系統管理員可能會想要參考[商務用 Skype Online 服務說明](https://technet.microsoft.com/library/skype-for-business-online-service-description.aspx)，以瞭解可用之不同方案的詳細資訊。

 在您部署或升級至商務用 Skype 之前，請先檢查您組織中已使用的用戶端。 使用[商務用 Skype 的桌面用戶端功能比較](desktop-feature-comparison.md)，瞭解對這些用戶端的功能支援影響。 這可協助您向使用者傳達變更、節奏向外延展程式，以及充分瞭解升級為最新用戶端的益處。

## <a name="ways-to-deploy-the-client-to-your-users"></a>將用戶端部署至使用者的方式
<a name="BK_User_Deploy"> </a>

用戶端安裝程式適用于 MSI 及隨選即用類型安裝程式。 商務用 Skype 用戶端-sustainment 策略可能會影響您的選擇，所以您應該瞭解下列幾點：

- 在一般商務用 Skype 中，不會將新功能新增至先前發佈的用戶端

- 在一般的商務用 Skype 中，不會在初次發行後，計畫在商務用 Skype MSI 中運送新功能。 版本間的 MSI 增強功能主要是高品質/安全性。

- 在商務用 Skype 2019 隨選即用安裝程式中，會找到最新和最大的商務用 Skype 用戶端體驗。

您可以像在[商務用 Skype Server 中的 [自訂 Windows 用戶端安裝](../../deploy/deploy-clients/customize-windows-client-installation.md)] 中所述，執行用戶端的自訂部署。 在[部署商務用 Skype Server 的用戶端](../../deploy/deploy-clients/deploy-clients.md)中，更詳細地說明安裝方法

### <a name="click-to-run"></a>隨選即用

隨選即用是 Microsoft 資料流程和虛擬化技術，您可以用來安裝及更新包含商務用 Skype 的 Office 產品。 這些流式處理和虛擬化功能是以 Microsoft Application Virtualization （App-V）技術為基礎。 隨選即用具有下列優點：

- 產生短安裝時間的 Office 套件資料流程安裝

- 已集成的更新及修補程式

- 佔用較少的磁碟空間

- 使用者基礎授權：每位使用者5安裝5個

- 可透過 XML 編輯器自訂，以安裝獨立程式

您可能想要針對這種類型的安裝程式使用[Office 部署工具](https://www.microsoft.com/download/details.aspx?id=49117)。

您可以使用隨選即用安裝程式來使用基本和完整的用戶端版本（可以選擇32和64位版本），您的使用者可以在[所有裝置上下載商務用 Skype](https://products.office.com/skype-for-business/download-app?tab=tabs-3)。

### <a name="msi"></a>MSI

MSI 是一種較為傳統的安裝方法，用於商務用 Skype 2015 和2016用戶端。 它可讓您手動安裝更新及修補程式、使用大量授權和啟用，也可以透過[Office 自訂工具](https://www.microsoft.com/download/details.aspx?id=49030)加以自訂。 您可以使用 Microsoft Endpoint Configuration Manager 或使用協力廠商工具來套用群組原則，以散佈用戶端。

商務用 skype Server 2019 和商務用 Skype Online 支援下列先前發佈的用戶端：商務用 Skype 2016、商務用 Skype 2015、Lync 2013。

商務用 Skype Server 2015 支援下列先前發佈的用戶端： Lync 2013、Lync 2010、Lync 2010 Mobile、Lync Phone Edition 和 Lync 2010 語音應答。 如需這些用戶端與其他伺服器搭配使用時的詳細資訊，請參閱 lync server [2013 的用戶端比較表](https://technet.microsoft.com/library/gg425836%28v=ocs.15%29.aspx)和[lync Server 2010 的用戶端比較表](https://technet.microsoft.com/library/gg425836%28v=ocs.14%29.aspx)。
