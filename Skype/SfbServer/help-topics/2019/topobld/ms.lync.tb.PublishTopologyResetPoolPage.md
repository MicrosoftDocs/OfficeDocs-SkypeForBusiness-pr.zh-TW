# <a name="publish-topology-reset-pool-page"></a>發行拓撲重設集區頁面
 
這是初步檔, 可能會變更。 空白主題是以預留位置形式提供。
  
若要將新電腦新增至現有的集區，您必須重設集區，以允許 Windows Fabric 重新定義該集區之所有成員間的連線。若要執行這項操作，請選取已新增伺服器的集區，然後按 [下一步]****。
