# <a name="office-web-apps-servers-scope-node"></a>Office Web Apps Server 範圍節點

您以滑鼠右鍵按一下 [ **Office Web Apps 伺服器**]，然後選取 [**新增 Office web apps 伺服器**] 來開啟嚮導。

![OfficeWebApps Server Scope 節點](../../../media/OfficeWebApps_Servers_Scope_Node.jpg)

- **拓撲**。您可以使用這個選項來定義新的拓撲、開啟現有的拓撲、下載目前拓撲、儲存拓撲的複本、發行拓撲、安裝資料庫、合併來自舊版的元件，或是移除部署。

- **說明**。您可以使用這個選項，存取作用中節點的說明。
