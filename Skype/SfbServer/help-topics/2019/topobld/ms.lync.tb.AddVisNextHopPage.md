# <a name="add-vis-next-hop"></a>新增 VIS 下一個躍點
 
在拓撲產生器中定義新的 VIS 或視頻閘道所需的步驟，會在 [在 [商務用 Skype Server 中建立 VIS 集](../../../deploy/deploy-video-interop-server/create-a-vis-pool.md)區] 中說明為線上。
  
在使用 Windows PowerShell 的伺服器上指派視頻 Interop 伺服器角色所需的步驟，會在在[商務用 Skype server 中的 DEPLOY VIS Server role 中](../../../deploy/deploy-video-interop-server/deploy-the-vis-server-role.md)說明為線上。
  
使用 Windows PowerShell 的新視頻 Interop 伺服器設定步驟，會在 [設定商務用 Skype server 中的視頻 Interop 伺服器](../../../deploy/deploy-video-interop-server/configure-the-vis.md)時，以線上說明。
  
 為了與 [商務用 Skype Server](../../../deploy/deploy-video-interop-server/configure-cucm-for-interoperation.md)互動，您必須在 Cisco GUI 中設定之視頻閘道的設定，才能與 VIS 交互操作。
  
 若要在 Cisco GUI 中設定之 VTC 的設定，以便與影片閘道和 VIS 進行交互操作，請在設定 [VTC 以與商務用 Skype Server 進行交互操作](../../../deploy/deploy-video-interop-server/configure-a-vtc-for-interoperation.md)時，在 online 中說明。
