# <a name="add-survivable-branch-appliance-pool"></a>新增 Survivable Branch Appliance 集區
 
若要**定義 Survivable 分支裝置 FQDN**，請在 [ **FQDN** ] 文字方塊中，輸入 [Survivable 分支裝置] 或 [Survivable 分支伺服器] 的完整功能變數名稱（FQDN）（在 [網域名稱系統（DNS）] 中定義）。 請按 [下一步]**** 繼續。
  
> [!NOTE]
> 除非另行說明，否則定義 Survivable 分支裝置或 Survivable 分支伺服器就是相同的程式。
