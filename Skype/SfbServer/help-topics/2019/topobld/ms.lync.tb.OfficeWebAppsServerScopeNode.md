# <a name="office-web-apps-server-scope-node-expander"></a>Office Web Apps Server 範圍節點展開工具
 
在拓撲產生器中，以滑鼠右鍵按一下 [ **Office 網頁應用程式伺服器**] 以執行下列作業：
  
- **新 Office Web apps server**：啟動用來定義新的 Office web apps server 的 [**定義新的 Office web apps server** ] 嚮導。
    
除了 [新增 Office Web Apps Server] 選項之外，您還可以執行下列一般工作：
  
![OfficeWebApps 伺服器範圍節點](../../../media/OfficeWebApps_Server_Scope_Node.jpg)
  
- **拓撲**。 您可以使用此選項來定義新的拓撲，開啟現有的拓撲，下載目前的拓撲，儲存拓撲的複本，發佈拓撲，安裝資料庫，從舊版中合併元件，或移除部署。
    
- **說明**。您可以使用這個選項，存取作用中節點的說明。
