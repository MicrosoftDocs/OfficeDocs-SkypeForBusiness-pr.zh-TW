# <a name="office-web-apps-server-scope-node-expander"></a>Office Web Apps Server 範圍節點展開工具
 
您以滑鼠右鍵按一下拓撲建立器中的**Office Web Apps 伺服器**來執行下列動作：
  
- **新的 Office Web Apps server**：啟動**定義新的 Office web apps 伺服器**嚮導，用來定義新的 office web apps 伺服器。
    
除了 [新的 Office Web Apps 伺服器] 選項之外，您還可以執行下列一般工作：
  
![Office WebApps Server Scope 節點](../../../media/OfficeWebApps_Server_Scope_Node.jpg)
  
- **拓撲**。您可以使用這個選項來定義新的拓撲、開啟現有的拓撲、下載目前拓撲、儲存拓撲的複本、發行拓撲、安裝資料庫、合併來自舊版的元件，或是移除部署。
    
- **說明**。您可以使用這個選項，存取作用中節點的說明。
