# <a name="add-edge-machine-nat-ip"></a>新增 Edge 電腦 NAT IP
 
如果由網路位址轉譯 (NAT) 來轉譯此 Edge 集區的公用 IP 位址，您必須提供要轉譯的公用 IP 位址。這應該是 A/V Edge Service 的外部 IP 位址。
  
若要將 Edge 集區變更為使用/不使用 NAT 轉譯，請以滑鼠右鍵按一下 Edge 集區的名稱，然後按一下 [編輯內容]。
