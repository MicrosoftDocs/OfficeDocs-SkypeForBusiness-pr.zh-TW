# <a name="monitoring-servers-scope-node"></a>[監視伺服器] 範圍節點
 
[**監視伺服器**] 節點會列出拓撲中的監視伺服器。 您可以執行下列動作：
  
- **新的監視伺服器**。 啟動 [**定義新的監視伺服器**] 嚮導, 您可以使用它將新的監視伺服器新增到拓撲。
    
除了 [監視伺服器] 選項之外, 您還可以執行下列一般工作:
  
- **拓撲**。 您可以使用這個選項來定義新的拓撲、開啟現有的拓朴、下載目前的拓撲、儲存拓撲複本、發佈拓撲、安裝資料庫、合併舊版 Office 通訊伺服器的元件, 或移除部署.
    
- **檢視**。您可以使用這個選項來自訂拓撲產生器中的檢視，方法是選取要在主控台樹狀目錄檢視中顯示及隱藏的項目。
    
- **說明**。您可以使用這個選項，存取作用中節點的說明。
