# <a name="add-av-mcu-machine"></a>新增 A/V MCU 電腦
 
在這個 A/V 會議池中，指定您想要新增為 A/V 會議伺服器的每個電腦的完整功能變數名稱（FQDN）。 將電腦新增至清單後，即可更新電腦的 FQDN，或在發行拓撲前，隨時將它從集區中移除。 發佈拓撲之後，變更 FQDN 需要先刪除拓撲建立器中的伺服器，然後使用新的 FQDN 將新的伺服器新增到該池。
