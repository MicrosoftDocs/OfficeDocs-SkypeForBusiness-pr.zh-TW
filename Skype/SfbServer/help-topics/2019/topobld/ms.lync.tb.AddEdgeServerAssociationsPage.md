# <a name="add-edge-server-associations"></a>新增 Edge Server 關聯
 
使用此頁面可選取哪些前端池和標準版伺服器將此 Edge 池用於 A/V 流量。 
  
每個前端池只能與一個 Edge 池建立關聯。 如果您選取的是與另一個邊緣池相關聯的池，則會變更與前一個 Edge 池的關聯。
  
您也可以選擇將此 Edge 伺服器或 Edge 池與中繼伺服器或轉送伺服器池進行關聯，以設定及啟用外部使用者的企業語音。
