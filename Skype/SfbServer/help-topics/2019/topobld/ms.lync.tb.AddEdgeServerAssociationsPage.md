# <a name="add-edge-server-associations"></a>新增 Edge Server 關聯
 
您可以使用此頁面來選取哪些前端集區和 Standard Edition Server 將使用此 Edge 集區處理 A/V 流量。 
  
每個前端集區都只能與一個 Edge 集區相關聯。如果您選取已經和另一個 Edge 集區相關聯的集區，它將會把先前 Edge 集區的關聯改掉。
  
您也可以選擇將此 Edge Server 或 edge 集區與轉送伺服器或轉送伺服器集區相關聯，以設定及啟用外部使用者的企業語音。
