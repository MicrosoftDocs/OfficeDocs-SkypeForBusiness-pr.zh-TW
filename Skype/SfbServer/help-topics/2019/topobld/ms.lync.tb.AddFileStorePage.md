# <a name="add-file-store"></a>新增檔案存放區
 
若要定義新的檔案存放區，請指定下列專案：
  
- 將託管新檔案存放區之伺服器的完整功能變數名稱（FQDN）。 在發佈拓朴前，該伺服器的 FQDN 必須存在於網域名稱系統（DNS）中。
    
- 主機伺服器上檔案共用的名稱。 發佈拓撲前，必須先存在檔案共用。
