# <a name="trunks-scope-node"></a>主幹範圍節點
 
在拓撲建立器中，以滑鼠右鍵按一下 [ **Trunks** ]，以執行下列動作：
  
- **新增主幹**：啟動用來定義新 SIP 主幹的 [定義新的主幹精靈]****。
    
除了 [新增主幹] 選項，您還可以執行下列一般工作：
  
![主幹範圍節點](../../../media/Trunks_Scope_Node.jpg)
  
- **拓撲**。您可以使用這個選項來定義新的拓撲、開啟現有的拓撲、下載目前拓撲、儲存拓撲的複本、發行拓撲、安裝資料庫、合併來自舊版的元件，或是移除部署。
    
- **說明**。您可以使用這個選項，存取作用中節點的說明。
