# <a name="legacy-merge-landing"></a>舊版合併登陸
 
[**指定邊緣設定**] 頁面可讓您設定舊版 Edge 伺服器資訊。 按一下 [**新增**]，指定舊版 Edge 伺服器版本的 Office 通訊伺服器部署。 如果您的舊版環境中沒有任何邊緣伺服器，請按 **[下一步]** 繼續。
