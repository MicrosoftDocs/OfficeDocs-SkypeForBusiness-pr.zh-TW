# <a name="legacy-merge-landing"></a>舊版合併登陸
 
「指定 Edge 安裝」頁面可讓您設定舊版 Edge Server 資訊。按一下 [新增] 以指定 Office Communications Server 部署的舊版 Edge Server。如果您的舊版環境中沒有任何 Edge Server，請按 [下一步] 繼續。
