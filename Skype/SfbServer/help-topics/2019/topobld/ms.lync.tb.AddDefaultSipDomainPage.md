# <a name="add-default-sip-domain"></a>新增預的 SIP 網域
 
指定您希望部署支援的主要會話初始通訊協定（SIP）網域。 主要 SIP 網域可以與您的 Active Directory 網域服務網域相同，或與其不同。 例如，您的 Active Directory 網域可能是 fabrikam.net，您可以指定要 contoso.net 的主要 SIP 網域。
