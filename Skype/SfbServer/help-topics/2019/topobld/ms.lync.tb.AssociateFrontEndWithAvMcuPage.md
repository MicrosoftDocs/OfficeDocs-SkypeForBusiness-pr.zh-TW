# <a name="associate-front-end-with-av-mcu"></a>將前端與 AV MCU 建立關聯
 
A/V 會議可在使用者之間即時進行音訊與視頻通訊, 也就是, 如果他們有適當的用戶端裝置, 例如音訊會議的耳機, 以及視訊會議的網路攝影機等功能。 如果您的部署支援會議, 而且您同時啟用了 [web 會議] 和 [A/V 會議], 您就可以 collocate 使用前端伺服器的 A/V 會議伺服器), 或部署一或多個獨立的 A/V 會議伺服器 (A/V 會議池). 如果您選擇部署獨立 A/V 會議伺服器的選項, 您必須在拓撲建立器中定義它。
  
如果使用方式不超過 A/V 會議伺服器的容量, 則在網站和多個中央網站的池中的所有池都可以使用相同的 A/V 會議伺服器。
