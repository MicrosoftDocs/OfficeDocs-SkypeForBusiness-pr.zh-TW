# <a name="associate-front-end-with-archiving"></a>將前端與封存關聯
 
每個前端池和 Survivable 分支裝置只能有一個與它相關聯的存檔伺服器。 針對網站啟用封存功能之後, 存檔伺服器會收集並記錄立即訊息 (IM) 訊息內容和網路會議內容, 以及來自駐留在關聯的 pool 及 Survivable 分支裝置上的使用者的交談。 
  
如果使用方式不超過存檔伺服器的容量, 則在網站和多個中央網站的池中的所有池都可以使用相同的存檔伺服器。
