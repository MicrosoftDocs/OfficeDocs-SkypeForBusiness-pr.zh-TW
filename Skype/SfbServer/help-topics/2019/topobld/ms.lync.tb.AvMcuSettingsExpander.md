# <a name="av-mcu-settings-expander"></a>AV MCU 設定展開工具
 
若要編輯您已定義的 A/V 會議伺服器集區的屬性，請執行下列動作：
  
## <a name="general-settings"></a>一般設定

若要更新 A/V 會議伺服器集區的完整功能變數名稱 (FQDN) ，請輸入在網域名稱系統中所定義的集區名稱 (DNS) 。 您必須有對應於 A/V 會議伺服器集區名稱的 DNS 主機 (A) 記錄。
  
## <a name="av-conferencing-service"></a>會議服務 A/V

若要與 A/V 會議伺服器集區建立關聯，請選取必須與 A/V 會議伺服器集區建立關聯的一或多個前端集區。
  
> [!NOTE]
> 如果 [ **選取要與此 A/V 會議集** 區] 窗格相關聯的前端集區沒有顯示前端集區，請組合 A/V 會議伺服器與前端集區，或在前端集區內容的 [ **功能** ] 區段中未選擇新增會議。
  

完成時，按一下 [確定] 完成 A/V 會議伺服器集區設定的編輯。按一下 [取消] 捨棄任何變更。
