# <a name="edit-office-web-apps-server-settings"></a>編輯 Office Web Apps Server 設定

您編輯已設定之 Office Web Apps Server 的屬性。 您可以編輯下列內容：

 **Office Web Apps SERVER FQDN**：這個屬性會定義 Office Web apps server 的完整功能變數名稱，而且應符合網域名稱系統 (DNS) 主機 a 或 AAAA (如果 IPv6 是用於) record。

 **Office Web Apps server 探索 URL**：統一資源定位器 (URL) 若要將用戶端存取至 Office Web Apps Server，您可能需要編輯此位址的預設值（如果伺服器放置於部署的內部網路以外的其他網路區域）。

如果此伺服器部署在周邊網路或內部防火牆外的其他網路區域 (此防火牆會將周邊網路、不受信任的網路及網際網路與您的內部部署分隔開來)，請選取 [Office Web Apps Server 部署在外部網路上] 核取方塊。
