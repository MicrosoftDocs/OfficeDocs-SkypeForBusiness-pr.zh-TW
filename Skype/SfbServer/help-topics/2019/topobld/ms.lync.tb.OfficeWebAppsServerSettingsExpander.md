# <a name="edit-office-web-apps-server-settings"></a>編輯 Office Web Apps Server 設定

您編輯已設定的 Office Web Apps 伺服器的屬性。 下列屬性可供編輯：

 **Office Web Apps 伺服器 FQDN**：這個屬性會定義 Office Web apps 伺服器的完整功能變數名稱，且應與網域名稱系統（DNS）主機 a 或 AAAA （如果使用 IPv6）記錄相符。

 **Office Web Apps 伺服器探索 URL**：適用于 Office Web apps server 的用戶端存取的統一資源定位器（URL），如果伺服器位於您部署的內部網路以外的其他網路區域中，您可能需要從預設值編輯此位址。

如果此伺服器是在您的周邊網路中部署，或從內部部署中分離周邊網路、較低信任的網路和網際網路，請選取核取方塊 [ **Office Web Apps 伺服器已部署在外部網路中**]。
