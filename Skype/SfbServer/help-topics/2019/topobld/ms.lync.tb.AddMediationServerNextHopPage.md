# <a name="add-mediation-server-nexthop"></a>新增中繼伺服器 NextHop
 
若要為您所定義的轉送伺服器定義下一個躍點，請從 **[下一個躍點集區]** 下拉式清單中，選取將充當轉送伺服器下一個躍點的 Enterprise Edition 伺服器或 Standard Edition 伺服器。 按 **[下一步]** 繼續。
