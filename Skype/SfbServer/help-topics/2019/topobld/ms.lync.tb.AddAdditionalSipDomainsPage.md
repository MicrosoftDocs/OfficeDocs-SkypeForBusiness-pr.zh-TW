# <a name="add-additional-sip-domains"></a>新增其他 SIP 網域
 
指定您想要在部署中支援之其他每個網域的功能變數名稱。 例如，如果您已將 contoso.net 指定為主要會話初始通訊協定（SIP）網域，而且您也想要在您的部署中支援 SIP 流量 fabrikam.net，請將 fabrikam.net 指定為額外的支援網域。
