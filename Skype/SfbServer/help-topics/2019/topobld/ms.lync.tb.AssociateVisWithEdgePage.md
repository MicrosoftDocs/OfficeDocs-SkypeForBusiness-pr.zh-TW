# <a name="associate-vis-with-edge"></a>建立 VIS 與 Edge 的關聯
 
在 [拓撲產生器] 中定義新的 VIS 或視頻閘道所需的步驟，在商務用 Skype Server 中的 [[建立 VIS 集](../../../deploy/deploy-video-interop-server/create-a-vis-pool.md)區] 中說明為線上。
  
在伺服器上使用 Windows PowerShell 指派視頻 Interop 伺服器角色所需的步驟，將在[商務用 Skype Server 中部署 VIS Server role 中](../../../deploy/deploy-video-interop-server/deploy-the-vis-server-role.md)說明為線上。
  
使用 Windows PowerShell 的新視頻 interop 伺服器設定步驟，會在[商務用 Skype Server 中設定影片的 interop 伺服器](../../../deploy/deploy-video-interop-server/configure-the-vis.md)上說明。
  
 若要在 Cisco GUI 中設定之視頻閘道的設定，以便與 VIS 交互操作，請參閱[Configure CUCM to with 商務用 Skype Server 的互用性](../../../deploy/deploy-video-interop-server/configure-cucm-for-interoperation.md)。
  
 若要在 Cisco GUI 中設定 VTC 的設定，以便與影片閘道和 VIS 進行交互操作，請參閱線上設定[VTC，以與商務用 Skype Server 進行交互操作](../../../deploy/deploy-video-interop-server/configure-a-vtc-for-interoperation.md)。
