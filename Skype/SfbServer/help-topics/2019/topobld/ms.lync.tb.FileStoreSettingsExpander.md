# <a name="file-store-settings-expander"></a>檔案存放區設定展開工具
 
若要編輯已定義的檔案存放區的內容：
  
在檔案共用名稱中，您可以定義新共用的名稱。接著，您必須在檔案伺服器完整網域名稱 (FQDN) 內容中所指名的伺服器上建立共用。
  
> [!NOTE]
> 您無法變更檔案伺服器 FQDN 內容中所指定的伺服器 FQDN。如果您需要將檔案存放區檔案共用放在不同伺服器上，請建立新的檔案存放區項目。
