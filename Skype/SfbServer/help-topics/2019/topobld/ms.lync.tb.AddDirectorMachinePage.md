# <a name="add-director-machine"></a>新增 Director 電腦
 
指定您要在此集區中，新增為 Director 之每台電腦的完整網域名稱 (FQDN)。將電腦新增至清單後，您即可更新電腦的 FQDN 或在發行拓撲前，隨時將其從集區中移除。發行拓撲後，若要變更 FQDN，就必須在拓撲產生器中刪除伺服器，然後再用新的 FQDN 將新伺服器新增至集區。
