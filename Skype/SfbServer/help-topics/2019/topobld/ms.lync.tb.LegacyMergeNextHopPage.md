# <a name="legacy-merge-next-hop"></a>舊版合併下一個躍點
 
**下一個躍點 FQDN**會識別舊版邊緣伺服器路由內部流量的下一個躍點伺服器。 輸入這個邊緣伺服器路由內部流量的下一個躍點伺服器的完整功能變數名稱 (FQDN), 然後選取它是否為**前端池**或**控制器**。
