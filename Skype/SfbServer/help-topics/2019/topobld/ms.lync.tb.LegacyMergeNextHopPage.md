# <a name="legacy-merge-next-hop"></a>舊版合併下一個躍點
 
[下一個躍點 FQDN] 識別供舊版 Edge Server 路由內部流量的下一部躍點伺服器。輸入供此 Edge Server 路由內部流量的下一部躍點伺服器的完整網域名稱 (FQDN)，然後選取其為 [前端集區] 或 [Director]。
