# <a name="topology-builder-startup-dialog"></a>拓撲產生器啟動對話方塊
 
啟動拓撲產生器以設定部署的屬性和設定。 在啟動時，拓撲產生器會顯示一個對話方塊，詢問拓撲檔的來源。 您可以選取下列三個選項之一：
  
- **從現有部署下載拓撲** 如果您已執行拓撲產生器，且已成功將拓撲發行至中央管理伺服器，請選取此選項。 此選項會要求及下載目前設定的本機複本，讓您可以編輯及設定現有的部署。
    
- **從本機檔案開啟拓撲** 如果您正在設定部署，但尚未發佈設定，請選取此選項。 此選項可讓您將設定工作的本機複本儲存在拓撲產生器中，然後再回一次。
    
- **新的拓撲** 選取這個選項，即可開始設定新的拓撲檔。 此選項會開啟您設定的新拓撲檔，以符合您將設定之部署的需求。
