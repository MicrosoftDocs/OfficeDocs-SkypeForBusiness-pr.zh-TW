# <a name="add-edge-server-fqdn"></a>新增邊緣伺服器 FQDN
 
您必須為存取邊緣服務指定完全限定的功能變數名稱 (FQDN)。 如果您沒有在 [**選取功能**] 頁面上選取 [**使用單一&amp; FQDN IP 位址**] 選項, 您也必須為 [Web 會議 Edge 服務] 或 [a/V 邊緣] 服務指定 FQDN。
  
此外, 如果您已選取 [**使用單一 FQDN &amp; IP 位址**] 選項, 則必須為每個 Edge 服務指定不同的埠號碼 (建議的埠設定: 444 以取得存取邊緣服務、8057網頁會議 Edge 服務, 以及 a/V 的 443)Edge 服務)。 如果您沒有選取此選項, 您可以針對所有三個服務使用相同的埠號碼 (例如 443)。
