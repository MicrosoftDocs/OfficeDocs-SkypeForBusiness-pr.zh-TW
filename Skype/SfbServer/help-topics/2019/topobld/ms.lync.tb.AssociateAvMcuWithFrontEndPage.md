# <a name="associate-av-mcu-with-front-end"></a>建立 AV MCU 與前端的關聯
 
所有啟用會議的企業版前端池都必須有 A/V 會議服務 collocated，或必須使用獨立的 A/V 會議池。 任何目前有 A/V 會議服務 collocated 的現有企業版前端池，都可能會選取改為使用這個新的獨立 A/V 會議池。
