# <a name="add-trusted-application-machine"></a>新增信任的應用程式機器
 
在此集區中，指定您要新增為信任的應用程式伺服器之每台電腦的完整功能變數名稱 (FQDN) 。 將電腦新增至清單後，您可以在發行拓撲之前，隨時更新電腦的 FQDN，或將它從集區移除。 發行拓撲後，若要變更 FQDN，就必須在拓撲產生器中刪除伺服器，然後再用新的 FQDN 將新伺服器新增至集區。
