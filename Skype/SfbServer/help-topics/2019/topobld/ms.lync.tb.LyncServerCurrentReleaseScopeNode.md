# <a name="skype-for-business-server-create-new-expander"></a>商務用 Skype Server建立新的展開器
 
您可以在拓撲產生器中建立新的伺服器角色，方法是以滑鼠右鍵按一下 [**商務用 Skype Server**]，選取 [**新增**]，然後選取要建立的伺服器角色。
