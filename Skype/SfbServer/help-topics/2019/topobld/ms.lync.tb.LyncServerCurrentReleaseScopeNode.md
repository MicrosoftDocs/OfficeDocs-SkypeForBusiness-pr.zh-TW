# <a name="skype-for-business-server-create-new-expander"></a>商務用 Skype Server 建立新的展開器
 
您可以用滑鼠右鍵按一下 [**商務用 Skype 伺服器**]，選取 [**新增**]，然後選取要建立的伺服器角色，在拓撲建立器中建立新的伺服器角色。
