# <a name="certificate-request-organization-information"></a>憑證要求 (組織資訊)
 
[組織資訊] 窗格可讓您指定您組織的名稱以及要求憑證的組織單位 (OU) (通常是部門)。當其他人檢視憑證時，您在 [組織] 和 [組織單位] 文字方塊中輸入的資訊會出現在憑證上，用來識別當初要求憑證的人員。
  
> [!NOTE]
> 憑證上的 OU 名稱不一定與 Active Directory 網域服務中的 OU 相同。 根據您設定 Active Directory 的方式，您使用的名稱可能會相同，也可能不相同。
