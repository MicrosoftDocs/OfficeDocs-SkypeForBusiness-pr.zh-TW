# <a name="certificate-request-certificate-signing-request-generated"></a>憑證要求 (已產生憑證簽署要求)
 
當您完成憑證的離線要求作業之後，會顯示「憑證要求檔案」頁面。若要顯示憑證要求並準備將其傳送至離線憑證授權單位 (CA) 或公用 CA，按一下 [檢視] 按鈕。依預設，憑證要求會顯示在 Windows 記事本中。您可以從此顯示內容中複製並貼上編碼過的憑證要求，或是遵循所選的離線 CA 或公用 CA 的指示完成憑證要求的處理作業。按一下 [完成] 繼續。
