# <a name="sql-server-reporting-services-groups"></a>SQL Server Reporting Services (群組)

在 [指定唯讀群組] 中，指定您要將監控報告唯讀存取權限授予哪個群組。 使用 format &lt; domain &gt; \\ &lt; access 群組 &gt; 。 例如：contoso\RTCUniversalReadOnlyAdmins。
  
為群組指派唯讀存取權限為選用動作。如果您現在不想要授予唯讀存取權限，可以在稍後直接在 SQL Server Reporting Services 上設定。
