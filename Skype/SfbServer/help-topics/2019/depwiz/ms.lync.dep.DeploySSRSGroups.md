# <a name="sql-server-reporting-services-groups"></a>SQL Server Reporting Services (群組)

在 [**指定唯讀] 群組**中, 指定您要授予其唯讀存取權的群組, 以進行監視報告。 使用 [設定&lt;網域&gt;\\&lt;格式]&gt;群組。 例如: contoso\RTCUniversalReadOnlyAdmins。
  
指派群組唯讀存取是選擇性的。 如果您目前不想授與唯讀存取權, 您可以在稍後直接在 SQL Server Reporting Services 中進行設定。
