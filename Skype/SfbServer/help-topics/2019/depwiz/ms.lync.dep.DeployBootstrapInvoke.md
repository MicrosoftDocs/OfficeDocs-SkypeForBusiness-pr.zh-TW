# <a name="setup-or-remove-skype-for-business-server-components-invoke"></a>安裝或移除商務用 Skype Server 元件 (叫用)
 
[**執行] 命令**會顯示執行來部署伺服器角色之工作的運行摘要。 完成工作之後（部署、啟動或停用和卸載），即會顯示摘要。 此外，您也可以在下拉式清單中，選取執行命令時所建立的任何記錄檔。 使用下拉式清單來選取您要查看的記錄檔，然後按一下 [**查看記錄**]。
  
審閱摘要及記錄檔之後，按一下 **[完成]**。
