# <a name="install-local-configuration-store-invoke"></a>安裝本機設定存放區 (叫用)
 
[**執行命令**] 窗格會顯示安裝中央管理儲存區本機唯讀複本的工作狀態。 當工作執行時，工作窗格會隨著命令執行而更新。 您應該查看任務狀態，並記下顯示的任何錯誤或警告。 完成工作後，就會顯示 [郵件工作**狀態：已完成**]，而記錄檔的下拉式清單會顯示記錄檔的名稱。 若要查看已建立的記錄檔，請選擇要查看的記錄檔，然後按一下 [**查看記錄**]。 查看所有記錄檔，以取得成功、錯誤及警告訊息。
  
審閱記錄檔之後，按一下 **[完成]** 以完成本機配置存儲區的安裝。
