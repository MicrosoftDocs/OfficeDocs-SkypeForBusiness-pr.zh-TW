# <a name="certificate-request-additional-subject-alternate-names"></a>憑證要求 (其他主體別名)
 
[ **設定其他主體** 別名] 選項可讓您在完成和要求憑證之前，將任何其他主體替代名稱新增 (SANs) 至憑證。 若要新增 SAN，請輸入 SAN 的完整網域名稱 (FQDN)，然後按一下 **[新增]**。 如果必須刪除您新增的 SAN，請反白顯示 SAN 項目，然後按一下 **[移除]**。 若要移除您已新增的所有 SAN 項目，請按一下 **[全部清除]**。
  
> [!NOTE]
> [ **移除** ] 或 [ **全部清除** ] 按鈕只會在此頁面上新增的 SAN 專案上運作。 這些按鈕不影響目前已在憑證上的 SAN 項目。
