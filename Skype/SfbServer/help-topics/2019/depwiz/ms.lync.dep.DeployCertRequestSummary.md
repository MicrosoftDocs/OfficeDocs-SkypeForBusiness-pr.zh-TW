# <a name="certificate-request-summary"></a>憑證要求 (摘要)
 
[憑證要求摘要]**** 頁面顯示憑證的所有選項和設定的詳細摘要。摘要會指出憑證要求所包含的內容，以及接下來將如何發出憑證。請仔細檢閱摘要，確定您選取的所有項目和選項都正確。如果選項正確，請按 [下一步]****，以繼續產生要求並準備將要求傳送給憑證授權單位 (CA)。
