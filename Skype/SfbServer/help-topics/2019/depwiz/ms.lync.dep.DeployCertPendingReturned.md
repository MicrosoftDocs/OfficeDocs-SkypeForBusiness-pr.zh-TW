# <a name="pending-certificate-returned"></a>擱置的憑證 (已傳回)
 
「線上憑證要求狀態」頁面會傳回憑證的目前狀態。如果憑證在憑證授權單位系統管理員那邊仍處於等待核准狀態 (亦即，如果憑證未獲核准)，則狀態會指出憑證仍在處理中。
  
如果已核准擱置要求且已成功擷取憑證，則狀態會指出已將憑證指派至本機憑證存放區，而狀態窗格也會顯示憑證指紋。
  
如果憑證授權單位系統管理員已拒絕憑證要求，狀態窗格會報告已拒絕所指定 ID (由憑證授權單位維護) 的擱置憑證的要求。
  
按一下 [完成] 完成擱置的憑證處理程序。
