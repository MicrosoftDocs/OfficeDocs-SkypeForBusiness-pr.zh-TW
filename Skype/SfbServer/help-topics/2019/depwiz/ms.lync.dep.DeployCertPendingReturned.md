# <a name="pending-certificate-returned"></a>擱置的憑證 (已傳回)
 
[**線上憑證要求狀態**] 頁面會傳回憑證的目前狀態。 如果憑證仍與憑證授權單位系統管理員（亦即尚未核准）中的擱置核准狀態相同，即表示該憑證仍在處理中。
  
如果擱置的要求已獲核准且證書的檢索成功，則狀態會指出已將憑證指派給本機憑證存放區，而狀態窗格也會顯示憑證指紋。
  
如果憑證授權單位管理員拒絕證書申請，則 [狀態] 窗格會將擱置憑證要求與指定的識別碼（由憑證授權單位維護）遭到拒絕。
  
按一下 **[完成]** 以完成擱置中的憑證處理。
