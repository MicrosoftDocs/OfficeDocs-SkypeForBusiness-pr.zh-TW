# <a name="certificate-request-online-offline"></a>憑證要求 (線上/離線)
 
若要建立延遲要求或離線要求，請選取 [立即準備此要求，但稍後再傳送 (離線憑證要求)]****。 若要建立對線上憑證授權單位的立即要求（通常是貴組織公開金鑰結構中的內部系統），請選取 **[立即將要求傳送給線上憑證授權單位**]。
  
選取之後，按 [下一步] **** 繼續。如果您要取消而不要變更，請按一下 [取消]****。若要回到上一個步驟，請按 [上一步]****。
