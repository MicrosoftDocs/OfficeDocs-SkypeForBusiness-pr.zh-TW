# <a name="certificate-request-geography"></a>憑證要求 (地理位置)
 
**[地理資訊]** 頁面要求您提供憑證要求所需的其他資訊。 在此頁面上，您可以選取 **[國家]**，然後在文字方塊中輸入 **[州/省]** 和 **[城市/位置]**。 請注意，您不應該使用縮寫，而應拼出州/省和城市/地點的完整名稱。
