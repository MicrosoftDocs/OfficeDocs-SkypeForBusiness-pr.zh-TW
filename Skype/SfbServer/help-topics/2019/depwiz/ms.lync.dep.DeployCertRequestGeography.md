# <a name="certificate-request-geography"></a>憑證要求 (地理)
 
[**地理資訊**] 頁面會要求提供憑證要求的其他資訊。 您可以在此頁面上選取 [**國家/地區**], 然後在文字方塊中輸入**州/省**和**市/縣**。 請注意, 您不應該使用縮寫, 而且應該拼出州/省和市/縣的完整名稱。
