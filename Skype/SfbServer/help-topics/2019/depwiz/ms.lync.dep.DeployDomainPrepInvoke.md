# <a name="prepare-current-domain-invoke"></a>準備目前的網域 (Invoke)
 
[**執行命令**] 頁面會在作業執行時顯示工作摘要。 查看 [摘要] 畫面並確認工作已成功完成, 且沒有任何錯誤。 請檢查警告, 確定它們應該正常運作。 在摘要窗格下方, 您應該會看到 [**任務狀態: 已完成**]。 在下拉式清單中, 應顯示 [**準備網域**] 的 [記錄檔]。 如果不是, 請選取記錄檔, 然後按一下 [**查看記錄**], 以查看任務的詳細動作和結果。 按一下 **[完成]** 以完成網域的準備作業。
