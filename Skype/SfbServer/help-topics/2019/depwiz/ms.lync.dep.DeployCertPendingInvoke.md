# <a name="pending-certificate-invoke"></a>擱置的憑證 (叫用)
 
**[執行命令]** 頁面會顯示正在擷取擱置憑證的命令的狀態。工作完成時，將產生 **[擷取擱置的憑證]** 記錄檔。按一下 **[檢視記錄檔]** 來檢閱處理和工作完成狀態。按 **[下一步]** 繼續。
  
> [!IMPORTANT]
> 如果憑證授權單位系統管理員尚未核准憑證，[擷取攔置的要求] 記錄檔會指出有擱置的要求。 如果憑證授權單位管理員已拒絕要求，記錄檔會指出失敗並報告錯誤，表示要求遭到拒絕，以及拒絕要求之人員的使用者名稱。
