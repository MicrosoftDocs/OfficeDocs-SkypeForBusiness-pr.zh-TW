# <a name="remove-certificate-invoke"></a>移除憑證 (叫用)
 
[ **執行命令** ] 頁面提供移除憑證的詳細資料。 查看憑證移除的摘要資訊。 請確定下拉式清單中顯示 [移除憑證記錄檔]，然後按一下 [ **查看記錄** 檔] 以查看憑證移除的詳細資料。 當您完成複查摘要和記錄檔時，請按一下 **[完成]**。
