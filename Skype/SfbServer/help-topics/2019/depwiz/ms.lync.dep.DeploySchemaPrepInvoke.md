# <a name="prepare-schema-invoke"></a>準備架構 (叫用)
 
[**執行命令**] 頁面會顯示為了擴充商務用 Skype Server 的架構而發出的命令摘要。 當工作完成時，[摘要] 窗格會指出工作是否成功，或是否失敗。 查看摘要畫面，並確定 [ **任務狀態：已完成** ] 顯示在摘要窗格下方。 [摘要] 窗格底下的下拉式清單可讓您選取及查看記錄檔 [準備架構]。 按一下 [ **查看記錄** 檔] 以查看記錄檔。 [！注意] 如果您在架構準備期間收到錯誤，則會在這裡記錄任務所遇到的錯誤。 您可以採取動作修正任何錯誤，並重新執行架構準備工作。
  
按一下 [完成] 完成工作。
