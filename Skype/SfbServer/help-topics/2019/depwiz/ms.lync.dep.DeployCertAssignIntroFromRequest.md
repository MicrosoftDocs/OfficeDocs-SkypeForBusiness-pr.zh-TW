# <a name="certificate-assignment-from-request"></a>憑證指派 (從要求)
 
[憑證指派] **** 頁面可讓您開始將憑證指派至角色伺服器上所要求的使用方式。按一下 [檢視憑證詳細資料] **** 來檢視憑證。您可以選取 [一般] **** 索引標籤、[詳細資料] **** 索引標籤或 [憑證路徑] **** 索引標籤。[一般] **** 索引標籤會顯示憑證的基本資訊，例如發行憑證授權單位和私密金鑰的狀態。[詳細資料] 索引標籤會顯示憑證資訊，例如主體名稱、主體別名和其他詳細資料。[憑證路徑] **** 索引標籤會顯示從憑證開始，經由任何發行或中繼憑授權單位 (如果存在)，最後到達根憑證授權單位的憑證「信任鏈結」。
  
按 [下一步] **** 繼續。
