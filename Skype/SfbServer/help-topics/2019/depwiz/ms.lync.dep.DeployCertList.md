# <a name="certificate-list"></a>憑證清單
 
若要指派憑證，請從本機憑證存放區選取憑證。按 **[下一步]** 繼續。
  
[從本機憑證存放區選取憑證]**** 窗格中可供選取的憑證，就是可指派至您需要之憑證使用方式的有效憑證。您可以按一下 [檢視憑證詳細資料]**** 按鈕，確認您選取的是正確的憑證。在 [詳細資料]**** 索引標籤上，您可以檢視憑證上設定的主體名稱和主體別名。
  
> [!IMPORTANT]
> 選取窗格中也可能不會列出任何憑證。發生此情況時，原因通常是所需伺服器上未安裝信任的根憑證或中繼憑證授權單位憑證，來向憑證授權單位驗證憑證，進而維護由憑證建立的信任鏈結。若要解決此問題，請要求並匯入憑證鏈結，其中通常包括根憑證授權單位 (CA) 憑證及任何中繼 CA 憑證和發行 CA 憑證。
