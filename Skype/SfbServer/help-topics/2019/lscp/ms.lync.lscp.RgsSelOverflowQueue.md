# <a name="select-overflow-queue"></a>選取溢位佇列
 
使用 [**選取溢出佇列**] 對話方塊來選取當回應群組佇列已滿時呼叫轉寄的佇列。
  
在搜尋方塊中，輸入您想要使用的所有或部分名稱。 若要傳回所有可用的佇列，請將 [搜尋] 方塊保留空白，然後按一下 [搜尋] 圖示。
  
在搜尋所傳回的佇列清單中，選取回應群組佇列已滿時要用來轉寄來電的佇列。
