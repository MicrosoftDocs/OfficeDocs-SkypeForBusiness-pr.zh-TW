# <a name="pin-set-successfully"></a>已成功設定 PIN
 
若要使用電子郵件將 PIN 傳送給使用者，請按一下 [**開啟我的電子郵件應用程式]，將新的 pin 傳送給使用者**。 若要將 PIN 複製到電子郵件訊息中，請選取 [**顯示 PIN**]。
