# <a name="pin-set-successfully"></a>已成功設定 PIN
 
若要使用電子郵件將 PIN 碼傳送給使用者，請按一下 [ **開啟我的電子郵件應用程式]，將新的 PIN 碼傳送給使用者**。 若要將 PIN 碼複製到電子郵件中，請選取 [ **顯示 PIN**]。
