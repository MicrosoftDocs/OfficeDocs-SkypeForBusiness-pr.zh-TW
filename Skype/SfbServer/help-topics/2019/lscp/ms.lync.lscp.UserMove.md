# <a name="move-user"></a>移動使用者

在 [ **移動使用者** ] 頁面上，您可以選取要將選取的使用者移至其中的目的地集區。

如需移動使用者的詳細資訊，請參閱＜[Move Users to Another Pool](/previous-versions/office/lync-server-2013/lync-server-2013-move-users-to-another-pool)＞。
