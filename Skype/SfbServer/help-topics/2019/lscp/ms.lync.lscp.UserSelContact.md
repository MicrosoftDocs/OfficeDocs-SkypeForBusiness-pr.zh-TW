# <a name="select-contacts-from-active-directory"></a>從 Active Directory 選取連絡人
 
使用 [**從 Active Directory 選取**] 對話方塊，尋找您要啟用商務用 Skype 的連絡人。 在搜尋方塊中，輸入您要尋找之連絡人的 SIP 位址或顯示名稱。 若要取得所有可用的連絡人，請不要在搜尋方塊中輸入任何內容。
