# <a name="hybrid-manageability-wizard"></a>混合式可管理性精靈

您可以使用混合式管理性嚮導執行某些設定工作，以使用商務用 Skype 線上設定混合部署。 如需如何設定混合部署的詳細資訊，請參閱 [使用混合式管理性嚮導設定環境](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx)。
