# <a name="select-groups"></a>選取群組

回應群組佇列會將呼叫保留給回應群組，直到代理接聽來電為止。 佇列已指派代理人群組。 當您建立或編輯佇列時，您可以使用 [ **選取群組** ] 對話方塊，選取要指派給佇列的一或多個代理人群組。

## <a name="ui-reference"></a>UI 參考

下列清單說明 [ **選取群組** ] 對話方塊中的控制項和欄位。

- 搜尋方塊會搜尋代理群組的名稱。 輸入全部或部分的名稱。 保留搜尋方塊為空白，以顯示組織的所有代理人群組。

- **名稱** 顯示傳回之代理人群組的名稱。

- **參與原則** 顯示針對代理人群組定義的參與原則：正式或非正式。

- **路由方法** 顯示為 agent 群組定義的路由類型。

如需使用佇列的詳細資訊，請參閱 Operations 檔中的 [管理回應群組佇列](https://technet.microsoft.com/library/1e91720c-ab67-4dfb-b30c-0ef2a8012310.aspx) 。
