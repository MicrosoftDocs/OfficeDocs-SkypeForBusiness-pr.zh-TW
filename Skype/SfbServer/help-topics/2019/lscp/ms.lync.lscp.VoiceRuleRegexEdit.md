# <a name="translation-rule-type-a-regular-expression"></a>轉譯規則：輸入規則運算式
 
在 **[比對此模式]** 欄位，指定將用來比對要轉譯之數字的模式。 在 **[轉譯規則]** 欄位中，指定轉譯數字格式的模式。 例如，如果您在 [比對 \+ {9} **此模式** ] 欄位中輸入 ^ ( \d \d +) $，且 [ **轉譯規則** ] 欄位中輸入 011 $ 1，則此規則會將 + 441235551010 轉譯為011441235551010。
