# <a name="hybrid-wizard-move-user"></a>混合式精靈：移動使用者

使用 [移動使用者] 嚮導，將駐留在內部部署池中的使用者帳戶移到要駐留在商務用 Skype Online 上。 如需詳細資訊，請參閱〈[Use the Hybrid Manageability wizard](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx)〉。
