# <a name="set-pin"></a>設定 PIN

若要產生使用者的 PIN 碼，請選取 [ **自動產生有效的 pin**]。

若要自行建立 PIN，請選取 **[手動輸入特定的 PIN]**，然後在文字方塊中輸入符合您 PIN 原則設定中所指定之 PIN 要求的 PIN。
