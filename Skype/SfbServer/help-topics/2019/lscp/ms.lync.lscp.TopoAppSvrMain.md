# <a name="topology-server-application"></a>拓撲：伺服器應用程式

[**拓撲**：**伺服器應用程式**] 頁面會顯示您部署中的伺服器應用程式。

## <a name="tasks-you-can-perform"></a>您可以執行的工作

您可以在 [**拓撲**：**伺服器應用程式**] 頁面上執行下列任務：

- 在您的部署中查看伺服器應用程式

- 啟用或停用伺服器應用程式

- 選取或取消選取應用程式為重要
