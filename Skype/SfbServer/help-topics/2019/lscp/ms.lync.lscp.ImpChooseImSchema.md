# <a name="select-hyperlink-prefix"></a>選取超連結首碼
 
使用 [**選取超連結前置**詞] 對話方塊來選取您要封鎖的超連結類型、傳送警告訊息，或選取（若要封鎖或傳送警告訊息）或清除（允許）其在**超連結首碼**底下的類別延伸的核取方塊。 按一下箭號以展開類別，並查看該類別的所有超連結首碼。 如果您想要選取某個類別的所有超連結首碼，請選取該類別的名稱。 
  
如果您沒有看到要選取的超連結前置詞，請在 [**超連結首碼**] 下的文字方塊中輸入**副檔名，然後****按一下 [新增]**。 
  
將所需的所有超連結首碼新增至 [**選取的超連結首碼**] 清單後，請按一下 **[確定]**。
