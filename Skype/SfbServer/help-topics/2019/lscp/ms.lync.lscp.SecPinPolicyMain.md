# <a name="pin-policy"></a>PIN 原則

您想要做什麼？ 按一下您要執行之任務的連結。

 **使用 IP 電話的 PIN 原則**此個人識別碼 (PIN) 原則支援使用 IP 電話連線到商務用 Skype Server 的使用者的驗證。
- [在商務用 Skype Server 中建立新的 PIN 原則](../../../manage/authentication/create-a-new-pin-policy.md)

- [在商務用 Skype Server 中修改現有的 PIN 原則](../../../manage/authentication/modify-an-existing-pin-policy.md)

- [在商務用 Skype Server 中刪除 PIN 原則](../../../manage/authentication/delete-a-pin-policy.md)

  **使用電話撥入式會議的 PIN 原則**這個 PIN 原則可讓在貴組織中擁有 Active Directory 網域服務認證的使用者, 透過輸入 PIN, 以已驗證的使用者身分加入電話撥入式會議。
- [Create or Modify Dial-in Conferencing PIN Settings for a Site or Group of Users](https://technet.microsoft.com/library/c29bab5c-2b93-48e0-ae0b-29564daaff9a.aspx)

- [Modify the Default Dial-in Conferencing PIN Settings](https://technet.microsoft.com/library/2d110e94-ad29-4755-b17f-d8c2da9b78a4.aspx)

- [Delete Dial-in Conferencing PIN Settings for a Site or Group of Users](https://technet.microsoft.com/library/15a9faee-d024-4c0e-b2a0-fe7e7dc00589.aspx)
