# <a name="assign-properties"></a>指派屬性
 
使用 [**指派屬性**] 對話方塊, 一次指定多個子網上的描述和網路網站識別碼。 在 [**描述**] 中, 輸入您想要用於您所選取之子網的描述。 在 [**網路網站識別碼**] 中, 選取您要指派給所選子網的網路網站 id。 若要將這些屬性指派給您所選取的子網, 請按一下 **[確定]**。
