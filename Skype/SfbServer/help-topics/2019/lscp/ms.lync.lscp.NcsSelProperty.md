# <a name="assign-properties"></a>指派屬性
 
使用 **[指派內容]** 對話方塊，一次指定多個子網路的描述和網路網站 ID。在 **[描述]** 中，輸入想要用於您選取之子網路的描述。在 **[網路網站 ID]** 中，選取想要指派給您選取之子網路的網路網站 ID。若要將這些內容指派給您選取的子網路，請按一下 **[確定]**。
