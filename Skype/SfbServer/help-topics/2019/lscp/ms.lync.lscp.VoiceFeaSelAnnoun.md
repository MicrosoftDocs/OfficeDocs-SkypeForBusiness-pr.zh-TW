# <a name="select-an-announcement-service"></a>選取宣告服務
 
使用 [**選取宣告服務**] 對話方塊來選取將會處理來電至未指派號碼範圍的宣告服務。
  
在 [搜尋] 方塊中，輸入您要使用之宣告服務的全部或部分名稱。 若要傳回所有可用的宣告服務，請將 [搜尋] 方塊保留空白，然後按一下 [搜尋] 圖示。
  
在 [搜尋] 傳回的宣告服務清單中，選取您要用來處理來電到未指派號碼範圍之電話的服務。
