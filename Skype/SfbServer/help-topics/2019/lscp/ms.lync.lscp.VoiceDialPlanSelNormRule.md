# <a name="select-normalization-rules"></a>選取正規化規則
 
正規化規則會使用 .NET Framework 規則運算式，將撥出的電話號碼轉譯成 E.164 格式。每個撥號對應表都必須被指派一或多個正規化規則。
  
若要使用現有的正規化規則，請按一下清單中的規則，再按一下 [確定]****。
