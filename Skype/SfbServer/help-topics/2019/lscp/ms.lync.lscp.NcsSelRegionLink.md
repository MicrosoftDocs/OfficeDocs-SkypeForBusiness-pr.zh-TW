# <a name="select-a-region-link"></a>選取地區連結
 
使用 [ **地區連結** ] 對話方塊，尋找您環境中可用的地區連結。 若要使用現有的地區連結，請按一下清單中的地區連結，然後按一下 **[確定]**。
  
> [!NOTE]
> 若要取得所有可用的地區連結，請不要在搜尋方塊中輸入任何內容，然後按一下搜尋圖示。
