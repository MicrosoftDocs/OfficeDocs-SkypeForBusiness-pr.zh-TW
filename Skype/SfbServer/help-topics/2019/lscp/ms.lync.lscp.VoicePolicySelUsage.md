# <a name="select-pstn-usage-record"></a>選取 PSTN 使用記錄
 
使用 [**選取 PSTN 使用記錄**] 對話方塊來尋找您可以使用的現有公用交換電話網絡 (PSTN) 使用記錄。
  
> [!NOTE]
> 若要檢索所有可用的 PSTN 使用記錄, 請勿在搜尋方塊中輸入任何內容, 然後按一下 [搜尋] 圖示。
