# <a name="select-pstn-usage-record"></a>選取 PSTN 使用方式記錄
 
使用 [選取 PSTN 使用方式記錄] 對話方塊，尋找您可以使用的現有公用交換電話網路 (PSTN) 使用方式記錄。
  
> [!NOTE]
> 若要擷取所有可用的 PSTN 使用方式記錄，請不要在搜尋方塊中輸入任何內容，然後按一下搜尋圖示。
