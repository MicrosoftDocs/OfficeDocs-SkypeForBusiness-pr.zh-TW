# <a name="mobility-planning-tool"></a>行動性 (規劃工具) 
 
商務用 Skype Server 中的行動功能可支援行動裝置上的商務用 Skype 功能。
  
商務用 Skype Server 中的行動功能與 Lync Server 2013 中的功能相同。 如需準備行動環境的詳細資訊，請參閱 [在 Lync Server 2013 中規劃行動](https://technet.microsoft.com/library/hh689981%28v=ocs.15%29.aspx)。
