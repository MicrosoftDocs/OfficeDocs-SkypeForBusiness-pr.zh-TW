# <a name="mobility-planning-tool"></a>行動性（規劃工具）
 
商務用 Skype Server 的行動功能功能支援行動裝置上的商務用 Skype 功能。
  
在 Lync Server 2013 中，行動功能在商務用 Skype 伺服器中是一樣的。 如需準備行動環境的詳細資料，請參閱[在 Lync Server 2013 中規劃行動](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx)。
