# <a name="voice-traffic-planning-tool"></a>語音流量（計畫工具）
 
規劃工具會使用使用者流量的相關資訊來判斷要在部署中使用的埠數。 這個數位依次決定所需的中繼伺服器和閘道數目。 如需詳細資訊，請參閱[估計商務用 Skype Server 的語音使用與流量](../../../plan-your-deployment/capacity/estimating-voice-traffic.md)。
