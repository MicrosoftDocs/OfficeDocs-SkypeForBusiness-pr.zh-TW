# <a name="web-conferencing-planning-tool"></a>Web 會議 （規劃工具）
 
Skype 商務伺服器 web 會議包含文件共同作業、 應用程式共用與桌面共用。
  
> [!NOTE]
> Skype 商務 Server 使用 Office Web Apps 與 Office Web Apps Server 來處理共用和 PowerPoint 簡報的轉譯。 一樣在 [Microsoft Lync 2013 的 web 會議功能運作完全相同。 如需安裝及設定 Office Web Apps Server 的詳細資訊，請參閱[設定整合 Office Web Apps Server 與 Lync Server 2013](https://technet.microsoft.com/library/jj204792%28v=ocs.15%29.aspx)。
