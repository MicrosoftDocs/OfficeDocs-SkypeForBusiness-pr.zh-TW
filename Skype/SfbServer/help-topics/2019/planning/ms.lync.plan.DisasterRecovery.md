如需準備環境以進行嚴重損壞修復的詳細資訊，請參閱[在商務用 Skype Server 中規劃高可用性和嚴重損壞修復](../../../plan-your-deployment/high-availability-and-disaster-recovery/high-availability-and-disaster-recovery.md)。
