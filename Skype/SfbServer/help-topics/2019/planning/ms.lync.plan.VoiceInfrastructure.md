# <a name="voice-infrastructure-planning-tool"></a>語音基礎結構 (規劃工具) 
 
商務用 Skype Server 會利用 sip 主幹、將 sip 連線到 PSTN 閘道，或將 sip 連線至 PBX，提供可靠、可伸縮的 PSTN 連線能力。 如需詳細資訊，請參閱[Plan for PSTN connectivity in 商務用 Skype Server](../../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md)。
