# <a name="enterprise-voice-planning-tool"></a>企業語音 (規劃工具) 
 
使用企業語音時，商務用 Skype Server 會提供獨立語音 over 網際網路通訊協定 (VoIP) 提供以加強或取代傳統專用交換機 (PBX) 系統。 企業語音使用者可以撥打組織 VoIP 網路或 PBX 上的同事，也可以撥打組織外部的傳統電話號碼。 企業語音方案包含常見呼叫功能（例如，答案、轉寄、轉移、保留、轉移、發行和寄存）及增強型 9-1-1 (E9-1-1) 呼叫 (E9-1-1 只適用于美國。 ) 企業語音也支援廣泛的目前和較舊的 IP 和 USB 裝置。
  
如需規劃企業語音的詳細資訊，請參閱[Plan for 企業語音 in 商務用 Skype Server](../../../plan-your-deployment/enterprise-voice-solution/enterprise-voice.md)。 如需部署獨立轉送伺服器的詳細資訊，請參閱商務用 Skype Server 中的拓撲產生器[部署轉送伺服器](../../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)。
