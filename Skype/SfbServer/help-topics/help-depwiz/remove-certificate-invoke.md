# <a name="remove-certificate-invoke"></a>移除憑證 (叫用)
 
[**執行命令**] 頁面提供移除憑證的詳細資料。 查看憑證移除的摘要資訊。 請務必在下拉式清單中看到 [移除憑證記錄檔]，然後按一下 [**查看記錄**]，以查看憑證移除的詳細資料。 當您完成查看摘要及記錄檔時，請按一下 **[完成]**。
