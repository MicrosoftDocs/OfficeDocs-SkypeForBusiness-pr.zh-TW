# <a name="certificate-assignment-invoke"></a>憑證指派 (叫用)
 
「執行命令」頁面顯示將憑證指派給伺服器的工作相關動作。工作完成之後，您可以檢視詳述憑證指派流程的記錄檔。若要檢視記錄檔，請按一下 [檢視記錄檔]。
