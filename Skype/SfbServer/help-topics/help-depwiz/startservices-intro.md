# <a name="startservices-intro"></a>StartServices (簡介)
 
若要在所部署的商務用 Skype Server 2015 角色服務器上啟動服務，請參閱簡介畫面。 請注意，發佈用以啟動商務用 Skype 伺服器的命令，不會等候服務啟動。 這是啟動服務的最佳作法，您應該使用 Windows 服務 Microsoft Management Console (MMC) 來監視服務的實際狀態。 按 [下一步] 開始執行 [啟動服務] 工作。
