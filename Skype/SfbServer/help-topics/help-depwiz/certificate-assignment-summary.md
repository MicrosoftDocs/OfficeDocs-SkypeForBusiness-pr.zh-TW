# <a name="certificate-assignment-summary"></a>憑證指派 (摘要)
 
在證書指派完成之前，您會看到 [**憑證指派摘要**]，其中列出將指派的憑證詳細資料。 請查看資訊，以確定主旨名稱和 subject 替換名稱正確無誤。 審閱完憑證摘要之後，請按 **[下一步]**。
