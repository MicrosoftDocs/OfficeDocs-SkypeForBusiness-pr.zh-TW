# <a name="request-install-or-assign-certificates"></a>要求、安裝或指派憑證
 
 [步驟 3：要求、安裝或指派憑證] **** 會在您按一下 [執行] **** 時啟動 [憑證精靈]。 透過嚮導所設定的憑證，是以 [由拓撲建立器] 配置併發布到中央管理儲存體的商務用 Skype Server 2015 拓朴的定義為基礎。 為了順利對組織中的線上憑證授權單位 (CA) 執行 [憑證精靈]，您必須以電腦本機系統管理員群組的成員身分登入電腦。 在電腦和 CA 所在的網域中，您也必須是經過驗證的網域使用者。 [憑證] 嚮導能提供指定備用認證以存取您組織的 CA 的功能。
  
> [!NOTE]
> 您也可以使用 [憑證精靈]，要求及處理傳送至公用 CA 或其他離線公開金鑰基礎結構 (PKI) 的離線憑證要求。若要產生離線要求，除了登入電腦所需的群組成員資格外，不需要其他特定群組成員資格。若要處理公用 CA 回應及指派憑證給電腦和角色，您必須以本機 Administrators 群組成員或相等的身分登入。
