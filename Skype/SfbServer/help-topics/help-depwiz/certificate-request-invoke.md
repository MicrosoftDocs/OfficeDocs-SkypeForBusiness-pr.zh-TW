# <a name="certificate-request-invoke"></a>證書申請 (Invoke)
 
[**執行命令**] 頁面會顯示 [任務狀態] 窗格, 就是執行證書要求時所執行的命令。 完成工作後, [摘要] 窗格下方的下拉式清單中會包含一份記錄檔案清單。 使用下拉式清單選取要查看的記錄檔, 然後按一下 [**查看記錄**]。 Internet Explorer 會啟動並顯示記錄檔。 如果要求失敗, 您應該查看記錄檔, 以判斷失敗的原因。 您可以重新執行憑證要求, 以解決問題, 修正記錄檔中指出的問題。 繼續執行憑證要求頁面, 並再次要求證書。
