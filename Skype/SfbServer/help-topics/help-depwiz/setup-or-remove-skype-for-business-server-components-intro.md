# <a name="setup-or-remove-skype-for-business-server-components-intro"></a>安裝或移除商務用 Skype Server 元件 (簡介)
 
您準備安裝及啟動或停用，並卸載商務用 Skype Server 2015 元件。 將執行的進程是由發佈至中央管理存放區之定義的目前設定所定義。 若要順利完成工作，您必須以本機系統管理員群組成員的身分登入，而且至少必須能夠讀取 Active Directory 網域服務。
  
按 **[下一步]** 繼續 **執行 [執行命令** ] 畫面。
