# <a name="pending-certificate-invoke"></a>擱置的憑證 (叫用)
 
[**執行命令**] 頁面會顯示正在取得未決憑證之命令的狀態。 完成工作後，就可以使用「**取得擱置**中的憑證」記錄檔。 按一下 [**查看記錄**]，查看處理及任務完成狀態。 請按 [下一步]**** 繼續。
  
> [!IMPORTANT]
> 如果證書尚未由憑證授權單位系統管理員核准，則檢索擱置的要求記錄盤案會指出有擱置的要求。 如果憑證授權單位管理員拒絕要求，則記錄檔會指出失敗並報告錯誤，指出要求遭到拒絕，以及拒絕要求的人員的使用者名稱。
