# <a name="certificate-request-organization-information"></a>憑證要求 (組織資訊)
 
[**組織資訊**] 窗格可讓您指定貴組織的名稱, 以及要求憑證的組織單位 (OU) (通常是部門或部門)。 您在**組織**中輸入的資訊和**組織單位**文字方塊會出現在憑證上, 並用於在其他人查看憑證時, 識別誰要求認證。
  
> [!NOTE]
> 憑證上的 OU 名稱不一定與 Active Directory 網域服務中的 OU 相同。 根據您設定 Active Directory 的方式而定, 您所使用的名稱可能會相同, 也可能不相同。
