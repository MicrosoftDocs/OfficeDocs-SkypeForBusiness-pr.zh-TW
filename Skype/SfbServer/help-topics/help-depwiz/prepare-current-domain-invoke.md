# <a name="prepare-current-domain-invoke"></a>準備目前的網域 (叫用)
 
「執行命令」頁面會顯示正在執行的工作的摘要。請檢閱摘要畫面，確認工作成功且沒有錯誤。請檢閱警告，確定沒有非預期的警告。在摘要窗格下，您應該會看到 **工作狀態：已完成**。在下拉式清單中，應該可以看到 [準備網域] 記錄檔。如果沒有，請選取記錄檔，然後按一下 [檢視記錄檔]，檢閱工作的詳細動作和結果。按一下 [完成] 完成網域的準備。
