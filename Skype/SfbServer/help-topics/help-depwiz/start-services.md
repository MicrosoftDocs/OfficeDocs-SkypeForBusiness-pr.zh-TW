# <a name="start-services"></a>啟動服務
 
若要在已部署的伺服器角色上啟動服務，請按一下 [**步驟4：啟動服務**] 中的 [**執行**]。 您必須以本機管理員群組或對等的成員的身分登入。 
  
> [!NOTE]
> 如果已執行 [**啟動服務**]，您可以按一下 [**再次執行**] 來重新執行命令。
