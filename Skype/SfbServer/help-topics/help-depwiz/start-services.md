# <a name="start-services"></a>啟動服務
 
若要在部署的伺服器角色上啟動服務，請在 **[步驟 4：啟動服務]** 中按一下 **[執行]**。 您必須以本機 Administrators 群組成員的身分登入或同等使用者的身分登入。 
  
> [!NOTE]
> 如果已執行 [ **啟動服務** ]，您可以按一下 [ **再次執行**] 以重新執行命令。
