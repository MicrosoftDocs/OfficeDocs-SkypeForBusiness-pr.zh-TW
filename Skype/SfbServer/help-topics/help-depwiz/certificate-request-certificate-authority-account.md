# <a name="certificate-request-certificate-authority-account"></a>憑證要求 (憑證授權帳戶)
 
若要送出要求，憑證授權單位 (CA) 可能會要求不同於目前用來登入之使用者身分的認證。 若要允許憑證要求為不同的使用者，請選取 [為 **憑證授權單位單位指定替代認證**] 核取方塊，然後輸入可要求憑證之使用者的使用者名稱或 _網域_ 使用者名稱 \   。 在 [密碼] 文字方塊中，輸入您指定的使用者的密碼。 然後，憑證要求程序中 (而不是在實際憑證要求中) 便會將使用者名稱和密碼傳送至線上 CA。
