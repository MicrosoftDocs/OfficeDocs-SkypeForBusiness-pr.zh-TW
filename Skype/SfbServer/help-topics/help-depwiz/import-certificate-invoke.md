# <a name="import-certificate-invoke"></a>匯入憑證 (叫用)
 
在 [**執行命令**] 頁面上，查看已匯入憑證的摘要資訊。 在下拉式清單中，選取 [匯入憑證記錄檔]。 請務必在下拉式清單中看到 [匯入憑證記錄檔]。 按一下 [**查看記錄**]，查看憑證匯入的詳細資料。 完成查看摘要及記錄檔後，請按一下 **[完成]**。
