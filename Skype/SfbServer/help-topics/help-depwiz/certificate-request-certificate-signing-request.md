# <a name="certificate-request-certificate-signing-request"></a>證書要求 (證書簽署要求)
 
若要建立離線要求, 您必須先建立證書簽署要求 (CSR)。 在 [檔案名] 文字方塊中, 輸入 CSR 的路徑**** 和檔案名。 您可以按一下 **[流覽]** , 選擇建立 CSR 的位置。 請按 [下一步]**** 繼續。
  
按一下 [**返回**], 返回上一個畫面。 按一下 [**取消**], 不需變更系統, 即可結束進行。
  
> [!IMPORTANT]
> 在這個步驟中建立的檔案會傳送到您的公用憑證授權單位 (CA)、內部離線 CA, 或提交給您的 CA 管理員進行處理。 請注意檔案位置和名稱, 以便您可以進行檢索並將 CSR 提交至 CA。 [預設位置] 是執行證書嚮導之人員的 [檔] 目錄。
