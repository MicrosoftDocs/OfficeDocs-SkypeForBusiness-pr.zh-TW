# <a name="certificate-request-specify-termplate"></a>憑證要求 (指定範本)
 
[指定其他憑證範本]**** 頁面可讓您定義非預設使用之 WebServer 憑證範本的憑證範本。請選取 [將其他憑證範本用於所選的憑證授權單位]**** 核取方塊，然後在 [憑證範本名稱]**** 文字方塊中定義其他憑證範本的名稱。您所使用的範本名稱，必須與憑證授權單位 (CA) 中所定義的名稱相同。按 [上一步]**** 回到前一頁；按 [取消]**** 結束憑證要求程序。
  
> [!CAUTION]
> 唯有當公用 CA 建議您使用其系統上定義的特定範本來發出憑證時，才應該使用這個選項。如果憑證是由您的內部 CA 發出，您的 CA 系統管理員應該建議其他憑證範本使用什麼名稱。當您的組織已定義新的 WebServer 範本，且停用預設的 WebServer 範本時，這個選項極具價值。
