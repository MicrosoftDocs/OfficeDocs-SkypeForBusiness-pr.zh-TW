# <a name="install-or-update-skype-for-business-server-system"></a>安裝或更新商務用 Skype Server 系統

若要開始安裝，或更新現有的商務用 Skype Server 2015 部署，請按一下 [**安裝或更新商務用 Skype server**]。 本節中的步驟會根據拓撲結構建立檔，來安裝特定伺服器角色專用的元件。 開始執行這系列步驟之前，您必須先在拓撲建立器中定義部署。 您必須以本機系統管理員以及伺服器所在網域的 Domain Admins 群組的成員身分，登入所部署的電腦。 完成此步驟之前，電腦必須已加入網域。

如果您要更新目前已部署的伺服器，您必須在拓撲建立器中進行您想要的變更。
