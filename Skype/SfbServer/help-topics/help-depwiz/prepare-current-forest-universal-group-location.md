# <a name="prepare-current-forest-universal-group-location"></a>準備目前的樹系 (萬用群組位置)
 
您必須指出您需要在林中建立通用群組的位置。 [**通用群組位置**] 頁面會使用您登入之電腦所屬的網域作為預設的成員。 如果您想要在預設**本地域**以外的網域中建立通用群組，請選取 [**網域 FQDN** ]，然後在文字方塊中輸入完整的功能變數名稱（FQDN）。 如果您是從您想要建立通用群組的網域執行林準備工作，則提供備用網域很有説明。 請按 [下一步]**** 繼續。
