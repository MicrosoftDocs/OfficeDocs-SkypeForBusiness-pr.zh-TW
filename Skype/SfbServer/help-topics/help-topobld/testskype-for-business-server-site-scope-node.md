# <a name="test---skype-for-business-server-site-scope-node"></a>測試 - 商務用 Skype Server 網站範圍節點
 
此節點會顯示您拓撲上定義的其中一個網站。您可以執行下列動作： 
  
- **新增。** 將新的集區或伺服器新增至網站。
    
- **編輯內容。** 編輯網站內容 (例如名稱和位置)，並指定網站是否要使用通話許可控制管理頻寬，以及是否要將此網站的 Director 集區或 Edge 集區用為同盟路由。
    
- **刪除。** 從您的拓撲移除此網站。
    
除了這些選項之外，您也可以執行下列一般工作：
  
- **拓撲。** 您可以使用此選項來定義新的拓撲，開啟現有的拓撲，下載目前的拓撲，儲存拓撲的複本，發佈拓撲，安裝資料庫，合併元件，或移除部署。
    
- **檢視。** 您可以使用這個選項來自訂拓撲產生器中的檢視，方法是選取要在主控台樹狀目錄檢視中顯示及隱藏的項目。
    
- **說明。** 您可以使用這個選項，存取作用中節點的說明。
