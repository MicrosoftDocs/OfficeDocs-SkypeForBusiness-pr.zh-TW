# <a name="define-the-root-trunk-for-a-new-ip-or-pstn-gateway"></a>定義新 IP 或 PSTN 閘道的根主幹

您可以設定下列各項，以定義 IP 或公開交換電話網絡（PSTN）的根主幹：

- **主幹名稱**：定義與主幹相關聯的完全限定功能變數名稱

- **IP/PSTN 閘道的偵聽埠**：定義此主幹將偵聽的埠

- **SIP 傳輸通訊協定**：根據主幹需求從清單中選取 [ **TCP** ] 或 [ **TLS**]。

- **關聯的中繼伺服器**：從您部署中的可用轉送伺服器清單中選取

- **關聯的中繼伺服器埠**：定義所選中繼伺服器正在偵聽的埠
