# <a name="add-front-end-associations-2010"></a>新增前端關聯 2010

您可以將伺服器角色與前端集區產生關聯，對需要部署其他伺服器的特定功能啟用支援。 您也可以稍後將伺服器角色與前端集區產生關聯。 可與前端集區相關聯的伺服器角色包括下列各項：

1. 封存伺服器。 如需規劃及實施封存支援的詳細資訊，請參閱規劃檔中的 [定義組織需求](/previous-versions/office/lync-server-2013/lync-server-2013-defining-your-requirements-for-archiving) ，以及 [定義及設定前端集區或 Standard Edition Server](/previous-versions/office/lync-server-2013/lync-server-2013-define-and-configure-a-front-end-pool-or-standard-edition-server) ，以及 [部署檔中的](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-archiving) 封存。

2. 監控伺服器。 如需關於實施監控支援的詳細資訊，請參閱規劃檔中的 [規劃監控](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-monitoring) 和 [定義及設定前端集區或 Standard Edition Server](/previous-versions/office/lync-server-2013/lync-server-2013-define-and-configure-a-front-end-pool-or-standard-edition-server) ，以及部署檔中的 [部署監控](/previous-versions/office/lync-server-2013/lync-server-2013-deploying-monitoring) 。

3. A/V Edge service。 如需有關執行 A/V Edge service 的詳細資訊，請參閱 [外部使用者存取的概述](/previous-versions/office/lync-server-2013/lync-server-2013-overview-of-external-user-access)。
