# <a name="add-front-end-sql-store-page-for-lync-server-2010"></a>新增 Lync Server 2010 的前端 SQL 存放區頁面
 
您可以為您要設定的新前端池定義 SQL Server 存儲的屬性。
  
> [!IMPORTANT]
> 如果您正在設定標準版 server，則會顯示此對話方塊，但您將無法設定設定。 請按 [下一步]**** 繼續。
  
您可以設定的設定：
  
- **使用先前定義的 Sql server store**：從 [前端] 池存放區的清單中選取現有的 sql server 存放區。
    
- 選取 **[定義新的 Sql server store** ] 來建立新的 sql server store。
    
    您在**Sql SERVER FQDN**中輸入 sql server 的完整功能變數名稱
    
    預設會先選取**預設實例**。 如果您想要定義實例，請選取 [**命名實例**]，然後輸入您需要建立的實例名稱。
    
  **返回**：回到上一個對話方塊
  
  **[下一**步]：提交變更並移至 [下一步] 對話方塊
  
  [**取消**]：捨棄變更並取消設定
  
  說明 **：顯示**說明資訊，例如本主題
