# <a name="legacy-merge-edge-type"></a>舊版合併 Edge 類型
 
不論是 [單一 Edge Server] 或 [負載平衡合併的 Edge] 伺服器，請按 [下一步] 指定組態資訊。 針對 **展開的 edge**，每個伺服器的設定必須變更為合併的 edge 設定，才能與 Lync server 2013 合併。
