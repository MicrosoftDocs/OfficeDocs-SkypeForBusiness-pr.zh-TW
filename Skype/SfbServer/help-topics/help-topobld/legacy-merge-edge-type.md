# <a name="legacy-merge-edge-type"></a>舊版合併 Edge 類型
 
針對**單一邊緣伺服器**或**負載平衡的整合邊緣**伺服器，按一下 **[下一步]** 以指定設定資訊。 針對延伸的**邊緣**，必須先將每個伺服器的設定變更為整合的邊緣配置，才能與 Lync server 2013 合併。
