# <a name="add-edge-machine-nat-ip"></a>新增 Edge 電腦 NAT IP
 
如果這個 Edge 池的公用 IP 位址是由網路位址轉譯（NAT）所轉譯，您必須指定要翻譯的公用 IP 位址。 這應該是 A/V Edge 服務的外部 IP 位址。
  
若要變更 Edge 池是否使用 NAT 轉譯，請以滑鼠右鍵按一下 Edge 池的名稱，然後按一下 [**編輯屬性**]。
