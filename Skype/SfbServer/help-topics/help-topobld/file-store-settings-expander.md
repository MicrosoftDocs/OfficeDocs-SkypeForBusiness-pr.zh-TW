# <a name="file-store-settings-expander"></a>檔案存放區設定展開工具
 
若要編輯已定義之檔案存放區的屬性：
  
若是檔案共用名稱稱，您可以定義新共用的名稱。 接著，您必須在 [檔案伺服器] 完整功能變數名稱（FQDN）屬性所命名的伺服器上建立共用。
  
> [!NOTE]
> 您無法變更 [檔案伺服器 FQDN] 屬性中指定的伺服器 FQDN。 如果您需要將檔案存放區檔案共用儲存在不同的伺服器上，您可以建立新的檔案存放區專案。
