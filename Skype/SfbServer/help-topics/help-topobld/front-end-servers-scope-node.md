# <a name="front-end-servers-scope-node"></a>前端伺服器範圍節點
 
[Enterprise Edition 前端集區]**** 節點會列出此網站上的前端集區，[Standard Edition 前端伺服器]**** 節點則會列出此網站上的 Standard Edition Server。您可以執行下列動作：
  
- **新增前端集區**。啟動 [定義新前端集區精靈]****，以便將新的前端集區或 Standard Edition Server 新增到此網站上的拓撲。
    
此外，您還可以執行下列一般工作：
  
- **拓撲**。您可以使用這個選項來定義新的拓撲、開啟現有的拓撲、下載目前拓撲、儲存拓撲的複本、發行拓撲、安裝資料庫、合併來自舊版的元件，或是移除部署。
    
- **檢視**。您可以使用這個選項來自訂拓撲產生器中的檢視，方法是選取要在主控台樹狀目錄檢視中顯示及隱藏的項目。
    
- **說明**。您可以使用這個選項，存取作用中節點的說明。
