# <a name="sql-instances-scope-node"></a>SQL 執行個體範圍節點
 
[ **SQL 書店**] 節點會列出拓撲中的 SQL Server 資料庫。 您可以執行下列動作：
  
- **新增 SQL 存放區**。 啟動 [**定義新的 Sql Store** ] 嚮導，您可以使用它將新的 sql Server 資料庫新增到拓撲。
    
除了 SQL Server 資料庫選項之外，您還可以執行下列一般工作：
  
- **拓撲**。您可以使用這個選項來定義新的拓撲、開啟現有的拓撲、下載目前拓撲、儲存拓撲的複本、發行拓撲、安裝資料庫、合併來自舊版的元件，或是移除部署。
    
- **檢視**。您可以使用這個選項來自訂拓撲產生器中的檢視，方法是選取要在主控台樹狀目錄檢視中顯示及隱藏的項目。
    
- **說明**。您可以使用這個選項，存取作用中節點的說明。
