# <a name="add-branch-site-name"></a>新增分支網站名稱

若要定義新的分支網站，請指定下列項目：

新分支網站的名稱。您定義的名稱在部署中必須是唯一的。您可以決定使用任何命名慣例。通常建議依城市、位置名稱、分公司名稱命名。名稱值是必要項目。

選用的描述欄位可讓您指定額外的描述性資訊，這不是基礎結構名稱的一部分，只是提供位置讓您輸入文字來進一步描述網站。

如需詳細資訊，請參閱〈[Sites](https://technet.microsoft.com/library/022cb6dd-37e2-4882-a53e-5ddfdbc6f53a.aspx)〉。
