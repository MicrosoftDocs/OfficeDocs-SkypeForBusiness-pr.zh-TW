# <a name="remove-deployment-introduction"></a>移除部署簡介
 
只有在您想要解除委任整個部署時，才執行 **移除部署嚮導** 。 這個嚮導會建立由單一前端伺服器組成的空拓撲。 在您執行此嚮導併發布空的拓撲之後，就無法還原為原始的拓撲。
  
> [!IMPORTANT]
> 在您按 **[下一步]** 之前，請務必移除任何非使用中的電腦、停用整合通訊中的所有使用者與端點，並移除嚮導第一頁所指定的所有會議目錄。
