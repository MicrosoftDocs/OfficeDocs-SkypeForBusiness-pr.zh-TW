# <a name="add-director-features"></a>新增 Director 功能

如果您希望主管池包含監視伺服器，請選取 **[監視] （CDR 與 QoE 度量值）**。

如需新增主管池的詳細資訊，請參閱[控制器的案例](https://technet.microsoft.com/library/d2cf384a-0860-4779-80ce-cba2543be322.aspx)和[設定導演](https://technet.microsoft.com/library/408b76f7-6fdd-4e50-8a3e-e87db12c1394.aspx)。
