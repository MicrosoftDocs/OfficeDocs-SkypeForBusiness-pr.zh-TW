# <a name="publish-topology-select-cms-page"></a>發佈拓撲選取 [CMS] 頁面

您可以使用 [拓撲建立器] 發佈已設定的拓撲。 系統會要求您從清單中選取前端伺服器或前端池將會擔任中央管理儲存區的角色。 在任何指定時間, 只有一個前端伺服器或前端池可以擁有此角色。

如需中央管理儲存區的詳細資訊, 請參閱[將 Lync server 2010 中央管理伺服器移至 Lync server 2013](https://technet.microsoft.com/library/30cc98f2-1916-4dbe-99d0-8df5368ed3ec.aspx)。
