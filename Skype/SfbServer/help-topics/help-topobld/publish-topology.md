# <a name="publish-topology"></a>發行拓撲後續步驟
 
當您開始發佈程式時，就會顯示 [**發佈拓撲**] 頁面。 繼續進行之前，請先閱讀並確認列出的工作已完成。 當您準備好要繼續時，請按 **[下一步]**。 如果您必須關閉發佈程式，請按一下 [**取消**]。
