# <a name="publish-topology"></a>發行拓撲
 
當您開始發行程式時，就會顯示 [ **發行拓撲** ] 頁面。 請閱讀並確認所列的工作已完成，再繼續進行。 當您準備好要繼續時，請按 **[下一步]**。 如果您必須關閉發佈程式，請按一下 [ **取消**]。
