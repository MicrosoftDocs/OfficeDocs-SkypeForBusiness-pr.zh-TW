# <a name="lync-server-2010-create-new-expander"></a>Lync Server 2010 建立新展開工具
 
您可以在拓撲產生器的 Lync Server 2010 節點中，以滑鼠右鍵按一下 [ **Lync server 2010**]，然後選取 [ **新增**]，再選取要建立的伺服器角色，以建立新的伺服器角色。
