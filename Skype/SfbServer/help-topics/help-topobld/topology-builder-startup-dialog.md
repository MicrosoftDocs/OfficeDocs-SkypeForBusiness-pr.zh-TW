# <a name="topology-builder-startup-dialog"></a>拓撲產生器啟動對話方塊

您啟動拓撲建立器，以設定您的 Microsoft Lync Server 2013 部署的屬性和配置。 在啟動時，拓撲建立器會顯示一個對話方塊，詢問拓撲檔的來源。 您可以選取三個選項的其中一個：

- **從現有部署下載拓撲**如果您已執行拓撲產生器，且已成功將拓撲發佈到中央管理伺服器，請選取此選項。 這個選項會要求並下載目前設定的本機複本，允許您編輯及設定現有的部署。

- **從本機檔案開啟拓撲**如果您正在設定部署，但尚未發佈設定，請選取此選項。 這個選項可讓您儲存配置工作在拓撲建立器中的本機複本，並在日後再次顯示。

- **新拓撲**選取這個選項，即可開始配置新的拓撲檔。 此選項會開啟您設定的新拓撲檔，以符合您將設定的部署需求。
