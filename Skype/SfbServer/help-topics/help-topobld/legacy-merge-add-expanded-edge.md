# <a name="legacy-merge-add-expanded-edge"></a>舊版合併新增擴充的 Edge
 
商務用 Skype Server 不支援合併舊版 Office 通訊伺服器 2007 R2 加寬 Edge 伺服器設定。 在合併 Edge Server 資訊之前，擴充的 Edge Server 必須先轉換為合併的 Edge Server 設定。
