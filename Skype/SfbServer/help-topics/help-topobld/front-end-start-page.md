# <a name="front-end-start-page"></a>前端開始頁面

使用此精靈來定義網站的新前端集區。 每個網站都至少必須有一個 Enterprise Edition 集區或 Standard Edition Server。 如需使用精靈建立新前端集區時之需求的詳細資訊，請參閱＜部署＞文件中的[定義和設定前端集區](https://technet.microsoft.com/library/713fc263-23dd-414a-b001-82932e4fe966.aspx)。
