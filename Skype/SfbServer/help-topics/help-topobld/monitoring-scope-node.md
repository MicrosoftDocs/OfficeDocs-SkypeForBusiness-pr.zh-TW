# <a name="monitoring-scope-node"></a>監控範圍節點
 
在 [Lync Server 2013 拓撲產生器] 中，您可以執行與執行監控之個別伺服器相關的工作，方法是在主控台樹中以滑鼠右鍵按一下執行監控的伺服器，按一下工具列中的 [ **動作** ]，或是按一下 [動作] 窗格中的工作，然後按一下執行監視之伺服器的下列其中一個選項：
  
- **編輯內容**。 開啟執行監控之伺服器的「**編輯** 內容」頁面，您可以在其中變更執行監控之伺服器的完整功能變數名稱 (FQDN) 或監視 SQL Server 資料庫。
    
- **刪除**。 從拓撲和監視 SQL Server 資料庫中刪除執行監控的伺服器。
    
除了本主題先前所描述之執行監控的伺服器選項，您還可以從 [監控] 節點使用一般的選項，這些選項也能從主控台樹狀目錄中的其他節點使用。包括下列選項：
  
- **拓撲**。 您可以使用此選項來定義新的拓撲，開啟現有的拓撲，下載目前的拓撲，儲存拓撲的複本，發佈拓撲，安裝資料庫，或移除部署。
    
- **說明**。您可以使用這個選項，存取作用中節點的說明。
