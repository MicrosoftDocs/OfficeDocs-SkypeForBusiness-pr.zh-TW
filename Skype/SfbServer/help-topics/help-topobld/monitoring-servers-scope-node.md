# <a name="monitoring-servers-scope-node"></a>監控伺服器範圍節點
 
**[監控伺服器]** 節點會列出您拓撲中的監控伺服器。您可以執行下列動作：
  
- **新增監控伺服器**。啟動 **[定義新的監控伺服器]** 精靈，以供您將新的監控伺服器新增至拓撲。
    
除了監控伺服器選項，您還可以執行下列一般工作：
  
- **拓撲**。您可以使用此選項來定義新的拓撲、開啟現有的拓撲、下載目前的拓撲、儲存拓撲的副本、發行拓撲、安裝資料庫、合併舊版 Office Communications Server 的元件，或移除部署。
    
- **檢視**。您可以使用這個選項來自訂拓撲產生器中的檢視，方法是選取要在主控台樹狀目錄檢視中顯示及隱藏的項目。
    
- **說明**。您可以使用這個選項，存取作用中節點的說明。
