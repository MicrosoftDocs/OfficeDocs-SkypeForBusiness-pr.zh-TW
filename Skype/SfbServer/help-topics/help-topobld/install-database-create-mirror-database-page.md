# <a name="install-database-create-mirror-database-page"></a>[安裝資料庫建立鏡像資料庫] 頁面
 
這是初步檔, 可能會變更。 空白主題是以預留位置形式提供。
  
如果您在發佈程式期間選擇不安裝 SQL Server 鏡像資料庫, 您可以使用 [安裝資料庫], 從 [拓撲建立器] 中的任何節點上按一下 [**安裝資料庫**]。 您可以選取要安裝的鏡像存放區。 按一下 [**設定**] 以定義任何高級設定。
