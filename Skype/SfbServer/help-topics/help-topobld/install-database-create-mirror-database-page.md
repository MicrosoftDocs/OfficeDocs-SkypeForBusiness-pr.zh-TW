# <a name="install-database-create-mirror-database-page"></a>安裝資料庫建立鏡像資料庫頁面
 
這是初步檔，而且可能會變更。 空白主題會以預留位置形式包含。
  
如果您選擇在發佈過程中未安裝 SQL Server 鏡像資料庫，您可以使用從右擊拓撲產生器中的任何節點來安裝 **這些資料庫。** 您可以選取要安裝的鏡像存放區。 按一下 [**設定**] 以定義任何高級設定。
