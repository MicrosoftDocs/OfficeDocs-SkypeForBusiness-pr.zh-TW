# <a name="add-director-association-for-lync-server-2010"></a>為 Lync Server 2010 新增 Director 關聯
 
您可以定義 Director 或 Director 集區是否要有相關聯的監控伺服器。 選取 [將監控伺服器與 Director 或 Director 集區建立關聯] 核取方塊。
  
> [!TIP]
> 若尚未建立監控伺服器資料庫 (和選用實例) ，您可以在稍後選取此核取方塊，將監控伺服器與 Director 或 Director 集區建立關聯。
