# <a name="add-director-association-for-lync-server-2010"></a>新增 Lync Server 2010 的導向關聯
 
您可以定義控制器或控制器池是否會有與其相關聯的監視伺服器。 選取核取方塊, 將監視伺服器與主管或主管池建立關聯。
  
> [!TIP]
> 如果您還沒有建立監視伺服器資料庫 (以及選用的實例), 您可以在稍後選取此核取方塊, 以將監視伺服器與主管或主管池建立關聯, 以便日後建立。
