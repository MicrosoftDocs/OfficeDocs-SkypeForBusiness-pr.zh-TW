# <a name="add-edge-machine-external-ip-2010"></a>新增 Edge 電腦外部 IP 2010

您可以在此頁面上設定屬性, 將 Edge 伺服器指向 [存取邊緣服務]、[Web 會議邊緣服務] 和 [A/V 邊緣服務] 的外部位置。

- 在 [**外部 IPv4 位址**] 底下, 于 [**存取邊緣服務**]、[ **Web 會議 Edge 服務**] 和 [ **A/V 邊緣**] 服務中輸入 Edge 伺服器的外部 ipv4 位址。

- 按一下 **[確定]** 儲存並提交您對拓撲檔所做的變更。

- 按一下 [**取消**] 放棄變更, 然後關閉 [**新增伺服器到 Edge 池**] 嚮導。

- 按一下**** [說明] 以閱讀此說明主題。

如需將邊緣伺服器新增至現有拓撲的詳細資料, 請參閱[定義您的邊緣拓撲](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)。 如需有關移植邊緣伺服器的資訊, 請參閱[階段 5: 將 Lync server 2013 Preview Edge 伺服器新增至試驗池](https://technet.microsoft.com/library/d2fd0a2f-8fec-4f86-a5b0-7f7d03172016.aspx)。
