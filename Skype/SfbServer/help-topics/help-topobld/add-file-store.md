# <a name="add-file-store"></a>新增檔案存放區
 
若要定義新的檔案存放區，請指定下列各項：
  
- 將裝載新檔案存放區的伺服器完整網域名稱 (FQDN)。 伺服器的 FQDN 必須存在於網域名稱系統中 (DNS) 才能發行拓撲。
    
- 主機伺服器上的檔案共用名稱。檔案共用必須先存在，您才能發行拓撲。
