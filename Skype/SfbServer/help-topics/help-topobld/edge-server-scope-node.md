# <a name="edge-server-scope-node"></a>Edge Server 範圍節點
 
此節點顯示集區的完整網域名稱 (FQDN) 與此 Edge 集區的其他基本設定。您可以執行下列動作：
  
- **新增伺服器**。啟動 [定義新的伺服器精靈]****，以便將其他 Edge Server 新增到此集區。
    
- **編輯內容**。開啟此伺服器的「編輯內容」**** 頁面，在此設定內部與外部集區 FQDN 及連接埠，啟用與停用同盟關係及網路位址轉譯 (NAT)，並指定下一步。
    
- **刪除**。從拓撲刪除集區。
    
除了這些選項之外，您也可以執行下列一般工作：
  
- **拓撲**。您可以使用這個選項來定義新的拓撲、開啟現有的拓撲、下載目前拓撲、儲存拓撲的複本、發行拓撲、安裝資料庫、合併元件，或是移除部署。
    
- **檢視**。您可以使用這個選項來自訂拓撲產生器中的檢視，方法是選取要在主控台樹狀目錄檢視中顯示及隱藏的項目。
    
- **說明**。您可以使用這個選項，存取作用中節點的說明。
