若要 **定義 Survivable 分支裝置 FQDN**，請輸入 (fqdn)  (中所定義的完整功能變數名稱， (DNS) # A5 中的 Survivable Branch 裝置或 Survivable Branch Server 的 [ **fqdn** ] 文字方塊中。 按 **[下一步]** 繼續。
