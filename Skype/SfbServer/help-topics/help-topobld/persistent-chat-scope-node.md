# <a name="persistent-chat-scope-node"></a>常設聊天室範圍節點
 
若要開始建立新的 Persistent Chat Server 或 Persistent Chat Server 集區，請在拓撲產生器中，以滑鼠右鍵按一下 **Persistent Chat** 集區，然後選取 [**新增持久聊天集** 區]。
  
- **拓撲**。您可以使用此選項來定義新的拓撲、開啟現有的拓撲、下載目前的拓撲、儲存拓撲的副本、發行拓撲、安裝資料庫、合併舊版 Office Communications Server 的元件，或移除部署。
    
- **說明**。您可以使用這個選項，存取作用中節點的說明。
