# <a name="persistent-chat-scope-node"></a>常設聊天室範圍節點
 
若要開始建立新的持續聊天伺服器或持久聊天伺服器池, 請在拓撲結構建立器中, 以滑鼠右鍵按一下 [**永久聊天**] 池, 然後選取 [**新增持久聊天] 池**
  
- **拓撲**。 您可以使用這個選項來定義新的拓撲、開啟現有的拓朴、下載目前的拓撲、儲存拓撲複本、發佈拓撲、安裝資料庫、合併舊版 Office 通訊伺服器的元件, 或移除部署.
    
- **說明**。 您可以使用這個選項，存取作用中節點的說明。
