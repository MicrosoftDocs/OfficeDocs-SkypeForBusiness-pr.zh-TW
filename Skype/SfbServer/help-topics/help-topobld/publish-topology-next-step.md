# <a name="publish-topology-next-step"></a>發行拓撲後續步驟

拓撲發佈完成後，就會顯示 [**發佈嚮導完成]** 頁面。

當您在發佈拓撲流程期間完成步驟時，就會建立記錄，其中包含所執行任務的相關資訊。 若要查看與特定步驟相關的記錄檔，請選取您要用來查看記錄檔的步驟，然後按一下 [**查看記錄**]。

> [!NOTE]
> 在發佈拓撲流程期間，不是所有步驟都會有相關聯的記錄檔。 如果您選取一個步驟，且有可用的 [**查看記錄**]，此步驟已建立一個記錄檔。 如果 [**查看記錄**] 按鈕無法使用（呈現灰色），則表示沒有為此步驟建立記錄檔。

若要查看完成拓撲所建議的後續步驟，請按一下連結 [**按一下這裡以開啟待辦事項清單**]。 按一下此連結開啟的文字檔，會指出您必須執行本機設定的伺服器，以及完成部署所需採取的任何其他步驟。 針對清單中的每個工作，請參閱部署檔中的適當章節。 如需詳細資訊，請參閱[部署 Lync Server 2013 Preview](https://technet.microsoft.com/library/b76795a4-4e71-4c70-a5c0-d1197fa8028c.aspx)，然後參閱待辦事項清單中所述之每個伺服器角色的相關章節。

按一下 **[完成]**

如需詳細資訊，請參閱[發佈拓撲](https://technet.microsoft.com/library/3b5a744b-b3a8-4538-a55e-e2e4f72dff47.aspx)。
