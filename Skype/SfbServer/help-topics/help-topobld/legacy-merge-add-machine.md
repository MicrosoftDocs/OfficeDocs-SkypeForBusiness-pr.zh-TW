# <a name="legacy-merge-add-machine"></a>舊版合併新增電腦
 
若要定義組成 Edge Server 集區的電腦，請輸入至少一部電腦的完整功能變數名稱 (FQDN) （此為舊版負載平衡、合併的 Edge 集區）。
