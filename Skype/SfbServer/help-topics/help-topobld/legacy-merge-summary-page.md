# <a name="legacy-merge-summary-page"></a>舊版合併摘要頁面

從 Microsoft Office 通訊伺服器 2007 R2 到 Lync Server 2013 的遷移使用拓撲產生器的功能來合併現有的拓撲。 [摘要] 畫面會指定您在將 Office 通訊伺服器 2007 R2 合併至 Lync Server 2013 拓撲結構建立程式配置期間所做的所有動作與選取專案。

查看摘要, 並判斷是否已正確定義您的選項。 如果您對摘要中反映的選項感到滿意, 請繼續合併。
