# <a name="associate-mediation-server-with-edge"></a>建立中繼伺服器與 Edge 的關聯
 
若要定義中繼伺服器，請指定將中繼伺服器與前端伺服器以及邊緣伺服器或池關聯的設定。 若要將中繼伺服器或池與 Edge 伺服器或 Edge 伺服器池關聯，請執行下列動作：
  
1. 從下拉式清單中，選取 [邊緣] 池或 [邊緣伺服器]，以與中繼伺服器建立關聯。
    
2. 請按 [下一步]**** 繼續。
    
> [!NOTE]
> 如果您在部署中還沒有定義 Edge 伺服器或 Edge 伺服器池，下拉式清單不會包含專案。 您可以按一下 **[下一步]** ，在您將 Edge 伺服器或池新增到您的部署之後，繼續並編輯轉送伺服器的設定。
  
如需其他詳細資料，請參閱[在商務用 Skype server 2015 的拓撲建立器中部署轉送伺服器](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)。
