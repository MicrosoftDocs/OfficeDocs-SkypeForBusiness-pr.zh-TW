# <a name="edit-office-web-apps-server-settings"></a>編輯 Office Web Apps 伺服器設定

您編輯已設定的 Office Web Apps 伺服器的屬性。 下列屬性可供編輯:

 **Office Web Apps 伺服器 FQDN**: 這個屬性會定義 Office Web apps 伺服器的完整功能變數名稱, 且應與網域名稱系統 (DNS) 主機 a 或 AAAA (如果使用 IPv6) 記錄相符。

 **Office Web Apps 伺服器探索 URL**: 適用于 Office Web apps server 的用戶端存取的統一資源定位器 (URL), 如果伺服器位於與您的內部網路不同的其他網路區域中, 您可能需要編輯此位址的預設值。部署.

如果此伺服器是部署在您的周邊網路中, 或在內部防火牆以外的區域網路、較不信任的網路, 且您不在網路上, 請選取 [ **Office Web Apps Server 部署在外部網路] 中**的核取方塊。內部部署的網際網路。
