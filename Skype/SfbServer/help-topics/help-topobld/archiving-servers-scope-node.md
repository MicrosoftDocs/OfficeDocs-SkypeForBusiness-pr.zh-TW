# <a name="archiving-servers-scope-node"></a>封存伺服器範圍節點
 
在拓撲產生器中，您可以執行與個別封存伺服器相關的工作，作法為在主控台樹狀目錄中的 [封存伺服器] **** 上按滑鼠右鍵，按一下工具列中的 [動作] ****(或按一下 [動作] 窗格中的工作)，然後按一下以下封存伺服器選項：
  
- **新增封存伺服器**。啟動 [定義新的封存伺服器精靈]****，以供您指定新封存伺服器的設定，包括完整網域名稱 (FQDN)、封存 SQL Server 資料庫、封存檔案存放區和集區關聯。
    
> [!NOTE]
> 如果您先前已在拓撲產生器中將封存伺服器新增至拓撲，則封存伺服器節點會包含封存伺服器。 不過，立即訊息或 web 會議（訊息）必須先設定存檔伺服器，這包括在伺服器上安裝商務用 Skype 伺服器（您要用來做為封存伺服器並設定支援）。方面. 
  
除了本主題稍早所述的封存伺服器選項，您在 [封存伺服器] 節點中還有主控台樹狀目錄中的其他節點也有的一般選項。包括下列選項：
  
- **拓撲**。 使用這個選項來定義新的拓撲、開啟現有的拓朴、下載目前的拓撲、儲存拓撲複本、發佈拓撲、安裝資料庫、合併元件，或移除部署。
    
- **檢視**。使用這個選項來自訂拓撲產生器中的檢視，方法是選取要在主控台樹狀目錄檢視中顯示及隱藏的項目。
    
- **說明**。使用這個選項存取作用中節點的說明。
