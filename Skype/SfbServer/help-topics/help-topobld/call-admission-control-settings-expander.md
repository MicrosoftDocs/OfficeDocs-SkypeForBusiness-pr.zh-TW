# <a name="call-admission-control-settings-expander"></a>通話許可控制服務設定展開工具
 
通話許可控制 (CAC) 是由地區、網站和子網路所構成的網路，可讓您根據可用頻寬來限制音訊和視訊傳輸。在設定 CAC 的網路之後，您必須啟用 CAS，才能強制執行頻寬限制。 
  
> [!NOTE]
> 您也可以使用控制台或管理命令介面 Cmdlet 來啟用 CAC。 
  
在網站的 **[編輯內容]** 對話方塊的 **[通話許可控制設定]** 區段中，您可以變更下列設定：
  
- **啟用通話許可控制** 選取此設定以啟用 CAC。 清除此設定，以停用整個網路的 CAC。 若要啟用 CAC，您必須將您的網路設定為 CAC。 如需詳細資訊，請參閱部署檔中的[商務用 Skype Server 2015 部署通話許可控制](../../deploy/deploy-enterprise-voice/deploy-call-admission-control.md)。
    
- **執行通話許可控制的前端集** 區如果您啟用 CAC，您可以變更執行它的集區。 從下拉式清單中選取集區。
