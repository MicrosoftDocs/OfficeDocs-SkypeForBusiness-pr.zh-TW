# <a name="av-mcu-settings-expander"></a>AV MCU 設定展開工具
 
若要編輯已定義之 A/V 會議伺服器池的屬性，請執行下列動作：
  
## <a name="general-settings"></a>一般設定

若要更新 A/V 會議伺服器池的完整功能變數名稱（FQDN），請輸入在網域名稱系統（DNS）中定義的該池的名稱。 您必須擁有與 A/V 會議伺服器池名稱相對應的 DNS 主機（A）記錄。
  
## <a name="av-conferencing-service"></a>A/V 會議服務

若要關聯 A/V 會議伺服器池，請選取您必須與 A/V 會議伺服器池建立關聯的前端池或池。
  
> [!NOTE]
> 如果在 [**選取要與此 A/v 會議池] 窗格建立關聯**的 [前端] 池] 中沒有顯示 [前端] 池，就表示您已 collocated 具有 [前端] 池的 A/v 會議服務器，或未選取在 [前端] 池屬性的 [**功能與功能**] 區段中新增會議。
  


完成後，按一下 **[確定]** 以完成 [A/V 會議伺服器池設定] 的編輯作業。 按一下 [取消]**** 捨棄任何變更。
