# <a name="director-general-settings-expander-for-lync-server-2010"></a>Director 一般設定展開工具 (適用於 Lync Server 2010)
 
若要設定 Director 或 Director 集區的 **一般** 屬性：
  
 **FQDN**：輸入 Director 或 director 集區的完整功能變數名稱
  
 **協會**
  
 **檔存放區**：從清單中選取 Director 或 director 集區的檔案存放區
  
按一下 [新增] 定義新的檔案存放區
  
選取 [ **關聯監控伺服器** ] 核取方塊，然後從清單中選取監控伺服器。
  
按一下 [ **新增** ] 定義新的監控伺服器
  
 **確定**：接受並認可對話方塊的變更。
  
 **取消**：捨棄變更，並關閉對話方塊。
  
 **說明**：顯示此說明畫面。
