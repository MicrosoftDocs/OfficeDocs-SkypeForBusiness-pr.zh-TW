# <a name="director-general-settings-expander-for-lync-server-2010"></a>Lync Server 2010 的主管一般設定擴展器
 
若要設定控制器或主管池的**一般**屬性:
  
 **FQDN**: 輸入主管或主管池的完整功能變數名稱
  
 **關聯**
  
 檔案**存放區**: 從清單中選取控制器或主管池的檔案存放區
  
按一下 [**新增**] 以定義新的檔案存放區
  
選取 [**關聯監視伺服器**] 核取方塊, 然後從清單中選取監視伺服器
  
按一下 [**新增**] 以定義新的監視伺服器
  
 **確定**：接受並認可對話方塊的變更。
  
 **取消**：捨棄變更，並關閉對話方塊。
  
 **說明**：顯示此說明畫面。
