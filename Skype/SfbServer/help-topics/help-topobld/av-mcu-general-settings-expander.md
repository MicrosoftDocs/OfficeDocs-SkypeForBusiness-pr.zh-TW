# <a name="av-mcu-general-settings-expander"></a>AV MCU 一般設定展開工具
 
若要編輯您已定義的 A/V 會議伺服器集區的內容，您可以執行下列動作：
  
## <a name="general-settings"></a>一般設定

透過輸入網域名稱系統 (DNS) 中所定義的集區名稱，更新 A/V 會議伺服器集區的完整網域名稱 (FQDN)。
  
## <a name="av-conferencing-service"></a>A/V 會議服務

若要與 A/V 會議伺服器集區建立關聯，請選取必須與 A/V 會議伺服器集區建立關聯的一或多個前端集區。
  
> [!NOTE]
> 如果 [選取要與此 A/V 會議集區產生關聯的前端集區] 窗格中未顯示前端集區，則表示您已組合 A/V 會議伺服器與前端集區，或在前端集區內容的 [功能] 區段中未選擇新增會議。
  


完成時，按一下 [確定] 完成 A/V 會議伺服器集區設定的編輯。按一下 [取消] 捨棄任何變更。
