# <a name="add-edge-server-support"></a>新增邊緣伺服器支援

您可以在部署初始拓撲時新增外部使用者存取的支援, 或在稍後的時間新增支援。 如需新增 Edge Server 至現有拓撲的詳細資訊，請參閱 Edge Server 部署文件中的〈[Define Your Edge Topology](https://technet.microsoft.com/library/787b23f1-8fa0-4c37-abf2-c516c5dd66f0.aspx)〉。
