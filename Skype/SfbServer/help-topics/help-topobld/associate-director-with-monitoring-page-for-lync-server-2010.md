# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>建立 Director 與監控的關聯頁面 (適用於 Lync Server 2010)
 
若要設定監控伺服器並與 Director 或 Director 集區建立關聯：
  
在 [ **監視伺服器** ] 底下，從 [監控伺服器] 清單中選取。
  
如果沒有監控伺服器，而您想要立即建立監控伺服器，請按一下 [ **新增**]。
  
 **上一步**：回到上一個對話方塊
  
 **完成**：認可變更，並完成精靈
  
 **取消**：捨棄變更，並取消設定
  
 **說明**：顯示說明資訊，例如此主題
