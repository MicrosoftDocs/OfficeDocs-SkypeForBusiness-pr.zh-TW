# <a name="associate-director-with-monitoring-page-for-lync-server-2010"></a>建立 Director 與 Lync Server 2010 監視頁面的關聯
 
若要設定監視伺服器，並將它與控制器或控制器池建立關聯：
  
在 [**監視伺服器**] 底下，從 [監視伺服器] 清單中選取。
  
如果沒有監視伺服器，且想要立即建立監視伺服器，請按一下 [**新增**]。
  
 **返回**：回到上一個對話方塊
  
 **完成**：提交變更並完成嚮導
  
 [**取消**]：捨棄變更並取消設定
  
 說明 **：顯示**說明資訊，例如本主題
