# <a name="add-monitoring-server"></a>新增監控伺服器

若要將監視伺服器納入拓撲設計中，您必須指定要部署監視伺服器之伺服器的完整功能變數名稱（FQDN）。 在發佈包括監視伺服器並在伺服器上安裝商務用 Skype 伺服器的拓撲之前，您應該完成部署監視的所有先決條件。 如需這些必要條件的詳細資訊，請參閱部署文件中的〈[Deploying Monitoring](https://technet.microsoft.com/library/117f4a3e-0670-4388-a553-b9854921145f.aspx)〉。
