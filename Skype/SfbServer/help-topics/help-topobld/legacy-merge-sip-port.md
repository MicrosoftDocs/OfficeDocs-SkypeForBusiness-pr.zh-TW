# <a name="legacy-merge-sip-port"></a>舊版合併 SIP 連接埠
 
與用戶端、舊版的前端集區及伺服器進行通訊的預設傳輸控制通訊協定 (TCP) SIP 連接埠為連接埠 5061。如果沒有使用預設值，則選取 [所有集區和伺服器的 SIP 連接埠:]**** 並輸入新值。
  
如果舊有的部署已經設為自動用戶端登入，請選取 [針對各集區 SIP 連接埠設定的自動探索] **** 並按 [下一步] **** 繼續。
