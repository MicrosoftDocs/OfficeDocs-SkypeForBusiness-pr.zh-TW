# <a name="define-the-pstn-gateway-fqdn"></a>定義 PSTN 閘道 FQDN

您可以指定公用交換電話網路 (PSTN) 閘道的完整功能變數名稱 (FQDN) ，以將您的企業語音元件連線至 PSTN 或專用交換機 (PBX) 。

如需詳細資訊，請參閱 [商務用 Skype Server 中的 PSTN 連線元件 2015](../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity.md)
