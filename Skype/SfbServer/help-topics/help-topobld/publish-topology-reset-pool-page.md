# <a name="publish-topology-reset-pool-page"></a>發行拓撲重設集區頁面
 
這是初步檔，而且可能會變更。 空白主題會以預留位置形式包含。
  
若要將新電腦新增至現有的集區，您必須重設集區，讓 Windows Fabric 重新定義集區所有成員之間的連線。 若要這麼做，請選取您已新增伺服器的集區，然後按 **[下一步]**。
