# <a name="associate-branch-office-appliance-with-edge"></a>建立 Branch Office Appliance 與 Edge 的關聯
 
如果您已部署邊緣伺服器且必須提供供外部使用的媒體服務，請選擇下拉式清單中要與 Survivable 分支裝置或 Survivable 分支伺服器關聯的邊緣伺服器或池。 請按 [下一步]**** 繼續。
  
> [!NOTE]
> 如果您沒有在拓撲中定義的邊緣伺服器或 Edge 伺服器池，您可以略過此步驟。 您可以在定義新的 Edge 伺服器或 Edge 伺服器池時，將 Survivable 分支裝置或 Survivable 分支伺服器與 Edge 伺服器或 Edge 伺服器池建立關聯，或當您編輯 Survivable 分支裝置或 Survivable 的屬性頁面時，在拓撲中定義 Edge 伺服器或 Edge 伺服器池之後的分支伺服器。
