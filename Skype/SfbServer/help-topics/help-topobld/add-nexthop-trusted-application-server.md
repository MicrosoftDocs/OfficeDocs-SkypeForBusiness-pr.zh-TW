# <a name="add-nexthop-trusted-application-server"></a>新增 NextHop 信任的應用程式伺服器
 
若要設定受信任的應用程式伺服器或池的下一個躍點，請執行下列動作：
  
在 [**下一個躍點池]** 下拉式清單中，選取您希望受信任的應用程式池作為下一個躍點的企業版池或標準版前端伺服器。
  
按一下 **[完成]** 以完成受信任的應用程式池建立。
