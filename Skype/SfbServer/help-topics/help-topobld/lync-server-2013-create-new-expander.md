# <a name="lync-server-2013-create-new-expander"></a>Lync Server 2013 建立新展開工具
 
您可以在 [拓撲建立器] 的 Lync Server 2013 節點中建立新的伺服器角色，方法是以滑鼠右鍵按一下**Lync server 2013**，選取 [**新增**]，然後選取要建立的伺服器角色。
