# <a name="add-archiving-server"></a>新增封存伺服器
 
若要在拓撲設計中部署封存伺服器，您必須為要部署封存的伺服器指定完整網域名稱 (FQDN)。 在發佈包含封存及安裝商務用 Skype Server 的拓撲之前，您應該已完成部署封存的所有必要條件。
