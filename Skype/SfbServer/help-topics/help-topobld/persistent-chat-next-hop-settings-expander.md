# <a name="persistent-chat-next-hop-settings-expander"></a>常設聊天室下一個躍點設定展開工具
 
您可以從清單中選取 [前端] 伺服器或 [前端] 池來設定持續聊天的 **[下一個躍點] 池**。
  
 **確定**：接受並認可對話方塊的變更。
  
 **取消**：捨棄變更，並關閉對話方塊。
  
 **說明**：顯示此說明畫面。
