# <a name="archiving-server-scope-node"></a>封存伺服器範圍節點
 
在拓撲建立器中，您可以使用滑鼠右鍵按一下主控台樹中的 [池]，然後選取 [**編輯屬性**]，來執行與執行封存作業伺服器相關的工作。 透過 [編輯內容]**** 頁面，您可以變更執行封存之伺服器、封存 SQL Server 資料庫以及封存檔案存放區的完整網域名稱 (FQDN)。
