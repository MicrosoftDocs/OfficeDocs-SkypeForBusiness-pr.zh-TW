# <a name="archiving-server-scope-node"></a>封存伺服器範圍節點
 
在 [拓撲產生器] 中，您可以執行與執行封存之伺服器相關的工作，方法是在主控台樹中以滑鼠右鍵按一下集區，然後選取 [ **編輯屬性**]。 在 [**編輯** 內容] 頁面中，您可以變更執行封存之伺服器的完整功能變數名稱 (FQDN) 、封存 SQL Server 資料庫及封存檔案存放區。
