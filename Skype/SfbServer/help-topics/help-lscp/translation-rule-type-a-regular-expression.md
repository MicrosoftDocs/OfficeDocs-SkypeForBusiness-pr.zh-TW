# <a name="translation-rule-type-a-regular-expression"></a>轉譯規則：輸入規則運算式
 
在 [**符合這個模式**] 欄位中，指定將用來符合要翻譯之數位的模式。 在 [**翻譯規則**] 欄位中，指定已翻譯數位格式的模式。 例如，如果您在 [**翻譯規則**]{9}欄位的 [**符合此模式**欄位與 011 $ 1] 中輸入 ^\+（\d \d +） $，規則會將 + 441235551010 轉換為011441235551010。
  
如需使用商務用 Skype Server [控制台] 所能執行的不同程式的詳細資訊，請參閱[管理商務用 Skype server 2015](../../manage/manage.md)。
