# <a name="select-a-region-link"></a>選取區域連結
 
使用 [地區連結]**** 對話方塊，尋找您環境中可用的地區連結。若要使用現有的地區連結，請按一下清單中的地區連結，再按一下 [確定]****。
  
> [!NOTE]
> 若要擷取所有可用的地區連結，請不要在搜尋方塊中輸入任何內容，然後按一下搜尋圖示。 
  
如需使用商務用 Skype Server [控制台] 所能執行的不同程式的詳細資訊, 請參閱[管理商務用 Skype server 2015](../../manage/manage.md)。
