# <a name="administrative-roles-and-lync-server-control-panel"></a>管理角色和 Lync Server 控制台

為了讓您委派管理工作，同時維持高安全性的安全性，商務用 Skype Server 支援以角色為基礎的存取控制 (RBAC) 。 使用 RBAC，您可藉由將預先定義的系統管理角色指派給使用者，而授與其系統管理權限。 商務用 Skype Server 包括一組豐富的內建管理角色，也可讓您建立新角色，並為每個新角色指定自訂的 Cmdlet 清單。 您也可以將 Cmdlet 的指令碼新增至預先定義與自訂 RBAC 角色之允許的工作中。 如需詳細資訊，包括商務用 Skype Server 中可用的預先定義的管理角色，請參閱[Role-Based 存取控制](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-role-based-access-control)。
