# <a name="move-all-users"></a>移動所有使用者

在 [**移動使用者**] 對話方塊中，您可以選取要在部署中將使用者帳戶從一個池中移到另一個池的來源和目標池。

如需移動使用者的詳細資訊，請參閱〈[Move Users to Another Pool](https://technet.microsoft.com/library/e7b4968c-0e9d-4d56-b5f1-9edf0f7206f8.aspx)〉。
