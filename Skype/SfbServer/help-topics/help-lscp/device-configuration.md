# <a name="device-configuration"></a>裝置設定
 
裝置設定是用於管理商務用 Skype Phone Edition 裝置（全域或網站內）的設定集合。 這些設定可讓您設定諸如所需的安全性模式、裝置記錄等級、語音服務品質 (QoS) 設定，以及電話是否應該在指定的非使用期限後自動鎖定等事項。 
  
## <a name="tasks-you-can-perform"></a>您可以執行的工作

您可以在「 **裝置** 設定」頁面上執行下列工作：
  
- 全域或針對特定網站新增裝置設定。
    
- 修改現有裝置設定的選項。
    
## <a name="ui-reference"></a>UI 參考

下列清單說明頁面上的功能表、命令、欄位及內容。
  
- **新** 您可以使用下列範圍新增新的裝置設定：
    
  - 全域
    
  - 網站
    
- **編輯** 您可以在清單中變更裝置設定的選項。 使用此選項，您可以執行下列作業：
    
  - **顯示詳細資料** 此選項會開啟一個對話方塊，您可以在其中變更裝置設定的選項。
    
  - **全選** 此選項會選取清單中的所有裝置設定。
    
  - **Delete** 此選項會刪除所有選取的裝置設定。
    
- **Refresh** 您可以重新整理裝置設定清單，以確認所有裝置設定選項的狀態。
