# <a name="select-routes"></a>選取路由
 
使用 [**選取路由**] 對話方塊來尋找您可以使用的現有語音路由。
  
> [!NOTE]
> 若要檢索所有可用的語音路由，請不要在搜尋方塊中輸入任何內容，然後按一下 [搜尋] 圖示。 
  
如需使用商務用 Skype Server [控制台] 所能執行的不同程式的詳細資訊，請參閱[管理商務用 Skype server 2015](../../manage/manage.md)。
