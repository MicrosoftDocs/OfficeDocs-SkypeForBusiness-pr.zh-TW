# <a name="edit-client-mobility-policy"></a>編輯用戶端行動性原則

您可在 [編輯行動性原則] 對話上編輯行動性原則的詳細資料。

## <a name="ui-reference"></a>UI 參考

下列清單說明頁面上的功能表、命令、欄位及內容。



- **認可** 按一下以儲存對原則設定所做的變更。

- **取消** 按一下以停止編輯原則，並捨棄對原則設定所做的任何變更。

- **名稱** 行動性原則的名稱。

- **描述** 行動性原則的描述。

如需行動性原則的詳細資訊，請參閱＜[Mobility Policies](/previous-versions/office/lync-server-2013/lync-server-2013-mobility-policies)＞。
