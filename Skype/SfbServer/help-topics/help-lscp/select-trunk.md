# <a name="select-trunk"></a>選取主幹
 
使用 [**選取主幹**] 對話方塊尋找現有的主幹, 您可以使用。
  
> [!NOTE]
> 若要檢索所有可用的 trunks, 請不要在搜尋方塊中輸入任何內容, 然後按一下 [搜尋] 圖示。 
  
如需使用商務用 Skype Server [控制台] 所能執行的不同程式的詳細資訊, 請參閱[管理商務用 Skype server 2015](../../manage/manage.md)。
