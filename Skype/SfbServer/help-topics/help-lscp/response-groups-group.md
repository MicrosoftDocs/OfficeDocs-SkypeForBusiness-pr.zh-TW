# <a name="response-groups-group"></a>回應群組：群組

回應群組由代理群組、佇列和工作流程所組成。 [代理程式群組] 是指派給回應群組通話的一組人員。

[**回應群組** - ]**群組**頁面會顯示針對您的組織定義的所有代理群組清單。

## <a name="tasks-you-can-perform"></a>您可以執行的工作

您可以從 [**回應群組** - ]**群組**頁面執行下列工作：

- 建立新的代理群組

- 變更現有的代理群組

- 刪除代理人群組

## <a name="ui-reference"></a>UI 參考

下列清單說明頁面上的命令。

- **新增**啟動新的 [代理] 群組。

- [**編輯**]開啟選取的代理群組以進行編輯、選取清單中的所有代理群組，或刪除選取的 agent 群組。

- **更新**刷新代理群組清單。

下列清單說明頁面上的欄位。

- **名稱**識別代理程式群組的唯一名稱。

- **服務**託管 [代理] 群組的**ApplicationServer**服務。

- **描述**[代理] 群組的描述。

如需回應群組功能與功能的詳細資訊，請參閱規劃檔中的[商務用 Skype Server 2015 中的回應群組應用程式規劃](../../plan-your-deployment/enterprise-voice-solution/response-group.md)。 如需管理代理人群組的詳細資訊，請參閱作業文件中的〈[Managing Agent Groups](https://technet.microsoft.com/library/36084cdc-38f1-4c45-922f-f81c7e86210c.aspx)〉。
