# <a name="select-an-announcement-service"></a>選取宣告服務
 
使用 [選取宣告服務] 對話方塊選取可將來電處理至未指派號碼範圍的宣告服務。
  
在搜尋方塊中，輸入想要使用之宣告服務的完整或部分名稱。若要傳回所有可用的宣告服務，請將搜尋方塊留白，然後按一下搜尋圖示。
  
在搜尋傳回的宣告服務清單中，選取您要用於將來電處理至未指派號碼範圍的服務。
