# <a name="select-exchange-um-auto-attendant"></a>選取 Exchange UM 自動語音應答
 
使用 [**選取 EXCHANGE Um 自動**語音應答] 對話方塊選取 Exchange um 自動語音應答，該應答會處理來電至未指定號碼的範圍。
  
在 [搜尋] 方塊中，輸入您要使用之 Exchange UM 自動語音應答的全部或部分名稱。 若要傳回所有可用的 Exchange UM 自動語音應答，請將 [搜尋] 方塊保留空白，然後按一下 [搜尋] 圖示。
  
在 [搜尋] 傳回的 Exchange UM 自動語音應答清單中，選取您要用來處理來電與未指派號碼範圍的自動語音應答。
