# <a name="select-exchange-um-auto-attendant"></a>選取 Exchange UM 自動語音應答
 
使用 [ **選取 EXCHANGE Um 自動** 語音應答] 對話方塊，選取 Exchange um 自動語音應答，該語音應答會將來電處理至未指派號碼的範圍。
  
在搜尋方塊中，輸入您要使用的 Exchange UM 自動語音應答的全部或部分名稱。 若要傳回所有可用的 Exchange UM 自動語音應答，請將搜尋方塊保留空白，然後按一下 [搜尋] 圖示。
  
在搜尋傳回的 Exchange UM 自動語音應答清單中，選取您要用於處理未指派號碼範圍內之來電的自動語音應答。
