# <a name="hybrid-wizard-move-user"></a>混合式精靈：移動使用者

使用 [移動使用者] 嚮導移動位於內部部署集區的使用者帳戶，使其位於商務用 Skype Online。 如需詳細資訊，請參閱 [使用混合式管理性嚮導](https://technet.microsoft.com/library/d777f79b-a740-4aba-a9e2-c91f0315b6f8.aspx)。
