# <a name="response-groups-workflow"></a>回應群組工作流程

回應群組中包含代理群組、佇列及工作流程。 回應群組工作流程定義回應群組應用程式接聽電話時所採取的動作。

[**回應群組**  -  **工作流程**] 頁面會顯示為組織定義的所有回應群組工作流程清單。

## <a name="tasks-you-can-perform"></a>您可以執行的工作

您可以從「**回應群組**  -  **工作流程**」頁面執行下列工作：

- 建立或變更群組搜尋工作流程

- 建立或變更互動式工作流程

## <a name="ui-reference"></a>UI 參考

下列清單說明頁面上的命令。

- **建立或編輯工作流程** 開啟 [回應群組] 設定工具，以建立或編輯工作流程。

- **Refresh** 重新整理工作流程清單。

下列清單說明頁面上的欄位。

- **名稱** 指派給工作流程的唯一名稱。

- **服務** 主控工作流程的 **ApplicationServer** 服務。

- **SIP 位址** 將接聽工作流程通話之群組的 SIP 位址。

- **電話** 呼叫以送達此回應群組的電話號碼。

- **語言** 用於語音辨識及文字轉換語音的語言。

- **IVR** 會指出工作流程是否為群組搜尋或互動式工作流程。

- **已啟用** 會指出是否要啟用工作流程以接收來電。

如需有關回應群組功能及功能的詳細資訊，請參閱規劃檔中的在[商務用 Skype Server 2015 中規劃回應群組應用程式](../../plan-your-deployment/enterprise-voice-solution/response-group.md)。 如需使用回應群組工作流程的詳細資訊，請參閱 Operations 檔中的 [管理回應群組工作流程](/previous-versions/office/lync-server-2013/lync-server-2013-managing-response-group-workflows) 。
