# <a name="set-pin"></a>設定 PIN

若要產生使用者的 PIN, 請選取 [**自動產生有效的 pin**]。

若要自行建立 PIN，請選取 [手動輸入特定 PIN]****，然後在文字方塊中輸入符合您 PIN 原則設定中所指定的 PIN 要求之 PIN。

如需使用商務用 Skype Server [控制台] 所能執行的不同程式的詳細資訊, 請參閱[管理商務用 Skype server 2015](../../manage/manage.md)。
