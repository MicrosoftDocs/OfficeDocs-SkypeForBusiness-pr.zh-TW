# <a name="users-view-persistent-chat"></a>使用者：檢視常設聊天室

**CsPersistentChatAdministrator**角色的成員（持續聊天系統管理員）可將聊天室管理員權利和許可權授與其他使用者及他們自己。

持續聊天管理員可以執行下列動作：

- 建立會議室類別和聊天室。

- 設定所有類別和聊天室的成員資格。

- 管理所有類別和聊天室的所有設定。

- 啟用或停用持久聊天伺服器的原則。

- 在持久性聊天伺服器池中設定及管理設定。

如需詳細資訊，請參閱在部署檔中[新增持續性聊天系統管理員](https://technet.microsoft.com/library/c107eb20-4e58-4463-b4f9-63fb5b1d9534.aspx)。
