# <a name="users-view-persistent-chat"></a>使用者：檢視常設聊天室

**CsPersistentChatAdministrator** 角色的成員 (Persistent Chat Administrators) 可將聊天室管理員權力和許可權授與其他使用者及自身。

Persistent Chat 系統管理員可以執行下列作業：

- 建立聊天室類別和聊天室。

- 設定所有類別和聊天室的成員資格。

- 管理所有類別和聊天室的所有設定。

- 啟用或停用持久聊天伺服器的原則。

- 設定和管理 Persistent Chat Server 集區上的設定。

如需詳細資訊，請參閱部署文件中的[Adding a Persistent Chat Administrator](https://technet.microsoft.com/library/c107eb20-4e58-4463-b4f9-63fb5b1d9534.aspx)。

## <a name="tasks-that-you-can-perform"></a>您可以執行的工作

您可以在 [ **使用者： View Persistent Chat** ] 頁面上執行下列工作：

- [讓使用者管理類別、聊天室及使用者權限](https://technet.microsoft.com/library/6c551be3-bc74-4d0e-9008-ddfabd86e940.aspx)

- [瞭解 Persistent Chat 成員資格](https://technet.microsoft.com/library/900392d6-6e9f-4dae-93d6-39d7474409ef.aspx)

如需您可以使用商務用 Skype Server 控制台執行的不同程式的詳細資訊，請參閱 [管理商務用 Skype server 2015](../../manage/manage.md)。
