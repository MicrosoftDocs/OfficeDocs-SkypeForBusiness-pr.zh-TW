行動功能與 Lync Server 2013 中的相同。 如需準備行動環境的詳細資料, 請參閱[在 Lync Server 2013 中規劃行動](https://technet.microsoft.com/en-us/library/hh689981%28v=ocs.15%29.aspx)。
