# <a name="skype-for-business-server-mobility-planning-tool"></a>商務用 Skype Server：行動計畫工具
 
商務用 Skype Server 2015 中的行動功能可支援行動裝置上商務用 Skype 功能。
  
行動功能與 Lync Server 2013 中的功能相同。 如需準備行動環境的詳細資訊，請參閱 [在 Lync Server 2013 中規劃行動](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-mobility)。
