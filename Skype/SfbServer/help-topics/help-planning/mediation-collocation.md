# <a name="mediation-collocation-planning-tool"></a>中繼組合 （規劃工具）
 
根據預設，中繼伺服器已組合與前端伺服器。 中繼伺服器也可以部署在獨立的集區，基於效能考量，或如果您部署 SIP 主幹，這種情況的獨立集區強烈建議。 Lync Server 2013 中，並未組合 skype for Business Server 函數，就像。 如需詳細資料，請參閱下列主題：
  
- [Lync Server 2013 中支援的伺服器共同配置](https://technet.microsoft.com/library/gg425885%28v=ocs.15%29.aspx)
    
- [在商務用 Skype Server 2015 中的中繼伺服器元件](../../plan-your-deployment/enterprise-voice-solution/mediation-server.md)
