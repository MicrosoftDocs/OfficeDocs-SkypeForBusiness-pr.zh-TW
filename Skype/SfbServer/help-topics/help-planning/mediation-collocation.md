# <a name="skype-for-business-server-mediation-collocation-planning-tool"></a>商務用 Skype Server中繼組合規劃工具
 
根據預設，轉送伺服器會與前端伺服器組合。 轉送伺服器也可以部署在獨立的集區中以取得效能原因，或者，如果您要部署 SIP 主幹，建議使用獨立集區。 商務用 Skype Server 功能中的組合，就像在 Lync Server 2013 中一樣。 如需詳細資料，請參閱下列主題：
  
- [Lync Server 2013 中支援的伺服器組合](/previous-versions/office/lync-server-2013/lync-server-2013-supported-server-collocation)
    
- [商務用 Skype Server 2015 的轉送伺服器元件](../../plan-your-deployment/enterprise-voice-solution/mediation-server.md)
