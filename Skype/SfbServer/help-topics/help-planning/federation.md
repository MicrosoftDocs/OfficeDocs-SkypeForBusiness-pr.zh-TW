# <a name="federation-planning-tool"></a>同盟 (規劃工具) 
 
您可以設定 Edge Server，以允許您的內部和外部使用者存取夥伴組織或服務上的連絡人。
  
 同盟的運作方式類似于 Microsoft Lync 2013。 如需針對同盟準備環境的詳細資訊，請參閱下列主題：
  
- [規劃 Lync Server 2013 和 Office 通訊伺服器同盟](https://technet.microsoft.com/library/jj205335%28v=ocs.15%29.aspx)
    
- [在 Lync Server 2013 中規劃公用立即訊息連線](https://technet.microsoft.com/library/jj205349%28v=ocs.15%29.aspx)
    
- [在 Lync Server 2013 中規劃可擴展郵件和顯示狀態通訊協定 (XMPP) 同盟](https://technet.microsoft.com/library/jj205107%28v=ocs.15%29.aspx)
