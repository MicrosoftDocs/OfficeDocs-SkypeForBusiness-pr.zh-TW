# <a name="web-conferencing-planning-tool"></a>網路會議（規劃工具）
 
商務用 Skype Server 2015 網路會議包含檔共同作業、應用程式共用和桌面共用。
  
> [!NOTE]
> 商務用 Skype Server 2015 使用 Office Web Apps 和 Office Web Apps 伺服器來處理 PowerPoint 簡報的共用和轉譯。 網路會議功能的運作方式與 Microsoft Lync 2013 中的功能完全相同。 如需安裝及設定 Office Web Apps 伺服器的詳細資料，請參閱設定[與 Office Web Apps server 和 Lync Server 2013 的整合](https://technet.microsoft.com/en-us/library/jj204792%28v=ocs.15%29.aspx)。
