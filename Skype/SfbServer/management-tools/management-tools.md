# <a name="skype-for-business-server-2015-management-tools"></a>Skype for Business Server 2015 管理工具
 
**摘要：** 深入了解服務管理工具中用 Skype Server 2015。
  
商務用 Skype Server 2015 通訊軟體 （先前稱為 Lync 伺服器），提供立即訊息 (IM)、 目前狀態、 會議及電話語音解決方案，支援企業層級的共同作業需求。 工具來管理這些服務非常有彈性，強大。
