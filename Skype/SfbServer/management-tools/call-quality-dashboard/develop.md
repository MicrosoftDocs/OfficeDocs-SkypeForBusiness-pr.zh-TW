# <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>開發商務用 Skype Server 的通話品質儀表板
 
**總結：** 瞭解如何開發通話品質儀表板的解決方案。 通話品質儀表板是商務用 Skype Server 的工具。
  
本主題供應商務用 Skype Server 之通話品質儀表板 (CQD) 的開發檔。
  
## <a name="develop-call-quality-dashboard-for-skype-for-business-server"></a>開發商務用 Skype Server 的通話品質儀表板

使用自訂開發，CQD 可以擴充並整合到報表環境。 
  
有兩組 web APIs 可用，一個用於存取 cube 中的資料 (透過資料 APIs) ，另一個用來存取透過存放庫 API (的報表中繼資料) 。 
  
- [通話品質儀表板的資料 API (CQD) 商務用 Skype Server](data-api.md)
    
- [用於通話品質儀表板的存放庫 API (CQD) 商務用 Skype Server](repository-api.md)
    
- [CQD 開發範例](cqd-development-samples.md)
