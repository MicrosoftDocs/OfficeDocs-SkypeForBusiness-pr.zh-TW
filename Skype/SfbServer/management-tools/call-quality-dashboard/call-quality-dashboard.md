# <a name="call-quality-dashboard-for-skype-for-business-server"></a>商務用 Skype Server 的通話品質儀表板
 
**摘要：** 深入瞭解通話品質儀表板，它是商務用 Skype Server 的報表工具。
 
商務用 Skype Server 呼叫品質儀表板 (CQD) 是在商務用 Skype Server 和 Lync Server 2013 中，位於監控伺服器上的經驗品質資料庫之上的報表層。 CQD 會使用 Microsoft SQL Server Analysis Services，提供匯總使用狀況及通話品質資訊，以及篩選和旋轉資料集。
  
本節包含下列主題，在部署通話品質儀表板之前應先加以複查。
  
- [規劃商務用 Skype Server 的通話品質儀表板](plan.md)
- [為商務用 Skype Server 部署通話品質儀表板](deploy-0.md)
- [使用商務用 Skype Server 的通話品質儀表板](use.md)
- [開發商務用 Skype Server 的通話品質儀表板](develop.md)
