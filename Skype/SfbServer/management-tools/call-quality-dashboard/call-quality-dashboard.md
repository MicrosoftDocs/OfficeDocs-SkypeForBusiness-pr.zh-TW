# <a name="call-quality-dashboard-for-skype-for-business-server"></a>商務用 Skype Server 的通話品質儀表板
 
**摘要：** 瞭解 [通話品質儀表板]，它是商務用 Skype Server 的報告工具。
 
商務用 Skype Server 通話品質儀表板（CQD）是 [商務用 Skype 伺服器] 和 [Lync Server 2013] 的 [監視伺服器] 的 [體驗品質] 資料庫上方的報告層。 CQD 使用 Microsoft SQL Server Analysis Services 來提供匯總使用量及呼叫品質資訊，以及篩選與旋轉資料集。
  
本區段中包含下列主題，請在部署通話品質儀表板之前先進行審查。

- [部署商務用 Skype Server 的通話品質儀表板](deploy-0.md)
- [使用商務用 Skype Server 的通話品質儀表板](use.md)
- [開發商務用 Skype Server 的通話品質儀表板](develop.md)
