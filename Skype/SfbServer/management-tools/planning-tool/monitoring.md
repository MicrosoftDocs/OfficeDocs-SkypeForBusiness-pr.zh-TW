# <a name="monitoring-planning-tool"></a>監控規劃工具

在商務用 Skype Server 2015 中，監控是用來收集使用狀況資訊和經驗品質 (QoE 有關您的使用者所參與之通訊會話的) 資料。

如需規劃監控伺服器的詳細資訊，請參閱 [規劃監控](/previous-versions/office/lync-server-2013/lync-server-2013-planning-for-monitoring)。
