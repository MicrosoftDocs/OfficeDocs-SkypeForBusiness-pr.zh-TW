# <a name="mobility-planning-tool"></a>行動性（規劃工具）
 
商務用 Skype Server 2015 中的行動功能支援行動裝置上的商務用 Skype 功能。
  
行動功能與 Lync Server 2013 中的相同。 如需準備行動環境的詳細資訊，請參閱[規劃商務用 Skype Server 的行動方案](../../plan-your-deployment/mobility.md)。
