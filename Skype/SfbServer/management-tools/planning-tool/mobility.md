# <a name="mobility-planning-tool"></a>行動計畫工具
 
商務用 Skype Server 2015 中的行動功能可支援行動裝置上商務用 Skype 功能。
  
行動功能與 Lync Server 2013 中的功能相同。 如需準備行動環境的詳細資訊，請參閱[規劃行動的商務用 Skype Server](../../plan-your-deployment/mobility.md)。
