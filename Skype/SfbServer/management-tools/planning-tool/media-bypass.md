# <a name="media-bypass-planning-tool"></a>媒體旁路（規劃工具）
 
「媒體旁路」是指當信號流經中繼伺服器時，從媒體路徑中移除轉送伺服器。 如需詳細資訊，請參閱[在商務用 Skype 2015 中規劃媒體旁路](../../plan-your-deployment/enterprise-voice-solution/media-bypass.md)。
