# <a name="voice-infrastructure-planning-tool"></a>語音結構（規劃工具）
 
商務用 Skype Server 2015 透過使用 SIP trunks、直接 SIP 連線至 PSTN 閘道，或直接將 SIP 連線至 PBX，提供可靠、可伸縮的 PSTN 連線。 如需詳細資訊，請參閱[在商務用 Skype Server 2015 中規劃 PSTN](../../plan-your-deployment/enterprise-voice-solution/pstn-connectivity-0.md)連線。
