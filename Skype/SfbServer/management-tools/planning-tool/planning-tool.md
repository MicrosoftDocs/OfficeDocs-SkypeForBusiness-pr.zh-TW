# <a name="skype-for-business-server-2015-planning-tool"></a>Skype for Business Server 2015 規劃工具
 
使用 Skype for Business Server 2015 規劃工具的指南。
  
Skype for Business Server 2015 規劃工具是驅動，精靈會詢問有關問題用 Skype Server 2015 拓撲您所設計的採訪類似的工具。 提供資訊，請規劃工具使用搭配慣用的拓撲設計和容量，以呈現提供的答案為基礎的建議的拓撲的作法。 您可以從[Skype for Business Server 2015 規劃工具](https://go.microsoft.com/fwlink/p/?LinkID=282725)下載規劃工具。
  
最後，規劃工具的目標是為了簡化潛在的設計完成用 Skype Server 2015 拓撲複雜性。 假設沒有可用連線到 Microsoft 網站的網際網路連線工具也可讓內工具]，規劃及部署 > 文件的內容相關式參考。
  
自訂之後的拓撲的基礎結構的 TCP/IP 位址與完整的網域名稱 (Fqdn)，規劃工具會提供一系列的涵蓋網域名稱系統 (DNS) 命名、 防火牆規則、 憑證、 等等的報告。 
  
使用此工具是第一個步驟中規劃實作。 下一步就是到[Skype for Business Server 2015 Capacity Calculator](https://www.microsoft.com/download/details.aspx?id=51196)輸入您網站的資訊細節，請視需要調整，然後使用[Skype for Business Server 2015 壓力及效能工具](https://www.microsoft.com/download/details.aspx?id=50367)來模擬，並確認實作會提供您的需求。
  
規劃工具也提供匯出資訊兩種格式的能力：
  
- Microsoft Excel （.xml 試算表）
    
- Microsoft Visio (.vdx)
