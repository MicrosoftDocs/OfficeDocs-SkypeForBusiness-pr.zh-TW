# <a name="enterprise-voice-planning-tool"></a>企業語音規劃工具
 
透過企業語音，商務用 Skype Server 2015 可提供獨立的語音 over 網際網路通訊協定 (VoIP) 提供以加強或取代傳統專用交換機 (PBX) 系統。 企業語音使用者可以撥打組織 VoIP 網路或 PBX 上的同事，也可以撥打組織外部的傳統電話號碼。 企業語音方案包含常見呼叫功能（例如，答案、轉寄、轉移、保留、轉移、發行和寄存）及增強型 9-1-1 (E9-1-1) 呼叫 (E9-1-1 只適用于美國。 ) 企業語音也支援廣泛的目前和較舊的 IP 和 USB 裝置。
  
如需規劃企業語音的詳細資訊，請參閱[商務用 Skype Server 2015 中的企業語音方案](../../plan-your-deployment/enterprise-voice-solution/enterprise-voice.md)。 如需部署獨立轉送伺服器的詳細資訊，請參閱[在商務用 Skype Server 2015 的拓撲產生器中部署轉送伺服器](../../deploy/deploy-enterprise-voice/deploy-a-mediation-server.md)。
