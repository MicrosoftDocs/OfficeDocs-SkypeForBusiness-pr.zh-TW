# <a name="voice-traffic-planning-tool"></a>語音流量 (規劃工具) 
 
規劃工具使用使用者流量的相關資訊來判斷要在部署中使用的埠數目。 這個數目又會決定所需的轉送伺服器和閘道數目。 如需詳細資訊，請參閱 [評估語音使用方式和流量（適用于商務用 Skype Server 2015](../../plan-your-deployment/capacity/estimating-voice-traffic.md)）。
