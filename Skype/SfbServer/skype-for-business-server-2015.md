# <a name="skype-for-business-server-2015"></a>商務用 Skype Server 2015

商務用 Skype 是一種通訊與共同作業平臺，透過 Skype 使用企業級的安全性、合規性及控制能力，將用戶端體驗融入在一起。 商務用 Skype 提供功能，包括目前狀態、IM、語音和視頻通話，以及線上會議。
