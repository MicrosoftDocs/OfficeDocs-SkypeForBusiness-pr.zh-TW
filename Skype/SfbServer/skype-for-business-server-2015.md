# <a name="skype-for-business-server-2015"></a>商務用 Skype Server 2015

商務用 Skype 是聚集所有同等級激發由 Skype 用戶端體驗與企業級安全性、 規範和控制項的通訊和共同作業平台。 商務用 Skype 提供包括目前狀態、 IM、 語音及視訊通話及線上會議的功能。
