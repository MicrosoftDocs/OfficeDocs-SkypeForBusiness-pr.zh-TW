# <a name="hardwareversions-table-in-skype-for-business-server-2015"></a>商務用 Skype Server 2015 中的 HardwareVersions 表格
 
HardwareVersions 資料表是支援資料表。 每個記錄儲存一個裝置硬體版本的相關資訊。
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**VersionId** <br/> |int  <br/> |首選  <br/> |識別這個硬體版本的唯一號碼。  <br/> |
|**版本** <br/> |Nvarchar （256）  <br/> | <br/> |硬體版本。  <br/> |
