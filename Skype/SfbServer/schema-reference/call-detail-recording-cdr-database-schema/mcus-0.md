# <a name="mcus-view"></a>Mcus 視圖
 
[Mcus] 視圖儲存已參與會議會話之 MCUs 的相關資訊。 此視圖已在 Microsoft Lync Server 2013 中推出。
  
|**左欄**|**資料類型**|**詳細資料**|
|:-----|:-----|:-----|
|**McuId** <br/> |int  <br/> |識別 MCU 的唯一號碼。  <br/> |
|**McuUri** <br/> |Nvarchar （450）  <br/> |MCU 的 URI。  <br/> |
|**McuUriType** <br/> |Nvarchar （256）  <br/> |MCU URI 的類型。 如需詳細資訊，請參閱[UriTypes 資料表](uritypes.md)。 <br/> |
