# <a name="mcus-view"></a>Mcus view
 
Mcus view 會儲存參與會議會話之 MCUs 的相關資訊。 此視圖已引進于 Microsoft Lync Server 2013。
  
|**欄**|**資料類型**|**詳細資料**|
|:-----|:-----|:-----|
|**McuId** <br/> |int  <br/> |用於識別 MCU 的唯一號碼。  <br/> |
|**McuUri** <br/> |Nvarchar (450)   <br/> |MCU 的 URI。  <br/> |
|**McuUriType** <br/> |Nvarchar (256)   <br/> |MCU URI 的類型。 如需詳細資訊，請參閱 [UriTypes 表格](uritypes.md) 。 <br/> |
