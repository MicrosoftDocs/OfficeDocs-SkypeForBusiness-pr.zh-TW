# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>商務用 Skype Server 2015 中的 [製造商] 表格
 
[製造商] 資料表是支援資料表。 每筆記錄儲存一個裝置（辦公桌電話）製造商的相關資訊。
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**[Manufacturerid** <br/> |int  <br/> |首選  <br/> |識別此製造商的唯一號碼。  <br/> |
|**製造商** <br/> |Nvarchar （256）  <br/> | <br/> |[製造商名稱]。  <br/> |
