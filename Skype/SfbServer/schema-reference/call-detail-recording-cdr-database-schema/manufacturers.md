# <a name="manufacturers-table-in-skype-for-business-server-2015"></a>商務用 Skype Server 2015 中的製造商表格
 
製造商資料表是支援的表格。 每個記錄會儲存一個裝置 (phone) 製造商的相關資訊。
  
|**欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**ManufacturerId** <br/> |int  <br/> |主要  <br/> |用於識別此製造商的唯一號碼。  <br/> |
|**製造商** <br/> |Nvarchar (256)   <br/> | <br/> |製造商名稱。  <br/> |
