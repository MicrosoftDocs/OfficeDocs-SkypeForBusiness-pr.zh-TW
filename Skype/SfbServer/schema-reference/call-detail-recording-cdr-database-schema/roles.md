# <a name="roles-table"></a>Roles 表格
 
Roles 表格是一個靜態表格，用來儲存可能的會議角色（例如出席者和簡報者）的清單。
  
|**欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**RoleId** <br/> |Tinyint  <br/> |主要  <br/> ||
|**Role** <br/> |Nvarchar (256)   <br/> || 允許的值： <br/>  0 - 未知 <br/>  1-簡報者 <br/>  2-出席者 <br/> |
