# <a name="roles-table"></a>Roles 表格
 
Roles 資料表是一個靜態表格，可儲存出席者與簡報者等可能的會議角色清單。
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**RoleId** <br/> |Tinyint  <br/> |首選  <br/> ||
|**角色** <br/> |Nvarchar （256）  <br/> || 允許的值： <br/>  0-未知 <br/>  1-簡報者 <br/>  2-出席者 <br/> |
