# <a name="edgeservers-table-in-skype-for-business-server-2015"></a>商務用 Skype Server 2015 中的 EdgeServers 表格
 
EdgeServers 表格是支援的表格。 每個記錄會儲存與具有資料庫中記錄的通話相關的一部 Edge Server 相關資訊。
  
|**欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**EdgeServerId** <br/> |int  <br/> |主要  <br/> |用於識別此 Edge Server 的唯一號碼。  <br/> |
|**Edgeserver atl-edge** <br/> |Nvarchar (256)   <br/> | <br/> |Edge Server 名稱。  <br/> |
