# <a name="edgeservers-table-in-skype-for-business-server-2015"></a>商務用 Skype Server 2015 中的 EdgeServers 表格
 
EdgeServers 資料表是支援資料表。 每個記錄都儲存在資料庫中有記錄的通話中所涉及之一台邊緣伺服器的相關資訊。
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**EdgeServerId** <br/> |int  <br/> |首選  <br/> |標識此 Edge 伺服器的唯一號碼。  <br/> |
|**EdgeServer** <br/> |Nvarchar (256)  <br/> | <br/> |Edge 伺服器名稱。  <br/> |
