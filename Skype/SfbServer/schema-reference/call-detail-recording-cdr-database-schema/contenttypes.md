# <a name="contenttypes-table-in-skype-for-business-server-2015"></a>商務用 Skype Server 2015 中的主控表格
 
主控資料表是一種支援表格，可儲存點對點工作階段和會議會話中所使用的內容類型清單。 資料表中的每一筆記錄代表一種內容類型。
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**ContentTypeId** <br/> |int  <br/> |首選  <br/> |識別內容類型的唯一號碼。  <br/> |
|**ContentType** <br/> |Nvarchar （256）  <br/> ||內容類型名稱。  <br/> |
