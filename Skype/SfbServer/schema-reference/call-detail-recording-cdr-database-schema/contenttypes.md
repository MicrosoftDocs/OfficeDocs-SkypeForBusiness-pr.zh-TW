# <a name="contenttypes-table-in-skype-for-business-server-2015"></a>商務用 Skype Server 2015 中的 ContentTypes 表格
 
ContentTypes 表格是一種支援表格，可儲存點對點工作階段和會議會話中所使用的內容類型清單。 表格中的每一筆記錄代表一種內容類型。
  
|**欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**ContentTypeId** <br/> |int  <br/> |主要  <br/> |用於識別內容類型的唯一號碼。  <br/> |
|**ContentType** <br/> |Nvarchar (256)   <br/> ||內容類型名稱。  <br/> |
