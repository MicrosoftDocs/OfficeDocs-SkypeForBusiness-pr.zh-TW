# <a name="pools-table"></a>Pool 表格
 
Pool 表格是一種支援資料表，可儲存各種集區的資訊。 資料表中的每一筆記錄都代表一個集區。
  
|**欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**PoolId** <br/> |int  <br/> |主要  <br/> |用於識別此集區的唯一號碼。  <br/> |
|**PoolFQDN** <br/> |Nvarchar (256)   <br/> | <br/> |集區 FQDN。  <br/> |
