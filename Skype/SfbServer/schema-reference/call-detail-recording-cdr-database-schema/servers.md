# <a name="servers-table"></a>Servers 表格
 
Servers 表格是一種支援資料表，可儲存各種伺服器的相關資訊。 資料表中的每一筆記錄都代表一部伺服器。
  
|**欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |主要  <br/> |用於識別此伺服器的唯一號碼。  <br/> |
|**ServerFQDN** <br/> |Nvarchar (256)   <br/> | <br/> |伺服器 FQDN。  <br/> |
