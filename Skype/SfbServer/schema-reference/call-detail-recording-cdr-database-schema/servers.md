# <a name="servers-table"></a>Servers 表格
 
[Servers] 資料表是一個支援資料表，可儲存各種伺服器的相關資訊。 資料表中的每一筆記錄代表一台伺服器。
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**ServerId** <br/> |int  <br/> |首選  <br/> |標識此伺服器的唯一號碼。  <br/> |
|**ServerFQDN** <br/> |Nvarchar （256）  <br/> | <br/> |伺服器 FQDN。  <br/> |
