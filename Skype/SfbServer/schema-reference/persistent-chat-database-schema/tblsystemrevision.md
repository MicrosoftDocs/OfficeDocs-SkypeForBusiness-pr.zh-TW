# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision 包含的修訂編號會與 tblAdminLock 表搭配使用，以達到多個系統管理員用戶端之間的一致性。
  
**Columns**

|**欄**|**類型**|**描述**|
|:-----|:-----|:-----|
|sysRevision  <br/> |bigint，非 null  <br/> |修訂版編號。  <br/> |
