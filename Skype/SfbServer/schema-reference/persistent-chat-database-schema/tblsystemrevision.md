# <a name="tblsystemrevision"></a>tblSystemRevision
 
tblSystemRevision 包含與 tblAdminLock 資料表搭配使用的修訂編號, 以達到多個系統管理員用戶端之間的一致性。
  
**分欄**

|**左欄**|**類型**|**說明**|
|:-----|:-----|:-----|
|sysRevision  <br/> |Bigint, not null  <br/> |修訂編號。  <br/> |
