# <a name="region-table"></a>地區表格
 
Region 資料表是支援表格。 每筆記錄代表 [網路設定] 中定義的一個國家/地區。
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**RegionKey** <br/> |int  <br/> |首選  <br/> |標識國家/地區的唯一號碼。  <br/> |
|**RegionName** <br/> |Nvarchar  <br/> |唯一  <br/> |國家/地區的名稱。  <br/> |
