# <a name="region-table"></a>地區表格
 
地區表格是一種支援資料表，其中的每一項記錄都代表網路組態設定中定義的一個國家/地區。
  
|**欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**RegionKey** <br/> |int  <br/> |主要  <br/> |用於識別國家/地區的唯一號碼。  <br/> |
|**RegionName** <br/> |Nvarchar (128)   <br/> |Unique  <br/> |國家/地區的名稱。  <br/> |
