# <a name="codecdescription-table"></a>CodecDescription 資料表
 
CodecDescription 表格會將唯一的編解碼器識別碼對應至其相對應的編解碼器。 編解碼器是用來編碼傳輸和廣播的數位信號, 然後解碼這些信號以進行播放。 此表格是在 Microsoft Lync Server 2013 中推出
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**CodecDescriptionKey** <br/> |Smallint  <br/> |首選  <br/> |指派給編解碼器的唯一識別碼。  <br/> |
|**CodecDescription** <br/> |Varchar (256)  <br/> |唯一  <br/> |對應至 CodecDescriptionKey 的編解碼器的唯一描述。  <br/> |
