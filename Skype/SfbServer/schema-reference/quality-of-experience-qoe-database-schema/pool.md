# <a name="pool-table"></a>Pool 表格
 
[Pool] 資料表是一個支援資料表, 可儲存各種前端池的相關資訊。 資料表中的每一筆記錄代表一個池。
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**PoolKey** <br/> |int  <br/> |首選  <br/> |識別這個池子的唯一號碼。  <br/> |
|**PoolName** <br/> |Nvarchar (256)  <br/> |唯一  <br/> |池 FQDN。  <br/> |
