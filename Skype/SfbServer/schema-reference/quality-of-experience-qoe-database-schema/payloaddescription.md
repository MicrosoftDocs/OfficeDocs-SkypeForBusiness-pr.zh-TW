# <a name="payloaddescription-table"></a>PayloadDescription 表格
 
PayloadDescription 表格是支援的表格。 每筆記錄代表一個編解碼器，在音訊或視頻會話中使用。
  
|**欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**PayloadDescriptionKey** <br/> |int  <br/> |主要  <br/> |識別編解碼器的唯一號碼。  <br/> |
|**PayloadDescription** <br/> |Nvarchar (256)   <br/> |Unique  <br/> |編解碼器名稱。  <br/> |
