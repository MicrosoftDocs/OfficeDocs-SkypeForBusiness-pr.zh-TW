# <a name="payloaddescription-table"></a>PayloadDescription 表格
 
PayloadDescription 資料表是支援資料表。 每個記錄代表一個在音訊或視頻會話中使用的編解碼器。
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**PayloadDescriptionKey** <br/> |int  <br/> |首選  <br/> |識別編解碼器的唯一號碼。  <br/> |
|**PayloadDescription** <br/> |Nvarchar (256)  <br/> |唯一  <br/> |編解碼器名稱。  <br/> |
