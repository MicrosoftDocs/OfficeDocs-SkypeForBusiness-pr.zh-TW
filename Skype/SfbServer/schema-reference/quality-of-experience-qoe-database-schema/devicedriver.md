# <a name="devicedriver-table"></a>DeviceDriver 表格
 
DeviceDriver 資料表是支援資料表。 每個記錄都代表擷取裝置或轉譯裝置所使用的驅動程式。
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**DeviceDriverKey** <br/> |int  <br/> |首選  <br/> |識別此裝置驅動程式記錄的唯一號碼。  <br/> |
|**DeviceDriver** <br/> |Varchar (256)  <br/> |唯一  <br/> |裝置驅動程式名稱。  <br/> |
