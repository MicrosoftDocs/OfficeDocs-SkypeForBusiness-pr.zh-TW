# <a name="devicedriver-table"></a>DeviceDriver 表格
 
DeviceDriver 表格是支援的表格。 每筆記錄都代表一個捕獲裝置或轉譯裝置使用的驅動程式。
  
|**欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**DeviceDriverKey** <br/> |int  <br/> |主要  <br/> |用於識別此裝置磁碟機記錄的唯一號碼。  <br/> |
|**DeviceDriver** <br/> |Varchar (256)   <br/> |unique  <br/> |裝置驅動程式名稱。  <br/> |
