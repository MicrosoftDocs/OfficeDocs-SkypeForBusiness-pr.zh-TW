|**欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |主要  <br/> |用以識別 Mac 位址的唯一號碼。  <br/> |
|**MacAddress** <br/> |Varchar (256)   <br/> |Unique  <br/> |Mac 位址字串。  <br/> |
