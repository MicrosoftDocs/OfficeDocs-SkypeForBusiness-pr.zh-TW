# <a name="macaddress-table"></a>MacAddress 表格
 
MacAddress 資料表是支援資料表。 每個記錄代表一個來源。
  
|**左欄**|**資料類型**|**索引鍵/索引**|**詳細資料**|
|:-----|:-----|:-----|:-----|
|**MacAddressKey** <br/> |int  <br/> |首選  <br/> |識別 Mac 位址的唯一號碼。  <br/> |
|**MacAddress** <br/> |Varchar （256）  <br/> |唯一  <br/> |Mac 位址字串。  <br/> |
