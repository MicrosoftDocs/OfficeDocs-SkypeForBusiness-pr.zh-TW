# <a name="manage-skype-for-business-server"></a>管理商務用 Skype Server 

**摘要：** 探索這些連結以瞭解如何管理及維護商務用 Skype Server。
  
商務用 Skype 伺服器提供立即訊息（IM）、目前狀態、會議、影片和電話解決方案，可支援企業級的共同作業需求。 本主題提供如何設定、監控及維護商務用 Skype Server 部署的相關資訊連結。 
  
隨著新資訊的推出及使用者提供意見反應，作者會在繼續基礎上更新文章。 此頁面將會經常更新，因此請經常查看。
