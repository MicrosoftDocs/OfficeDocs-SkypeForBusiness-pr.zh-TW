# <a name="manage-pin-settings-in-skype-for-business-server"></a>在商務用 Skype Server 中管理 PIN 碼設定
 
**摘要：** 在商務用 Skype Server 中管理 PIN 碼設定。
  
使用下列各節中的程式來管理商務用 Skype Server 中的 Pin 碼。
  
您可以從商務用 Skype server 控制台或商務用 Skype Server 管理命令介面來管理商務用 Skype Server PIN 碼原則。 使用下列程式可為您的組織設定 PIN 碼原則。
  
- [在商務用 Skype Server 中查看 PIN 原則資訊](view-pin-policy-information.md)
    
- [在商務用 Skype Server 中建立新的 PIN 原則](create-a-new-pin-policy.md)
    
- [在商務用 Skype Server 中修改現有的 PIN 原則](modify-an-existing-pin-policy.md)
    
- [在商務用 Skype Server 中刪除 PIN 原則](delete-a-pin-policy.md)
    
- [在商務用 Skype Server 中指派每個使用者的 PIN 原則](assign-a-per-user-pin-policy.md)
    
使用下列程式，從商務用 Skype Server 控制台的 [ **使用者** ] 區段中管理使用者的電話撥入式會議 pin。
  
- [在商務用 Skype Server 中設定使用者的電話撥入式會議 PIN](set-a-user-s-dial-in-conferencing-pin.md)
    
- [在商務用 Skype Server 中查看使用者 PIN 資訊](view-user-pin-information.md)
    
- [鎖定或解除鎖定使用者 PIN 碼中的商務用 Skype Server](lock-or-unlock-a-user-pin.md)
