# <a name="manage-pin-settings-in-skype-for-business-server"></a>在商務用 Skype Server 中管理 PIN 設定
 
**摘要：** 在商務用 Skype Server 中管理 PIN 設定。
  
使用下列各節中的程式來管理商務用 Skype Server 中的 Pin。
  
您可以從商務用 Skype Server 的 [控制台] 或 [商務用 Skype 伺服器管理] 命令介面管理商務用 Skype Server PIN 原則。 使用下列程式來設定貴組織的 PIN 原則。
  
- [在商務用 Skype Server 中查看 PIN 原則資訊](view-pin-policy-information.md)
    
- [在商務用 Skype Server 中建立新的 PIN 原則](create-a-new-pin-policy.md)
    
- [在商務用 Skype Server 中修改現有的 PIN 原則](modify-an-existing-pin-policy.md)
    
- [在商務用 Skype Server 中刪除 PIN 原則](delete-a-pin-policy.md)
    
- [在商務用 Skype Server 中指派每位使用者的 PIN 原則](assign-a-per-user-pin-policy.md)
    
使用下列程式，從商務用 Skype Server [控制台] 的 [**使用者**] 區段管理使用者的電話撥入式會議 pin。
  
- [在商務用 Skype Server 中設定使用者的電話撥入式會議 PIN](set-a-user-s-dial-in-conferencing-pin.md)
    
- [在商務用 Skype Server 中查看使用者 PIN 資訊](view-user-pin-information.md)
    
- [鎖定或解除鎖定商務用 Skype Server 中的使用者 PIN](lock-or-unlock-a-user-pin.md)
