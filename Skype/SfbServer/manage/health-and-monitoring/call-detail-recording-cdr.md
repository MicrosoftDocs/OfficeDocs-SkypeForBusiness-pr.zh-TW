# <a name="call-detail-recording-cdr-in-skype-for-business-server"></a>在商務用 Skype Server 中呼叫詳細資料錄製（CDR）
 
**摘要：** 瞭解商務用 Skype Server 中使用的通話詳細資料錄製（CDR）記錄。
  
通話詳細資料錄製（CDR）記錄使用狀況與對等活動的相關診斷資訊，包括實例訊息、透過網際網路通訊協定（VoIP）通話、應用程式共用、檔案傳輸及會議。 使用資料可用來計算投資回報率（ROI），而診斷資料可用來針對對等活動和會議進行疑難排解。 當您安裝商務用 Skype Server 時，您也會安裝預先定義的 CDR 全域配置設定集合。 使用本節中的主題來設定 CDR。
