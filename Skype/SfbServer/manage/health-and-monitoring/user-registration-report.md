# <a name="user-registration-report-in-skype-for-business-server"></a>商務用 Skype Server 中的使用者註冊報告
 
**摘要：** 深入瞭解商務用 Skype Server 中的使用者註冊報告。
  
使用者註冊報告提供使用者登入活動的概覽，最值得注意的是在指定期間 (每小時、每日、每週、每月) 的商務用 Skype Server 的使用者人數。 請記住，報告只會告訴您已登入的人員人數。 它不會告訴您登入的人員。 監視報告不提供使用商務用 Skype Server (的特定使用者相關資訊，也不會) 。 不過，您可以使用 [使用者活動報告] 來取得使用者資訊的粗略估計。
  
使用者註冊報告在提供使用者登入的相關資訊時，會提出兩項重要差異。 首先，其會將登入分為兩種主要類別：內部登入及外部登入。 內部登入代表從組織防火牆內部登入的使用者 (也就是連線到公司網路時)。 外部登入代表透過 Edge Server 從防火牆外部登入的使用者 (例如，從網際網路咖啡館登入的使用者會算作外部登入) 。 如果您需要知道您有多少使用者從防火牆外部登入，使用者註冊報告可以提供您此資訊。
  
此外，使用者註冊報告會記錄在指定的時段內，有多少「作用中」使用者存在。 作用中的使用者是在立即訊息 (IM) 會話中參與的使用者，參與商務用 Skype Server 會議，進行或接收通話，或在該期間內使用商務用 Skype Server。 這不同於登入但實際上從未使用系統的使用者。
  
## <a name="accessing-the-user-registration-report"></a>存取使用者註冊報告

您只能從監視報告首頁存取使用者註冊報告。使用者註冊報告沒有連結至任何其他報告。
  
## <a name="making-the-best-use-of-the-user-registration-report"></a>善加利用使用者註冊報告

在您部署商務用 Skype Server 一個一般要求的問題是：如何知道我的使用者是否真的使用這項新技術？ 雖然在這方面有些許限制，但是使用者註冊報告可以協助您回答這個問題。 若要判斷使用者是否正在使用商務用 Skype Server，您需要執行兩項操作。 首先，從使用者註冊報告取得唯一登入使用者計量值。 這個值會告訴您已登入商務用 Skype Server 的不同個人人數。
  
相比之下，total 登入度量單位會顯示每個人登入商務用 Skype Server 的總時間。 例如，假設 Ken Myer 登入到一天中商務用 Skype Server 五個不同的時間。 在此情況下，就總登入計量而言，Ken Myer 會計為五個不同的登入工作階段，但是就唯一登入使用者計量而言，只有一位登入使用者。 同樣地，一位使用者從多個裝置或多個位置登入並無不同。 例如，使用者可以使用她的桌上型電腦（她的膝上型電腦）登入，而她可以擁有可自動登入商務用 Skype Server 的 IP 電話。 在此範例中，共有一位唯一使用者登入三次。
  
為進一步說明總登入及唯一登入之間的差異，請想想下表中特定時段內的登入。
  
|**使用者**|**登入時間**|
|:-----|:-----|
|Ken Myer  <br/> |7/7/2015 8:45 AM  <br/> |
|Ken Myer  <br/> |7/7/2015 8:46 AM  <br/> |
|Pilar Ackerman  <br/> |7/7/2015 9:17 AM  <br/> |
|Ken Myer  <br/> |7/7/2015 9:22 AM  <br/> |
|Pilar Ackerman  <br/> |7/7/2015 9:31 AM  <br/> |
   
請注意，總共有五次登入；不過，只有兩位唯一登入使用者：Ken Myer (登入三次) 及 Pilar Ackerman (登入兩次)。這就是登入與唯一登入使用者之間的差異。
  
除了瞭解唯一登入的數量之外，您還需要知道已啟用商務用 Skype Server 的使用者總數。 您可以開啟商務用 Skype Server 管理命令介面，並執行下列 Windows PowerShell 命令，以找回此值：
  
```PowerShell
(Get-CsUser).Count
```

如果上述命令傳回的值為1236，且唯一登入使用者度量會傳回平均值667，則表示您每日的商務用 Skype 實際登入系統的一半 (也就是667除以1236，也就是大約 54% ) 。
  
> [!CAUTION]
> 請記住，登入度量會記錄在指定期間內實際登入的使用者。 他們不會追蹤已經登入系統的使用者。 例如，如果您的唯一登入使用者度量顯示667登入，而您有1236使用者，則表示您的使用者在登入系統時大約有一半。 不過，假設您在開始檢查登入資料時，已登入系統的300使用者。 這表示您實際上有近1000個使用者登入商務用 Skype Server，這表示您的使用者近80% 的使用者已登入。 
  
您也應該要比較唯一登入使用者值與唯一作用中使用者計量值。 「作用中的作用中使用者」度量可告訴您實際使用商務用 Skype Server 多少個唯一的使用者：他們撥打了電話、加入商務用 Skype Server 會議，或是參與 IM 會話。 這是很有用的資訊，因為商務用 Skype Server 可以設定為每當使用者啟動 Windows 時，自動啟動。 因此，您可能會有大量的使用者在每日登入 Windows 時自動登入商務用 Skype，但在此期間內，則不會實際使用商務用 Skype Server。
  
「作用中的「作用中使用者」度量也會在組織中提供更有意義的資料，而使用者通常不會在一天結束時登出 Windows。 相反地，他們只是鎖定其電腦，並 Windows 和商務用 Skype 執行中。 在此情況下，您可能會最後一次登入，因為您的使用者在數天前登入，而且永不登出。 不過，唯一的作用中使用者會告訴您使用者目前使用商務用 Skype 還是其他商務用 Skype Server 用戶端。
  
## <a name="filters"></a>篩選

篩選器可以讓您傳回更精確的資料集，或者以不同方法檢視傳回的資料。 例如，使用者註冊報告可讓您查看所有註冊區集區和 Edge Server 的資料，或查看個別集區的資料。 您也可以選擇資料的分組方式。 在這種情況下，註冊會按照小時、日、星期或月來分組。
  
下表列出您可以搭配使用者註冊報告的篩選。
  
**使用者註冊報告篩選**

|**名稱**|**描述**|
|:-----|:-----|
|**From** <br/> |時間範圍的開始日期和時間。若要按照小時檢視資料，請輸入開始日期和時間，如下所示：  <br/> 7/7/2015 1:00 PM  <br/> 如果您未輸入開始時間，報告會自動從指定日期凌晨 12 點開始。若要按照日期檢視資料，只要輸入日期即可：  <br/> 7/7/2015  <br/> 若要按星期或月份檢視，請輸入當週或該月您想檢視的日期 (您不必輸入當週或該月的第一天)：  <br/> 7/3/2015  <br/> 星期永遠是從星期日開始星期六結束。  <br/> |
|**To** <br/> |時間範圍的結束日期和時間。若要按照小時檢視資料，請輸入結束日期和時間，如下所示：  <br/> 7/7/2015 1:00 PM  <br/> 如果您未輸入結束時間，報告會自動在指定日期凌晨 12 點結束。若要按照日期檢視資料，只要輸入日期即可：  <br/> 7/7/2015  <br/> 若要按星期或月份檢視，請輸入當週或該月您想檢視的日期 (您不必輸入當週或該月的第一天)：  <br/> 7/3/2015  <br/> 星期永遠是從星期日開始星期六結束。  <br/> |
|**Interval** <br/> | 時間間隔。請選取下列其中一項： <br/>  每小時 (最多可以顯示 25 個小時) <br/>  每日 (最多可以顯示 31 天) <br/>  每週 (最多可以顯示 12 週) <br/>  每月 (最多可以顯示 12 個月) <br/>  若開始與結束日期超出所選間隔允許的上限值，將只會顯示上限值 (從開始日期開始顯示)。 例如，如果您選取 [開始日期 7/7/2015] 和 [結束2/28/2015 日期] 的 [每日間隔]，將會顯示 8/7/2015 12:00 AM 到 9/7/2015 12:00 AM (的資料，也就是有31天的資料) 總計。 <br/> |
|**集區** <br/> |登錄器集區或 Edge Server 的完整網域名稱 (FQDN)。 您可以選取個別的集區，或是選擇 [全部]，以檢視所有集區的資料。 此下拉式清單會自動將資料庫內的資料填入。 <br/> |
