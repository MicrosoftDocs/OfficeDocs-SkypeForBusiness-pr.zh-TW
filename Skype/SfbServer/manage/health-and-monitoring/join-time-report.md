# <a name="conference-join-time-report-in-skype-for-business-server"></a>商務用 Skype Server 中的會議加入時間報告
 
**總結：** 深入瞭解商務用 Skype Server 中的會議加入時間摘要報告。
  
會議加入時間摘要可讓您判斷使用者加入會議所需的時間。 報告會以毫秒) 顯示平均加入時間 (，也可讓您知道有多少使用者可以在2秒內加入會議，以及在2到5秒的時間內加入會議，等等。
  
## <a name="accessing-the-conference-join-time-report"></a>存取會議加入時間報告

會議加入時間報告可以從監控報告首頁進行存取。
  
## <a name="filters"></a>篩選

篩選器可以讓您傳回更精確的資料集或者以不同方法檢視傳回的資料。 下表列出您可以搭配會議加入時間報告使用的篩選器。
  
**會議加入時間報表篩選**

|**名稱**|**描述**|
|:-----|:-----|
|**From** <br/> |時間範圍的開始日期/時間。若要按照小時檢視資料，請輸入開始日期和時間，如下所示：  <br/> 7/7/2015 1:00 PM  <br/> 如果您未輸入開始時間，報告會自動從指定日期凌晨 12 點開始。若要按照日期檢視資料，只要輸入日期即可：  <br/> 7/7/2015  <br/> 若要按星期或月份檢視，請輸入當週或該月您想檢視的日期 (您不必輸入當週或該月的第一天)：  <br/> 7/3/2015  <br/> 星期永遠是從星期日開始星期六結束。  <br/> |
|**To** <br/> |時間範圍的結束日期/時間。若要按照小時檢視資料，請輸入開始日期和時間，如下所示：  <br/> 7/7/2015 1:00 PM  <br/> 如果您未輸入結束時間，報告會自動在指定日期凌晨 12 點結束。若要按照日期檢視資料，只要輸入日期即可：  <br/> 7/7/2015  <br/> 若要按星期或月份檢視，請輸入當週或該月您想檢視的日期 (您不必輸入當週或該月的第一天)：  <br/> 7/3/2015  <br/> 星期永遠是從星期日開始星期六結束。  <br/> |
|**Interval** <br/> | 時間間隔。請選取下列其中一項： <br/>  每小時 (最多可以顯示 25 個小時) <br/>  每日 (最多可以顯示 31 天) <br/>  每週 (最多可以顯示 12 週) <br/>  每月 (最多可以顯示 12 個月) <br/>  若開始與結束日期超出所選間隔允許的上限值，將只會顯示上限值 (從開始日期開始顯示)。 例如，如果您選取 [開始日期 7/7/2015] 和 [結束2/28/2015 日期] 的 [每日間隔]，將會顯示 8/7/2015 12:00 AM 到 9/7/2015 12:00 AM (的資料，也就是有31天的資料) 總計。 <br/> |
|**集區** <br/> |登錄器集區或 Edge Server 的完整網域名稱 (FQDN)。您可以選取個別的集區，或是按一下 [全部] 檢視所有集區的資料。此下拉式清單會自動將資料庫內的資料填入。<br/> |
|**會議會話** <br/> | 會話類型。 允許的值為： <br/>  一切 <br/>  焦點會話 (焦點是線上會議的中央原則和狀態管理員，並協調會議的各個方面 <br/>  應用程式共用 <br/>  A/V 會議 <br/>  如果您選取 [全部]，會議加入時間的總數將會顯示在報告的頂端。 請注意，這些總和只適用于使用 microsoft Exchange 或 microsoft Outlook 排程的會議。 <br/> |
   
## <a name="metrics"></a>度量

下表列出會議加入時間報告中提供的資訊。
  
**會議加入時間報表計量**

|**名稱**|**可以排序這個項目嗎？**|**描述**|
|:-----|:-----|:-----|
|**Date** <br/> 此度量值的實際標題會因所選取的間隔而異。  <br/> |否  <br/> |召開會議的日期與時間。  <br/> |
|**工作階段總數** <br/> |否  <br/> |工作階段的總數，包括成功的工作階段、失敗的工作階段 (預期失敗與未預期失敗) 以及未分類的工作階段。  <br/> |
|**平均 (毫秒)** <br/> |否  <br/> | (以毫秒為單位) 參與者加入會議所需的平均時間量。  <br/> |
|* * 會話 \< 2 seconds, Volume** <br/> |否  <br/> |可在2秒內加入會議的參與者人數。  <br/> |
|* * 會話 \< 2 seconds, Percentage** <br/> |否  <br/> ||
|**會話2-5 秒、磁片區** <br/> |否  <br/> |需要2秒到5秒才能加入會議的參與者人數。  <br/> |
|**會話2-5 秒、百分比** <br/> |否  <br/> |在2秒到5秒內加入會議的總通話參與者百分比。  <br/> |
|**會話5-10 秒、磁片區** <br/> |否  <br/> |需要5秒和10秒才能加入會議的參與者人數。  <br/> |
|**會話5-10 秒、百分比** <br/> |否  <br/> |在5秒及10秒內加入會議的總通話參與者百分比。  <br/> |
|**會話 \> 10 秒、磁片區** <br/> |否  <br/> |需要超過10秒才能加入會議的參與者人數。  <br/> |
|**會話 \> 10 秒、百分比** <br/> |否  <br/> |需要超過10秒才能加入會議的通話總人數百分比。  <br/> |
