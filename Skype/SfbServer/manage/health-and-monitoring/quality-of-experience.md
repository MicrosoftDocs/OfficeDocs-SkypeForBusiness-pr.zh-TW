# <a name="quality-of-experience-qoe-in-skype-for-business-server"></a>商務用 Skype Server 中的體驗品質（QoE）
 
**摘要：** 管理商務用 Skype Server 中的體驗品質（QoE）。
  
經驗品質（QoE）會記錄數位資料，指出媒體質量與參與者的相關資訊、裝置名稱、驅動程式、IP 位址，以及通話和會話中所涉及的端點類型。 當您安裝商務用 Skype Server 時，您也會安裝 QoE 的全域設定的預先定義集合。 使用本節中的主題來設定 QoE 設定。
