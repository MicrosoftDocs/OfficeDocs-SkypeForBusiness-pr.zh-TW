# <a name="media-quality-diagnostic-reports-in-skype-for-business-server"></a>商務用 Skype Server 中的媒體質量診斷報告
 
**總結：** 深入瞭解商務用 Skype Server 中的媒體質量診斷報告。
  
媒體質量診斷報告提供有關通話品質的資訊，以及失敗通話的診斷與疑難排解資訊。
  
## <a name="in-this-section"></a>本節內容

- [Media Quality Summary Report in 商務用 Skype Server](summary.md)提供不同端點類型的整體品質資料，包括企業語音對等通話、企業語音電話會議，以及至少在公用交換電話網路 (PSTN) 上的呼叫（至少是部分）。
    
- [Media Quality Comparison Report in 商務用 Skype Server](comparison.md)可比較各種音訊 (通話類型的通話品質值，例如，透過無線網路撥打的通話，以及透過有線連線) 進行的通話。
    
- [Server Performance Report in 商務用 Skype Server](server-performance.md)會根據這類重要品質度量值的測量，以降低遇到最大問題的伺服器，因為這類重要品質度量值為降低、封包遺失及抖動。
    
- [商務用 Skype Server 的位置報告](location-report.md)提供網路位置清單，以及每個地點所發生之通話的媒體質量摘要。 針對此報告的目的，位置是以 IP 子網為基礎。
    
- [中的裝置報告商務用 Skype Server](device-report.md)提供用於企業語音通話的裝置摘要，也包括裝置通話的平均媒體質量。
    
- [商務用 Skype Server 中的通話清單報告](call-list-report-0.md)提供組織內撥打或接收之電話的詳細資訊。
    
- [商務用 Skype Server 中的詳細通話報告](call-detail-report.md)提供組織內撥打或接收電話的詳細資訊。
    
- [商務用 Skype Server 中的伺服器媒體質量趨勢報告](server-media-quality-trend-report.md)為您提供一種方式，可讓您以圖形方式比較多達五台伺服器的經驗品質計量，例如通話數量、不良通話百分比、封包遺失及抖動。
    
- [商務用 Skype Server 中的媒體質量計量散佈報告](media-quality-metrics-distribution-report.md)提供的圖形顯示經驗品質度量（如抖動或封包遺失）的散佈價值。
    
- [商務用 Skype Server 的位置趨勢報告](location-trend-report.md)可提供網路位置的通話品質趨勢資訊。
