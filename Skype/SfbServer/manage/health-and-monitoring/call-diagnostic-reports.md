# <a name="call-diagnostic-reports-in-skype-for-business-server"></a>在商務用 Skype Server 中呼叫診斷報告
 
**摘要：** 瞭解商務用 Skype Server 中使用的多使用者呼叫診斷報告。
  
通話診斷報告提供失敗的對等與會議會話的摘要資訊與診斷資料。
  
## <a name="in-this-section"></a>本節內容

- [在商務用 Skype Server 中呼叫診斷摘要報告](summary-report.md)提供失敗的點對點工作階段及會議會話的整體摘要。 點對點工作階段是只涉及兩個參與者的會話。 會議會話涉及三個或更多參與者。
    
- [商務用 Skype Server 中的對等活動診斷報告](peer-to-peer-activity-diagnostic-report.md)提供失敗的點對點工作階段的整體趨勢視圖。 點對點工作階段只涉及兩個參與者。
    
- [商務用 Skype Server 中的會議診斷報告](conference-diagnostic-report.md)提供每個會議模式的失敗會議會話和趨勢視圖的整體趨勢視圖。 會議會話至少包含三個參與者。
    
- [商務用 Skype Server 中的 [熱門失敗] 報表](top-failures-report.md)提供最常見失敗及其趨勢的清單。
    
- [商務用 Skype Server 中的失敗發佈報告](failure-distribution-report.md)提供失敗會話的分析。
    
- [商務用 Skype Server 中的 [失敗清單] 報告](failure-list-report.md)提供有關失敗的會議中所涉及之個別參與者的詳細資訊。
    
- [商務用 Skype Server 中的診斷報告](diagnostic-report.md)針對失敗的會話提供診斷和疑難排解資訊（包括 SIP 回應代碼及診斷標頭和識別碼）。
