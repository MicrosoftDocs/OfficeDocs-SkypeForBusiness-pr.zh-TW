# <a name="whats-in-skype-for-business-server-2019"></a>商務用 Skype Server 2019 的功能

**摘要：** 若要瞭解商務用 Skype Server 2019 中的新功能，請閱讀本主題。  

商務用 Skype Server 2019 中的新功能包括下列各項：
  
- 雲端語音信箱  
- 通話資料連接器
- 並行遷移

## <a name="unified-messaging-services-cloud-voicemail"></a>整合訊息服務：雲端語音信箱

當您將商務用 Skype 2019 與 Exchange 2013 或 Exchange 2016 整合時，Exchange UM 仍可在商務用 Skype Server 2019 中使用。 由於 Exchange 2019 中的支援變更，因此需要取消 Exchange UM 整合，以取代雲端語音信箱和雲端自動語音應答功能。  

雲端語音信箱可讓您所有的商務用 Skype 2019 使用者&#x2014;無論是駐留在內部部署或線上&#x2014;，都能存取 Microsoft 雲端中的相同語音信箱服務。 雲端語音信箱可為您的內部部署和線上使用者提供下列好處：

- 使用商務用 Skype Online、團隊或 Outlook 用戶端，存取 Exchange 信箱中的語音信箱
- 能夠使用以 web 為基礎的入口網站來管理其語音信箱選項

如需詳細資訊，請參閱[規劃雲端語音信箱服務](../sfbhybrid/hybrid/plan-cloud-voicemail.md)並[規劃商務用 Skype Server 和 Exchange server 遷移](../sfbhybrid/hybrid/plan-um-migration.md)。
  
## <a name="call-monitoring-call-data-connector"></a>通話監視：呼叫資料連線器

呼叫資料連線器極大地簡化了混合式環境中的呼叫監視，因為您不再需要使用不同的內部部署和線上工具來監控所有使用者的通話品質。  無論您的使用者是駐留在內部部署還是線上，您都可以選擇在線上查看整個組織的通話品質。

使用 [呼叫資料連線器]，您可以使用單一工具集來執行下列工作：

- 在 Microsoft 團隊、商務用 Skype Online 和商務用 Skype Server 上監控您的使用者體驗。
- 在您的網路上查看問題並進行疑難排解
- 指派呼叫分析的 [支援人員] 和 [系統管理員] 角色，讓技術支援人員能查看並疑難排解其職責範圍。

如需詳細資訊，請參閱[規劃通話資料連線器](../sfbhybrid/hybrid/plan-call-data-connector.md)。
