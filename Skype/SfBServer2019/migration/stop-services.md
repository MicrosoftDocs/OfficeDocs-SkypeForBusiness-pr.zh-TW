# <a name="stop-legacy-services"></a>停止舊版服務

您可以使用商務用 Skype Server 的 [控制台] 來啟動或停止在特定電腦上執行的所有舊版服務，或是啟動或停止特定的舊版服務。
  
## <a name="to-start-or-stop-all-skype-for-business-server-services-on-a-computer"></a>在電腦上啟動或停止所有商務用 Skype Server 服務

1. 開啟商務用 Skype Server 的 [控制台]。
    
2. 在左側導覽列中，按一下 [**拓撲**]，然後按一下 [**狀態**]。
    
3. 在 [**狀態**] 頁面上，視需要排序或搜尋清單，以尋找執行您要開始或停止之服務的電腦，然後按一下它。 
    
4. 按一下 [**動作**]。
    
5. 按一下 [**啟動所有服務**] 或 [**停止所有服務**]。
    
## <a name="to-start-or-stop-a-specific-service"></a>啟動或停止特定服務

1. 開啟商務用 Skype Server 的 [控制台]。
    
2. 在左側導覽列中，按一下 [**拓撲**]，然後按一下 [**狀態**]。
    
3. 在 [**狀態**] 頁面上，視需要排序或搜尋清單，以尋找執行您要開始或停止之服務的電腦，然後按一下它。 
    
4. 按一下 [**屬性**]。
    
5. 如有需要，請排序服務清單，然後按一下您要開始或停止的服務。
    
6. 按一下 [**動作**]。
    
7. 按一下 [**開始服務**] 或 [**停止服務**]。
    
8. 按一下 [**關閉**]。
