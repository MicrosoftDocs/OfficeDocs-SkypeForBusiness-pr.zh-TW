# <a name="back-up-systems-and-data"></a>備份系統和資料

開始遷移之前, 請先執行完整的系統備份並記錄您現有的系統, 包括駐留在每個池中的使用者帳戶清單, 以便在必要時回滾。 有多個工具和程式可用於備份及還原資料、設定和系統。

如需詳細資訊與程式, 請參閱您版本的備份及還原程式。
