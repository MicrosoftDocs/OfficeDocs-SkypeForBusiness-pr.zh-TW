# <a name="back-up-systems-and-data"></a>備份系統與資料

開始遷移之前，請先執行完整系統備份並記錄現有系統，包括位於每個集區的使用者帳戶清單，這樣就能在必要時復原。 您可使用多種工具和程式來備份和還原資料、設定和系統。

如需詳細資訊與程式，請參閱備份及還原版本的程式。
