# <a name="administering-servers-after-migration"></a>遷移之後管理伺服器

一般來說, 您必須使用對應到您想要管理之伺服器版本的系統管理工具。 您無法在同一部電腦上安裝不同版本的管理工具。
  
> [!IMPORTANT]
> 部署商務用 Skype Server 試驗池之後, 您就無法使用 [拓撲建立器] 或 [控制台] 來管理任何2019資源。 您必須使用2019工具來管理所有目前及之前的版本資源。
