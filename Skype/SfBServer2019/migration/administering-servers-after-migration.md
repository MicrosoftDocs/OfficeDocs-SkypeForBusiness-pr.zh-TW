# <a name="administering-servers-after-migration"></a>在移轉後管理伺服器

一般來說，您必須根據所要管理的伺服器版本，使用對應的系統管理工具。 您無法在同一部電腦上安裝不同版本的系統管理工具。
  
> [!IMPORTANT]
> 部署商務用 Skype Server 試驗集區之後，就無法使用拓撲產生器或控制台來管理任何2019資源。 您必須使用2019工具來管理所有目前和舊版資源。
