# <a name="view-current-users-in-legacy-pool"></a>在舊版池中查看目前的使用者

在瞭解您可以在不同的池中移動使用者的各種方式之前, 我們必須先判斷舊版池中存在哪些使用者。 [**註冊機構池**] 欄會識別為舊版池設定的使用者。 以下是我們將移至商務用 Skype Server 2019 池的測試使用者。
  
### <a name="to-see-the-list-of-users-in-the-legacy-pool"></a>若要查看舊版池中的使用者清單

1. 使用 RTCUniversalServerAdmins 群組成員的帳戶或 CsAdministrator 或 CsUserAdministrator 系統管理角色的成員登入舊版前端伺服器。
    
2. 開啟**商務用 Skype Server**的 [控制台]。
    
3. 按一下 [**使用者**], 按一下 [**搜尋**], 然後按一下 [**尋找**]。
