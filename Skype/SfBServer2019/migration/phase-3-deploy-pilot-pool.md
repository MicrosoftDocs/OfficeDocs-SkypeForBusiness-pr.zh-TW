# <a name="phase-3-deploy-skype-for-business-server-2019-pilot-pool"></a>階段 3: 部署商務用 Skype Server 2019 試生產池

本節涵蓋部署商務用 Skype Server 2019 的試用版池所需的步驟。 部署商務用 Skype Server 2019 需要使用拓撲建立器來定義您的拓撲及您想要部署的元件, 以準備部署商務用 Skype Server 2019 元件的環境, 發佈您的拓撲在第一個前端伺服器上進行設計, 然後針對您部署的元件, 安裝和設定商務用 Skype Server 2019 軟體。 完成後, 您的商務用 Skype Server 2019 試驗池部署將與現有的舊版池共存。
