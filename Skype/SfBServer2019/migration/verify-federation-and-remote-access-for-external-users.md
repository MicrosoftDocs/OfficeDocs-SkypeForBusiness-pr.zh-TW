# <a name="verify-federation-and-remote-access-for-external-users"></a>驗證同盟及外部使用者的遠端存取

將同盟路由轉換為商務用 Skype Server 2019 Edge Server 之後，您應該執行一些功能測試，以確認同盟如預期般執行。 外部使用者存取的測試應該包含組織支援的每種外部使用者類型，包括下列任一項或所有項目。
  
### <a name="test-connectivity-of-external-users-and-external-access"></a>測試外部使用者與外部存取的連線能力

- 至少一個同盟網域中的使用者、商務用 Skype Server 2019 上的內部使用者，以及舊版安裝中的使用者。 測試立即訊息 (IM)、目前狀態、音訊/視訊 (A/V) 及桌面共用。
    
- 您的組織所支援的每個公用 IM 服務提供者的使用者 (及已完成的布建) 商務用 Skype Server 2019 上的使用者和舊版安裝上的使用者進行通訊。 
    
- 確認匿名使用者可以加入會議。
    
- 使用遠端使用者存取的舊版安裝所主控的使用者 (記錄我想要從內部網路以外的 Lync Server/商務用 Skype 進行記錄，但沒有 VPN) 與商務用 Skype Server 2019 上的使用者，以及舊版安裝中的使用者。 測試 IM、目前狀態、A/V 及桌面共用。
    
- 使用遠端使用者存取方式商務用 Skype Server 2019 上的使用者 (登入至內部網路以外的商務用 Skype Server 2019，但沒有 VPN) 與使用者在商務用 Skype Server 2019 上，以及舊版安裝中的使用者。 測試 IM、目前狀態、A/V 及桌面共用。
