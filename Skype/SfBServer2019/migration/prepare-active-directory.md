# <a name="prepare-active-directory-for-skype-for-business-server"></a>為商務用 Skype Server 準備 Active Directory

在共存狀態中部署商務用 Skype Server 2019 之前, 您必須先執行一些其他的 Active Directory 工作, 才能設定商務用 Skype Server 2019 的架構、林及網域。 架構延伸會新增商務用 Skype Server 2019 所需的 Active Directory 類別和屬性。 <!-- For additional information, see the topic 
[Preparing Active Directory Domain Services for Skype for Business Server 2019](../deployment/deploying-lync-server-2013/preparing-active-directory-domain-services-for-lync-server-2013.md). -->
  
### <a name="to-prepare-active-directory-for-skype-for-business-server-2019"></a>為商務用 Skype Server 2019 準備 Active Directory

1. 在商務用 Skype Server 2019 前端伺服器上, 執行商務用 Skype Server 2019 設定。
    
2. 選取 [**準備 Active Directory**]。
  
3. 完成嚮導中的步驟1到5。
