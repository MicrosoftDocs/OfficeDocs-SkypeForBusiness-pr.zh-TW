# <a name="configure-clients-for-migration"></a>設定用戶端以進行移轉

本主題包含您在遷移至商務用 Skype Server 2019 之前所應採取的建議用戶端部署步驟。 您應該在 Lync Server 2013 或商務用 Skype Server 2015 上進行這些設定變更， (**從**) 遷移的版本。
  
### <a name="to-configure-clients-before-migration"></a>在遷移之前設定用戶端

1. 針對現有的安裝，部署最新的伺服器、用戶端及裝置更新 (修復程式) 。
    
2. 在舊版的商務用 Skype Server 中，使用用戶端版本篩選，只允許安裝了最新更新的用戶端。
