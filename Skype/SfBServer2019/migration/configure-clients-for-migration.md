# <a name="configure-clients-for-migration"></a>設定用戶端以進行移轉

本主題包含移轉至 Skype for Business Server 2019 之前應該採取的建議的用戶端部署步驟。 這些設定應該變更在 Lync Server 2013 或 Skype Server 2015 （**要移轉**的版本）。
  
### <a name="to-configure-clients-before-migration"></a>移轉前設定用戶端

1. 部署最新版伺服器、 用戶端及裝置更新 (hotfix) 用於您現有的安裝。
    
2. 在舊版的 Skype for Business 伺服器中，使用用戶端版本篩選，只允許用戶端安裝最新版更新的。
