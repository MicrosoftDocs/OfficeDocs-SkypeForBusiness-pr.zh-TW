# <a name="skype-for-business-server-2019-management-tools"></a>商務用 Skype Server 2019 管理工具
 
**摘要:** 瞭解商務用 Skype Server 2019 中的服務管理工具。
  
商務用 Skype Server 2019 提供可支援企業級共同作業需求的立即訊息 (IM)、目前狀態、會議和電話方案。 管理這些服務的工具既靈活又強大。
