# <a name="skype-for-business-server-2019-management-tools"></a>商務用 Skype Server 2019 管理工具
 
**摘要：** 深入瞭解商務用 Skype Server 2019 中的服務管理工具。
  
商務用 Skype Server 2019 提供立即訊息 (IM) 、顯示狀態、會議及電話語音解決方案，以支援企業層級的共同作業需求。 管理這些服務的工具既靈活又強大。
