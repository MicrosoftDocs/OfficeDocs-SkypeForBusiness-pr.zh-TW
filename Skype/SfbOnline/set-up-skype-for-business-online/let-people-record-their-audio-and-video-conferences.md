# <a name="let-people-record-their-audio-and-video-conferences"></a>讓人員錄製音訊和視訊會議

## <a name="choose-the-default-recording-setting-for-a-single-user-or-everyone-in-your-organization"></a>針對單一使用者或貴組織中的所有人選擇預設的錄製設定

![](../images/sfb-logo-30x30.png) **使用商務用 skype 系統管理中心**顯示商務用 skype 標誌的圖示

1. 移至 [系統管理中心] > [**商務用 Skype**]，然後按一下 [**使用者**]。
    
2. 在 [**使用者**] 頁面上，從清單中選取使用者或使用者，然後按一下 [**編輯**]。
    
3. 在 [**選項] 頁面**上，按一下 [**記錄交談和會議**]，然後按一下 [**儲存**]。
    
開啟 [錄製] 時，召集人或簡報者可以錄製商務用 Skype Online 會議或電話會議。 開啟或關閉錄製會影響貴組織中的每個人。
  
|**包含在錄製中**|**不包含在錄製中**|
|:-----|:-----|
|音訊及 HD 影片  <br/> PowerPoint 與白板簡報  <br/> 桌面或程式共用會話  <br/> |共用的 OneNote 頁面  <br/> PowerPoint 注釋  <br/> 共用檔案（附件）  <br/> |
   
如需詳細資訊，請參閱[Lync 錄製儲存與發佈](https://support.office.com/en-US/Article/83cf1507-0f05-40f5-a520-5a6c133bc33e)及[Lync 錄製管理員](https://support.office.com/en-US/Article/fa915c55-ff55-45c0-b372-1ef5596da10b)。
