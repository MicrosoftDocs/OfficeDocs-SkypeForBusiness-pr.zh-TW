# <a name="why-cant-i-use-the-skype-for-business-online-admin-center-right-now"></a>為什麼我目前無法使用 商務用 Skype管理中心？

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]?

我們知道，當您無法完成工作時，這很令人沮喪，因此我們會說明這裡發生的一切，以及為什麼值得等待。 
  
首先，以下是技術說明：
  
我們正在將您的 商務用 Skype Online 服務 (，這表示您的使用者和組織) 移至離您較近的另一個 Microsoft 資料中心。 這會改善您的服務並減少延遲。 
  
有關更多技術詳細資料，請參閱 [資料移動期間和之後]( https://go.microsoft.com/fwlink/?LinkId=526418)。
  
## <a name="ok-so-what-does-that-mean"></a>好的，這代表什麼？

首先，讓我們將幾個字詞分門別項。
  
- **資料中心** 這是儲存您檔案或郵件Microsoft 365 Office 365位置，例如檔案和電子郵件訊息。 如果您真的想要深入瞭解資料中心Microsoft 365 Office 365，請查看[本文](https://www.microsoft.com/online/legal/v2/?docid=25)。
    
- **移移** 這和「移動」幾乎一樣。 在此案例中，這表示我們將您的線上商務用 Skype和設定從一個資料中心移往另一個離您較近的資料中心，以改善您的服務。
    
- **延遲** 這是您存取帳戶、進行Microsoft 365 系統管理中心，然後儲存這些變更所花的時間。
    
- **相關識別碼** 您可能在剛剛收到的郵件中看到此訊息。 Microsoft 支援工程師會使用這項資訊來説明您疑難排解錯誤。 如果您與 Microsoft 支援服務聯繫，系統可能會要求您提供相關識別碼。
    
因此，這一切代表我們正在將您所有的線上使用者商務用 Skype服務設定移動到離您較近的另一個位置。 越接近越好。 好消息是，經過這一段短時間之後，您的線上商務用 Skype服務將會改善。
  
![在 Microsoft 365 或 Office 365 中移Office 365。](../images/77502071-36fe-4833-a5ff-3b9ca7676542.png)
  
## <a name="what-skype-for-business-online-features-will-still-work"></a>線上商務用 Skype哪些功能仍然能夠使用？

雖然您無法存取 商務用 Skype Online 系統管理中心，但下列 商務用 Skype Online 功能在移移期間仍然可以使用：
  
- 線上會議
    
- 目前狀態資訊
    
## <a name="can-i-get-other-work-done"></a>我可以完成其他工作嗎？

確定。 當我們移商務用 Skype Online 服務時，您仍可使用 Microsoft 365 (中的其他系統管理中心，例如 Microsoft 365 和 Exchange 系統管理中心) 。 不過，除了 商務用 Skype Online 系統管理中心外，在移轉期間商務用 Skype線上遠端 PowerShell Cmdlet。
