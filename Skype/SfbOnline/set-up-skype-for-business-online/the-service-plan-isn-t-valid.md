# <a name="the-users-skype-for-business-online-service-plan-isnt-valid"></a>使用者的商務用 Skype Online 服務方案無效

很抱歉，指派給此使用者的商務用 Skype Online 服務方案無法辨識或無效。
  
移至*Microsoft 365 系統管理中心** > 作用中的**使用者**，並將有效的服務方案指派給此使用者。
