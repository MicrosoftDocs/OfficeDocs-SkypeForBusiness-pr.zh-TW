# <a name="user-statistics-and-organization-information"></a>使用者統計資料和組織資訊

商務用 Skype 儀表板提供您服務設定方式的快速快照。 如果您尚未設定商務用 Skype，請參閱設定商務用 [Skype Online](set-up-skype-for-business-online.md) 以逐步指示。
  
## <a name="how-to-change-skype-for-business-online-user-statistics"></a>如何變更商務用 Skype Online 使用者統計資料

|||
|:-----|:-----|
|**屬性** <br/> |**如何變更** <br/> |
|商務用 Skype Online 使用者  <br/> |若要新增或封鎖人員使用商務用 Skype 應用程式，請參閱指派或移除商務用 [Microsoft 365](https://support.office.com/article/997596b5-4173-4627-b915-36abac6786dc) 授權，或移除 [商務用 Microsoft 365](https://support.office.com/article/9b497c85-d0a4-4735-80fa-d3565bc05bd1)中的使用者授權。  <br/> 有關使用者離職時該做什麼的完整清單，請參閱從 [Microsoft 365](https://support.office.com/article/44d96212-4d90-4027-9aa9-a95eddb367d1)移除離職員工 。  <br/> |
|已啟用音訊會議的使用者  <br/> |若要啟用音訊會議人員，請參閱在 [Microsoft 365 或 Office 365](../audio-conferencing-in-office-365/try-or-purchase-audio-conferencing-in-office-365.md)中試用或購買音訊會議。  <br/> |
|雲端使用者  <br/> |這是擁有 Microsoft 365 或 Office 365 帳戶的商務用 Skype Online 使用者數目。  <br/> 若要將人員新增到您的 Microsoft 365 或 Office 365 訂閱 (也就是說，為使用者建立 Microsoft 365 或 Office 365 帳戶) ，請參閱個別或大量新增使用者至 [Office 365 -](https://support.office.com/article/1970f7d6-03b5-442f-b385-5880b9c256ec)系統管理協助 。  <br/> |
|線上同步和家用的使用者  <br/> |這是在商務用 Skype Server 內部部署中建立，但已同步處理至 Microsoft 365 或 Office 365 的混合式使用者數目。  <br/> 若要設定商務用 Skype 混合式連線，請參閱在商務用 Skype Server 與商務用 Skype Online 之間部署 [混合式連線](../../SfbHybrid/hybrid/configure-hybrid-connectivity.md?bc=%2fSkypeForBusiness%2fbreadcrumb%2ftoc.json&toc=%2fSkypeForBusiness%2ftoc.json)。  <br/> |
   
> [!NOTE]
> 如果您看到列出的無法使用，表示沒有資訊或發生錯誤。 
  
## <a name="how-to-change-information-about-your-organization"></a>如何變更貴組織的資訊

|||
|:-----|:-----|
|**屬性** <br/> |**如何變更** <br/> |
|組織名稱  <br/> |這是您註冊 Microsoft 365 或 Office 365 時輸入的名稱。 若要變更，請參閱變更貴組織的位址、 [技術連絡人等](https://support.office.com/article/a36e5a52-4df2-479e-bb97-9e67b8483e10)。  <br/> |
|組織識別碼  <br/> |您無法變更組織識別碼。  <br/> |
|已建立組織  <br/> |無法變更此值。  <br/> |
|Active Directory 同步處理于  <br/> |這是貴組織是否與內部部署商務用 Skype Online 組織同步處理。  <br/> 若要設定您的內部部署 Active Directory，請參閱設定 [Microsoft 365 或 Office 365 的目錄同步處理](https://support.office.com/article/1b3b5318-6977-42ed-b5c7-96fa74b08846)。  <br/> |
|組織網域  <br/> |當您最初設定 Microsoft 365 或 Office 365 時，這些網域就是已註冊的網域。  <br/> 若要新增網域，請參閱新增 [多個網域至 Microsoft 365 或 Office 365](https://support.office.com/article/2d2fa996-b760-411d-a5cc-190d63f13207)。  <br/> 若要移除網域，請參閱 [從 Microsoft 365 或 Office 365 移除網域](https://support.office.com/article/f09696b2-8c29-4588-a08b-b333da19810c)。  <br/> |
