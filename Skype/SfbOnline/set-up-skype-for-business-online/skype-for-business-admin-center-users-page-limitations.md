# <a name="skype-for-business-online-admin-center-users-page-limitations"></a>商務用 Skype線上系統管理中心使用者頁面限制

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

為了提升商務用 Skype **系統管理中心** 會顯示使用者清單中的前 1，000 個使用者。 如果您有超過 1，000 個使用者，請使用篩選篩選和搜尋搜尋來尋找您要找 ![ ](../images/d360528a-a05a-4f0d-a7fd-25a6c696206a.png)  ![ ](../images/5cc29d50-4eec-48a4-bdad-bbeae230f9ae.png) 的特定人員。
  
## <a name="bulk-editing"></a>大量編輯
<a name="__top"> </a>

1. 選取清單頂端的顯示名稱旁的方塊，選取前 1，000 個使用者 ，或是篩選或搜尋所退回的每個人。
    
2. 按一下 **[編輯** ![ 編輯 ](../images/2f8948c1-e4f3-4022-b9cd-37fed066056e.png) > 。
    
您可以使用清單底部的控制項，變更每個頁面上顯示的使用者數目：
  
![變更頁面上使用者數量的使用者頁面流覽](../images/7d8ba24e-e3e8-4ff8-92f7-98ac143aab6d.png)
