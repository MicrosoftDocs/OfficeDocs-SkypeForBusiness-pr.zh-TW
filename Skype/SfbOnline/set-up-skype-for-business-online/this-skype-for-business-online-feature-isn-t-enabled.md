# <a name="this-skype-for-business-online-feature-isnt-enabled"></a>未啟用此商務用 Skype Online 功能

很抱歉，看起來使用者並未獲此功能的授權。
  
|**可能的原因**|**建議的動作**|
|:-----|:-----|
|您有可用的授權，但您尚未開啟此使用者的功能。  <br/> |移至**Microsoft 365 管理中心** > [作用中的**使用者**] 以授權此使用者。 <br/> |
|使用者位於無法使用此功能的位置。  <br/> |如果可能的話，請移至**Microsoft 365 系統管理中心** > 的 [作用中**使用者**]，並將此使用者指派到可使用此功能的位置。 <br/> |
|您的組織尚未開啟該功能。  <br/> |移至**Microsoft 365 系統管理中心** > **商務用 Skype** ，並為您的整個組織開啟功能。 <br/> |
