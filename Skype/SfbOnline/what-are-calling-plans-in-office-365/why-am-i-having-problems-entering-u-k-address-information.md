# <a name="why-am-i-having-problems-entering-uk-address-information"></a>為什麼我在輸入英國時遇到問題。 位址資訊？

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]?

以下是在英法兩國取得正確驗證位址時 (問題) 。
  
## <a name="whats-wrong-with-the-company-name"></a>公司名稱有什麼問題？

如果您在輸入公司名稱時發生問題，請記得：
  
- 它可以有字母、數位及下列特殊字元：、 / - + 。 ' &amp; " _ 
    
    > [!CAUTION]
    > 在英國 (英國) ，公司名稱不能有 + 符號。 
  
- 其必須小於 80 個字元。
    
- 它不能以特殊字元開始。
    
## <a name="whats-wrong-with-the-house-number"></a>房屋號碼有什麼問題？

如果輸入房屋號碼時發生問題，請記得：
  
- 它可以有字母、數位及下列特殊字元：- / [ ]
    
- 它必須從數位開始。
    
- 其必須小於 20 個字元。
    
- 它可以有大寫或小寫字母。
    
## <a name="whats-wrong-with-the-house-name"></a>房屋名稱有什麼問題？

如果輸入房屋名稱時發生問題，請記得：
  
- 它可以有字母、數位及下列特殊字元：、/ - + 。 ' &amp; " _
    
- 其必須小於 50 個字元。
    
- 它不能以特殊字元開始。
    
## <a name="whats-wrong-with-the-street-name"></a>街地道名有什麼問題？

如果您在輸入街地名稱時發生問題，請記得：
  
- 它可以有字母、數位及下列特殊字元：、/ - + 。 ' &amp; " _ 
    
- 其必須小於 50 個字元。
    
- 它不能以特殊字元開始。 
    
## <a name="whats-wrong-with-the-city-or-town"></a>城市或鄉鎮有什麼問題？

如果您在輸入縣/市名稱時發生問題，請記得：
  
- 它可以有字母、數位及下列特殊字元：、/ - + 。 ' &amp; " _
    
- 其必須小於 50 個字元。
    
- 它不能以特殊字元開始。 
    
## <a name="whats-wrong-with-the-postal-code"></a>郵遞區號有什麼問題？

如果輸入郵遞區號時發生問題，請記得：
  
- 它只能有字母、數位或空格。
    
- 它必須包含 8 個或 8 個以下的字母，包括數位和空格。
    
- 請記住，如果您在英國 (英國 **) ，** 您必須將郵遞區號放在下列其中一種格式中，其中字母 A = 字元 A-Z、# 符號 = 0-9 中的數位，以及 _ (強調符號) = 空格。 以下是一些範例：
    
|**格式**|**範例**|
|:-----|:-----|
|AA#A_#AA  <br/> |EC3B 2BD  <br/> |
|A#A_#AA  <br/> |E1W 1AP  <br/> |
|A#_#AA  <br/> |F1 2AG  <br/> |
|A##_#AA  <br/> |C44 8HT  <br/> |
|AA#_#AA  <br/> |DR3 6HG  <br/> |
|AA##_#AA  <br/> |FN35 1TH  <br/> |
