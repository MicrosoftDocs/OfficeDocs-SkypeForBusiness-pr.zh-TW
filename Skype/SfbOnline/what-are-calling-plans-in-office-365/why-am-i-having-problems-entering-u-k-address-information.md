# <a name="why-am-i-having-problems-entering-uk-address-information"></a>為什麼我無法輸入英國 位址資訊？

以下是為什麼您無法在英國（英國）正確驗證位址或位址的一些原因。
  
## <a name="whats-wrong-with-the-company-name"></a>公司名稱有什麼問題？

如果您在輸入公司名稱時發生問題，請記住：
  
- 它可以包含字母、數位，以及下列特殊字元：，/-+。 ' &amp; " _ 
    
    > [!CAUTION]
    > 在英國（英國），公司名稱不能有 + 符號。 
  
- 它必須不超過80個字元。
    
- 它不能以特殊字元為開頭。
    
## <a name="whats-wrong-with-the-house-number"></a>房屋編號有什麼問題？

如果您輸入房屋電話號碼時發生問題，請記住：
  
- 它可以包含字母、數位及下列特殊字元：-/[]
    
- 它必須從數位開始。
    
- 它必須不超過20個字元。
    
- 它可以有一個大寫或小寫字母。
    
## <a name="whats-wrong-with-the-house-name"></a>房屋名稱有什麼問題？

如果您在輸入房屋名稱時發生問題，請記住：
  
- 它可以包含字母、數位及下列特殊字元：，/-+。 ' &amp; " _
    
- 它必須不超過50個字元。
    
- 它不能以特殊字元為開頭。
    
## <a name="whats-wrong-with-the-street-name"></a>街道名稱有什麼問題？

如果您在輸入街道名稱時遇到問題，請記住：
  
- 它可以包含字母、數位，以及下列特殊字元：，/-+。 ' &amp; " _ 
    
- 它必須不超過50個字元。
    
- 它不能以特殊字元為開頭。 
    
## <a name="whats-wrong-with-the-city-or-town"></a>城市或城鎮有什麼問題？

如果您在輸入城市名稱或城鎮時遇到問題，請記住：
  
- 它可以包含字母、數位，以及下列特殊字元：，/-+。 ' &amp; " _
    
- 它必須不超過50個字元。
    
- 它不能以特殊字元為開頭。 
    
## <a name="whats-wrong-with-the-postal-code"></a>郵遞區號有什麼問題？

如果您在輸入郵遞區號時遇到問題，請記住：
  
- 只能有字母、數位或空格。
    
- 它必須有8個或更少的字母，包括數位和空格。
    
- **請記住，如果您使用的是英國（英國）**，則**必須**使用下列其中一種格式，即字母 a = 字元 a-z、# sign = 0-9 中的數位，以及 _ （底線） = 空格。 以下是一些範例：
    
|**設置**|**範例**|
|:-----|:-----|
|AA # A_ # AA  <br/> |EC3B 2BD  <br/> |
|# A_ # AA  <br/> |E1W 1AP  <br/> |
|# _ # AA  <br/> |F1 2AG  <br/> |
|# #_ # AA  <br/> |C44 8HT  <br/> |
|AA # _ # AA  <br/> |DR3 6HG  <br/> |
|AA # #_ # AA  <br/> |FN35 1TH  <br/> |
