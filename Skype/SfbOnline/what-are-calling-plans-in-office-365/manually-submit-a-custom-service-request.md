# <a name="manually-submit-a-custom-service-request"></a>手動提交自訂服務要求

如果您有電話撥入式會議橋接、自動語音應答或其他服務號碼、免付費電話號碼，或您需要傳送至商務用 Skype 的999使用者（訂閱者）電話號碼，您必須[管理手機貴組織的數位](/microsoftteams/manage-phone-numbers-for-your-organization)。

在某些國家/地區，您也必須手動提交自訂服務要求，如果您想要取得電話號碼，或是釋放號碼或變更位址。 若要查看每個國家/地區所需的內容，或深入瞭解編號移植，請參閱[管理貴組織的電話號碼](/microsoftteams/manage-phone-numbers-for-your-organization)。


## <a name="if-you-need-to-create-and-submit-a-service-request-follow-these-steps"></a>如果您需要建立並提交服務要求，請依照下列步驟執行：

## <a name="for-organizations-150-or-less-follow-these-steps"></a>針對貴組織的150或較少，請依照下列步驟進行
1. 在系統管理中心中，選擇 [**需要協助**嗎？]
2. 在 [**需要協助嗎？** ] 窗格中，告訴我們您需要協助，然後選擇 [**取得**說明]。
3. 如果 [**視圖解決方案**] 區段中的結果不是 [說明]，請選擇下列其中一項：
    - **讓我們撥打電話給您（建議使用）**：確認您的聯絡人電話號碼，然後按 [**呼叫我**]。 來電的預期等待時間會顯示在 [**需要協助嗎？** ] 窗格中。
    - 傳送**電子郵件給我們**：確認您的電子郵件地址，然後選擇 [**傳送**]。 通話回應時間會顯示在 [**需求**說明] 窗格中。

### <a name="for-organizations-with-more-than-150-users-follow-these-steps-to-create-a-service-request"></a>針對有超過150使用者的組織，請遵循下列步驟來建立服務要求：

1. 使用您的公司或學校帳戶，以系統管理員身分登入 Office 365。
    
2. 在系統管理中心的左導覽中，按一下 [**支援** > **新的服務要求**]。
    
3. 在 [**服務要求**] 底下，按一下 [**新增**]。
    
4. 在 [**建立服務要求**] 頁面上，按一下 [**線上**共同作業]。
    
5. 在 [**識別問題**] 頁面上，選取並輸入以下內容：
    
   - **功能：** 選取 [**國內通話方案**] 和/或 [**國內與國際計畫**]。
    
   - **症狀：** 輸入**緊急通話**。
    
   - **問題摘要：** 輸入**位址驗證**。
    
   - **問題詳細資料：** 輸入您想要驗證的位址的任何詳細資料，例如：
    
      - 街道號碼
    
      - 街道名稱
    
      - 城鎮或城市
    
      - 國家或地區
    
     > [!IMPORTANT]
     > **在您想要在指派電話號碼時驗證緊急位址的 Office 365 中，輸入國家/地區。**
  
      - 郵政或郵遞區號
    
6. 按一下 **[下一頁]** ，按一下 **[是，繼續**繼續]。
    
7. 在 [**新增詳細資料**] 頁面上，選取並輸入下列專案：
    
   - **您的服務無法使用嗎？** 選取 [**否**]。
    
   - **受影響的使用者人數為何？** 選取 [**一些使用者**]。
    
   - **輸入受此問題影響之人員的電子郵件地址，** 或保留空白。
    
   - **選取您想要列出的網域**
    
   - 如果您有多個需要驗證的位址，請**附加**檔案。
    
   - 按一下 **[下一步]**。
    
   - 輸入您的連絡人電話號碼。
    
8. 查看資訊，然後按一下 [**提交要求**]。
    
> [!TIP]
> 參考編號會列在 Microsoft 365 系統管理中心的 [**服務要求**] 頁面上。
  
## <a name="what-else-should-you-know-about-number-porting"></a>關於數位移植，您還應該知道什麼？

- 若要使用通話方案，您必須購買並指派授權給使用者。 [商務用 Skype 和 Microsoft 團隊附加元件授權](../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md)。
    
- 您必須為每位使用者指派您所擁有的新電話號碼。 [指派、變更或移除使用者的電話號碼](/microsoftteams/assign-change-or-remove-a-phone-number-for-a-user)。
