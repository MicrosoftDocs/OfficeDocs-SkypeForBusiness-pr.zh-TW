# <a name="set-up-your-computer-for-windows-powershell"></a>設定 Windows PowerShell 電腦

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

只有在執行下列商務用 Skype之一的 64 位電腦上，才支援使用 Microsoft PowerShell 遠端系統管理 Microsoft Online：

- Windows Server 2016

- Windows 10
    
- Windows 8.1
    
- Windows 8
    
- Windows Server 2012R 2
    
- Windows Server 2012
    
- WindowsServer 2008
    
- Windows 7
    
除了支援的作業系統之外，電腦也必須同時進行下列操作：
  
- PowerShell
    
- TeamsPowerShell 模組 
    
您可以在下列主題中尋找有關下載及安裝這些專案的資訊：
  
- [下載並安裝 powershell Teams模組](/MicrosoftTeams/teams-powershell-install#install-the-teams-powershell-module)
    
- [下載並安裝 Windows PowerShell 5.1](download-and-install-windows-powershell-5-1.md)

- [使用 商務用 Skype PowerShell 管理 Office 365 Online](/office365/enterprise/powershell/manage-skype-for-business-online-with-office-365-powershell)
