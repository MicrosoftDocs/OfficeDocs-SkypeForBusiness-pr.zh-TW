# <a name="peer-to-peer-activity-report"></a>對等活動報告

[!INCLUDE [sfbo-retirement](../../Hub/includes/sfbo-retirement.md)]

報表 **儀表板** 會顯示您組織中所有Microsoft 365 Office 365活動概觀。 它可讓您深入查看個別的產品層級報告，以進一步深入瞭解每個產品內的活動。 例如，您可以使用 商務用 Skype對等活動報告來查看使用者使用 IM、音訊、視像、應用程式共用和傳輸檔案的量。 

請查看報告 [概觀](https://support.office.com/article/0d6dfb17-8582-4172-a9a9-aed798150263)。
  
此報告與其他報表商務用 Skype一起，提供您整個組織活動的詳細資訊。 當您調查、規劃及為貴組織做出其他商務決策時，這些詳細資料非常實用。 
  
> [!NOTE]
> 當您以系統管理員的商務用 Skype登入時，可以看到所有Microsoft 365 系統管理中心。 
  
## <a name="how-to-get-to-the-skype-for-business-peer-to-peer-activity-report"></a>如何取得商務用 Skype對等活動報告

1. 請前往系統管理中心>**報告**  >  **使用方式**。
    
2. 在使用 **狀況** 頁面上 **，商務用 Skype** 左側的選取報表清單上的對等  >  **** 活動。 或者，按一下 [商務用 Skype **活動** 小工具，然後按一下 商務用 Skype活動清單上的 [對等商務用 Skype **活動**。

![數位 8](../images/sfbcallout8.png)<br/>下表顯示每個使用者的對等活動明細。 這會顯示已指派商務用 Skype使用者及其對等活動。 您可以新增其他欄至資料表。

*    **已刪除** 表示使用者授權已移除。 <br/> <br/> **注意：**  只要在所選時段內獲得授權，已刪除使用者的活動仍然會顯示在報告中。 刪除 **欄** 可協助您注意，使用者可能已不再作用中，但已對報告中的資料有影響。  <br/><br/>
*    **刪除日期** 是移除使用者授權的日期。 
*    **UTC (上次**) 日期是該使用者 (UTC) 活動日期。
*    **IM** 會顯示使用者使用的點對點工作階段總數。

*    **影片** 顯示使用視像的點對點工作階段總數。
*    **應用程式共用** 會顯示對等應用程式共用會話總數。
*    **檔案傳輸** 會顯示對等檔案傳輸會話總數。
*    **音訊分鐘** 數會顯示整個組織使用的音訊分鐘總數。 
*    **影片分鐘** 數顯示整個組織使用的視音訊分鐘總數。
