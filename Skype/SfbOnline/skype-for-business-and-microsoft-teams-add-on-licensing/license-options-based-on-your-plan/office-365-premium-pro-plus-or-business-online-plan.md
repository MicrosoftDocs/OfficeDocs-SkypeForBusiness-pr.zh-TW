# <a name="microsoft-365-apps-for-enterprise-or-skype-for-business-online-standalone-plan-2"></a>適用于企業版或商務用 Skype Online 獨立方案2的 Microsoft 365 應用程式

如果您已購買適用于企業版或商務用[Skype Online 獨立方案 2](https://products.office.com/skype-for-business/online)的[Microsoft 365 應用程式](https://products.office.com/en/business/office-365-proplus-business-software)，以下是您需要購買才能取得更多商務用 skype 功能的附加元件授權。

> [!IMPORTANT]
> 雖然音訊會議、電話系統和通話方案可以使用商務用 Skype Online 方案2購買，才能啟用語音通話功能，我們建議您評估企業版 E5 方案，以查看其成本是否較低。

|**音訊會議** <br/> 為打算排程或主持撥入會議的每位人員購買1個**音訊會議**授權。 出席者不需要授權。 <br/> |**音訊會議**附加元件 <br/>購買授權後，Microsoft 就會成為您的音訊會議提供者。 |
|**電話撥入存取的免付費電話號碼，以及從會議撥出給其他人，透過撥打世界電話號碼來新增人員的功能** <br/> |[通訊點數](/microsoftteams/set-up-communications-credits-for-your-organization)|
|**電話系統** <br/> |1.[切換到其他商務用 Microsoft 365 方案](https://support.office.com/article/73318661-8f33-478b-bcc7-fb8d69dbb22a)。 如果您有適用于企業的 Microsoft 365 應用程式，或商務用 Skype Online 獨立方案2，我們建議您將方案升級為企業版 E5 方案，以取得雲端語音功能。 如果您升級至企業版 E1 或 E3 方案，您會看到購買**電話系統**附加元件的選項。 <br/> 2.**電話系統**附加元件 <br/>  3. 通話方案：購買[通話方案](/MicrosoftTeams/calling-plans-for-office-365)。 <br/>  或者，請[從您的現有提供者使用電話語音](../../skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing.md#bkmk_existing)。  <br/> |
